# 슬라이드 3 — 전반부 핵심 논증 종합

## 1~6주 핵심 논증 한눈에 보기

```
프롤로그+1장  "기술 발전 ≠ 자동적 번영" — 밴드왜건은 자동 작동하지 않는다
    ↓
2장   기술 → 잉여 → 불평등의 구조적 패턴 (고대 메소포타미아~중세 유럽)
    ↓
3장   설득 권력: 이야기로 불평등을 정당화 (중세 교회 → 산업혁명 → 빅테크)
    ↓
4장   엥겔스의 일시정지: GDP↑ + 임금→ = 밴드왜건 실패의 역사적 증거
    ↓
5장   미국: 다른 조건 → 부분적 성공 → 그러나 배제된 집단 존재
    ↓
6-7장 대항 권력의 형성 → 공유된 번영의 시대 (1870~1970) — 투쟁의 결과
```

# 슬라이드 4 — 천 년간 반복된 패턴

## 기술-권력-대항 권력의 삼각 구도

| 단계 | 내용 | 역사적 사례 |
|---|---|---|
| ① 기술 혁신 | 새로운 기술 등장 → 생산성 향상 | 관개, 풍차, 방직기, AI |
| ② 잉여 집중 | 비전 설정자가 잉여 독점 | 사제·왕, 공장주, 빅테크 |
| ③ 설득 권력 | 불평등을 "당연한 것"으로 정당화 | "신의 뜻", "자유 시장", "혁신" |
| ④ 피해 누적 | 다수의 삶은 개선되지 않거나 악화 | 농노, 아동 노동자, 긱 워커 |
| ⑤ 대항 권력 | 시민·노동자의 조직화된 저항 | 노동조합, 참정권, 복지국가 |
| ⑥ 제도 변화 | 밴드왜건이 작동하는 조건 형성 | 공유된 번영의 시대 |

> **어항 토론 질문:** "천 년의 역사를 돌아보았을 때, '기술이 모두에게 이롭게 되는 조건'은 무엇이라고 정리할 수 있는가?"
