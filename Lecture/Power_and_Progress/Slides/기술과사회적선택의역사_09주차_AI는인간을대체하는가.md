# 슬라이드 3 — 자동화 편향의 정의와 원인

### 01 / 자동화 편향
**슬라이드 1 / 2** — AI 개발을 지배하는 편향

**자동화 편향 (Automation Bias)**
> 인간 노동을 대체하는 방향으로 기술을 개발하려는 체계적 경향

**왜 이런 편향이 발생하는가?**

| 원인 | 설명 |
|---|---|
| 💰 즉각적 이윤 | 인간을 AI로 대체하면 인건비가 **곧바로** 절감 |
| 🔮 AGI 비전 | "기계가 모든 면에서 인간을 능가하는 미래"라는 비전에 사로잡힘 |
| 📊 투자 구조 | 벤처 투자자는 "인건비 절감" 스토리에 투자 |
| 📣 설득 권력 | "AI가 모든 것을 바꿀 것이다" → 대체 방향을 불가피한 것으로 포장 |

> **핵심:** 대체 vs 보완의 선택은 기술적 필연이 아니라 **사업 모델의 선택**이다

# 슬라이드 6 — 두 방향의 비교

### 02 / 자동화 vs 보완
**슬라이드 1 / 2** — 핵심은 기술이 인간을 어떻게 다루느냐

| | ❌ 자동화 (인간 대체) | ✅ 보완 (인간 역량 강화) |
|---|---|---|
| 방향 | 인간 노동을 기계로 대체 | 인간의 능력을 확장·보조 |
| 의료 | AI가 의사를 대체하여 진단 | 의사 + AI 진단 보조 → 정확도↑ |
| 교육 | AI 튜터가 교사를 대체 | 교사 + 학생 개별 데이터 → 맞춤 지도 |
| 제조업 | 로봇이 노동자를 대체 | 협업 로봇 → 안전·생산성↑ |
| 결과 | 일자리 감소, 소득 격차↑ | 생산성↑ + 고용 유지 |

> **저자들의 핵심 주장:** 기술의 방향은 "대체"만이 아니다. 같은 AI를 "보완" 방향으로 설계하는 것이 가능하며, 더 바람직하다.

# 슬라이드 7 — 일본의 긍정적 사례

### 02 / 자동화 vs 보완
**슬라이드 2 / 2** — 같은 기술, 다른 선택

**도요타의 사례: 자동화 + 새 업무 창출 병행**

| 미국 자동차 산업 | 일본 도요타 |
|---|---|
| 자동화 → 대규모 해고 | 자동화 + 노동자 재배치 |
| 비용 절감에 집중 | 숙련 노동자의 지식 활용 |
| 노동자 = 대체 대상 | 노동자 = 품질 향상의 핵심 |
| 결과: 대량 실업 | 결과: 상대적 고용 유지 |

> **교훈:** 기술의 방향은 **사회적 선택**이다. 같은 기술을 가지고도 전혀 다른 결과를 만들 수 있다.
> 이것이 이 책 전체의 핵심 메시지이며, AI 시대에도 동일하게 적용된다.

# 슬라이드 9 — AI의 두 방향: 교육 적용

### 03 / 교사와 AI
**슬라이드 1 / 2** — 교육에서 "대체"와 "보완"은 구체적으로 어떻게 다른가

| | AI가 교사를 **대체** | AI가 교사를 **보완** |
|---|---|---|
| 수업 설계 | AI가 지도안을 자동 생성 | 교사가 AI 초안을 기반으로 수정·맥락화 |
| 평가 | AI가 채점·피드백 전담 | AI가 기초 분석 → 교사가 맥락적 피드백 |
| 학생 상담 | AI 챗봇이 정서 지원 | AI가 학생 패턴 분석 → 교사가 상담 |
| 학부모 소통 | AI가 자동 알림·보고 | AI가 데이터 정리 → 교사가 소통 |

> **핵심 질문:** AI가 한 부분과 교사가 **꼭** 해야 할 부분의 경계는 어디인가?

# 슬라이드 10 — 교사의 본질적 가치

### 03 / 교사와 AI
**슬라이드 2 / 2** — AI가 대체할 수 없는 교사의 역할

**AI가 잘 하는 것:**
- 정보 전달, 패턴 분석, 반복 연습, 기초 평가, 데이터 정리

**AI가 대체할 수 없는 것:**
- 🫂 학생과의 관계 형성 — 신뢰, 안전감, 소속감
- 🎯 맥락적 판단 — "이 학생에게 지금 필요한 것은 무엇인가"
- 🌱 가치관 교육 — 공감, 협력, 시민성
- ❓ 비판적 질문 제기 — "왜?", "다른 관점은?"
- 🤝 갈등 중재와 공동체 형성

> **아세모글루의 프레임으로:** 교사가 AI의 "보완 기술"이 되는 것이 아니라, **AI가 교사의 "보완 도구"가 되어야 한다.** 주체는 교사다.
