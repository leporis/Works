# 슬라이드 5 — 주의경제의 구조

### 02 / 주의경제와 감시 자본주의
**슬라이드 1 / 2** — 당신의 주의(attention)가 상품이다

**주의경제 (Attention Economy)**
> 인간의 관심(attention)을 자원으로 추출하여 광고주에게 판매하는 경제 구조

```
[이용자] → "무료" 서비스 사용 → [플랫폼이 수집하는 것]
                                     ↓
                              시간 / 데이터 / 심리적 에너지
                                     ↓
                              [광고주에게 판매] → 플랫폼 수익
```

**알고리즘의 목표:** 이용자의 주의를 최대한 오래 붙잡기
→ 자극적이고 극단적인 콘텐츠 우선 노출 → 중독, 불안, 사회적 분열

> **핵심:** "무료 서비스를 사용하는 당신이 상품이다" — 이용자는 고객이 아니라, 광고주에게 판매되는 **상품**

# 슬라이드 6 — 감시 자본주의

### 02 / 주의경제와 감시 자본주의
**슬라이드 2 / 2** — 데이터 수집 → 행동 예측 → 행동 조작

**감시 자본주의 (Surveillance Capitalism)**
> 개인의 모든 온라인 활동을 수집하여 행동 예측 모델을 구축하는 시스템

| 수집되는 데이터 | 활용 방식 |
|---|---|
| 검색어, 클릭 패턴 | 관심사 예측 → 맞춤 광고 |
| 체류 시간, 스크롤 패턴 | 콘텐츠 추천 알고리즘 최적화 |
| 위치 정보, 이동 경로 | 오프라인 행동까지 추적 |
| 구매 패턴, 카드 내역 | 소비 성향 예측 → 구매 유도 |

> **필터 버블 (Filter Bubble):** 알고리즘이 이용자가 선호하는 정보만 보여주는 현상
> → 자신과 다른 관점을 접할 기회 상실 → 사회적 분열 심화

# 슬라이드 8 — 두 혁명의 비교

### 03 / 산업혁명 vs 디지털혁명
**슬라이드 1 / 1** — 구조적 유사성에 주목하라

| 비교 항목 | 산업혁명 초기 | 디지털 혁명 |
|---|---|---|
| 착취 대상 | 공장의 **노동력** | 플랫폼의 **데이터/주의** |
| 규제 부재 | 노동 시간 규제 없음 | 스크린 타임 규제 없음 |
| 아동 문제 | 아동 노동 | 미성년 소셜 미디어 사용 |
| 교섭력 | 노동자의 교섭력 부재 | 이용자의 협상력 부재 |
| 설득 권력 | "자유 시장이 모두에게 이롭다" | "정보의 민주화" |

> **저자들의 핵심 메시지:** 이 모든 것은 디지털 기술의 **불가피한 결과가 아니다.** 알고리즘은 다르게 설계될 수 있다. 문제는 기술이 아니라 기술을 만드는 사람들의 **선택**이다.
