# 슬라이드 5 — 대항 권력의 정의와 형성 과정

### 02 / 대항 권력의 형성
**슬라이드 1 / 2** — 권리는 저절로 주어지지 않았다

**대항 권력 (Countervailing Power)**
> 지배적 권력에 맞서는 조직화된 시민사회의 힘

**형성 과정 (19세기 후반 ~ 20세기 중반):**

```
① 노동조합 합법화 (1871~ 영국)
      ↓
② 참정권 확대 (19세기 말~20세기 초)
      ↓
③ 사회보장제도 도입 (비스마르크 1880s, 뉴딜 1930s, 복지국가 1940s)
      ↓
④ 공교육 보편화 → 노동자 기술 수준 ↑ → 교섭력 ↑
      ↓
⑤ 결과: "공유된 번영의 시대" (1870~1970)
```

> 오늘날 당연하게 누리는 최저임금, 8시간 노동, 아동 노동 금지, 보통선거권, 사회보장제도는 모두 **쟁취**의 결과다.

# 슬라이드 6 — 공유된 번영의 시대

### 02 / 대항 권력의 형성
**슬라이드 2 / 2** — 밴드왜건이 가장 잘 작동한 시기

**공유된 번영의 시대 (1870~1970):**
- 기술 발전의 혜택이 비교적 널리 공유됨
- 중산층 두터워짐 / 불평등 감소
- **밴드왜건 3대 조건이 모두 충족된 유일한 시기**

| 조건 | 실현 방식 |
|---|---|
| ① 새 일자리 | 산업 다각화 + 서비스업 성장 |
| ② 교섭력 | 노동조합 합법화 + 단체 협약 |
| ③ 재분배 | 복지국가 + 누진세 + 공교육 |

> **그러나 이것은 기술의 자연스러운 결과가 아니라, 투쟁의 결과였다.**

# 슬라이드 8 — 집합 행동의 딜레마

### 03 / 무임승차 문제
**슬라이드 1 / 1** — 대항 권력 형성의 가장 큰 장벽

**무임승차 문제 (Free Rider Problem)**
> 집합 행동의 비용은 참여자가 부담하지만, 혜택은 참여하지 않은 사람에게도 돌아간다

| | 참여의 비용 | 참여의 혜택 |
|---|---|---|
| 파업 | 임금 손실, 해고 위험, 시간 투자 | 임금 인상, 노동 조건 개선 |
| 비용 부담 | 참여자만 부담 | 참여 안 해도 **동일하게** 누림 |

> **결과:** 모두가 이익을 원하지만 아무도 비용을 부담하려 하지 않으면 → 집합 행동 자체가 일어나지 않음

**그런데도 역사적으로 대항 권력이 형성된 이유:**
- 공동체 의식과 연대감 / 도덕적 분노 / 리더십의 역할 / 제도적 지원(합법화)

기술과 사회적 선택의 역사 | 6주차 | 6-7장
