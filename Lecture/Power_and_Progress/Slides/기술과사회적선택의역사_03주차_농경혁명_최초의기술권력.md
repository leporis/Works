# 슬라이드 3 — 농경혁명: 밝은 면과 어두운 면

### 01 / 농경혁명의 두 얼굴
**슬라이드 1 / 2** — 같은 기술이 만든 정반대의 결과

| 밝은 면 ☀️ | 어두운 면 🌑 |
|---|---|
| 정착 생활 가능 → 문명의 기초 | 잉여 독점 → 최초의 불평등 |
| 식량 잉여 → 인구 증가 | 노동 강도 ↑ (수렵채집보다 더 고된 노동) |
| 전문 직업 분화 (장인, 사제, 군인) | 계층 분화 (지배자 vs. 피지배자) |
| 문자·법·행정 시스템 탄생 | 전염병 확산 (밀집 거주 + 가축 근접) |

> 유발 하라리의 도발적 주장: "밀이 인간을 길들인 것이지, 인간이 밀을 길들인 것이 아니다"
> 아세모글루·존슨의 분석: 기술 자체가 아니라, 잉여를 **누가 차지하느냐**가 결과를 결정했다

# 슬라이드 4 — 기술 → 잉여 → 불평등 패턴

### 01 / 농경혁명의 두 얼굴
**슬라이드 2 / 2** — 이후 수천 년간 반복되는 구조적 패턴

```
[기술 혁신] → [생산성 향상] → [잉여 생산] → [잉여를 누가 차지하는가?]
                                                    ↓
                                        ┌──────────┴──────────┐
                                   소수 엘리트 독점          사회적 공유
                                        ↓                        ↓
                                   불평등 심화              공유된 번영
```

> **저자들의 핵심 논제:** 역사적으로 잉여가 소수에게 집중된 경우가 훨씬 많았다.
> 공유된 번영은 **의도적인 사회적 선택과 제도 설계**가 있을 때만 가능했다.

# 슬라이드 6 — 메소포타미아의 비전 설정자

### 02 / 비전 설정자의 등장
**슬라이드 1 / 2** — 신전과 궁전이 기술의 방향을 결정했다

**메소포타미아의 관개 시스템**

| 요소 | 내용 |
|---|---|
| 🔧 기술 | 대규모 관개 수로 건설 — 농업 생산성 비약적 향상 |
| 👑 비전 설정자 | 신전의 사제 + 궁전의 왕 |
| 📣 비전 | "신의 뜻에 따라 사회를 조직한다" |
| 💰 잉여의 귀속 | 대부분 신전과 궁전에 귀속 — 거대 건축물, 군대, 관료제 |
| 🔒 농민의 현실 | 수확의 50~70%를 세금·공물로 납부, 강제 노동 동원 |

> 관개 시스템은 누가 건설했나? → 농민. 누가 혜택을 누렸나? → 엘리트.
> 기술 → 잉여 → 권력 집중의 최초 사례

# 슬라이드 9 — 수에즈 vs 파나마

### 03 / 운하의 비전
**슬라이드 1 / 2** — 같은 사람, 같은 비전, 완전히 다른 결과

| | 수에즈 운하 (1869) ✅ | 파나마 운하 (1881~) ❌ |
|---|---|---|
| 지형 | 평탄한 사막 | 험준한 산악 + 열대 우림 |
| 공법 | 해수면 높이 운하 가능 | 해수면 높이 운하 불가능 |
| 기술자 의견 | "가능합니다" | "불가능합니다" — **무시됨** |
| 결과 | 성공 → 세계 무역 혁신 | 2만 명 이상 사망 + 재정 파탄 |

**레셉스의 비전:** "수에즈에서 성공했으니 파나마에서도 된다"
→ 비판적 목소리를 차단한 **비전의 과잉 확신**

> 피해자: 이집트 농민(수에즈), 카리브해 흑인 노동자(파나마) — 사회적 약자

| "수에즈에서 성공했으니 파나마도 된다" | "ChatGPT가 잘 하니 AI가 모든 것을 바꿀 것이다" |

| 이익을 얻은 사람: 투자자 | 이익을 얻는 사람: 빅테크 기업 |

> **공통 교훈:** 비전이 과잉 확신으로 이어지면, 비판이 차단되고, 비용은 가장 취약한 사람이 치른다

# 슬라이드 12 — 고대 운하 vs 현대 데이터센터

### 04 / 고대와 현대의 비교
**슬라이드 1 / 1** — 기술은 변했지만 권력 구조는 닮아 있다

| 비교 항목 | 고대 운하 건설 | 현대 데이터센터 건설 |
|---|---|---|
| 기술 | 관개 수로 | 서버 팜·해저 케이블 |
| 비전 설정자 | 왕·사제 | 빅테크 CEO |
| 비전 | "신의 뜻에 따른 사회 조직" | "AI가 모든 것을 바꿀 것이다" |
| 잉여의 귀속 | 신전·궁전 | 빅테크 주주 |
| 노동자의 현실 | 강제 노동, 세금 | 긱 워커, 저임금 데이터 라벨링 |
| 비판의 운명 | "신성모독" | "기술 발전을 가로막는 시대착오" |

> **저자들의 메시지:** 기술의 형태는 완전히 바뀌었지만, "소수가 기술의 방향을 정하고 다수가 비용을 치르는" 구조는 수천 년간 반복되고 있다.
