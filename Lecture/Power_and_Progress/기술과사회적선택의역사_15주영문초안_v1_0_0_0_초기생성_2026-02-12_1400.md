# Ⅰ. 도입 (1-2주)

---

## 1주차: 오리엔테이션 — 기술은 우리를 행복하게 하는가?

**읽기 범위:** 없음 (첫 수업)

### 사전 과제
- "내 삶에서 기술이 가장 큰 변화를 만든 순간" 3가지를 포스트잇 크기로 메모해 오기

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-20분 | **수업 안내** | 15주 과정 소개, 평가 방법, 토론 그라운드 룰 합의 |
| 20-40분 | **아이스브레이커** | "기술 변화 타임라인" — 각자의 포스트잇을 칠판에 연도순으로 배치 → 세대별 기술 경험 비교 |
| 40-70분 | **사전 인식 조사 토론** | 아래 질문으로 자유 토론 |
| 70-100분 | **저자·도서 소개** | 아세모글루(2024 노벨경제학상), 존슨(전 IMF 수석 이코노미스트) 소개 + 핵심 논제 미리보기 영상 |
| 100-115분 | **모둠 구성** | 4인 1조 학기 모둠 편성, 역할 분담(사회자·기록자·발표자·타임키퍼 순환) |
| 115-120분 | **다음 주 안내** | 프롤로그 + 1장 읽기 배정 |

### 토론 질문 (사전 인식 탐색)

1. "스마트폰이 등장한 이후 나의 삶은 전반적으로 더 나아졌다" — 동의/비동의 스펙트럼 위에 서기 (물리적 이동)
2. "기술이 발전하면 일자리는 늘어날까, 줄어들까?" — 자유 발언
3. "기술의 혜택을 가장 많이 받는 사람은 누구인가?" — 포스트잇에 적어 분류

### 핵심 개념
- 기술 낙관주의(techno-optimism) / 기술 결정론(technological determinism)

## 2주차: "기술 발전 = 진보"라는 신화

**읽기 범위:** 프롤로그 (진보란 무엇인가) + 1장 (테크놀로지에 대한 통제)

### 사전 과제
- 지정 범위 읽기
- 질문지 작성: ① 가장 공감한 문장 1개 필사 + 이유, ② 가장 의문이 든 문장 1개 필사 + 질문

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-15분 | **도입 강의** | "생산성 밴드왜건"의 작동 원리와 실패 조건 |
| 15-30분 | **필사 나눔** | 모둠 내 공감 문장·의문 문장 공유 → 모둠별 베스트 질문 1개 선정 |
| 30-75분 | **소크라테스 세미나** | 모둠별 베스트 질문을 중심으로 전체 토론, 교수자가 꼬리 질문으로 심화 |
| 75-110분 | **개념 매핑** | 모둠별로 1장의 핵심 논증 구조를 큰 종이에 개념 지도로 그리기 → 갤러리 워크 |
| 110-120분 | **정리** | "비전(vision)"의 역할 정리 + 다음 주 안내 |

### 토론 주제

1. **소크라테스 질문:** 아세모글루는 "기술 발전이 자동으로 번영을 가져다준다는 믿음"을 왜 위험하다고 보는가?
2. **확장 질문:** 한국 사회에서 "기술 발전 = 진보"라는 믿음이 가장 강하게 작동하는 영역은 어디인가?
3. **교육 연결:** 초등학교 실과 교과서에서 기술을 어떤 관점으로 서술하고 있는가? 비판적 관점이 포함되어 있는가?

### 핵심 개념
- 생산성 밴드왜건(productivity bandwagon) / 비전(vision) / 기술 낙관주의의 함정

# Ⅱ. 역사 속 기술과 권력 (3-6주)

---

## 3주차: 농경혁명 — 최초의 기술 권력

**읽기 범위:** 2장 (운하의 비전)

### 사전 과제
- 지정 범위 읽기
- 짧은 글쓰기: "만약 내가 고대 메소포타미아의 농민이라면" (A4 반쪽, 하루 일과 상상)

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-15분 | **도입 강의** | 농경혁명의 두 얼굴: 잉여 생산 vs. 계층 분화 |
| 15-35분 | **상상 글쓰기 공유** | 2인 1조 교차 발표 → 전체 공유 |
| 35-80분 | **핵심 토론** | 아래 토론 주제 활용 |
| 80-110분 | **비교 분석 활동** | "고대 운하 건설"과 "현대 데이터센터 건설"의 권력 구조 비교표 작성 |
| 110-120분 | **정리** | 기술과 잉여, 그리고 통제의 관계 |

### 토론 주제

1. **분석 토론:** 농경 기술은 인류의 삶을 개선했는가, 악화시켰는가? 유발 하라리의 "밀이 인간을 길들였다"는 주장과 비교하여 논의하라.
2. **구조 토론:** 잉여 생산물이 생기면 왜 불평등이 발생하는가? "기술 → 잉여 → 권력 집중"의 메커니즘을 설명하라.
3. **교육 연결:** 초등 사회과에서 "문명의 발생"을 다룰 때, 기술과 권력의 관계를 어떻게 포함시킬 수 있을까?

### 핵심 개념
- 잉여와 불평등 / 비전 설정자(vision setter) / 엘리트의 기술 통제

---

## 4주차: 설득 권력 — 누가 "진보의 이야기"를 쓰는가

**읽기 범위:** 3장 (설득 권력)

### 사전 과제
- 지정 범위 읽기
- 미디어 분석 과제: 최근 1주간 뉴스에서 "AI가 ~을 혁신할 것이다"라는 보도 1건을 찾아오기 → 누가, 어떤 근거로, 누구에게 말하는지 분석

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-15분 | **도입 강의** | "설득 권력"의 정의와 역사적 작동 방식 |
| 15-35분 | **미디어 분석 공유** | 모둠 내 기사 분석 발표 → 공통 패턴 추출 |
| 35-80분 | **핵심 토론** | 아래 토론 주제 활용 |
| 80-110분 | **내러티브 분석 워크숍** | "ChatGPT가 교육을 바꿀 것이다"라는 주장을 설득 권력 프레임으로 해부하기 |
| 110-120분 | **정리** | 설득 권력의 현대적 형태 정리 |

### 토론 주제

1. **비판적 토론:** 빅테크 CEO들이 "AI는 모든 사람에게 이롭다"고 주장할 때, 이것은 아세모글루가 말하는 '설득 권력'의 현대적 형태인가?
2. **역사 비교 토론:** 중세 교회의 설득 권력과 실리콘밸리의 설득 권력은 어떤 점에서 유사하고 다른가?
3. **교육 연결:** 초등학생에게 "누군가의 주장을 그대로 믿지 않고 질문하는 법"을 어떻게 가르칠 수 있을까?

### 핵심 개념
- 설득 권력(persuasion power) / 내러티브 통제 / 기술 담론의 프레이밍

## 5주차: 산업혁명의 명암

**읽기 범위:** 4장 (비참함의 육성) + 5장 (중간 정도의 혁명)

### 사전 과제
- 지정 범위 읽기
- 데이터 분석: 교수자가 제공하는 "영국 산업혁명 시기 GDP vs. 실질 임금 추이" 그래프를 보고, 두 지표의 괴리가 의미하는 바를 A4 반쪽으로 분석

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-15분 | **도입 강의** | "엥겔스의 일시정지(Engels' pause)": GDP는 성장하는데 임금은 정체하는 현상 |
| 15-40분 | **역할극 토론 준비** | 3개 역할(공장주 / 노동자 / 의회 정치인) 배정 → 입장 정리 |
| 40-75분 | **역할극 토론** | "아동 노동법을 제정해야 하는가?" — 1833년 영국 의회 토론 재현 |
| 75-100분 | **역할 해제 디브리핑** | 역할에서 벗어나 느낀 점 공유 → 핵심 토론 주제로 전환 |
| 100-115분 | **현대 비교** | "오늘날의 엥겔스 일시정지는 무엇인가?" 브레인스토밍 |
| 115-120분 | **정리** | 기술 발전과 노동자 복지의 시간차 |

### 토론 주제

1. **역할극 핵심 질문:** 기술 혁신으로 사회 전체의 부가 증가하더라도, 특정 집단의 삶이 악화된다면 그것은 "진보"라고 할 수 있는가?
2. **분석 토론:** 산업혁명 초기 100년간 노동자 생활이 개선되지 않은 것은 기술의 문제인가, 제도의 문제인가?
3. **교육 연결:** 초등 사회과에서 "산업혁명"을 가르칠 때, 발명가 중심 서사에서 벗어나 노동자의 경험을 포함시키는 수업을 어떻게 설계할 수 있을까?

### 핵심 개념
- 엥겔스의 일시정지 / 착취적 기술 활용 / 아동 노동과 규제

## 6주차: 투쟁을 통한 진보

**읽기 범위:** 6장 (진보의 피해자) + 7장 (투쟁으로 점철된 경로)

### 사전 과제
- 지정 범위 읽기
- 사례 조사: "기술 변화에 맞서 시민/노동자가 권리를 쟁취한 역사적 사례" 1건 조사 (한국 또는 해외)

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-15분 | **도입 강의** | 대항 권력(countervailing power)의 형성 조건 |
| 15-35분 | **사례 발표** | 모둠 내 각자 조사한 사례 공유 → 공통 성공 요인 추출 |
| 35-80분 | **핵심 토론** | 아래 토론 주제 활용 |
| 80-110분 | **전반부 종합 활동** | 2-6주 내용을 "기술 → 권력 → 대항 → 제도 변화" 흐름으로 정리하는 포스터 제작 |
| 110-120분 | **정리** | 전반부 핵심 메시지 정리 + 중간 프로젝트 안내 |

### 토론 주제

1. **찬반 토론:** "기술 발전의 피해자에 대한 보상은 국가의 의무인가, 기업의 책임인가, 개인의 적응 문제인가?"
2. **구조 토론:** 대항 권력이 형성되려면 어떤 조건이 필요한가? 6-7장의 역사적 사례에서 공통점을 추출하라.
3. **교육 연결:** "시민으로서 권리를 주장하고 연대하는 법"을 초등학생에게 가르치는 것은 가능한가? 어떤 방식이 적절한가?

### 핵심 개념
- 대항 권력(countervailing power) / 집합 행동(collective action) / 무임승차 문제 / 포용적 제도

# Ⅲ. 디지털 시대의 권력 (7-8주)

---

## 7주차: 중간 점검 — 전반부 종합 토론

**읽기 범위:** 없음 (전반부 복습)

### 사전 과제
- **중간 프로젝트 제출:** "1-7장에서 배운 '권력과 기술' 프레임을 활용하여, 내가 관심 있는 현대 기술 1가지를 분석하는 보고서" (A4 3쪽)
  - 분석 대상 기술 선정 및 현황
  - 누가 이 기술의 비전을 설정하고 있는가?
  - 이 기술의 혜택은 누구에게 돌아가고 있는가?
  - 대항 권력은 존재하는가?
  - 교육적 시사점

### 수업 흐름

| 시간 | 활동 | 세부 내용 |
|------|------|----------|
| 0-10분 | **도입** | 전반부 키워드 워드클라우드 (멘티미터 활용) |
| 10-50분 | **중간 프로젝트 발표** | 모둠별 발표 (각 8분) + 질의응답 (각 4분) |
| 50-60분 | **휴식** | |
| 60-100분 | **어항 토론 (Fishbowl)** | 전반부 종합 질문으로 운영 |
| 100-115분 | **상호 피드백** | 발표에 대한 동료 피드백 작성 |
| 115-120분 | **후반부 안내** | 8장 이후 디지털·AI 파트 미리보기 |

### 어항 토론 핵심 질문

> "천 년의 역사를 돌아보았을 때, '기술이 모두에게 이롭게 되는 조건'은 무엇이라고 정리할 수 있는가?"

## 부록 B: 토론 그라운드 룰

1. **경청 우선:** 상대의 발언이 끝난 후 3초간 침묵한 뒤 발언한다.
2. **근거 제시:** "저는 ~라고 생각합니다. 왜냐하면 책의 ~장에서..."와 같이 텍스트 근거를 활용한다.
3. **건설적 반론:** "그 의견에 동의하지 않습니다" 대신 "다른 관점에서 보면..."으로 시작한다.
4. **질문 환영:** 모르는 것을 질문하는 것은 토론의 핵심이다.
5. **평등한 참여:** 한 사람이 연속 2회 이상 발언하지 않는다.
6. **존중:** 의견을 비판하되 사람을 비판하지 않는다.
7. **열린 마음:** 토론 후 내 생각이 바뀌는 것은 성장이다.
