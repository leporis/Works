# 🔧 MakeCode 시뮬레이터 — LED 이름 표시 개발일지

**문서 버전: v1.0.0.0**
**시뮬레이터 버전: v1.0.2.4**

---

## 1. 프로젝트 개요

### 1.1 목적

micro:bit MakeCode 환경을 웹 브라우저에서 체험할 수 있는 교육용 블록 코딩 시뮬레이터를 개발한다. 학생들이 실제 하드웨어 없이도 블록을 드래그 앤 드롭하여 LED 매트릭스에 이름을 표시하는 프로그래밍 과정을 경험한다.

### 1.2 교육 목표

| 목표 | 내용 |
|------|------|
| **블록 코딩 이해** | MakeCode 블록 구조와 프로그래밍 기초 개념 학습 |
| **이벤트 개념 학습** | "시작하면 실행"과 "무한반복 실행"의 차이 이해 |
| **순차 실행 체험** | 블록 순서에 따라 LED가 표시되는 순차 실행 원리 체험 |
| **입출력 이해** | 문자열 → LED 매트릭스 출력 과정 이해 |

### 1.3 배경

청주교육대학교 컴퓨터교육과 「사물인터넷과 빅데이터」 과목 1주차 강의자료로 활용한다. 대상은 4학년 예비 초등교사로, 프로그래밍 경험이 없는 학생이 대부분이다. 실제 micro:bit 기기를 사용하기 전에 시뮬레이터를 통해 블록 코딩의 기초를 체험하도록 설계하였다.

## 2. 시스템 설계

### 2.1 입출력 설계

| 구분 | 항목 | 설명 |
|------|------|------|
| **입력** | 블록 드래그 앤 드롭 | 팔레트에서 코딩 영역으로 블록 배치 |
| **입력** | 블록 파라미터 편집 | 문자열, 숫자, 시간(ms) 값 직접 수정 |
| **입력** | 버튼 조작 | 실행, 정지, 리셋, A/B 버튼 |
| **출력** | 5×5 LED 매트릭스 | 문자, 숫자, 아이콘 패턴 표시 |
| **출력** | 상태 표시줄 | 실행 상태 메시지 (준비됨/실행 중/정지됨) |
| **출력** | 블록 하이라이트 | 현재 실행 중인 블록 강조 |

### 2.2 블록 카테고리

| 카테고리 | 블록 | 기능 |
|----------|------|------|
| **기본** | 문자열 보여주기 | LED에 문자열을 한 글자씩 스크롤 표시 |
| **기본** | 숫자 보여주기 | LED에 숫자를 한 자리씩 표시 |
| **기본** | 아이콘 보여주기 | LED에 미리 정의된 아이콘(❤️) 표시 |
| **기본** | 일시정지 | 지정된 시간(ms) 동안 대기 |
| **LED** | 화면 지우기 | 모든 LED를 끄기 |

### 2.3 이벤트 핸들러

| 이벤트 | 실행 방식 | 용도 |
|--------|----------|------|
| **시작하면 실행** (onStart) | 1회 실행 | 프로그램 시작 시 초기 동작 |
| **무한반복 실행** (forever) | 반복 실행 | 지속적으로 반복할 동작 |

## 5. 기술 구현

### 5.1 사용 기술

| 기술 | 용도 |
|------|------|
| **HTML5** | 문서 구조, 드래그 앤 드롭 API |
| **CSS3** | 레이아웃(Grid/Flexbox), 애니메이션, 반응형 디자인 |
| **JavaScript (ES6+)** | 시뮬레이션 엔진, 이벤트 처리, async/await |
| **Google Fonts** | Noto Sans KR, JetBrains Mono |

### 5.2 파일 구조

단일 HTML 파일(Single File Application)로 구현. CSS와 JavaScript를 모두 포함한다.

```
MakeCode시뮬레이터LED이름표시_v1.0.2.4_카테고리헤더수정_2026-02-19_2125.html
├── <style>        CSS (약 650줄)
│   ├── 헤더 스타일
│   ├── 블록 팔레트 스타일
│   ├── 코딩 영역 스타일
│   ├── 시뮬레이터 패널 스타일
│   ├── LED 매트릭스 스타일
│   └── 반응형 미디어 쿼리
├── <body>         HTML 구조 (약 100줄)
│   ├── 헤더
│   ├── 블록 팔레트
│   ├── 코딩 영역 (onStart + forever)
│   ├── 시뮬레이터 패널
│   └── 도움말 팝업
└── <script>       JavaScript (약 700줄)
    ├── LED 문자 패턴 정의 (A~Z, 0~9, 특수문자)
    ├── 드래그 앤 드롭 시스템
    ├── 시뮬레이션 엔진
    ├── LED 제어 함수
    └── 유틸리티 함수
```

### 5.3 핵심 구현 포인트

**LED 문자 패턴**: 5×5 비트맵 배열로 알파벳(A~Z), 숫자(0~9), 한글 자모(ㄱ~ㅎ, ㅏ~ㅣ), 특수문자를 정의. 문자열 스크롤 시 한 글자씩 순차 표시한다.

**드래그 앤 드롭**: HTML5 Drag and Drop API를 활용. 팔레트 블록의 `draggable` 속성과 코딩 영역의 `dragover`/`drop` 이벤트로 구현. 블록 복제(clone) 방식으로 팔레트 원본을 유지한다.

**비동기 실행 엔진**: `async/await` + `setTimeout` 래퍼로 블록 간 대기 시간을 구현. `isRunning` 플래그로 정지 시 즉시 중단. `forever` 루프는 `while(isRunning)` 조건으로 반복한다.

**실행 블록 하이라이트**: `executeBlocks()` 함수에서 현재 실행 중인 블록에 `executing` CSS 클래스를 토글하여 시각적 피드백을 제공한다.
