## 1. 개요

### 1.1 프로그램 소개

MakeCode 시뮬레이터 — LED 이름 표시는 micro:bit의 블록 코딩 환경을 웹 브라우저에서 체험할 수 있는 교육용 시뮬레이터이다. 실제 micro:bit 기기 없이도 블록을 드래그 앤 드롭하여 프로그램을 구성하고, 5×5 LED 매트릭스에서 문자열, 숫자, 아이콘의 표시 결과를 즉시 확인할 수 있다.

### 1.2 실행 환경

| 항목 | 요구사항 |
|------|---------|
| 웹 브라우저 | Chrome 90+, Edge 90+, Firefox 88+, Safari 15+ |
| 화면 해상도 | 375px(모바일) ~ 1920px(데스크탑) 지원 |
| 인터넷 연결 | 최초 로딩 시 필요 (Google Fonts 다운로드) |
| 설치 | 불필요 (HTML 파일 더블클릭으로 실행) |

### 1.3 문서 구성

| 문서 | 목적 | 깊이 |
|------|------|:----:|
| 퀵가이드 | 즉시 실행, 빠른 참조 | ⭐ |

### 2.1 전체 레이아웃

시뮬레이터는 크게 **헤더**, **블록 팔레트**, **코딩 영역**, **시뮬레이터 패널**의 4개 영역으로 구성된다.

**좁은 화면 (1100px 이하):** 3열 구조가 1열 세로 스크롤로 전환된다. 블록 팔레트 → 코딩 영역 → 시뮬레이터 패널 순서로 위에서 아래로 배치된다.

### 2.2 헤더 영역

| 요소 | 위치 | 기능 |
|------|------|------|
| 🔮 아이콘 + 제목 | 좌측 | "1주차 강의자료 / LED에 이름 표시하기" 표시 |
| **🔄 새로 만들기** 버튼 | 우측 | 모든 블록 제거 + LED 초기화 (리셋) |
| **❓ 도움말** 버튼 | 우측 | 사용 방법 안내 팝업 표시 |

### 2.3 블록 팔레트

블록 팔레트는 화면 왼쪽(또는 좁은 화면에서 상단)에 위치하며, 코딩 영역으로 드래그할 수 있는 블록 목록을 제공한다.

#### 기본 카테고리

| 블록명 | 색상 | 입력값 | 기본값 |
|--------|:----:|--------|:------:|
| 문자열 보여주기 | 보라 | 텍스트 (문자열) | `Hello!` |
| 숫자 보여주기 | 보라 | 숫자 (정수) | `0` |
| 아이콘 보여주기 ❤️ | 보라 | 없음 (고정: 하트) | — |
| 일시정지 | 보라 | 시간 (밀리초) | `1000` |

#### LED 카테고리

| 블록명 | 색상 | 입력값 | 기본값 |
|--------|:----:|--------|:------:|
| 화면 지우기 | 청록 | 없음 | — |

> 💡 팔레트의 블록은 **원본**이 유지된다. 드래그 시 **복사본**이 코딩 영역에 생성되므로 같은 블록을 여러 번 사용할 수 있다.

### 2.4 코딩 영역

코딩 영역은 두 개의 이벤트 컨테이너로 구성된다.

#### 시작하면 실행 (onStart)

프로그램이 시작될 때 **1회만** 실행되는 블록을 배치하는 영역이다. 실제 MakeCode의 `on start` 블록에 대응한다. 보라색 점선 테두리로 표시되며, 블록이 없을 때 "블록을 여기에 드래그하세요" 안내 메시지가 나타난다.

#### 무한반복 실행 (forever)

프로그램이 실행되는 동안 **무한히 반복** 실행되는 블록을 배치하는 영역이다. 실제 MakeCode의 `forever` 블록에 대응한다. 파란색 점선 테두리로 표시된다.

#### 블록 배치 및 관리

| 동작 | 방법 |
|------|------|
| 블록 배치 | 팔레트에서 블록을 드래그하여 이벤트 영역에 드롭 |
| 값 수정 | 배치된 블록의 입력 필드를 클릭하여 직접 편집 |
| 블록 삭제 | 배치된 블록 우측의 **×** 버튼 클릭 |
| 순서 | 위에서 아래 순서로 실행 (배치된 순서) |

### 2.5 시뮬레이터 패널

#### 마이크로비트 시뮬레이터

| 요소 | 설명 |
|------|------|
| **micro:bit 제목** | "마이크로비트 시뮬레이터" 라벨 |
| **5×5 LED 매트릭스** | 25개 LED 격자. 켜짐(빨간색/밝음)/꺼짐(어두운 회색) |
| **A 버튼** | 좌측 원형 버튼 (현재 버전에서는 시각적 표시만) |
| **B 버튼** | 우측 원형 버튼 (현재 버전에서는 시각적 표시만) |
| **핀 커넥터** | 하단 금색 막대 (시각적 표시만) |

#### 제어 버튼

| 버튼 | 기능 | 상태 변화 |
|:----:|------|----------|
| **▶️ 실행** | 코딩 영역의 블록 프로그램 실행 | 상태바: "실행 중..." |
| **⏹️ 정지** | 실행 중인 프로그램 즉시 중단 | 상태바: "정지됨" |
| **🔄 리셋** | 블록 모두 제거 + LED 초기화 | 상태바: "준비됨" |

#### 상태 표시줄

현재 시뮬레이터 상태를 텍스트로 표시한다.

| 메시지 | 의미 |
|--------|------|
| "준비됨 - 블록을 배치하고 실행하세요" | 초기 상태 |
| "실행 중..." | 블록 프로그램 실행 중 |
| "정지됨" | 사용자가 정지 버튼을 누름 |

#### .hex 파일 다운로드

"⬇️ .hex 파일 다운로드 (실제 마이크로비트용)" 버튼을 클릭하면, 실제 micro:bit에 프로그램을 전송하는 방법을 안내하는 대화 상자가 나타난다. 본 시뮬레이터에서는 직접 .hex 파일을 생성하지 않으며, MakeCode 공식 사이트를 통한 다운로드를 안내한다.

## 3. 블록 상세 설명

### 3.1 문자열 보여주기

| 항목 | 내용 |
|------|------|
| **기능** | 입력된 문자열을 5×5 LED에 한 글자씩 순차 표시 |
| **입력** | 텍스트 문자열 (기본값: `Hello!`) |
| **동작** | 각 문자의 LED 패턴을 약 600ms 간격으로 순차 표시 |
| **지원 문자** | 영어 대문자(A~Z), 소문자(a~z, 대문자로 변환), 숫자(0~9), 한글 자모(ㄱ~ㅎ, ㅏ~ㅣ), 일부 특수문자(!?.,-) |
| **미지원 문자** | 공백은 LED 전체 꺼짐으로 표시, 정의되지 않은 문자는 건너뜀 |

### 3.2 숫자 보여주기

| 항목 | 내용 |
|------|------|
| **기능** | 입력된 숫자를 문자열로 변환하여 한 자리씩 LED에 표시 |
| **입력** | 정수 (기본값: `0`) |
| **동작** | 숫자를 문자열로 변환 후 문자열 보여주기와 동일하게 동작 |
| **예시** | `42` 입력 → "4" 표시 → "2" 표시 |

### 3.3 아이콘 보여주기

| 항목 | 내용 |
|------|------|
| **기능** | 미리 정의된 아이콘 패턴을 LED에 표시 |
| **입력** | 없음 (현재 버전: 하트 ❤️ 고정) |
| **동작** | 하트 모양 5×5 패턴을 즉시 표시 |

하트 패턴:
```
.  #  .  #  .
#  #  #  #  #
#  #  #  #  #
.  #  #  #  .
.  .  #  .  .
```

> `#` = LED 켜짐, `.` = LED 꺼짐

### 3.4 일시정지

| 항목 | 내용 |
|------|------|
| **기능** | 지정된 시간(밀리초) 동안 프로그램 실행을 일시 중단 |
| **입력** | 시간 (ms, 기본값: `1000`) |
| **동작** | 입력된 ms만큼 대기 후 다음 블록 실행 |
| **참고** | 1000ms = 1초. 값을 0으로 설정하면 즉시 다음 블록 실행 |

### 3.5 화면 지우기

| 항목 | 내용 |
|------|------|
| **기능** | 5×5 LED 매트릭스의 모든 LED를 끔 |
| **입력** | 없음 |
| **동작** | 25개 LED를 모두 꺼짐 상태로 전환 |

## 4. 드래그 앤 드롭 시스템

### 4.1 블록 드래그 시작

팔레트 블록에 마우스를 올리면 손가락 커서로 변하며, 마우스 버튼을 누른 채 끌면 드래그가 시작된다. 드래그 중인 블록은 반투명(50%)으로 표시된다.

### 4.2 드롭 영역

블록을 드래그하여 이벤트 컨테이너(시작하면 실행 또는 무한반복 실행) 위로 가져가면 영역이 하이라이트되어 드롭 가능 상태를 시각적으로 알려준다. 이벤트 영역 밖에 드롭하면 블록이 배치되지 않는다.

### 4.3 배치된 블록

배치된 블록은 코딩 영역 내에서 위에서 아래 순서로 나열된다. 각 블록에는 다음 요소가 포함된다.

| 요소 | 설명 |
|------|------|
| 블록 라벨 | 블록 이름 (예: "문자열 보여주기") |
| 입력 필드 | 편집 가능한 값 (텍스트, 숫자) |
| × 삭제 버튼 | 블록 우측 상단, 클릭 시 해당 블록 삭제 |

### 4.4 블록 값 수정

배치된 블록의 입력 필드(흰색 배경)를 클릭하면 직접 편집할 수 있다. 문자열 블록은 텍스트를, 숫자/일시정지 블록은 숫자를 수정한다. 값 변경은 즉시 반영되며 별도 저장 동작이 필요 없다.

## 5. 시뮬레이션 실행

### 5.1 실행 순서

**▶️ 실행** 버튼을 클릭하면 다음 순서로 동작한다.

```
[1] 실행 플래그(isRunning) 활성화
[2] LED 매트릭스 초기화
[3] "시작하면 실행" 영역의 블록을 위→아래 순서로 1회 실행
[4] "무한반복 실행" 영역의 블록을 위→아래 순서로 반복 실행
    └→ 정지 버튼 클릭 시까지 반복
```

### 5.2 실행 중 시각 피드백

| 피드백 | 설명 |
|--------|------|
| **블록 하이라이트** | 현재 실행 중인 블록의 테두리가 노란색으로 강조 |
| **LED 점등** | 문자/아이콘 패턴에 따라 LED가 빨간색으로 켜짐 |
| **상태 표시줄** | "실행 중..." 메시지 표시 |

### 5.3 정지 및 리셋

| 동작 | 설명 |
|------|------|
| **⏹️ 정지** | 실행 플래그를 비활성화하여 현재 블록 실행 후 즉시 중단. LED 상태는 마지막 표시 상태 유지 |
| **🔄 리셋** | 실행 중단 + 모든 배치 블록 제거 + LED 전체 초기화 |
| **🔄 새로 만들기** (헤더) | 리셋과 동일한 기능 |

## 6. LED 매트릭스 상세

### 6.1 좌표 체계

LED 매트릭스는 5행 × 5열 격자이며, 좌상단이 (0, 0)이다.

```
     열0  열1  열2  열3  열4
행0 [ ]  [ ]  [ ]  [ ]  [ ]
행1 [ ]  [ ]  [ ]  [ ]  [ ]
행2 [ ]  [ ]  [ ]  [ ]  [ ]
행3 [ ]  [ ]  [ ]  [ ]  [ ]
행4 [ ]  [ ]  [ ]  [ ]  [ ]
```

### 6.2 LED 상태

| 상태 | 시각적 표현 |
|------|------------|
| **꺼짐** | 어두운 회색 (#333) |
| **켜짐** | 밝은 빨간색 (#ff0000) + 발광 효과(box-shadow) |

### 6.3 문자 패턴

알파벳, 숫자, 한글 자모 각각에 대해 5×5 비트맵 패턴이 사전 정의되어 있다. 총 정의된 문자 수는 영어 26자 + 숫자 10자 + 한글 자모 24자(자음 14 + 모음 10) + 특수문자 약 10자이다.

### 6.4 문자열 표시 방식

문자열의 각 글자를 순서대로 하나씩 LED에 표시한다. 한 글자 표시 후 약 600ms 대기 → 다음 글자 표시를 반복한다. MakeCode의 실제 스크롤 방식(좌로 밀기)과 달리 글자 단위 전환 방식이다.

## 7. 도움말 기능

### 7.1 자동 도움말

프로그램 최초 실행 시 도움말 팝업이 화면 우하단에 자동 표시되며, 5초 후 자동으로 닫힌다.

### 7.2 수동 도움말

헤더의 **❓ 도움말** 버튼을 클릭하면 도움말 팝업이 표시된다. **×** 버튼 또는 팝업 외부 영역 클릭으로 닫을 수 있다.

### 7.3 도움말 내용

```
💡 사용 방법
1. 왼쪽 팔레트에서 블록을 드래그하세요
2. "시작하면 실행" 또는 "무한반복" 영역에 놓으세요
3. 블록의 입력값을 수정할 수 있어요
4. ▶️ 실행 버튼을 눌러 결과를 확인하세요!
```
