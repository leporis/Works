## 5. 주요 내용

### 5.1 강좌 소개 (20분)

**목표**: 강좌 전체 흐름과 평가 방식 이해

- 강좌명: 사물인터넷과 빅데이터
- 핵심 흐름: 센서 → 데이터 수집 → 시각화 → 분석 → AI 모델
- 평가: 주간과제 80% + 최종프로젝트 15% + 성찰보고서 5%
- 2022 개정 교육과정 연계 (피지컬 컴퓨팅, 데이터 수집·분석)

### 5.2 바이브코딩 프로그램 시연 (30분)

**목표**: 바이브코딩의 가능성을 직접 확인

**수업집중도분석프로그램** 5개 탭 시연:
1. **탭 1 - 데이터 입력**: 온도, 소음, 집중도 데이터 입력
2. **탭 2 - 데이터 분석**: 산점도, 상관계수, 패턴 분석
3. **탭 3 - 학생 모델**: 규칙 기반 AI (IF-THEN)
4. **탭 4 - 교수 모델**: 머신러닝 AI
5. **탭 5 - 모델 비교**: 정확도 비교, 해석 가능성 논의

**핵심 질문**: "이 프로그램, 코드를 한 줄도 직접 작성하지 않고 만들었어요!"

### 5.3 예시 프로젝트 소개 (10분)

**목표**: 15주 프로젝트 흐름 이해

"수업 집중도" 프로젝트 단계:
1. 기획 → 2. 제작 → 3. 수집 → 4. 분석 → 5. 발표

**핵심 질문**: "온도가 높으면 집중도가 낮아질까?"  
→ 상관관계 vs 인과관계 구분 필요성 강조

### 5.4 마이크로비트 LED 실습 (30분)

**목표**: 바이브코딩 첫 체험

**실습 단계**:
1. Claude에게 MakeCode 블록 코드 요청
2. MakeCode에서 프로그램 작성
3. 시뮬레이터에서 결과 확인
4. 실제 마이크로비트에서 실행
5. 실행 모습 사진 촬영

**프롬프트 예시**:
> "마이크로비트 LED에 '홍길동'을 한 글자씩 순서대로 표시하는 MakeCode 블록 코드를 알려줘. 각 글자는 1초간 표시되고 다음 글자로 넘어가도록 해줘."

### 5.5 마무리 및 과제 안내 (10분)

**2주차 예고**:
- 바이브코딩 개념 심화
- Claude 사용법
- 센서 탐색 (내장 + Grove)

**1주차 과제**:
- 제출물: 실행 사진 + 대화목록 전문(PDF) + 대화목록 요약본(PDF)
- 기한: 일요일 자정 (LMS)
- 특이사항: 평가 점검용 (모든 학생 10% 만점 부여)
