# 사물인터넷과 빅데이터

청주교육대학교 컴퓨터교육과 강의 공개 자료입니다.

## 파일 목록

- **Syllabus** (강의계획서) — 교과목 기본 정보, 교과목 개요, 학습 목표, 2022 개정 교육과정 연계, 수업 운영 방식, 주차별 수업 개요, 평가 방법, 수업 자료 및 도구 안내를 포함합니다.

- **Lubric** (평가 기준) — 평가 배점 요약, 주간 과제 운영 방식 및 주차별 제출물, 등급 기준, 대화목록 평가 기준(프롬프트 적절성·대화 과정 적절성), 피어 평가, 최종 프로젝트(보고서·발표·성찰) 세부 기준, 제출 방법을 포함합니다.

각 자료는 MD와 PDF 두 가지 형식으로 제공됩니다.
