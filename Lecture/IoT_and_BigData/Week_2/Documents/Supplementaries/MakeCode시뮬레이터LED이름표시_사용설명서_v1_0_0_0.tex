\section*{3. 기능 상세}

\subsection*{3.1 블록 팔레트}

기본·입력·음악·LED·논리 5개 카테고리


\subsection*{3.2 드래그 앤 드롭}

블록을 코딩 영역에 드래그하여 프로그램 구성


\subsection*{3.3 이벤트 블록}

시작하면 실행, 버튼A/B 누르면, 흔들면 3가지


\subsection*{3.4 LED 시뮬레이터}

5\texttimes{}5 매트릭스에 문자열·숫자·아이콘 표시


\subsection*{3.5 실행/정지}

프로그램 실행 및 정지 제어


\subsection*{3.6 블록 삭제}

휴지통 영역에 드래그하여 삭제


\subsection*{3.7 문자열 입력}

블록의 텍스트 필드에 직접 이름 입력


\subsection*{3.8 스크롤 표시}

긴 문자열은 LED에서 자동 스크롤



\section*{4. 조작 방법}

\subsection*{4.1 기본 워크플로우}

1. \textbf{블록 선택}: 좌측 팔레트에서 원하는 블록 클릭 또는 드래그

2. \textbf{블록 배치}: 코딩 영역의 이벤트 블록 아래에 드롭

3. \textbf{값 입력}: 블록의 텍스트/숫자 필드에 값 입력

4. \textbf{실행}: \emoji{play-button} 실행 버튼 클릭 \textrightarrow{} LED 시뮬레이터에서 결과 확인

5. \textbf{수정}: 블록 드래그로 순서 변경 또는 휴지통에 삭제



\section*{5. 자주 묻는 질문 (FAQ)}

\textbf{Q: 한글 이름도 표시되나요?}


A: 네, 한글을 포함한 모든 유니코드 문자가 LED에 스크롤 표시됩니다.


\textbf{Q: 블록을 잘못 놓았어요.}


A: 블록을 드래그하여 하단 \emoji{wastebasket} 휴지통 영역에 놓으면 삭제됩니다.


\textbf{Q: 실행이 안 돼요.}


A: 이벤트 블록(시작하면 실행 등) 아래에 블록이 연결되어 있는지 확인하세요.
