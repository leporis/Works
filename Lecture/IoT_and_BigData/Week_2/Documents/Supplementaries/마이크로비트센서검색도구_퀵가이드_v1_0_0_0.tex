\section*{\emoji{rocket} 빠른 시작 (2가지 방법)}

\subsection*{방법 1: \emoji{speech-balloon} 텍스트로 검색}

\textbf{"무엇을 하고 싶은지"} 키워드로 검색합니다.


1. 검색창에 키워드 입력 (예: "온도", "거리", "색깔")

2. \textbf{검색} 클릭 또는 Enter

3. 결과: 해당 키워드와 관련된 센서 목록 + 추천


\begin{tcolorbox}[tipbox, title={\emoji{light-bulb} 팁}]
\emoji{light-bulb} 추천 키워드 버튼을 활용하면 더 빠릅니다!

\end{tcolorbox}

\subsection*{방법 2: \emoji{bar-chart} 데이터로 검색}

\textbf{"어떤 데이터가 필요한지"} 체크박스로 선택합니다.


1. \textbf{\emoji{bar-chart} 데이터로 검색} 탭 클릭

2. 필요한 데이터 항목에 체크 (예: \emoji{check-mark-button} 온도, \emoji{check-mark-button} 습도)

3. 자동으로 필요한 센서 조합 추천


\begin{tcolorbox}[tipbox, title={\emoji{light-bulb} 팁}]
\emoji{light-bulb} 내장 센서로 충분한 항목은 별도로 알려줍니다. 불필요한 외장 센서를 아낄 수 있어요!

\end{tcolorbox}
