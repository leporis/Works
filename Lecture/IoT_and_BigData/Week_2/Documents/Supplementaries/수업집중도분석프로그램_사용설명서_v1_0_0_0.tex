\section*{3. 기능 상세}

\subsection*{3.1 데이터 수집}

샘플 로드, 랜덤 생성, CSV 업로드, 직접 입력 4가지 방식


\subsection*{3.2 데이터 분석}

통계 카드 4개 + 산점도·히스토그램·상관관계 차트


\subsection*{3.3 학생 모델}

슬라이더로 온도·소음 기준 설정 \textrightarrow{} if-else 규칙 기반 예측


\subsection*{3.4 교수 모델}

최소자승법 다중 회귀 \textrightarrow{} 수식·계수·R² 자동 계산


\subsection*{3.5 모델 비교}

학생/교수 모델 예측 곡선 비교 + 정확도 대조


\subsection*{3.6 예측 테스트}

온도·소음 입력 \textrightarrow{} 두 모델의 예측 결과 비교


\subsection*{3.7 CSV 내보내기}

생성된 데이터를 CSV 파일로 다운로드


\subsection*{3.8 차트 시각화}

Chart.js 기반 실시간 차트 (4종)



\section*{4. 조작 방법}

\subsection*{4.1 기본 워크플로우}

1. \textbf{데이터 준비}: \emoji{inbox-tray} 수집 탭에서 샘플/랜덤/CSV/직접 입력 중 선택

2. \textbf{데이터 분석}: \emoji{bar-chart} 분석 탭에서 통계·차트 관찰

3. \textbf{학생 모델}: \emoji{man-student} 탭에서 슬라이더로 규칙 설정 \textrightarrow{} 예측 테스트

4. \textbf{교수 모델}: \emoji{man-teacher} 탭에서 학습 시작 \textrightarrow{} 수식·R² 확인

5. \textbf{비교}: \emoji{balance-scale} 탭에서 두 모델 정확도 대조



\section*{5. 자주 묻는 질문 (FAQ)}

\textbf{Q: 데이터를 얼마나 넣어야 하나요?}


A: 의미 있는 분석을 위해 최소 10건 이상을 권장합니다. 샘플 데이터(12건)로 시작해보세요.


\textbf{Q: R²가 낮으면 모델이 나쁜 건가요?}


A: 반드시 그렇지 않습니다. 실세계 데이터는 다양한 요인이 작용하므로 R²가 낮을 수 있습니다. 중요한 것은 모델의 한계를 이해하는 것입니다.


\textbf{Q: 학생 모델이 교수 모델보다 정확할 수도 있나요?}


A: 네, 데이터 패턴에 따라 가능합니다. 이것이 "정답은 하나가 아니다"라는 핵심 학습 포인트입니다.
