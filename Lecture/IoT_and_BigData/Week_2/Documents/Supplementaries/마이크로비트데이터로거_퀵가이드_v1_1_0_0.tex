\section*{\emoji{open-book} 이 문서 사용법}

\begin{tcolorbox}[tipbox]
\textbf{이 퀵가이드는 데이터로거 다이어그램과 함께 보는 내비게이션입니다.}

이 자료는 \textbf{정적 인포그래픽}입니다 — 클릭하거나 조작하는 기능은 없고, 보면서 이해하는 참고 자료입니다.

\section*{\emoji{rocket} 빠른 시작 — 이렇게 읽으세요}

\subsection*{\emoji{keycap-1} 데이터 수집 흐름 (위\textrightarrow{}아래)}

\begin{tcolorbox}[formulabox]
\begin{verbatim}
센서 → 마이크로비트 → 데이터로거(RTC+SD) → CSV 파일 → PC에서 분석
\end{verbatim}
\end{tcolorbox}

이 다이어그램은 이 흐름의 \textbf{첫 단계(센서 종류)}를 보여줍니다.

\section*{\emoji{thought-balloon} 생각해보기}

\begin{tcolorbox}[summarybox]
\textbf{센서가 측정한 데이터는 어디에 저장되나요?}

\vspace{0.3em}
마이크로비트 메모리? SD카드? 클라우드?

데이터를 저장하지 않으면 어떤 문제가 생길까요?

\vspace{0.3em}
\textbf{프로젝트에 꼭 외장 센서가 필요할까요?}

\vspace{0.3em}
내장 센서 6종만으로도 온도·조도·가속도를 측정할 수 있습니다.

"내장 센서로 충분한가?"를 먼저 판단하면 불필요한 비용을 줄일 수 있어요.

\end{tcolorbox}
