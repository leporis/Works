\section*{3. 기능 상세}

\subsection*{3.1 내장 센서 영역}

마이크로비트 V2 내장 센서 6종 (초록 테두리)


\subsection*{3.2 외장 센서 영역}

Grove 확장 센서 10종 (파랑 테두리)


\subsection*{3.3 데이터 흐름}

센서 \textrightarrow{} 마이크로비트 \textrightarrow{} RTC+SD \textrightarrow{} CSV 파이프라인 시각화


\subsection*{3.4 센서 카드}

각 센서의 측정 항목, 단위, 사용 예시 표시


\subsection*{3.5 색상 코딩}

내장(초록), 외장(파랑), 고정 장치(보라) 구분


\subsection*{3.6 반응형}

화면 크기에 따라 레이아웃 자동 조정

\section*{5. 자주 묻는 질문 (FAQ)}

\textbf{Q: 이 다이어그램을 조작할 수 있나요?}


A: 아닙니다. 정적 인포그래픽이므로 보면서 구조를 이해하는 참고자료입니다.


\textbf{Q: RTC가 없으면 시각 기록이 안 되나요?}


A: 마이크로비트 자체에 RTC가 없으므로, 정확한 시각 기록을 위해 외부 RTC 모듈(DS3231)이 필요합니다.


\textbf{Q: 데이터는 어디에 저장되나요?}


A: 마이크로비트에 연결된 SD카드에 CSV 파일로 저장됩니다. 이후 PC에서 USB로 읽습니다.
