\section*{\emoji{bullseye} 실습 활동}

\subsection*{활동 1: 내 이름 표시하기}

1. \textbf{문자열 보여주기} 블록을 \textbf{시작하면 실행}에 배치

2. 입력칸에 자기 이름 입력

3. \emoji{play-button} 실행 \textrightarrow{} LED에 이름이 나오는지 확인


\subsection*{활동 2: 이름 + 하트 반복하기}

1. \textbf{무한반복 실행}에 블록 3개 순서대로 배치:

\hspace{1em}$\bullet$ \emoji{memo} 문자열 보여주기: "이름"

\hspace{1em}$\bullet$ \emoji{pause-button} 일시정지: 500ms

\hspace{1em}$\bullet$ \emoji{red-heart} 아이콘 보여주기: 하트

2. \emoji{play-button} 실행 \textrightarrow{} 이름과 하트가 번갈아 나타나는지 관찰
