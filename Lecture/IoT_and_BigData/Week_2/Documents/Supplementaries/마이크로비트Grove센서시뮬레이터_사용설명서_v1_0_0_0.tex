\section*{3. 기능 상세}

\subsection*{3.1 센서 탐색}

10개 외장 센서 + 6개 내장 센서 카드 확인


\subsection*{3.2 드래그 앤 드롭}

센서 카드를 Grove Shield 포트(1\textasciitilde{}6)에 연결


\subsection*{3.3 키워드 추천}

13가지 카테고리에서 관련 센서 하이라이트


\subsection*{3.4 검색 필터}

텍스트 검색으로 센서 이름·설명 필터링


\subsection*{3.5 측정 데이터}

연결된 센서 조합의 데이터 항목 실시간 표시


\subsection*{3.6 데이터 조합}

최대 848가지 센서 조합 탐색 가능


\subsection*{3.7 포트 재배치}

연결된 센서를 다른 포트로 드래그 이동


\subsection*{3.8 초기화}

전체 포트 초기화 (모든 센서 제거)



\section*{4. 조작 방법}

\subsection*{4.1 기본 워크플로우}

1. \textbf{센서 탐색}: 우측 센서 카드 확인 또는 추천 키워드 활용

2. \textbf{센서 연결}: 카드를 드래그하여 중앙 포트에 드롭

3. \textbf{데이터 확인}: 하단 측정 데이터 패널에서 항목 확인

4. \textbf{조합 변경}: 포트의 센서 드래그로 이동 또는 제거



\section*{5. 자주 묻는 질문 (FAQ)}

\textbf{Q: 센서를 6개 넘게 연결할 수 있나요?}


A: Grove Shield의 I2C 포트가 6개이므로 최대 6개까지 연결 가능합니다.


\textbf{Q: 같은 센서를 2개 연결하면?}


A: 같은 I2C 주소를 가진 센서는 1개만 연결 가능합니다. 시뮬레이터에서 중복 연결을 방지합니다.


\textbf{Q: 내장 센서는 왜 드래그할 수 없나요?}


A: 내장 센서는 마이크로비트에 고정되어 있어 항상 사용 가능합니다. 별도 연결이 필요 없습니다.
