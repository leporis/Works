\section*{3. 기능 상세}

\subsection*{3.1 텍스트 검색}

키워드로 센서 이름·설명 검색 (예: '온도', '거리')


\subsection*{3.2 데이터 검색}

필요한 측정 데이터 항목 체크 \textrightarrow{} 센서 조합 추천


\subsection*{3.3 추천 키워드}

미리 정의된 키워드 버튼으로 빠른 검색


\subsection*{3.4 내장 충분 판정}

내장 센서만으로 충분한지 자동 판별


\subsection*{3.5 조합 추천}

필요한 데이터를 충족하는 최소 센서 조합 계산


\subsection*{3.6 결과 표시}

센서별 I2C 주소, 측정 항목, 특징 상세 표시



\section*{4. 조작 방법}

\subsection*{4.1 기본 워크플로우}

1. \textbf{방법 선택}: \emoji{speech-balloon} 텍스트 또는 \emoji{bar-chart} 데이터 탭 선택

2. \textbf{검색 입력}: 키워드 입력 또는 데이터 항목 체크

3. \textbf{결과 확인}: 추천 센서 목록과 내장 충분 여부 확인

4. \textbf{조합 결정}: 프로젝트에 사용할 센서 조합 선택



\section*{5. 자주 묻는 질문 (FAQ)}

\textbf{Q: 848가지 조합은 어떻게 계산되나요?}


A: 내장 센서 7항목은 항상 포함되고, 외장 10개 센서의 조합(2\textasciicircum{}10 - 1 = 1,023)에서 유효한 조합을 필터링한 결과입니다.


\textbf{Q: 검색 결과가 없어요.}


A: 키워드를 더 넓은 범위로 바꿔보세요. 예: '미세먼지' \textrightarrow{} '공기', '감지' \textrightarrow{} '측정'.


\textbf{Q: 내장으로 충분하다는데 외장을 써야 하나요?}


A: 프로젝트 목표에 따라 다릅니다. 대략적 추세만 필요하면 내장으로 충분하고, 정확한 수치가 필요하면 외장을 추천합니다.
