\section*{\emoji{rocket} 빠른 시작 (3단계)}

\subsection*{\emoji{keycap-1} 센서 탐색하기}

오른쪽 \textbf{\emoji{electric-plug} 외장 Grove 센서} 영역에 10개의 센서 카드가 있습니다. 카드를 \textbf{마우스 오버}하면 센서 상세 설명이 툴팁으로 나타납니다.


무엇을 측정할지 모르겠다면? \textbf{\emoji{clipboard} 추천 키워드} 버튼을 클릭하세요. "온도 습도", "밝기 조도", "거리 장애물" 등 13가지 카테고리에서 원하는 키워드를 선택하면 관련 센서가 \textbf{하이라이트}됩니다.


\subsection*{\emoji{keycap-2} 센서 연결하기}

하이라이트된 센서 카드를 \textbf{드래그}하여 중앙 Grove Shield의 빈 포트(1\textasciitilde{}6)에 \textbf{드롭}합니다. 포트에 센서가 연결되면 포트 색상이 변하고, 센서 이름이 포트에 표시됩니다.


최대 \textbf{6개 센서}를 동시에 연결할 수 있습니다. 연결된 센서를 다른 포트로 옮기려면 포트의 센서를 드래그하세요.


\subsection*{\emoji{keycap-3} 측정 데이터 확인하기}

화면 하단 \textbf{\emoji{bar-chart} 측정 데이터} 패널에서 현재 연결된 센서들의 데이터 항목을 확인합니다. 내장 센서 7항목은 항상 표시되고, 외장 센서 데이터는 연결할 때마다 추가됩니다.
