## 💻 실습 도구 안내

### ⭐⭐⭐ 1. 센서 시뮬레이터 v9

| 항목 | 내용 |
|------|------|
| 파일 | `microbit-sensor-simulator-v9.html` |
| 용도 | 마이크로비트 센서 조합 체험 (848가지 조합 가능) |
| 사용법 | Chrome 또는 Edge 브라우저로 파일 열기 |

**실습 과제**
1. 좌측 센서 목록에서 센서를 선택하여 드래그 앤 드롭으로 연결해 보세요
2. 내장 센서 3개 이상 조합해 보기
3. Grove 센서 2개 이상 추가해 보기
4. 교실 환경 측정용 센서 조합 만들어 보기

### ⭐⭐⭐ 2. 센서 검색 도구

| 항목 | 내용 |
|------|------|
| 파일 | `마이크로비트센서검색도구_v1_0_2_0_인코딩복원_2026-02-19.html` |
| 용도 | 자연어로 측정 목적을 입력하면 AI가 적합한 센서를 추천 |
| 사용법 | Chrome 또는 Edge 브라우저로 파일 열기 |

**입력 예시**
- "교실 공기질을 측정하고 싶어요"
- "복도에서 사람이 지나가는 걸 감지하고 싶어요"
- "운동장 온도 변화를 기록하고 싶어요"

### ⭐⭐ 3. I2C 연결도

| 항목 | 내용 |
|------|------|
| 파일 | `마이크로비트GroveI2C연결도_v1_0_2_0_인코딩복원_2026-02-19.html` |
| 용도 | I2C 주소 확인 및 센서 연결 시뮬레이션 |
| 사용법 | Chrome 또는 Edge 브라우저로 파일 열기 |

**주의사항**: 같은 I2C 주소를 가진 센서는 동시에 사용할 수 없어요!
- 예) VL53L0X(0x29) + TCS34725(0x29) → ❌ 충돌
- 예) SHT40(0x44) + BMP280(0x76) → ✅ 정상
