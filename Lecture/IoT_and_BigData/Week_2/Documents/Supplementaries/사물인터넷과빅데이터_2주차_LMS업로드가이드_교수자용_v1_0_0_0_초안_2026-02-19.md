## 업로드 순서 (권장)

1. LMS에 **2주차 강의자료** 폴더 생성
2. 하위 폴더 4개 생성: 복습 자료, 실습 자료, 참고 자료, 센서 이미지
3. 강의교재 PDF + 슬라이드 PDF를 루트에 업로드
4. 각 HTML 파일을 해당 폴더에 업로드
5. 센서 이미지를 센서 이미지 폴더에 업로드
6. 학생용 안내문 게시 (별도 문서 참조)

## 수업 중 활용 순서

| 시간 | 활동 | 사용 도구 |
|:----:|------|-----------|
| 0~5분 | 1주차 복습 | LED 시뮬레이터 (#3) |
| 5~25분 | 바이브코딩 소개 + Claude 실습 | 슬라이드 (#2) |
| 25~45분 | 센서 탐색 실습 | 센서 시뮬레이터 (#4) + 검색 도구 (#5) |
| 45~55분 | I2C 연결 이해 | I2C 연결도 (#6) |
| 55~65분 | 데이터로거·프로젝트 소개 | 데이터로거 (#7) + 집중도 (#8) |
| 65~75분 | 프로젝트 브레인스토밍 + 과제 안내 | 슬라이드 (#2) |
