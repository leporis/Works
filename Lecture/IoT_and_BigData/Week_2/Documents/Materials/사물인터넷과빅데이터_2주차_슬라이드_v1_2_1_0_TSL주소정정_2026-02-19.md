### 슬라이드 9: 효과적인 프롬프트 작성법

1. **구체적으로 요청하기**
   - 나쁜 예: "웹페이지 만들어줘"
   - 좋은 예: "자기소개 웹페이지를 만들어줘. 이름, 전공, 취미를 포함하고 파란색 테마로"

2. **형식 지정하기**
   - "HTML 파일 하나로 만들어줘"
   - "CSS는 인라인으로 넣어줘"

3. **단계적으로 수정하기**
   - 일단 만들고 → 확인하고 → 수정 요청
   - "배경색을 연한 회색으로 바꿔줘"

---

### 슬라이드 10: 프롬프트 예시 1 — 기본 웹페이지

프롬프트 예시 (코드 박스):
```
자기소개 웹페이지를 만들어줘.
이름: 김철수
전공: 수학교육과
취미: 등산, 독서
심플한 디자인으로 HTML 파일 하나로 만들어줘.
```

결과:
- Claude가 HTML 코드 생성
- 파일로 저장 → 브라우저에서 열기
- 마음에 안 들면 수정 요청

---

### 슬라이드 11: 프롬프트 예시 2 — 수정 요청

수정 프롬프트 (코드 박스):
```
방금 만든 웹페이지에서:
1. 배경색을 연한 파란색으로 바꿔줘
2. 글자 크기를 조금 키워줘
3. 사진을 넣을 수 있는 공간을 추가해줘
```

팁:
> 한 번에 여러 가지 수정을 요청해도 됩니다! 단, 너무 많으면 단계적으로 나누세요.

### 슬라이드 15: 센서로 무엇을 측정할 수 있을까?

학교 현장 적용 예시:
- **교실 환경 측정**: 온도, 습도, CO2 → 쾌적한 학습 환경 조건 찾기
- **복도 소음 분석**: 시간대별 소음 패턴 파악
- **급식실 대기 분석**: 거리 센서로 줄 서는 시간 측정
- **운동장 활동량**: 가속도 센서로 체육 시간 활동량 측정
- **도서관 이용 패턴**: 빛, 소음으로 조용한 시간대 찾기

경고 블록:
> 센서는 **현재 상태**를 측정할 뿐, **원인이나 미래를 예측하지 않습니다!**

### 슬라이드 21: 센서 선택 가이드

내장 센서로 충분한 경우:
- 간단한 온도/소음/빛 측정
- 기울기/방향 감지
- 교육용 데모

Grove 센서가 필요한 경우:
- 정밀한 측정이 필요할 때 (온습도, 기압)
- 내장 센서에 없는 기능 (CO₂, 거리, 컬러)
- 측정값을 화면에 표시 (OLED)
- 여러 환경 데이터 동시 수집 (BME680)

팁:
> 프로젝트 주제에 맞는 센서를 선택하세요!

### 슬라이드 22: Grove I2C 연결 방식

TikZ 다이어그램:
- micro:bit (상단, 녹색 보드)
- ↓ 연결선
- Grove Shield (중앙, 파란색 보드, I2C 포트 5개)
- ↓ 센서 연결선 3개
- 센서1, 센서2, 센서3 (하단, 빨간색)
- "최대 6개", "V2" 라벨

설명:
- I2C는 **2개 선(SDA, SCL)**으로 여러 센서 연결
- 각 센서는 고유 **주소**로 구분 (예: 0x44, 0x76)
- Grove Shield로 **쉽게 연결** (납땜 불필요)
