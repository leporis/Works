| 슬라이드 | 시간 | 교사 활동 | 발화 예시 |
|:--------:|:----:|----------|----------|

> **교수 팁**: 핵심 메시지를 강조하세요: "AI는 판단하지 않는다, 계산할 뿐이다"와 "코딩을 의심하라"를 이 시간에 다시 상기시키세요. 학생들이 바이브코딩을 '쉬운 방법'이 아니라 '다른 방법'으로 인식하도록 유도하세요.

### 3.2 I2C 연결도

| I2C 주소 | 해당 센서들 | 동시 사용 가능 여부 |
|:--------:|-----------|:-----------------:|
| 0x18 | LIS3DHTR (가속도) | 단독 사용 가능 |
| 0x29 | TSL2561 (조도), TCS34725 (컬러), VL53L0X (거리) | 3개 중 **1개만** 선택 |

| 0x44 | SHT40 (온습도) | 단독 사용 가능 |
| 0x76 | BMP280 (기압), BME680 (복합 환경) | 2개 중 **1개만** 선택 |

**충돌/정상 예시:**
- 충돌: VL53L0X (0x29) + TCS34725 (0x29) → ✗
- 정상: SHT40 (0x44) + BMP280 (0x76) + 내장 마이크 → ✓

## 7. 예상 문제 및 대응

### 7.1 기술적 문제

| 문제 상황 | 대응 방법 |
|----------|----------|
| Claude 계정이 없는 학생 | 수업 시간에 바로 가입 가능. 구글 계정으로 빠르게 가입. 가입 전에는 교수자 화면으로 시연 관찰 |
| Claude가 영어로만 응답 | 프롬프트에 '한국어로 답해줘' 추가 요청 |
| Artifact 패널이 안 보임 | Chrome/Edge 브라우저 사용 권장. 다른 탭 닫기 후 재시도 |
| HTML 파일 다운로드 버튼 안 보임 | Artifact 패널 오른쪽 상단 '...' 메뉴 확인 |
| 센서 시뮬레이터 로딩 느림 | 브라우저 캐시 삭제, 다른 탭 닫기 |
| I2C 연결도 작동 안 함 | Chrome/Edge에서 파일을 로컬로 열기. 브라우저 보안 설정 확인 |

### 7.2 수업 진행 문제

| 문제 상황 | 대응 방법 |
|----------|----------|
| 실습 시간 부족 (20분 내 완성 못 함) | 기본 페이지 완성 목표로 하향. 추가 수정은 과제로 이어서 하도록 안내 |
| 센서 탐색 시간 부족 | Grove 센서 갤러리(#18-20) 생략, I2C + 시뮬레이터 핵심만 진행 |
| 학생 질문이 너무 많음 | 핵심 질문만 수업 중 답변, 나머지는 LMS Q&A 게시판으로 유도 |
| 프로젝트 아이디어가 안 떠오르는 학생 | 예시 주제 5가지(슬라이드 #25)를 그대로 선택해도 된다고 안내 |
