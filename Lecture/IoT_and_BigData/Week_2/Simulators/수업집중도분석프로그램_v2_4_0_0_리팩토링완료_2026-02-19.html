#날짜시간대온도(℃)소음(dB)집중도
데이터가 없습니다

👨‍🎓 학생이 만드는 규칙 기반 AI 학생이 직접 "기준"을 정하고, 그 기준에 따라 집중도를 판단하는 프로그램입니다.
📋
"규칙의 기준은 누가 정했나요?"
→ 바로 여러분입니다!

⚙️ 규칙 설정

// 학생이 만든 집중도 판단 규칙
if (온도 >= 26℃ || 소음 >= 70dB) {
  return "🔴 낮음";
} else if (온도 <= 24℃ && 소음 <= 50dB) {
  return "🟢 높음";
} else {
  return "🟡 보통";
}
🔴 "낮음" 기준 온도 (이상)26℃
🔴 "낮음" 기준 소음 (이상)70dB
🟢 "높음" 기준 온도 (이하)24℃
🟢 "높음" 기준 소음 (이하)50dB

🌡️ 온도25℃
🔊 소음55dB

📐 학습된 모델

집중도 = 12.50 + (온도 × -0.30) + (소음 × -0.05)
-0.30
온도 계수 (β₁)
-0.05
소음 계수 (β₂)
-
R² (설명력)

📖 계수 해석 방법

온도 계수 (β₁)음수 → 온도가 올라가면 집중도 감소
예: -0.15면 온도 1℃↑ → 집중도 0.15↓
소음 계수 (β₂)음수 → 소음이 커지면 집중도 감소
예: -0.03이면 소음 10dB↑ → 집중도 0.3↓
R² (설명력)0~100% 사이 값, 100%에 가까울수록 설명력 높음
67%면 "변동의 67%를 온도+소음으로 설명"
