## 1. 개요

### 1.1 목적

이 문서는 **교육용 시뮬레이터의 설계·구현·교육 기능**을 위한 기술 가이드입니다. 코드 품질, 교육 기능, 사용자 경험을 일관되게 유지하여 교육 효과를 극대화합니다.

공통 규칙(버전 관리, 이미지 관리, 코드 리뷰)은 **버전관리지침** 및 **문서관리지침**을 참조합니다. 시뮬레이터 완성 후 **문서화·SVG 제작·품질 점검**은 **코딩관리지침**을, 기존 시뮬레이터의 **구조 변경·규모 확장**은 **ML시뮬레이터 확장리팩토링 지침서**를 참조합니다.

### 1.2 적용 대상

| 분야 | 시뮬레이터 예시 |
|------|----------------|
| **머신러닝** | 선형 회귀, KNN, K-means, 의사결정트리 |
| **물리** | 포물선 운동, 진자, 파동, 회로 |
| **경제** | 수요-공급, 게임 이론, 시장 균형 |
| **생태** | 포식자-피식자, 개체군 동태, 전염병 확산 |
| **화학** | 반응 속도, 평형, 적정 |
| **수학** | 함수 그래프, 확률 분포, 기하 변환 |

### 1.3 핵심 설계 원칙

| 원칙 | 설명 | 적용 예 |
|------|------|---------|
| **일관성** | 모든 시뮬레이터에서 동일한 레이아웃 | 3열 그리드, 모드 탭 |
| **직관성** | 복잡한 개념을 시각적으로 표현 | 애니메이션, 실시간 피드백 |
| **단계적 학습** | 설정 → 실행 분리 | 모드 탭으로 구분 |
| **즉각적 피드백** | 파라미터 변경 시 실시간 반영 | 슬라이더 → 차트 연동 |
| **소프트코딩** | 설정값의 중앙 집중 관리 | CSS 변수, CONFIG 객체 |
| **코드 품질** | 유지보수 가능한 깔끔한 코드 | DRY, 네이밍 컨벤션 |
| **교육적 가치** | 이해·체험·확인의 3요소 충족 | 설명 패널, 재생 기능 |
| **문서 정합성** | 코드-문서-이미지 버전 일치 | 동기화된 버전 관리 |

### 1.4 개발 산출물

시뮬레이터 개발 시 다음 산출물을 함께 작성합니다:

```
시뮬레이터 프로젝트
├── [프로젝트명]_시뮬레이터_v버전.html    ← 실행 파일
├── [프로젝트명]_개발일지_v버전.md        ← 기술 문서 (상세)
├── [프로젝트명]_퀵가이드_v버전.md        ← 사용 안내 (요약)
└── (선택) 이미지/                        ← 스크린샷, 다이어그램
```

## 3. 코드 품질 원칙

### 3.1 네이밍 컨벤션

| 대상 | 규칙 | ❌ 나쁜 예 | ✅ 좋은 예 |
|------|------|----------|----------|
| 변수 | camelCase, 의미 명확 | `d`, `x1` | `dataCount`, `currentStep` |
| 상수 | UPPER_SNAKE_CASE | `maxvalue` | `MAX_STEPS` |
| 함수 | camelCase, 동사 시작 | `data()` | `generateData()` |
| 불리언 | is/has/can 접두사 | `running` | `isRunning` |
| CSS 클래스 | kebab-case | `btnPrimary` | `btn-primary` |

### 3.2 코드 구조화 원칙

#### DRY (Don't Repeat Yourself)

```javascript
// ❌ 중복 코드
function drawChart1() { ctx.clearRect(0, 0, w, h); ctx.strokeStyle = '#BCAAA4'; drawAxes(); }
function drawChart2() { ctx.clearRect(0, 0, w, h); ctx.strokeStyle = '#BCAAA4'; drawAxes(); }

// ✅ 공통 함수 추출
function initCanvas(ctx, w, h) { ctx.clearRect(0, 0, w, h); ctx.strokeStyle = '#BCAAA4'; drawAxes(); }
```

#### 단일 책임 원칙

```javascript
// ❌ 너무 많은 일 → function runSimulation() { /* 150줄... */ }

// ✅ 책임 분리
function runSimulation() {
    if (!validateInput()) return;
    initializeState();
    executeSimulationLoop();
    updateUI();
}
```

### 3.3 데드 코드 제거

| 유형 | 예시 | 조치 |
|------|------|------|
| 사용되지 않는 변수 | `let oldData = [];` | 삭제 |
| 호출되지 않는 함수 | `function legacyCalc() {}` | 삭제 |
| 주석 처리된 코드 | `// doOldThing();` | 삭제 |

### 3.4 에러 처리

```javascript
function loadFile(file) {
    if (!file) { showError(UI_TEXT.ERROR_NO_FILE); return; }
    if (!isValidFileType(file.name)) { showError(UI_TEXT.ERROR_FILE_FORMAT); return; }
    const reader = new FileReader();
    reader.onload = (e) => {
        try { processData(parseData(e.target.result)); }
        catch (err) { showError(UI_TEXT.ERROR_PARSE_FAILED); }
    };
    reader.onerror = () => showError(UI_TEXT.ERROR_READ_FAILED);
    reader.readAsText(file);
}
```

### 7.2 3가지 입력 방식

| 방식 | 특징 | 교육적 목적 |
|------|------|------------|

```javascript
let trainingHistory = [];
let isPlaying = false, playbackIndex = 0, playbackTimer = null;

function recordTrainingFrame(epoch, params, loss) {
    trainingHistory.push({
        epoch, params: JSON.parse(JSON.stringify(params)), loss, timestamp: Date.now()
    });
}

function playTrainingHistory() {
    if (trainingHistory.length === 0) { showToast('먼저 학습을 실행해주세요'); return; }
    isPlaying = true; playbackIndex = 0; updatePlaybackUI();
    function playNextFrame() {
        if (!isPlaying || playbackIndex >= trainingHistory.length) { stopPlayback(); return; }
        const frame = trainingHistory[playbackIndex];
        restoreState(frame.params); drawChart(); updateLossChart(playbackIndex);
        updateProgressBar(playbackIndex / trainingHistory.length);
        playbackIndex++;
        playbackTimer = setTimeout(playNextFrame, getPlaybackDelay());
    }
    playNextFrame();
}

function stopPlayback() {
    isPlaying = false;
    if (playbackTimer) { clearTimeout(playbackTimer); playbackTimer = null; }
    if (trainingHistory.length > 0) {
        restoreState(trainingHistory[trainingHistory.length - 1].params); drawChart();
    }
    updatePlaybackUI();
}

function getPlaybackDelay() {
    const speed = parseInt(document.getElementById('playbackSpeed').value);
    return CONFIG.EDUCATION.PLAYBACK.DELAY_MAX - 
           (speed - 1) * (CONFIG.EDUCATION.PLAYBACK.DELAY_MAX - CONFIG.EDUCATION.PLAYBACK.DELAY_MIN) / 9;
}
```
