## 🧠 K-means 클러스터링이란?

```
데이터를 K개의 그룹으로 자동 분류하는 비지도 학습 알고리즘

📌 핵심 아이디어
1. K개의 중심점 설정
2. 각 데이터를 가장 가까운 중심점의 클러스터에 할당
3. 중심점을 클러스터 데이터들의 평균으로 이동
4. 수렴할 때까지 2~3 반복
```

### 1️⃣ 데이터 생성
```
[학습 모드] → 데이터 개수 설정 → [✨ 데이터 생성] 클릭
```
- 데이터 개수: 20~200개
- 클러스터 개수 (K): 2~8개

```
초기화 방법 선택 → [⚡ 클러스터링 실행] 클릭
```

- **Random**: 랜덤하게 중심점 선택

### 3️⃣ 클러스터 할당 예측
```
[🔮 추론 모드] 클릭 → X, Y 좌표 조절 → 자동으로 클러스터 할당
```

## 🔍 초기화 방법 비교

### Random 초기화
```
장점: 단순, 빠름
단점: 결과가 매번 다름, 국소 최적해에 빠질 수 있음
```

### K-means++ 초기화
```
장점: 중심점을 멀리 떨어뜨려 빠른 수렴, 안정적
단점: 초기화에 약간 더 오래 걸림
```

💡 **권장**: 대부분의 경우 **K-means++**가 더 좋은 결과를 제공합니다.

## 🎯 실습 활동 예시

### 활동 1: K값 변화 관찰
1. K=2로 클러스터링 → Inertia 확인
2. K=3으로 클러스터링 → Inertia 확인
3. K=4로 클러스터링 → Inertia 확인
4. K값이 증가하면 Inertia는 어떻게 변하나요?

### 활동 2: 초기화 방법 비교
1. Random 초기화로 3번 실행 → 결과 비교
2. K-means++ 초기화로 3번 실행 → 결과 비교
3. 어느 방법이 더 안정적인가요?

### 활동 3: 분산 영향 관찰
1. 분산 '낮음'으로 데이터 생성 → 클러스터링
2. 분산 '높음'으로 데이터 생성 → 클러스터링
3. 분산이 높으면 Inertia는 어떻게 변하나요?

### 활동 4: 경계 탐색
1. 추론 모드로 전환
2. 두 클러스터 경계 근처 좌표 입력
3. 어느 클러스터에 할당되는지 확인

## ❓ 자주 묻는 질문

**Q: 클러스터링이 끝나지 않아요**
→ 최대 반복 횟수를 늘려보세요. 또는 데이터가 너무 분산되어 있을 수 있습니다.

**Q: 같은 데이터로 실행했는데 결과가 다릅니다**
→ Random 초기화는 매번 다른 중심점을 선택합니다. K-means++를 사용하거나, Random으로 여러 번 실행 후 가장 낮은 Inertia를 선택하세요.

**Q: 빈 클러스터가 생겼어요**
→ K값이 너무 크거나 데이터가 특정 영역에 밀집되어 있습니다. K값을 줄이거나 분산을 높이세요.

**Q: Inertia가 낮을수록 항상 좋나요?**
→ 네, 하지만 K값을 늘리면 자동으로 낮아집니다. 적절한 K값을 찾는 것이 중요합니다.

**Q: 재생이 너무 빠르거나 느려요**
→ 재생 속도 슬라이더를 조절하세요. 1(매우 느림) ~ 10(최고속)

## 🔑 핵심 용어

| 용어 | 의미 |
|------|------|
| **K-means** | 데이터를 K개의 그룹으로 나누는 클러스터링 알고리즘 |
| **클러스터** | 유사한 데이터들의 그룹 |
| **중심점 (Centroid)** | 클러스터의 중심, 데이터들의 평균 위치 |
| **할당 (Assignment)** | 각 데이터를 가장 가까운 중심점의 클러스터에 배정 |
| **Inertia** | 클러스터 내 제곱합, 클러스터의 밀집도 측정 (작을수록 좋음) |
| **수렴 (Convergence)** | 중심점 이동이 거의 없어지는 상태 |
| **K-means++** | 중심점을 멀리 떨어뜨려 초기화하는 개선된 방법 |
| **비지도 학습** | 레이블 없는 데이터에서 패턴을 찾는 학습 방법 |

## 📐 수식 이해하기

### 유클리드 거리
```
distance = √[(x₁ - x₂)² + (y₁ - y₂)²]

예: (30, 40)과 (50, 60) 사이 거리
   = √[(30-50)² + (40-60)²]
   = √[400 + 400]
   = √800
   ≈ 28.28
```

### Inertia 계산
```
Inertia = Σ(각 점 - 중심점)²

예: 클러스터 1에 3개 점, 중심점 (50, 50)
   점1 (48, 52): (48-50)² + (52-50)² = 8
   점2 (51, 49): (51-50)² + (49-50)² = 2
   점3 (50, 50): (50-50)² + (50-50)² = 0
   Inertia = 8 + 2 + 0 = 10
```
