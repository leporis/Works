# K-means 클러스터링 시뮬레이터 개발일지

**문서 버전: v1.0.0.1**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
K-means 클러스터링 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 **K-means 클러스터링**의 원리를 직관적으로 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표
| 목표 | 설명 |
|------|------|
| 비지도 학습 이해 | 레이블 없는 데이터를 자동으로 그룹화하는 원리 학습 |
| 클러스터링 과정 체험 | 중심점 이동과 재할당의 반복적 최적화 과정 시각화 |
| 초기화 방법 비교 | Random vs K-means++ 초기화 방법의 차이 이해 |
| 클러스터 할당 학습 | 새로운 데이터가 어떤 클러스터에 속하는지 예측 체험 |

### 1.4 개발 배경
- 비지도 학습의 가장 기본적인 알고리즘인 K-means를 직관적으로 학습
- 중심점 이동 과정을 애니메이션으로 시각화하여 알고리즘 이해 향상
- 2D 좌표계에서 클러스터 형성 과정을 명확하게 표현

## 2. 시스템 설계

### 2.1 입출력 변수

| 구분 | 변수명 | 범위 | 설명 |
|:----:|--------|------|------|
| 입력 | X 좌표 | 0 ~ 100 | 데이터의 X축 위치 |
| 입력 | Y 좌표 | 0 ~ 100 | 데이터의 Y축 위치 |
| 파라미터 | K | 2 ~ 8 | 클러스터 개수 |
| 출력 | 클러스터 ID | 0 ~ K-1 | 할당된 클러스터 번호 |

### 2.2 K-means 알고리즘

```
1. 초기화: K개의 중심점 설정
   - Random: 랜덤하게 K개 선택
   - K-means++: 멀리 떨어진 점들을 우선 선택

2. 반복:
   a. 할당 (Assignment):
      각 데이터를 가장 가까운 중심점의 클러스터에 할당
      
   b. 업데이트 (Update):
      각 클러스터의 중심점을 해당 클러스터 데이터들의 평균으로 이동
      
   c. 수렴 체크:
      중심점 이동이 임계값 이하면 종료
      
3. 최종 결과: K개의 클러스터와 중심점
```

### 2.3 거리 계산 (유클리드 거리)

```javascript
distance(p1, p2) = √[(x₁ - x₂)² + (y₁ - y₂)²]
```

### 2.4 품질 지표

| 지표 | 설명 | 계산식 |
|------|------|--------|
| **Inertia** | 클러스터 내 제곱합 | Σ(각 점 - 중심점)² |
| **반복 횟수** | 수렴까지 걸린 반복 | - |
| **수렴 여부** | 임계값 이하 수렴 | \|이전 Inertia - 현재 Inertia\| < 0.01 |

### 2.5 시뮬레이션 기본 설정

| 항목 | 기본값 | 조절 범위 |
|------|--------|-----------|
| 데이터 개수 | 100개 | 20 ~ 200개 |
| 클러스터 개수 (K) | 3개 | 2 ~ 8개 |
| 분산 정도 | 보통 | 낮음 / 보통 / 높음 |
| 최대 반복 횟수 | 50회 | 10 ~ 100회 |
| 초기화 방법 | K-means++ | Random / K-means++ |

### 5.2 핵심 알고리즘

#### K-means++ 초기화
```javascript
function initCentroidsKMeansPlusPlus() {
    const cents = [];
    // 첫 번째 중심점: 랜덤 선택
    cents.push(random point);
    
    // 나머지 중심점: 거리 비례 확률로 선택
    while (cents.length < K) {
        distances = 각 점의 최소 거리;
        totalDist = sum(distances);
        r = random(0, totalDist);
        // 확률적 선택
        선택된 점 추가;
    }
    return cents;
}
```

#### 클러스터 할당
```javascript
function assignClusters() {
    return data.map(p => {
        let minDist = Infinity;
        let cluster = 0;
        centroids.forEach((c, idx) => {
            const d = distance(p, c);
            if (d < minDist) {
                minDist = d;
                cluster = idx;
            }
        });
        return cluster;
    });
}
```

#### 중심점 업데이트
```javascript
function updateCentroids() {
    const newCentroids = [];
    for (let k = 0; k < K; k++) {
        const points = data.filter((p, i) => assignments[i] === k);
        if (points.length === 0) {
            // 빈 클러스터: 기존 중심점 유지
            newCentroids.push(centroids[k]);
        } else {
            // 평균 계산
            const avgX = mean(points.map(p => p.x));
            const avgY = mean(points.map(p => p.y));
            newCentroids.push({ x: avgX, y: avgY });
        }
    }
    return newCentroids;
}
```
