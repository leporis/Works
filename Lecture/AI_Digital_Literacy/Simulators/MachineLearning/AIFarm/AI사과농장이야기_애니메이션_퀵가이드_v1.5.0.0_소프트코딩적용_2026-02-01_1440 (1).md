## 📚 장면별 학습 포인트

### 1장: 딥러닝 CNN 🌳
```
📷 카메라 촬영 → 🧠 CNN 분석 → ✅ 92% 익음!

💡 핵심: 수천 장의 사진으로 미리 학습한 AI가
         새로운 사과 사진을 보고 익었는지 판단해요.
```
**지도 학습** - 정답(익음/안익음)을 알려주고 학습

### 2장: KNN (K-최근접 이웃) 🔍
```
📊 특성 측정 → 🔍 가장 가까운 3개 찾기 → 🏷️ 다수결!

💡 핵심: 당도, 산도, 무게가 비슷한 사과들 중
         가장 많은 품종으로 결정해요.
```
**지도 학습** - 이미 품종을 아는 사과로 학습

### 3장: K-means 클러스터링 📦
```
📦 사과들 → 🎯 3개 그룹으로 자동 분류 → 상품/중품/하품

💡 핵심: 정답 없이 크기가 비슷한 사과들끼리
         스스로 3개 그룹을 만들어요.
```
**비지도 학습** - 정답 없이 패턴을 찾아 분류

### 4장: 선형 회귀 💰
```
품질 점수 → 📈 가격 = 30 × 품질 + 1000

💡 핵심: 과거 판매 데이터에서 품질과 가격의
         관계(규칙)를 찾아 공식을 만들어요.
```
**지도 학습** - 과거 가격 정답을 보고 학습

### 5장: 결정 트리 🚚
```
등급 = 상품? → 예: 백화점 🏬
             → 아니오 → 중품? → 예: 마트 🛒
                              → 아니오: 온라인/주스 📱

💡 핵심: 예/아니오 질문을 따라가면
         자동으로 출하처가 결정돼요.
```
**지도 학습** - 과거 출하 데이터로 학습

## ❓ 자주 묻는 질문

**Q: 소리가 안 나와요**
→ 🔊 버튼이 눌려있는지 확인하세요. 음소거 상태면 🔇로 표시됩니다.

**Q: 재생이 멈췄어요**
→ 장면 전환 중입니다. 2~3초 후 자동으로 다음 장면이 시작됩니다.

**Q: 특정 장면으로 바로 가고 싶어요**
→ 하단의 ● 점(장면 인디케이터)을 클릭하세요.

**Q: TTS 목소리가 이상해요**
→ 브라우저마다 TTS 품질이 다릅니다. Chrome을 권장합니다.

**Q: 속도를 조절하고 싶어요**
→ 속도 드롭다운에서 0.5x~1.5x 선택 가능합니다.

## 🔑 핵심 용어 정리

| 용어 | 의미 |
|------|------|
| **CNN (Convolutional Neural Network)** | 이미지 분석에 특화된 딥러닝 |
| **KNN (K-Nearest Neighbors)** | K개의 가장 가까운 이웃으로 분류 |
| **K-means** | K개의 그룹으로 자동 군집화 |
| **선형 회귀** | 직선 공식으로 값 예측 |
| **결정 트리** | 예/아니오 질문으로 분류 |
| **지도 학습** | 정답을 알려주고 학습 |
| **비지도 학습** | 정답 없이 패턴 발견 |
