## 4. 핵심 기술 결정사항

### 4.1 이미지 표시 시스템

```javascript
// 이미지 라이브러리 (20개 SVG)
const imageLibrary = {
    ripeApple: { svg: `<svg>...</svg>`, label: '🍎 잘 익은 사과' },
    unripeApple: { svg: `...`, label: '🍏 덜 익은 사과' },
    cnnAnalysis: { ... },
    // ... 총 20개
};

// 나레이션 데이터에 image 속성 추가
{ speaker: 'farmer', text: '빨갛게 잘 익었지?', image: 'ripeApple' }
```

### 4.2 이미지 카테고리

| 카테고리 | 이미지 키 |
|----------|----------|
| 사과 | ripeApple, unripeApple |
| 분석 | cnnAnalysis, result92, knnAnalysis, kmeansCluster |
| 품종 | hongro, busa, aori, mixedApples |
| 등급 | gradeL, gradeM, gradeS |
| 가격 | qualityScore, priceFormula |
| 출하 | department, mart, online, juice, decisionTree |
| 캐릭터 | farmer, robot |

### 4.3 CSS 구조

```css
.narration-image {
    position: fixed;
    top: 50%; left: 50%;
    transform: translate(-50%, -50%);
    z-index: 90;
    /* 페이드 애니메이션 */
}
.narration-display {
    display: none;  /* v1.4.0.3에서 비활성화 */
}
```
