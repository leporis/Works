# AI 사과 농장 이야기 애니메이션 개발일지

**문서 버전: v1.5.0.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
AI 사과 농장 이야기 - 인터랙티브 스토리텔링 애니메이션

### 1.2 목적
초등 예비교사를 대상으로 **머신러닝의 5가지 핵심 알고리즘**을 스토리텔링 방식으로 학습할 수 있도록 인터랙티브 애니메이션을 개발한다.

### 1.3 교육 목표

| 목표 | 설명 |
|------|------|
| ML 알고리즘 이해 | 딥러닝(CNN), KNN, K-means, 선형회귀, 결정트리의 원리 학습 |
| 실생활 연결 | 사과 농장이라는 친숙한 맥락에서 AI 활용 사례 체험 |
| 지도/비지도 학습 구분 | 각 알고리즘의 학습 유형 이해 |
| 문제-도구 매칭 | 문제 유형에 따른 적절한 알고리즘 선택 능력 |

### 1.4 개발 배경
- 기존 ML 시뮬레이터의 개별 알고리즘 학습을 통합된 스토리로 연결
- 캐릭터(농부 김사과, AI 로봇 알파)를 통한 감정적 몰입 유도
- TTS 음성 나레이션으로 청각적 학습 경험 제공

## 2. 시스템 설계

### 2.1 콘텐츠 구조

| 장면 | 주제 | ML 알고리즘 | 학습 유형 |
|:----:|------|------------|----------|
| 프롤로그 | 등장인물 소개 | - | - |
| 1장 | 수확의 시작 | 딥러닝 CNN | 지도 학습 |
| 2장 | 품종 구분 | KNN (K-최근접) | 지도 학습 |
| 3장 | 등급 분류 | K-means 클러스터링 | 비지도 학습 |
| 4장 | 가격 예측 | 선형 회귀 | 지도 학습 |
| 5장 | 출하 결정 | 결정 트리 | 지도 학습 |
| 에필로그 | 요약 및 마무리 | 전체 통합 | - |

### 2.2 캐릭터 설정

| 캐릭터 | 역할 | 음성 설정 |
|--------|------|----------|
| 내레이터 | 장면 전환, 설명 | rate: 1.0, pitch: 1.0 |
| 농부 김사과 | 문제 제기, 질문 | rate: 0.9, pitch: 0.85 (낮고 느린 목소리) |
| AI 로봇 알파 | 해결책 제시, 설명 | rate: 1.05, pitch: 1.4 (높고 빠른 목소리) |

### 2.3 이미지 라이브러리 (v1.4.0.0 신규)

| 카테고리 | 이미지 키 | 용도 |
|----------|----------|------|
| 사과 | ripeApple, unripeApple | 익은/덜 익은 사과 |
| CNN | cnnAnalysis, result92 | 딥러닝 분석 과정 |
| 품종 | hongro, busa, aori, mixedApples | 품종별 사과, KNN |
| 등급 | gradeL, gradeM, gradeS, kmeansCluster | K-means 시각화 |
| 가격 | qualityScore, priceFormula | 선형 회귀 수식 |
| 출하 | department, mart, online, juice, decisionTree | 결정 트리 |
| 캐릭터 | farmer, robot | 등장인물 |

## 3. 주요 기능

### 3.1 재생 컨트롤

| 기능 | 설명 |
|------|------|
| ▶️/⏸️ 재생/일시정지 | 스페이스바 또는 버튼 클릭 |
| 🔊/🔇 음소거 | M키 또는 버튼 클릭 |
| 속도 조절 | 0.5x ~ 1.5x (5단계) |
| 진행 바 | 클릭으로 특정 위치 이동 |
| 장면 인디케이터 | 점 클릭으로 장면 직접 이동 |
| ← → 키보드 | 이전/다음 장면 이동 |

### 3.2 TTS (Text-to-Speech)

| 기능 | 설명 |
|------|------|
| 한국어 음성 | Web Speech API 활용 |
| 캐릭터별 음성 | rate, pitch 차별화 |
| Chrome 버그 대응 | 5초 간격 resume() 호출 |
| 음소거 시 대기 | 글자당 80ms 기준 자동 대기 |
| 상태 표시 | 준비됨/재생중 상태 표시 |

### 3.3 시각화 요소

| 요소 | 설명 |
|------|------|
| 플로우 다이어그램 | 순차적 애니메이션 |
| 클러스터 시각화 | K-means 그룹 표현 |
| 회귀 차트 | 점과 회귀선 애니메이션 |
| 결정 트리 | 분기 구조 시각화 |
| 대사별 이미지 | SVG 기반 20개 이미지 라이브러리 |

### 3.4 장면 전환 (v1.3.0.0 신규)

| 기능 | 설명 |
|------|------|
| 전환 효과음 | Web Audio API로 C-E-G 차임벨 멜로디 |
| 자동 장면 전환 | 나레이션 완료 후 자동 다음 장면 |
| 페이드 애니메이션 | 0.8초 opacity 전환 |

### 5.2 소프트코딩 구현 (v1.5.0.0 신규)

#### CONFIG 객체

```javascript
const CONFIG = {
    // 장면별 지속 시간 (ms)
    SCENE_DURATION: {
        PROLOGUE: 30000,
        CHAPTER: 45000,
        CHAPTER_SHORT: 40000,
    },
    
    // 애니메이션/타이밍
    PAUSE_BETWEEN_NARRATION: {
        SAME_SPEAKER: 1000,
        DIFFERENT_SPEAKER: 1500,
    },
    TRANSITION_SOUND_DURATION: 2500,
    NARRATION_START_DELAY: 500,
    
    // 음성 설정
    VOICE: {
        NARRATOR: { rate: 1.0, pitch: 1.0, icon: '📖' },
        FARMER: { rate: 0.9, pitch: 0.85, icon: '👨‍🌾' },
        ROBOT: { rate: 1.05, pitch: 1.4, icon: '🤖' }
    },
    // ...
};
```

#### UI_TEXT 객체

```javascript
const UI_TEXT = {

```

```css

    /* 1단계: 기본 색상 */

    /* 2단계: 의미적 색상 */

    /* 간격 시스템 */
```

### 5.3 핵심 기술 결정사항

| 결정 | 이유 |
|------|------|
| Web Speech API | 외부 의존성 없이 브라우저 내장 TTS 활용 |
| Web Audio API | 파일 없이 코드로 효과음 생성 |
| SVG 이미지 라이브러리 | 외부 이미지 파일 의존성 제거, 인라인 렌더링 |
| 캡션 숨김 | 이미지와 음성으로 충분한 정보 전달 |
| CONFIG/UI_TEXT 객체 | 소프트코딩으로 유지보수성 향상 |
