## 2. 프로젝트 개요

### 2.1 목적
사과 과수원 운영 과정에서 5가지 머신러닝 기술이 어떻게 활용되는지를 **스토리텔링 애니메이션**으로 제작하여 초등 예비교사에게 AI 개념을 쉽게 전달

### 2.2 핵심 콘셉트
- **농부 김사과**와 **AI 로봇 알파**가 대화하며 협력하는 스토리
- 각 단계에서 농부가 문제를 제기하고, 로봇이 AI 기술로 해결책을 설명
- 나레이터가 장면 전환 및 요약을 담당

### 2.3 5단계 AI 파이프라인

| 단계 | 작업 | AI 기술 | 학습 유형 |
|:----:|------|---------|-----------|
| 1 | 🌳 수확 | 딥러닝 CNN | 지도 학습 |
| 2 | 🔍 품종 구분 | KNN | 지도 학습 |
| 3 | 📦 등급 분류 | K-means | 비지도 학습 |
| 4 | 💰 가격 예측 | 선형 회귀 | 지도 학습 |
| 5 | 🚚 출하 결정 | 결정 트리 | 지도 학습 |

## 3. 스토리 개발 과정

### 3.1 초기 아이디어 (대화를 통한 스토리 구성)

사용자 요청:
> "사과 과수원에서 수확-선별-가격책정-출하 전 과정에 5가지 머신러닝 기술을 적용하는 과정을 교육용 스토리로 만들어달라"

**스토리 구조 설계:**
1. 사과가 한꺼번에 익지 않는 문제 → **딥러닝 CNN**으로 익은 사과 판별
2. 수확 후 품종이 섞이는 문제 → **KNN**으로 특성 기반 품종 분류
3. 매년 달라지는 크기 기준 문제 → **K-means**로 자동 그룹화
4. 품질에 따른 적정 가격 산정 → **선형 회귀**로 가격 예측
5. 등급별 출하처 결정 문제 → **결정 트리**로 자동 분류

### 3.2 스토리 풍부화 요청

사용자 요청:
> "농부와 로봇이 같이 작업하면서 대화를 나눕니다. 농부가 '이 사과는 빨간색이야 잘 익었군... 그런데 로봇 너는 어떻게 구분하니?' 물으면 로봇이 'CNN이라는 지도학습 딥러닝 방법으로...' 대답하는 형식"

**적용 결과:**
- 각 장면에 **농부-로봇 대화** 추가 (8~10개 대사/장면)
- 세 캐릭터의 **음성 차별화** (나레이터/농부/로봇)
- 교육 내용을 자연스러운 대화로 전달

### 3.3 결정 트리 수정 요청

사용자 지적:
> "결정트리에서 백화점, 대형마트, 온라인, 주스공장으로 나누어야 하는데 대형마트는 빠졌네요"

**수정 결과:**
```
등급 = 상품? → 예 → 🏬 백화점
           → 아니오 → 중품? → 예 → 🛒 대형마트
                          → 아니오 → 크기 OK? → 예 → 📱 온라인
                                              → 아니오 → 🧃 주스공장
```

## 5. 핵심 기술 결정사항

### 5.1 TTS 음성 시스템

| 항목 | 결정 | 이유 |
|------|------|------|
| TTS 방식 | Web Speech API | 별도 파일 없이 브라우저에서 즉시 사용 |
| 나레이션 큐 | Promise + AbortController | 대사 중단 버그 완전 해결 |
| Chrome 버그 대응 | 5초마다 resume() 호출 | 긴 텍스트 멈춤 방지 |

### 5.2 캐릭터별 음성 설정

| 캐릭터 | 아이콘 | 속도 (rate) | 톤 (pitch) | 말풍선 색상 |
|--------|:------:|:-----------:|:----------:|-------------|
| 나레이터 | 📖 | 1.0 | 1.0 | 검정 (#000) |
| 농부 김사과 | 👨‍🌾 | 0.9 (느림) | 0.85 (낮음) | 갈색 (#8D6E63) |
| 로봇 알파 | 🤖 | 1.05 (빠름) | 1.4 (높음) | 파랑 (#2196F3) |

### 5.3 대화 간격 설정

| 상황 | 대기 시간 |
|------|----------|
| 같은 캐릭터 연속 대사 | 1,000ms |
| 다른 캐릭터로 전환 | 1,500ms |
| 장면 시작 후 첫 대사 | 500ms |

## 6. 장면별 나레이션 스크립트 요약

### 프롤로그 (5개 대사)
- 나레이터: 도입
- 농부: 일손 부족 고민
- 로봇: 자기소개
- 농부: 기대 표현
- 나레이터: 협력 시작

### 1장: 수확 - 딥러닝 CNN (9개 대사)
- 농부가 익은/덜 익은 사과 설명
- 농부: "넌 어떻게 구분하니?"
- 로봇: CNN, 수천 장 사진 학습, 92% 확률 설명
- 농부: 지도 학습 이해

### 2장: 품종 - KNN (9개 대사)
- 농부: 품종 섞임 문제 제기
- 로봇: KNN, 당도/산도/무게 측정, 가장 가까운 친구들
- 농부: 다수결 원리 이해

### 3장: 등급 - K-means (9개 대사)
- 농부: 매년 달라지는 크기 기준 문제
- 로봇: K-means, 비지도 학습, 자동 그룹화
- 농부: 정답 없이도 분류 가능함에 놀람

### 4장: 가격 - 선형 회귀 (8개 대사)
- 농부: 적정 가격 고민
- 로봇: 선형 회귀, 품질-가격 관계 학습
- 농부: 품질 1점당 30원 상승 이해

### 5장: 출하 - 결정 트리 (10개 대사)
- 농부: 4개 출하처 중 선택 고민
- 로봇: 결정 트리, 예/아니오 질문 흐름
- 상품→백화점, 중품→대형마트, 하품(크기OK)→온라인, 하품(크기X)→주스공장

### 에필로그 (8개 대사)
- 나레이터: 5가지 AI 기술 요약
- 농부/로봇: 감사 인사
- 농부: "문제에 맞는 도구 선택" 깨달음
- 나레이터: 마무리
