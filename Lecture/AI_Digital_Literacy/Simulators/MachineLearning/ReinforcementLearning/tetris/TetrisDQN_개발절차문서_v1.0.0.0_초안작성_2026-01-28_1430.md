## 1. 프로젝트 개요

### 1.1 목적

DQN(Deep Q-Network) 강화학습 알고리즘을 사용하여 테트리스 게임을 스스로 학습하는 AI를 개발합니다. 사용자가 AI의 학습 과정을 실시간으로 관찰하고, 학습 결과를 분석할 수 있는 교육용 시뮬레이터입니다.

### 1.2 주요 특징

| 특징 | 설명 |
|------|------|
| **실시간 학습 시각화** | AI가 학습하는 과정을 그래프와 지표로 실시간 확인 |
| **초보자 친화적** | 한글 설명과 직관적인 UI로 DQN 개념 이해 가능 |
| **다양한 모드** | 학습, 테스트, 관전 모드 지원 |
| **학습 저장/불러오기** | 브라우저 및 파일로 학습 결과 저장 가능 |
| **자동 정지 기능** | 목표 성공률 달성 시 자동 정지 |

### 1.3 개발 기간

2026년 1월 28일 (단일 세션 집중 개발)

## 3. 개발 단계별 진행

### 3.1 1단계: 기본 구조 (Iteration 1-3)

**목표:** DQN 기반 테트리스 AI 기본 동작 구현

**구현 내용:**
- 테트리스 게임 보드 (10×20)
- 7종 테트로미노 구현 (I, O, T, S, Z, J, L)
- 기본 게임 로직 (회전, 이동, 라인 클리어)
- DQN 에이전트 기본 구조
- 5가지 특징 추출 (높이, 구멍, 울퉁불퉁함, 최대 높이)

**핵심 코드 구조:**

```
게임 보드 → 특징 추출 → 신경망 → Q-값 → 행동 선택 → 보상 → 학습
```

### 3.2 2단계: 대시보드 구현 (Iteration 4-5)

**목표:** 학습 과정 시각화 대시보드 개발

**구현 내용:**
- 실시간 그래프 4종 (점수, 생존 피스, 성공 점수, 손실)
- 학습 단계 진행바 (초기→탐색→학습→응용→숙련)
- 핵심 지표 표시 (Best, Avg, ε, 등급)
- 속도 조절 슬라이더

**학습 단계 정의:**

| 단계 | 게임 수 | 아이콘 | 설명 |
|------|---------|--------|------|
| 초기 | 0-100 | 🌱 | 무작위 탐색 시작 |
| 탐색 | 100-500 | 🔍 | 패턴 발견 중 |
| 학습 | 500-2000 | 📚 | 전략 학습 중 |
| 응용 | 2000-5000 | 🎯 | 전략 최적화 |
| 숙련 | 5000+ | 🏆 | 고급 플레이 |

### 3.3 3단계: 한글화 및 설명 추가 (Iteration 6)

**목표:** 초보자를 위한 한글 설명 및 가이드 추가

**구현 내용:**
- 그래프별 해석 가이드
- DQN 개념 설명 패널
- 학습 단계별 설명
- 상태 표시 (좋음/주의/안정)

### 3.4 4단계: 저장/불러오기 (Iteration 7)

**목표:** 학습 결과 영속성 구현

**구현 내용:**
- localStorage 자동 저장 (100게임마다)
- 수동 저장/불러오기 버튼
- JSON 파일 내보내기/가져오기
- 저장 상태 피드백

**저장 데이터 구조:**

```javascript
{
  version: 3,
  timestamp: Date.now(),
  agent: {
    net: { layers, weights, biases, ... },
    target: { ... },

    mem: [...]
  },
  stats: {
    gen: 500,
    bestScore: 1200,

```

### 3.5 5단계: 테스트/관전 모드 (Iteration 8)

**목표:** AI 성능 평가 및 관찰 기능 추가

**구현 내용:**
- 🧪 **테스트 모드**: 학습 중지 후 현재 실력 평가
- 👁️ **관전 모드**: 학습 없이 AI 플레이 관찰
- **성공 점수 시스템**: 100점 만점 종합 평가
- **등급 시스템**: S/A/B/C/D/F

**성공 점수 계산:**

```
성공 점수 = (생존 피스/200 × 50%) + (점수/5000 × 30%) + (라인/20 × 20%)
```

**등급 기준:**

| 등급 | 점수 | 색상 | 설명 |
|------|------|------|------|
| S | 90+ | 금색 | 완벽! |
| A | 80+ | 초록 | 훌륭함 |
| B | 70+ | 파랑 | 좋음 |
| C | 50+ | 노랑 | 보통 |
| D | 30+ | 주황 | 부족 |
| F | 30- | 빨강 | 더 학습 필요 |

### 3.6 6단계: 학습 분석 기능 (Iteration 9)

**목표:** AI가 배운 내용을 시각화

**구현 내용:**
- **효율 그래프**: 학습량 vs 성공률 추이
- **최적 학습 지점 감지**: 수익 체감점 자동 탐지
- **AI 인사이트**: 가중치 분석 기반 학습 내용 표시
- **학습 리포트 모달**: 종합 분석 보고서

**AI 인사이트 분석 항목:**

| 항목 | 아이콘 | 설명 |
|------|--------|------|
| 구멍 회피 | 🕳️ | 빈 공간 최소화 |
| 높이 관리 | 📏 | 낮게 쌓기 |
| 평탄화 | 🏔️ | 표면 평평하게 |
| 최대높이 제한 | ⚠️ | 한쪽 치우침 방지 |

### 3.7 7단계: 도움말 및 UX 개선 (Iteration 10)

**목표:** 사용자 경험 향상

**구현 내용:**
- ❓ **도움말 모달**: 첫 방문 시 자동 표시
- 📋 **학습 리포트 모달**: 상세 분석
- 버튼 크기 통일
- 마우스 호버 툴팁

## 4. 핵심 알고리즘

### 4.1 신경망 구조

```
입력층 (4)  →  은닉층1 (64)  →  은닉층2 (32)  →  출력층 (1)
   ↓              ↓               ↓              ↓
[특징벡터]    [ReLU 활성화]   [ReLU 활성화]   [Q-값]
```

### 4.2 특징 추출

```javascript
features = [
  총 높이 / 200,      // 정규화된 총 블록 높이
  구멍 수 / 50,       // 정규화된 구멍 개수
  울퉁불퉁함 / 20,    // 정규화된 인접 열 높이 차이 합
  최대 높이 / 20      // 정규화된 가장 높은 열
]
```

### 4.3 보상 함수

```javascript
reward = 
  (클리어 라인)² × 10    // 라인 클리어 보너스
  + (이전 구멍 - 현재 구멍) × 5    // 구멍 감소 보상
  + (이전 높이 - 현재 높이) × 2    // 높이 감소 보상
  + (이전 울퉁불퉁 - 현재 울퉁불퉁) × 1  // 평탄화 보상
  + 0.1    // 생존 보너스
```

### 4.4 DQN 학습 과정

```
1. 현재 상태에서 가능한 모든 배치 위치 계산
2. ε-greedy로 행동 선택:
   - 확률 ε: 무작위 선택 (탐색)
   - 확률 1-ε: Q-값 최대 선택 (활용)
3. 행동 실행 후 보상 계산
4. 경험 (s, r, s', done) 메모리에 저장
5. 배치 샘플링하여 학습:
   - Q_target = r + γ × max(Q_target_net(s'))
   - Loss = (Q_predicted - Q_target)²
6. Adam 옵티마이저로 가중치 업데이트
7. 500스텝마다 타겟 네트워크 동기화
8. ε 점진적 감소 (1.0 → 0.01)
```

### 4.5 하이퍼파라미터

| 파라미터 | 값 | 설명 |
|----------|-----|------|
| γ (gamma) | 0.99 | 할인율 |
| ε 초기값 | 1.0 | 탐색률 시작값 |
| ε 최소값 | 0.01 | 탐색률 최소값 |
| ε 감소율 | 0.9992 | 매 스텝 감소 |
| 학습률 | 0.001 | Adam learning rate |
| 배치 크기 | 32 | 샘플 배치 |
| 메모리 크기 | 30,000 | 경험 리플레이 |
| 타겟 업데이트 | 500 | 스텝 주기 |

## 7. 사용 방법

### 7.1 시작하기

1. React 환경에서 `tetris-v3.jsx` 파일 실행
2. 첫 방문 시 도움말 모달 자동 표시
3. ▶️ 학습 버튼 클릭하여 AI 학습 시작

### 7.2 학습 설정

1. **속도 조절**: 슬라이더로 5ms~100ms 조절 (낮을수록 빠름)
2. **자동 정지**: 토글 ON 후 목표 점수 설정 (30~95점)

### 7.3 학습 관찰

| 그래프 | 좋은 신호 |
|--------|----------|
| 📊 점수 | 우상향 |
| 🧩 생존 | 우상향 |
| ⭐ 성공 점수 | 우상향 |
| 📉 손실 | 하향 |

### 7.4 평가하기

1. 🧪 테스트 버튼 클릭
2. AI가 1게임 플레이
3. 게임 오버 시 성적표 확인

### 7.5 저장하기

| 방법 | 용도 |
|------|------|
| 💾 브라우저 저장 | 같은 브라우저에서 계속 학습 |
| 📤 파일 내보내기 | 다른 기기로 이동 또는 백업 |
