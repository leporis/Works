## 📖 이 문서 사용법

> **이 퀵가이드는 RL 틱택토 시뮬레이터와 함께 보는 내비게이션입니다.**

| 상황 | 참조 섹션 |
|------|----------|
| 처음 시작할 때 | ⚡ 초속성 실습 |
| AI를 학습시키고 싶을 때 | 🚀 빠른 시작 |
| 화면이 뭔지 모르겠을 때 | 🖥️ 화면 구성 |
| 파라미터를 바꿔보고 싶을 때 | ⚙️ 파라미터 설명 |
| 뭘 관찰해야 할지 모를 때 | 🔍 관찰 포인트 |
| 문제가 생겼을 때 | 🆘 트러블슈팅 |
| 용어가 이해 안 될 때 | 🔑 핵심 용어 |

## ⚡ 1분 초속성 실습

| 단계 | 행동 | 확인 |
|:---:|------|:----:|
| 1 | **[▶️ 학습 시작]** 버튼을 클릭하세요 | 그래프가 움직이기 시작하나요? |
| 2 | 30초 기다리세요 (속도 **1ms** 상태) | 에피소드 수가 빠르게 올라가나요? |
| 3 | **[👤 사람 대전]** 버튼을 클릭하세요 | 3×3 보드가 크게 나타나나요? |
| 4 | 빈 칸을 클릭해서 돌을 놓으세요 | AI가 바로 응수하나요? 🎉 |

> 💡 AI를 이긴다면 아직 학습이 부족한 것입니다! 더 학습시켜 보세요.

## 🚀 빠른 시작 (3단계)

### 1️⃣ AI 학습시키기

1. 파일(`rl-tictactoe-v3.html`)을 브라우저에서 엽니다
2. 좌측 패널에서 **📋 학습 방식**을 확인합니다 → `🔄 자기대국`이 기본입니다
3. **[▶️ 학습 시작]** 클릭 → 우측 그래프가 실시간으로 변합니다
4. 좌측 상단 진행바를 관찰하세요 → `🌱초기 → 🔍탐색 → 📚학습 → 🎯응용 → 🏆숙련`

> 🎯 **5,000판** 이상이면 효과가 보이기 시작하고, **20,000판** 이상이면 최적에 가까워집니다.

### 2️⃣ AI와 대국하기

1. **[👤 사람 대전]** 클릭 → 대국 화면으로 전환됩니다
2. 보드의 빈 칸을 클릭하여 돌을 놓으세요
3. AI가 즉시 응수합니다 → 오른쪽 **🧠 AI 판단 근거** 히트맵을 확인하세요
4. 이기기 어렵다면 **🎚️ AI 난이도** 슬라이더를 낮추세요

### 3️⃣ 실험하기

1. **[📊 대시보드]**로 돌아와서 학습을 정지합니다
2. 좌측 **⚙️ 파라미터**에서 학습률(α)을 바꿔봅니다
3. 다시 **[▶️ 학습 시작]** → 그래프 변화를 관찰하세요

## ⚙️ 파라미터 설명

| 파라미터 | 학술 용어 | 기본값 | 쉬운 설명 |
|----------|----------|:------:|----------|
| **학습률 α** | Learning Rate | 0.30 | 한 번의 경험에서 얼마나 배울지. 높으면 빠르지만 불안정 |
| **할인율 γ** | Discount Factor | 0.90 | 미래 보상을 얼마나 중요하게 볼지. 높으면 장기적 전략 |
| **ε 감소율** | Epsilon Decay | 0.9995 | 탐험을 줄이는 속도. 1에 가까울수록 천천히 줄임 |
| **속도** | — | 1ms | 학습 속도. 1ms가 가장 빠름 (500판/틱) |
| **AI 난이도** | — | 100% | 사람 대전에서 AI가 최적수를 두는 확률 |

### 파라미터 실험 가이드

| 실험 | 설정 | 관찰 포인트 |
|------|------|------------|
| 빠른 학습 vs 느린 학습 | α를 0.9 → 0.01로 변경 | 승률 그래프의 수렴 속도 차이 |
| 탐험 많이 vs 적게 | ε 감소율을 0.99 → 0.9999로 | 탐험률 그래프, 학습 후반 성능 |
| 단기 전략 vs 장기 전략 | γ를 0.1 → 0.99로 변경 | AI 플레이 스타일 변화 |

## 🎯 실습 활동

### 활동 1: AI의 성장 관찰

**목표**: 강화학습의 학습 과정을 직접 관찰합니다.

| 단계 | 행동 |
|:---:|------|
| 1 | 속도를 **1ms**로 설정하고 **[▶️ 학습 시작]** 클릭 |
| 2 | **1,000판** 학습 후 정지 → AI와 대국 → 결과 기록 |
| 3 | **10,000판** 학습 후 정지 → AI와 대국 → 결과 기록 |
| 4 | **50,000판** 학습 후 정지 → AI와 대국 → 결과 기록 |

**기록표**:

| 학습량 | AI 강도 | 대국 결과 (승/무/패) | 관찰 내용 |
|:------:|:-------:|:-------------------:|----------|
| 1,000판 | | | |
| 10,000판 | | | |
| 50,000판 | | | |

### 활동 2: 파라미터의 영향

**목표**: 학습률(α)이 학습에 미치는 영향을 실험합니다.

| 단계 | 행동 |
|:---:|------|
| 1 | **[🗑️ 초기화]**로 학습 데이터 삭제 |
| 2 | α를 **0.01**로 설정 → 10,000판 학습 → 강도 기록 |
| 3 | 초기화 → α를 **0.5**로 → 10,000판 학습 → 강도 기록 |
| 4 | 초기화 → α를 **0.99**로 → 10,000판 학습 → 강도 기록 |

**기록표**:

| 학습률 α | 10,000판 후 강도 | 그래프 모양 | 관찰 내용 |
|:--------:|:---------------:|:-----------:|----------|
| 0.01 | | | |
| 0.50 | | | |
| 0.99 | | | |

|------|------|--------|
| 화면이 안 뜨거나 하얀 화면만 보여요 | 인터넷 연결 필요 (CDN) | 인터넷 연결 확인 후 새로고침 |
| 학습 시작했는데 그래프가 안 움직여요 | 이미 정지 상태 | **[▶️ 학습 시작]** 버튼을 다시 클릭 |
| AI가 너무 쉬워요 | 학습 부족 | 속도 1ms로 20,000판 이상 학습 |
| AI를 절대 이길 수 없어요 | AI가 최적 학습 완료 | **🎚️ AI 난이도**를 50% 이하로 낮추세요 |
| 이전 학습 데이터가 사라졌어요 | 브라우저 데이터 초기화 | **💾** 버튼으로 저장하거나 **📤** 파일 내보내기 |
| 그래프가 심하게 요동쳐요 | 학습률(α)이 너무 높음 | α를 **0.1 이하**로 낮추세요 |
| 글씨가 잘 안 보여요 (프로젝터) | 다크 테마가 밝은 환경에 부적합 | **[☀️ 밝게]** 버튼으로 라이트 모드 전환 |

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| **Q-값** | Q-value | 상태에서 행동의 가치를 나타내는 수치 | "이 칸에 두면 얼마나 좋을까?" 점수 |
| **탐험률(ε)** | Epsilon (ε) | 무작위 행동을 선택할 확률 | "모험할까, 잘하는 걸 할까?" 비율 |
| **학습률(α)** | Learning Rate | 새로운 경험을 얼마나 반영할지 | "한 번 경험에서 얼마나 배울까?" |
| **할인율(γ)** | Discount Factor | 미래 보상의 현재 가치 비율 | "나중 보상도 소중하게 여길까?" |
| **보상** | Reward | 행동의 결과에 대한 피드백 | 이기면 +점수, 지면 -점수 |
| **에피소드** | Episode | 한 판의 게임 (시작~종료) | 한 판 = 한 에피소드 |
| **Q-테이블** | Q-table | 모든 상태-행동 쌍의 Q-값 저장소 | AI의 "경험 수첩" |
| **정규화** | Canonicalization | 같은 모양의 보드를 하나로 통합 | 돌려도 같은 모양이면 같은 걸로 취급 |
