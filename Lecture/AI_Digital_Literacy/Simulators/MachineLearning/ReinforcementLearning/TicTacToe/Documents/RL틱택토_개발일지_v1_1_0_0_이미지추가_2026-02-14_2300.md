## 1. 프로젝트 개요

### 1.1 목적

틱택토(Tic-Tac-Toe) 게임에서 **Q-Learning 강화학습** 알고리즘이 시행착오를 통해 스스로 게임 전략을 학습하는 과정을 시각적으로 관찰하고 체험할 수 있는 교육용 시뮬레이터입니다.

### 1.2 교육 목표

| 목표 | 내용 |
|------|------|
| **개념 이해** | 강화학습의 핵심 개념(상태, 행동, 보상, Q-값)을 직관적으로 이해 |
| **과정 관찰** | AI가 무작위 탐험에서 최적 전략으로 수렴하는 과정을 실시간 관찰 |
| **체험 학습** | AI와 직접 대국하며 학습 수준을 체감 |
| **파라미터 실험** | 학습률, 할인율, 탐험률 등을 조절하며 학습에 미치는 영향 탐구 |

### 1.3 배경

초등 예비교사 대상 「컴퓨팅사고와 AI교육」 강좌에서, 강화학습의 원리를 직접 눈으로 확인할 수 있는 도구가 필요하여 개발하였습니다. 틱택토는 규칙이 단순하면서도 최적 전략이 존재하여 강화학습 교육에 적합합니다.

## 2. 시스템 설계

### 2.1 알고리즘: Q-Learning

| 항목 | 설명 |
|------|------|
| 알고리즘 | Q-Learning (오프-폴리시 시간차 학습) |
| 상태 | 보드 배치 (3×3, 각 칸 = X / O / 빈칸) |
| 행동 | 빈 칸 중 하나에 돌 놓기 (0~8) |
| Q-테이블 | 상태→행동별 가치 매핑 |
| 갱신 식 | Q(s,a) ← Q(s,a) + α × (r - Q(s,a)) |

### 2.2 대칭성 정규화

보드의 **8가지 대칭 변환**(4회전 × 2반사)을 활용하여, 동일 모양의 보드를 하나의 정규 상태로 통합합니다.

| 항목 | 미적용 | 적용 |
|------|:------:|:----:|
| 첫수 고유 상태 | 9 | **3** (중앙/모서리/변) |
| Q-테이블 최대 크기 | ~5,000 | **~765** |
| 학습 효율 | 1× | **~8×** |

### 2.3 보상 체계

| 결과 | 보상값 | 설명 |
|------|:------:|------|
| 승리 | +1.0 | 이기면 보상 |
| 패배 | -1.0 | 지면 패널티 |
| 무승부 | +0.3 | 비기면 소량 보상 |
| 매 수 | -0.01 | 단계 패널티 (빠른 승리 유도) |

### 2.4 에이전트 구성

| 항목 | 기본값 | 범위 |
|------|:------:|------|
| 학습률 α | 0.3 | 0.01~1.0 |
| 할인율 γ | 0.9 | 0.1~1.0 |
| 초기 탐험률 ε | 1.0 | — |
| 최소 탐험률 | 0.01 | — |
| ε 감소율 | 0.9995 | 0.99~1.0 |

## 3. 주요 기능

### 3.1 기능 목록

| # | 기능 | 설명 |
|:-:|------|------|
| 1 | **자기대국 학습** | X↔O 에이전트가 상호 대국하며 학습 |
| 2 | **단독 학습** | 한쪽만 학습, 상대는 랜덤 (X 단독 / O 단독) |
| 3 | **AI 대전 관전** | 학습된 두 AI의 대국을 실시간 관전 |
| 4 | **사람 vs AI** | 사용자가 AI와 직접 대국, 난이도 조절 가능 |
| 5 | **대칭성 정규화** | 8가지 대칭 변환으로 Q-테이블 압축 (8배 효율) |
| 6 | **단계 패널티** | 매 수 -0.01 보상으로 최단 경로 승리 유도 |
| 7 | **배치 학습** | 속도별 동적 배치 크기 (1ms=500판, 100ms=10판) |
| 8 | **Q-값 히트맵** | 각 칸의 Q-값을 색상으로 시각화 |
| 9 | **실시간 그래프** | 승률, 탐험률, 강도, Q-상태 수 추이 |
| 10 | **학습 단계 표시** | 초기→탐색→학습→응용→숙련 5단계 진행바 |
| 11 | **AI 난이도 조절** | 0~100% 슬라이더로 AI 강도 조절 |
| 12 | **저장/불러오기** | localStorage 자동저장 + JSON 파일 내보내기/가져오기 |
| 13 | **학습 보고서** | Markdown 형식 보고서 다운로드 |
| 14 | **다크/라이트 테마** | 토글 스위치로 즉시 전환, localStorage 기억 |
| 15 | **파라미터 실시간 조절** | α, γ, ε 감소율 슬라이더 (동기/개별 모드) |

### 3.2 학습 모드

| 모드 | X 에이전트 | O 에이전트 | 용도 |
|------|:----------:|:----------:|------|
| 자기대국 | 학습 | 학습 | 두 AI 상호 발전 |
| X 단독 | 학습 | 랜덤 | X 전용 강화 |
| O 단독 | 랜덤 | 학습 | O 전용 강화 (후공 전략) |

### 3.3 난이도 체계 (사람 대전)

| 난이도 | 범위 | 색상 | 의미 |
|--------|:----:|:----:|------|
| 🐣 초보 | 0~19% | 🟢 | 거의 랜덤, 누구나 승리 가능 |
| 중급 | 20~49% | 🟡 | 가끔 좋은 수, 적당한 도전 |
| 고급 | 50~79% | 🟠 | 대부분 최적수, 실수 유도 필요 |
| 🤖 최강 | 80~100% | 🔴 | 거의 완벽, 충분히 학습 시 무승부만 가능 |

## 5. 기술 구현

### 5.1 기술 스택

| 항목 | 기술 |
|------|------|
| UI 프레임워크 | React 18 (CDN) |
| JSX 변환 | Babel Standalone (CDN) |
| 언어 | JavaScript (ES6+) |
| 렌더링 | React DOM, 인라인 스타일 |
| 저장 | localStorage, JSON 파일 |
| 배포 형태 | **단일 HTML 파일** (CDN 의존) |

### 5.2 핵심 아키텍처

```
┌── QAgent 클래스 ─────────────────────────────────────────┐
│  qTable: { canonicalState → [Q₀, Q₁, ..., Q₈] }       │
│                                                          │
│  chooseAction(board, available, training)                │
│    └→ canonicalize(board) → {state, transformIdx}       │
│    └→ ε-greedy 선택 (정규 좌표계)                        │
│    └→ 역변환하여 원본 좌표 반환                           │
│                                                          │
│  updateMoves(history, finalReward)                      │
│    └→ 역방향 순회: Q(s,a) += α×(r - Q(s,a))           │
│    └→ r = step_penalty + γ × r (단계 할인)              │
│                                                          │
│  getQForBoard(board) → 원본 좌표의 Q-값 배열             │
└──────────────────────────────────────────────────────────┘

┌── 대칭성 변환 ───────────────────────────────────────────┐
│  SYMMETRIES[8]: 원본→변환 매핑 (4회전 × 2반사)          │
│  SYMMETRIES_INV[8]: 변환→원본 역매핑                     │
│  canonicalize(board): 사전순 최소 표현 선택              │
└──────────────────────────────────────────────────────────┘

┌── 테마 시스템 ───────────────────────────────────────────┐
│  THEMES = { dark: {...}, light: {...} }                  │
│  ThemeContext → useTheme() 훅으로 전체 컴포넌트 전파      │
│  20+ 속성: bg, text, panel, cell, border, modal 등      │
└──────────────────────────────────────────────────────────┘
```
