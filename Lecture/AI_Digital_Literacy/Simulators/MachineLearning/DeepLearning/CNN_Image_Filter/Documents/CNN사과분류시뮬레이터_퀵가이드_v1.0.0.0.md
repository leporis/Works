## 🚀 빠른 시작 (3단계)

### 1️⃣ 준비 — 학습 데이터 만들기

왼쪽 **📦 학습 데이터** 패널에서 설정합니다.

| 슬라이더 | 추천 값 | 의미 |
|---------|:------:|------|
| 데이터 수 | 100 | 사과 이미지 100개 (익은 50 + 덜 익은 50) |
| 노이즈 강도 | 15% | 약간의 잡음 → 현실적인 데이터 |
| 밝기 변동폭 | 20% | 다양한 밝기 조건 |

👉 [🎲 학습 데이터 생성] 버튼을 클릭하면 "✅ 생성 완료" 메시지가 나타납니다.

### 2️⃣ 학습 — CNN 모델 훈련하기

오른쪽 **⚙️ 학습 설정** 패널에서 시작합니다.

| 설정 | 추천 값 | 의미 |
|-----|:------:|------|
| 학습률 | 0.01 (보통) | 가중치 업데이트 크기 |
| 에포크 수 | 30 | 전체 데이터를 30번 반복 |
| 시각화 간격 | 5 에포크마다 | 차트 갱신 빈도 |

👉 [▶ 시작]을 클릭하면 학습이 진행됩니다. 중앙 차트에서 손실이 내려가고 정확도가 올라가는 것을 관찰하세요!

| 버튼 | 기능 |
|------|------|
| ▶ 시작 | 학습 시작 또는 재개 |
| ⏸ 일시정지 | 학습 일시 중단 |
| ⏭ 단계 | 1 에포크만 실행 |

### 3️⃣ 분류 — 학습된 모델로 사과 분류하기

[🔮 분류 모드] 탭으로 전환합니다.

| 조작 | 결과 |
|------|------|
| 🍎/🍏 버튼으로 사과 타입 선택 | 분류할 사과 종류 결정 |
| 밝기·노이즈·회전 슬라이더 조절 | 사과 이미지 실시간 변환 |
| [🎲 랜덤 샘플] 클릭 | 무작위 사과 자동 설정 |
| [🔍 CNN 분류 실행] 클릭 | CNN이 8단계로 분석 후 결과 표시 |

👉 확률 바에서 🍎 익은 사과 / 🍏 덜 익은 사과 확률을 확인하고, 판정이 맞는지 ✅/❌로 확인하세요!

| 관찰 대상 | 찾아볼 장면 | 의미 |
|----------|------------|------|
| 📉 손실 그래프 | 곡선이 급격히 내려가는 구간 | 모델이 빠르게 배우는 중 |
| 📈 정확도 그래프 | 90% 이상에서 평평해지는 구간 | 학습 수렴 — 더 배울 게 없다는 신호 |

| 🔬 특성맵 | Conv1의 밝은 테두리선 | 필터가 사과 윤곽(에지)을 감지한 결과 |
| 📋 분류 결과 | 높은 확률 vs 낮은 확률 | CNN의 확신도 — 90% 이상이면 확실! |
| 학습률 0.1 | 손실이 울렁거리는 그래프 | 너무 빠른 학습 → 불안정 |

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| 합성곱 | Convolution | 필터로 이미지 특징 추출 | 🔍 돋보기로 부분을 살피는 것 |
| 풀링 | Max Pooling | 핵심만 남기고 크기 축소 | 📝 중요한 메모만 남기는 것 |
| 에포크 | Epoch | 전체 데이터 1회 학습 | 📚 교과서 1번 읽기 |
| 학습률 | Learning Rate | 가중치 업데이트 크기 | 🚶 걸음 크기 (크면 빠르지만 위험) |
| 손실 | Loss | 정답과 예측의 차이 | 📉 틀린 정도 (작을수록 좋음) |
| 특성맵 | Feature Map | 필터 적용 결과 이미지 | 🖼️ 필터를 씌운 사진 |
| Softmax | Softmax | 확률 정규화 (합 = 1) | 🎯 "몇 %로 확신하나요?" |
| 가중치 | Weight | 학습되는 파라미터 | ⚖️ 특징의 중요도 |
