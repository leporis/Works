# 의사결정트리 머신러닝 시뮬레이터 수업지도안

**문서 버전: v1.0.1.0**

---

## 1. 수업 개요

| 항목 | 내용 |
|------|------|
| 과목명 | 컴퓨팅사고와 AI교육 |
| 단원 | 머신러닝의 이해 — 분류 알고리즘 |
| 차시 | 의사결정트리를 활용한 데이터 분류 |
| 대상 | 초등 예비교사 (대학 2~3학년) |
| 수업 시간 | 45분 (1차시) |
| 수업 형태 | 실습 중심 + 토론 |
| 교구 | 의사결정트리 머신러닝 시뮬레이터 v1.0.1.0, 개인 PC/노트북 |

---

## 2. 학습 목표

### 2.1 성취 기준

| # | 학습 목표 | 평가 기준 |
|:-:|-----------|-----------|
| 1 | 의사결정트리의 구조(분기 노드, 리프 노드)를 설명할 수 있다 | 트리 구조 요소를 정확히 명명하고 역할을 설명 |
| 2 | 지니 불순도의 개념을 이해하고 간단한 예시를 계산할 수 있다 | 2~3종 데이터의 지니 불순도를 계산 |
| 3 | 파라미터 변경이 모델 성능에 미치는 영향을 관찰하고 설명할 수 있다 | 깊이/특성 수 변경에 따른 정확도 변화를 설명 |
| 4 | 과적합의 개념을 이해하고 트리 깊이 실험으로 확인할 수 있다 | 과적합 현상을 실험 결과로 근거하여 설명 |

### 2.2 핵심 역량

| 역량 | 관련 활동 |
|------|-----------|
| 컴퓨팅 사고력 | 알고리즘 원리 이해, 조건 기반 분류 추적 |
| 데이터 리터러시 | 데이터 분석, 혼동 행렬 해석, 성능 평가 |
| 비판적 사고 | 과적합 토론, 모델 한계 인식 |
| 협력적 문제 해결 | 팀별 최적 모델 찾기 활동 |

## 3. 수업 흐름 (45분)

### 3.1 전체 흐름도

```
도입 (8분) → 탐색 (7분) → 실습 (20분) → 정리 (10분)
```

### 3.2 상세 수업 전개

---

### 📌 도입 단계 (8분)

#### 활동 1: 일상 속 의사결정 (3분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | "여러분은 동물원에서 펭귄 3종을 어떻게 구분할까요?" 질문 |
| 학생 활동 | 자유 발표: 크기, 색상, 모양 등 특성 기반 구분 기준 제시 |
| 교구 | 없음 (자유 토론) |

**발문 예시**:
- "처음 보는 펭귄이 있다면, 어떤 특징을 먼저 확인하시겠어요?"
- "그 판단을 순서도로 그린다면 어떤 모양이 될까요?"

#### 활동 2: 의사결정트리 개념 도입 (5분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | 일상 의사결정 예시(우산 판단)를 트리로 구조화하여 설명 |
| 핵심 전달 | "AI도 이런 '질문 순서도'를 데이터로부터 스스로 만듭니다" |
| 용어 도입 | 분기 노드, 리프 노드, 조건, 분류 |

**칠판 설명 예시**:
```
    날씨 흐린가?
    /         \
  예           아니오
  ↓             ↓
우산 가져감    안 가져감
```

→ "이것이 의사결정트리입니다. AI는 이런 트리를 **데이터에서 자동으로** 만듭니다."

---

### 📌 탐색 단계 (7분)

#### 활동 3: 시뮬레이터 첫 실행 (7분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | 시뮬레이터 화면 구성 안내, 데이터셋 소개 |
| 학생 활동 | 시뮬레이터 실행, 데이터 관찰, 기본 설정으로 트리 생성 |
| 교구 | 시뮬레이터 (각자 PC) |

**안내 사항**:

| 단계 | 지시 |
|:----:|------|
| 1 | 시뮬레이터를 브라우저에서 열어주세요 |
| 2 | 오른쪽 [📊 데이터 정보] 패널에서 데이터 분포를 확인하세요 |
| 3 | [📋 데이터 보기]를 클릭하여 학습 데이터를 살펴보세요 |
| 4 | 기본 설정(깊이 4, 최소 샘플 2) 그대로 [🌳 트리 생성]을 클릭하세요 |
| 5 | 생성된 트리를 관찰하세요 |

**확인 발문**:
- "트리의 맨 위 노드에서 어떤 특성으로 나누나요?"
- "리프 노드에는 무엇이 표시되나요?"
- "학습 정확도와 테스트 정확도는 각각 얼마인가요?"

---

### 📌 실습 단계 (20분)

#### 활동 4: 깊이 실험 (10분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | "깊이를 1, 4, 8로 바꿔가며 정확도 변화를 관찰해봅시다" |
| 학생 활동 | 개별 실습, 기록지 작성 |
| 핵심 개념 | 과적합, 과소적합 |

**기록지 배포**:

| 최대 깊이 | 학습 정확도 | 테스트 정확도 | 관찰 메모 |
|:---------:|:-----------:|:------------:|-----------|
| 1 | | | |
| 2 | | | |
| 4 | | | |
| 6 | | | |
| 8 | | | |

**중간 발문** (5분 경과 후):
- "깊이를 늘릴수록 학습 정확도는 어떻게 되나요?"
- "테스트 정확도도 계속 올라가나요?"
- "학습 100%인데 테스트가 낮다면, 이 모델은 '공부는 잘했는데 시험은 못 본' 것과 같지 않나요?"

**과적합 개념 정리** (실험 후):
- "학습 데이터만 잘 맞추고, 처음 보는 데이터에서 떨어지는 것을 **과적합**이라고 합니다"
- "시험 범위를 달달 외웠지만 응용문제를 못 푸는 것과 같습니다"

#### 활동 5: 추론 과정 추적 (10분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | 추론 모드 전환 안내, 경로 추적 시범 |
| 학생 활동 | 개별 추론 실습, 정오답 확인 |
| 핵심 개념 | 추론(Inference), 경로 추적 |

**안내 사항**:

| 단계 | 지시 |
|:----:|------|
| 1 | [🔍 추론 모드] 탭을 클릭하세요 |
| 2 | 테스트 데이터 #1을 선택하세요 |
| 3 | 트리 위의 **노란색 경로**를 따라가 보세요 |
| 4 | 아래 단계별 설명을 읽으며, 각 분기점에서 왜 그 방향으로 갔는지 확인하세요 |
| 5 | [🎲 랜덤]으로 3~5개 더 추론해보세요 |
| 6 | 틀린 예측이 있나요? 왜 틀렸을까요? |

**발문**:
- "트리가 '부리 깊이 ≤ 15.6'이라고 물을 때, 이 펭귄의 부리 깊이는 얼마인가요?"
- "왼쪽으로 갔나요, 오른쪽으로 갔나요? 왜 그런가요?"
- "최종적으로 어떤 종으로 분류되었나요? 실제 종과 같나요?"

---

### 📌 정리 단계 (10분)

#### 활동 6: 핵심 개념 정리 및 토론 (10분)

| 항목 | 내용 |
|------|------|
| 교수 활동 | 핵심 개념 정리, 토론 진행, 확장 질문 |
| 학생 활동 | 토론 참여, 정리 퀴즈 응답 |

**핵심 정리 (5분)**:

| # | 핵심 개념 | 한 줄 정리 |
|:-:|-----------|-----------|
| 1 | 의사결정트리 | "질문의 순서도로 분류하는 AI" |
| 2 | 지니 불순도 | "데이터가 얼마나 섞여있는지 나타내는 숫자" |
| 3 | 과적합 | "공부만 열심히 하고 시험은 못 보는 AI" |
| 4 | 테스트 정확도 | "AI의 진짜 실력을 보여주는 숫자" |

**토론 (5분)**:

| 토론 주제 | 기대 답변 방향 |
|-----------|----------------|
| "AI의 판단을 항상 믿을 수 있을까요?" | AI도 틀릴 수 있음, 혼동 행렬에서 확인 가능 |
| "초등학생에게 의사결정트리를 어떻게 설명하시겠어요?" | 20가지 질문 게임, 동물 분류 퀴즈 등 일상 비유 |
| "이 알고리즘은 어떤 실생활 문제에 사용될 수 있을까요?" | 의료 진단, 스팸 필터, 고객 분류 등 |

## 4. 심화 수업 (2차시, 45분) — 선택

2차시가 가능한 경우의 추가 활동입니다.

### 4.1 수업 흐름

```
복습 (5분) → 특성 중요도 실험 (15분) → 최적 모델 찾기 (15분) → 발표 및 정리 (10분)
```

### 4.2 활동 7: 특성 중요도 실험 (15분)

| 단계 | 활동 |
|:----:|------|
| 1 | 특성을 하나씩만 선택하여 정확도 비교 |
| 2 | 2개 조합으로 정확도 비교 |
| 3 | "어떤 특성이 가장 중요한가?" 토론 |

### 4.3 활동 8: 팀별 최적 모델 찾기 (15분)

| 단계 | 활동 |
|:----:|------|
| 1 | 2~3명 팀 구성 |
| 2 | 깊이, 최소 샘플, 특성 자유 조합 실험 |
| 3 | 가장 높은 **테스트 정확도** 달성 경쟁 |
| 4 | 팀별 결과 발표 (설정값, 정확도, 전략) |

### 4.4 발표 및 정리 (10분)

- 팀별 최적 설정 비교
- "좋은 모델이란 무엇인가?" 종합 토론
- AI 윤리: "AI가 분류를 틀리면 어떤 문제가 생길까?"

## 5. 평가 계획

### 5.1 형성 평가 (수업 중)

| 평가 시점 | 평가 방법 | 평가 내용 |
|-----------|-----------|-----------|
| 탐색 단계 | 관찰 | 시뮬레이터 조작 능력 |
| 깊이 실험 | 기록지 확인 | 정확도 기록 정확성, 관찰 내용 |
| 추론 추적 | 발문 응답 | 경로 추적 이해도 |
| 정리 단계 | 토론 참여 | 핵심 개념 이해, 비판적 사고 |

### 5.2 총괄 평가 (수업 후)

**정리 퀴즈 (5문항)**:

| # | 문항 | 정답 |
|:-:|------|------|
| 1 | 의사결정트리에서 최종 분류 결과를 내놓는 노드를 무엇이라 하나요? | 리프 노드 |
| 2 | 지니 불순도가 0이면 데이터는 어떤 상태인가요? | 한 종만 존재 (완전 순수) |
| 3 | 학습 정확도는 높은데 테스트 정확도가 낮은 현상을 무엇이라 하나요? | 과적합 (Overfitting) |
| 4 | 혼동 행렬에서 대각선 숫자가 높을수록 좋은가요, 나쁜가요? | 좋다 (정확히 분류한 수) |
| 5 | 트리의 맨 위 노드에 사용된 특성은 분류에서 어떤 의미를 가지나요? | 가장 중요한 특성 |

## 6. 교수자 준비 사항

### 6.1 사전 준비

| # | 준비 항목 | 확인 |
|:-:|-----------|:----:|
| 1 | 시뮬레이터 파일 학생 PC에 배포 | ☐ |
| 2 | 브라우저 정상 실행 확인 (Chrome/Edge 권장) | ☐ |
| 3 | 인터넷 연결 확인 (Google Fonts 로딩) | ☐ |
| 4 | 기록지 인쇄 또는 디지털 배포 | ☐ |
| 5 | 사용설명서/활용가이드 숙지 | ☐ |

### 6.2 예상 질문과 대응

| 예상 질문 | 대응 |
|-----------|------|
| "왜 매번 같은 결과가 나와요?" | 시드 기반 셔플로 재현성 보장. 교실에서 모두 같은 결과를 보는 것이 장점 |
| "테스트 정확도가 왜 더 낮아요?" | 학습 데이터는 이미 본 문제, 테스트는 처음 보는 문제. 시험 비유 사용 |
| "AI가 100% 맞추면 되는 거 아니에요?" | 과적합 개념 연결. 교과서 달달 외우기 vs 진짜 이해의 차이 |
| "깊이를 무한히 늘리면?" | 데이터 수만큼 리프가 생겨 모든 학습 데이터를 암기. 새 데이터에서는 의미 없음 |

### 6.3 수업 시간 조절

| 상황 | 조절 방법 |
|------|-----------|
| 시간 부족 | 활동 5(추론 추적)를 축소 (5분으로) |
| 시간 여유 | 특성 중요도 실험 추가 |
| 학생 수준 높음 | 지니 불순도 직접 계산 활동 추가 |
| 학생 수준 낮음 | 깊이 실험을 깊이 1, 4 두 가지만 비교 |
