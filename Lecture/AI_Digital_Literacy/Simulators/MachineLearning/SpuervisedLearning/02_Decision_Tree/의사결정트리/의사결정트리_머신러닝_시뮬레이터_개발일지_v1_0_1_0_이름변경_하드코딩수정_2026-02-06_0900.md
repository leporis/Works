# 의사결정트리 머신러닝 시뮬레이터 개발일지

**문서 버전: v1.0.1.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
의사결정트리 머신러닝 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 머신러닝의 핵심 알고리즘인 **의사결정트리(Decision Tree)**의 분류 원리를 직관적으로 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표

| 목표 | 설명 |
|------|------|
| 의사결정트리 이해 | 특성 기반 분기 규칙으로 데이터를 분류하는 원리 학습 |
| 지니 불순도 체험 | 데이터를 가장 잘 나누는 기준 선택 과정 이해 |
| 추론 과정 추적 | 트리의 규칙을 따라가며 분류 과정을 단계별로 체험 |
| 모델 성능 평가 | 학습/테스트 정확도, 혼동 행렬 해석 능력 배양 |

### 1.4 개발 배경
- 사과가격예측 선형회귀 시뮬레이터(v1.2.4.1)와 동일한 교육적 설계 철학 적용
- Palmer Penguins 데이터셋을 활용한 3종 분류 문제
- 의사결정트리의 구조를 시각적으로 정확하게 표현하는 데 주안점

## 2. 시스템 설계

### 2.1 데이터셋: Palmer Penguins

| 구분 | 변수명 | 단위 | 범위 | 설명 |
|:----:|--------|------|------|------|
| 입력 | bill_length_mm | mm | 33.5~59.6 | 부리 길이 |
| 입력 | bill_depth_mm | mm | 13.1~21.5 | 부리 깊이 |
| 입력 | flipper_length_mm | mm | 172~230 | 날개(플리퍼) 길이 |
| 입력 | body_mass_g | g | 2700~6300 | 체중 |
| 출력 | species | - | 3종 | Adelie / Chinstrap / Gentoo |

### 2.2 데이터 분할

| 항목 | 값 | 설명 |
|------|-----|------|
| 전체 데이터 | 134개 | Adelie 50, Chinstrap 34, Gentoo 50 |
| 학습 데이터 | 80% (107개) | 트리 생성용 |
| 테스트 데이터 | 20% (27개) | 추론 정확도 평가용 |
| 분할 방식 | 시드 기반 셔플 | seed=42, 재현 가능 |

### 2.3 의사결정트리 알고리즘: CART

```
알고리즘: CART (Classification and Regression Trees)
분할 기준: 지니 불순도 (Gini Impurity)

Gini = 1 - Σ(pᵢ²)
  pᵢ = 클래스 i의 비율

가중 지니 = (|left|/|total|) × Gini(left) + (|right|/|total|) × Gini(right)
```

### 2.4 트리 파라미터

| 파라미터 | 기본값 | 조절 범위 | 설명 |
|----------|--------|-----------|------|
| 최대 깊이 | 4 | 1~8 | 트리의 최대 깊이 제한 |
| 최소 샘플 수 | 2 | 1~10 | 노드 분할에 필요한 최소 데이터 수 |
| 지니 임계값 | 0.01 | 고정 | 이 이하면 리프 노드로 확정 |
| 특성 선택 | 4개 전체 | 사용자 선택 | 체크박스로 개별 특성 선택 가능 |

### 5.2 핵심 알고리즘

#### 지니 불순도 계산
```javascript
function giniImpurity(data) {
    const counts = {};
    data.forEach(d => counts[d.species] = (counts[d.species] || 0) + 1);
    let gini = 1;
    Object.values(counts).forEach(c => {
        const p = c / data.length;
        gini -= p * p;
    });
    return gini;
}
```
