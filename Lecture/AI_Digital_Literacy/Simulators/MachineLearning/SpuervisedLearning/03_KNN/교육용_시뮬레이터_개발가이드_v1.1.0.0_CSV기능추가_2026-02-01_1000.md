# Part 1: 개발 원칙

---

## 1. 개요

### 1.1 목적

이 문서는 **교육용 시뮬레이터**를 개발하고 문서화하기 위한 종합 가이드입니다. 일관된 코드 품질, 사용자 경험, 문서 체계를 유지하여 교육 효과를 극대화합니다.

### 1.2 적용 대상

| 분야 | 시뮬레이터 예시 |
|------|----------------|
| **머신러닝** | 선형 회귀, KNN, K-means, 의사결정트리 |
| **물리** | 포물선 운동, 진자, 파동, 회로 |
| **경제** | 수요-공급, 게임 이론, 시장 균형 |
| **생태** | 포식자-피식자, 개체군 동태, 전염병 확산 |
| **화학** | 반응 속도, 평형, 적정 |
| **수학** | 함수 그래프, 확률 분포, 기하 변환 |

### 1.3 핵심 설계 원칙

| 원칙 | 설명 | 적용 예 |
|------|------|---------|
| **일관성** | 모든 시뮬레이터에서 동일한 레이아웃 | 3열 그리드, 모드 탭 |
| **직관성** | 복잡한 개념을 시각적으로 표현 | 애니메이션, 실시간 피드백 |
| **단계적 학습** | 설정 → 실행 분리 | 모드 탭으로 구분 |
| **즉각적 피드백** | 파라미터 변경 시 실시간 반영 | 슬라이더 → 차트 연동 |
| **소프트코딩** | 설정값의 중앙 집중 관리 | CSS 변수, CONFIG 객체 |
| **코드 품질** | 유지보수 가능한 깔끔한 코드 | DRY, 네이밍 컨벤션 |
| **문서 정합성** | 코드-문서-이미지 버전 일치 | 동기화된 버전 관리 |

### 1.4 개발 산출물

시뮬레이터 개발 시 다음 산출물을 함께 작성합니다:

```
시뮬레이터 프로젝트
├── [프로젝트명]_시뮬레이터_v버전.html    ← 실행 파일
├── [프로젝트명]_개발일지_v버전.md        ← 기술 문서 (상세)
├── [프로젝트명]_퀵가이드_v버전.md        ← 사용 안내 (요약)
└── (선택) 이미지/                        ← 스크린샷, 다이어그램
```

## 3. 코드 품질 원칙

### 3.1 네이밍 컨벤션

| 대상 | 규칙 | ❌ 나쁜 예 | ✅ 좋은 예 |
|------|------|----------|----------|
| 변수 | camelCase, 의미 명확 | `d`, `x1` | `dataCount`, `currentStep` |
| 상수 | UPPER_SNAKE_CASE | `maxvalue` | `MAX_STEPS` |
| 함수 | camelCase, 동사 시작 | `data()` | `generateData()` |
| 불리언 | is/has/can 접두사 | `running` | `isRunning` |
| CSS 클래스 | kebab-case | `btnPrimary` | `btn-primary` |

### 3.2 코드 구조화 원칙

#### DRY (Don't Repeat Yourself)

```javascript
// ❌ 중복 코드
function drawChart1() {
    ctx.clearRect(0, 0, width, height);
    ctx.strokeStyle = '#BCAAA4';
    drawAxes();
}

function drawChart2() {
    ctx.clearRect(0, 0, width, height);
    ctx.strokeStyle = '#BCAAA4';
    drawAxes();
}

// ✅ 공통 함수 추출
function initCanvas(ctx, width, height) {
    ctx.clearRect(0, 0, width, height);
    ctx.strokeStyle = '#BCAAA4';
    drawAxes();
}
```

#### 단일 책임 원칙

```javascript
// ❌ 너무 많은 일
function runSimulation() {
    // 150줄...
}

// ✅ 책임 분리
function runSimulation() {
    if (!validateInput()) return;
    initializeState();
    executeSimulationLoop();
    updateUI();
}
```

| 유형 | 예시 | 조치 |
|------|------|------|
| 사용되지 않는 변수 | `let oldData = [];` | 삭제 |
| 호출되지 않는 함수 | `function legacyCalc() {}` | 삭제 |

### 3.4 에러 처리

```javascript
function loadFile(file) {
    if (!file) {
        showError(UI_TEXT.ERROR_NO_FILE);
        return;
    }
    
    if (!isValidFileType(file.name)) {
        showError(UI_TEXT.ERROR_FILE_FORMAT);
        return;
    }
    
    const reader = new FileReader();
    reader.onload = (e) => {
        try {
            const data = parseData(e.target.result);
            processData(data);
        } catch (err) {
            showError(UI_TEXT.ERROR_PARSE_FAILED);
        }
    };
    reader.onerror = () => showError(UI_TEXT.ERROR_READ_FAILED);
    reader.readAsText(file);
}
```

```javascript
// 전역 변수
let rawCsvData = [];
let csvColumns = [];

// 파일 업로드 처리
function handleCSVUpload(event) {
    const file = event.target.files[0];
    if (!file) return;
    
    const reader = new FileReader();
    reader.onload = (e) => {
        try {
            parseCSV(e.target.result, file.name);
        } catch (err) {
            showError('CSV 파일 파싱에 실패했습니다.');
        }
    };
    reader.readAsText(file);
}

// CSV 파싱
function parseCSV(csvText, fileName) {
    const lines = csvText.trim().split('\n');
    if (lines.length < 2) {
        showError('CSV 파일에 데이터가 부족합니다.');
        return;
    }
    
    // 헤더 파싱
    const headers = lines[0].split(',').map(h => h.trim().replace(/"/g, ''));
    
    // 데이터 파싱
    rawCsvData = [];
    for (let i = 1; i < lines.length; i++) {
        const values = lines[i].split(',').map(v => v.trim().replace(/"/g, ''));
        if (values.length === headers.length) {
            const row = {};
            headers.forEach((h, idx) => { row[h] = values[idx]; });
            rawCsvData.push(row);
        }
    }
    
    // 수치형 컬럼 찾기
    csvColumns = headers.filter(col => {
        const values = rawCsvData.map(r => parseFloat(r[col]));
        return values.every(v => !isNaN(v));
    });
    
    if (csvColumns.length < 2) {
        showError('숫자형 컬럼이 부족합니다. (최소 2개 필요)');
        return;
    }
    
    // UI 업데이트
    updateCSVStatus(fileName, rawCsvData.length, headers.length);
    populateCSVSelects(headers);
}

// 드래그 앤 드롭 초기화
function initFileUpload() {
    const uploadArea = document.getElementById('fileUploadArea');
    if (!uploadArea) return;
    
    uploadArea.addEventListener('click', () => {
        document.getElementById('csvFileInput').click();
    });
    
    uploadArea.addEventListener('dragover', (e) => {
        e.preventDefault();
        uploadArea.classList.add('dragover');
    });
    
    uploadArea.addEventListener('dragleave', () => {
        uploadArea.classList.remove('dragover');
    });
    
    uploadArea.addEventListener('drop', (e) => {
        e.preventDefault();
        uploadArea.classList.remove('dragover');
        const file = e.dataTransfer.files[0];
        if (file && file.name.endsWith('.csv')) {
            handleCSVUpload({ target: { files: [file] } });
        }
    });
}
```

```javascript
function playHistory() {
    isPlayingBack = true;
    const speed = parseInt(document.getElementById('playbackSpeed').value);
    const delay = CONFIG.DELAY_MAX - speed * (CONFIG.DELAY_MAX - CONFIG.DELAY_MIN) / CONFIG.SPEED_MAX;
    
    let frameIndex = 0;
    
    function playFrame() {
        if (!isPlayingBack || frameIndex >= history.length) {
            stopPlayback();
            return;
        }
        restoreState(history[frameIndex]);
        updateVisualization();
        frameIndex++;
        playbackTimer = setTimeout(playFrame, delay);
    }
    playFrame();
}

function stopPlayback() {
    isPlayingBack = false;
    if (playbackTimer) {
        clearTimeout(playbackTimer);
        playbackTimer = null;
    }
    restoreState(history[history.length - 1]);
}
```
