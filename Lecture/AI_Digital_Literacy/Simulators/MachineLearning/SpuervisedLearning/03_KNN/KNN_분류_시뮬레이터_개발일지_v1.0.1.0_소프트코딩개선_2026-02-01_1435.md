# KNN 분류 시뮬레이터 개발일지

**문서 버전: v1.0.1.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
KNN 분류 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 머신러닝의 핵심 분류 알고리즘인 **K-최근접 이웃(K-Nearest Neighbors)**의 원리를 직관적으로 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표

| 목표 | 설명 |
|------|------|
| KNN 원리 이해 | 가장 가까운 K개 이웃을 찾아 다수결로 분류하는 원리 |
| 거리 개념 학습 | 유클리드/맨해튼 거리의 차이와 의미 |
| K값의 영향 파악 | K값 변화에 따른 분류 경계 변화 관찰 |
| 결정 경계 이해 | 분류 영역이 어떻게 형성되는지 시각화 |

### 1.4 개발 배경
- 사과가격예측 선형회귀 시뮬레이터 v1.2.4.1을 기반으로 설계
- ML시뮬레이터 디자인설계가이드 v1.0.0.0 적용
- 교육용 시뮬레이터 개발가이드 v1.0.0.0 준수

## 2. 시스템 설계

### 2.1 입출력 변수

| 구분 | 변수명 | 범위 | 설명 |
|:----:|--------|------|------|
| 입력 | X₁ (특성 1) | 0 ~ 100 | 첫 번째 특성값 |
| 입력 | X₂ (특성 2) | 0 ~ 100 | 두 번째 특성값 |
| 출력 | 클래스 | 2 ~ 4개 | 분류 결과 |

### 2.2 KNN 알고리즘

```
1. 새로운 데이터 점 입력
2. 모든 학습 데이터와의 거리 계산
3. 거리가 가장 가까운 K개 이웃 선택
4. K개 이웃의 클래스를 다수결 투표
5. 가장 많은 표를 받은 클래스로 분류
```

### 2.3 거리 측정 방식

| 방식 | 수식 | 특징 |
|------|------|------|
| 유클리드 | √((x₁-x₂)² + (y₁-y₂)²) | 직선 거리, 가장 일반적 |
| 맨해튼 | \|x₁-x₂\| + \|y₁-y₂\| | 격자 거리, 축 방향만 이동 |

### 2.4 시뮬레이션 기본 설정

| 항목 | 기본값 | 조절 범위 |
|------|--------|-----------|
| 데이터 개수 | 60개 | 20 ~ 150개 |
| 클래스 수 | 3개 | 2 ~ 4개 |
| K값 | 5 | 1 ~ 15 |
| 데이터 분산도 | 보통 | 밀집 ~ 분산 |

### 5.2 핵심 알고리즘

#### 거리 계산
```javascript
function calculateDistance(p1, p2, method = 'euclidean') {
    if (method === 'euclidean') {
        return Math.sqrt((p1.x1 - p2.x1) ** 2 + (p1.x2 - p2.x2) ** 2);
    } else {
        return Math.abs(p1.x1 - p2.x1) + Math.abs(p1.x2 - p2.x2);
    }
}
```

#### KNN 분류
```javascript
function knnClassify(point, k, method) {
    // 1. 모든 데이터와의 거리 계산
    const distances = data.map((d, idx) => ({
        idx,
        distance: calculateDistance(point, d, method),
        class: d.class
    }));

    // 2. 거리순 정렬
    distances.sort((a, b) => a.distance - b.distance);
    
    // 3. K개 이웃 선택
    const neighbors = distances.slice(0, k);

    // 4. 다수결 투표
    const votes = {};
    neighbors.forEach(n => {
        votes[n.class] = (votes[n.class] || 0) + 1;
    });

    // 5. 최다 투표 클래스 반환
    let maxVotes = 0;
    let predictedClass = 0;
    Object.entries(votes).forEach(([cls, count]) => {
        if (count > maxVotes) {
            maxVotes = count;
            predictedClass = parseInt(cls);
        }
    });

    return { predictedClass, neighbors, votes, confidence };
}
```
