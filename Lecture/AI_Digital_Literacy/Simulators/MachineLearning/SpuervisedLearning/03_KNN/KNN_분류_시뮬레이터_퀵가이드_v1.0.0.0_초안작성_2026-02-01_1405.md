## 📌 KNN이란?

**K-Nearest Neighbors (K-최근접 이웃)**

```
새로운 데이터가 들어오면...
1️⃣ 모든 기존 데이터와의 거리 계산
2️⃣ 가장 가까운 K개 이웃 선택
3️⃣ K개 이웃이 다수결 투표
4️⃣ 가장 많은 표를 받은 클래스로 분류!
```

| K=3 예시 | 이웃 | 클래스 |
|----------|------|--------|
| 가장 가까운 1번째 | 🔴 | A |
| 가장 가까운 2번째 | 🔴 | A |
| 가장 가까운 3번째 | 🟢 | B |
| **투표 결과** | **A: 2표, B: 1표** | **→ 클래스 A** |

## ⚙️ 주요 파라미터

| 파라미터 | 권장값 | 설명 |
|----------|--------|------|
| **K값** | 3~7 (홀수) | 이웃 수. 홀수 권장 (동점 방지) |
| 데이터 개수 | 60~100개 | 너무 적으면 과적합 |
| 클래스 수 | 2~3개 | 시작은 2개로, 익숙해지면 증가 |
| 분산도 | 보통 | 밀집: 쉬움, 분산: 어려움 |

### 💡 K값 선택 팁

| K값 | 특성 |
|-----|------|
| **K=1** | 가장 가까운 1개만 → 노이즈에 민감 |
| **K=3~7** | 균형 잡힌 선택 → 권장! |
| **K=15** | 많은 이웃 참조 → 경계가 부드러움 |

## 📏 거리 측정 방식

### 유클리드 거리 (기본)
```
√((x₁-x₂)² + (y₁-y₂)²)
```
- 직선 거리 (새가 날아가는 거리)
- 가장 일반적으로 사용

### 맨해튼 거리
```
|x₁-x₂| + |y₁-y₂|
```
- 격자 거리 (택시가 이동하는 거리)
- 축 방향으로만 이동

## 📊 결과 해석

### 정확도
| 정확도 | 상태 | 의미 |
|--------|------|------|
| 90% 이상 | ✅ 우수 | 매우 잘 분류됨 |
| 70~90% | ⚠️ 양호 | 대부분 분류 성공 |
| 70% 미만 | ❌ 개선필요 | K값 또는 데이터 조정 |

### 혼동행렬 읽는 법
```
          예측
        A    B    C
실   A [ 18   2    0 ]  ← A를 A로 18개, B로 2개 예측
제   B [  1  17    2 ]
     C [  0   3   17 ]
              ↑
         대각선이 클수록 좋음!
```

## 🗺️ 결정 경계란?

- **결정 경계**: 서로 다른 클래스 영역의 경계선
- K값이 작으면: 경계가 복잡하고 울퉁불퉁
- K값이 크면: 경계가 부드럽고 단순

```
K=1                    K=15
┌──────────────┐       ┌──────────────┐
│🔴🔴🟢🟢🟢🔴🔴│       │🔴🔴🔴🟢🟢🟢🟢│
│🔴🔴🔴🟢🔴🔴🔴│       │🔴🔴🔴🟢🟢🟢🟢│
│🔴🔴🔴🔴🔴🔴🔴│       │🔴🔴🔴🔴🔴🔴🔴│
└──────────────┘       └──────────────┘
   복잡한 경계             부드러운 경계
```

## 🔮 추론 모드 상세

### 1단계: 입력값 설정
```
📏 특성 1 (X₁): 슬라이더로 0~100 조절
📐 특성 2 (X₂): 슬라이더로 0~100 조절
```

### 2단계: K개 이웃 확인
```
🔗 K개의 최근접 이웃
#1 (거리: 5.2)  🔴 사과
#2 (거리: 7.1)  🔴 사과
#3 (거리: 8.9)  🟢 배
```

### 3단계: 투표 결과
```
🗳️ 다수결 투표 결과
사과 [████████] 2표
배   [████    ] 1표
귤   [        ] 0표
```

### 4단계: 최종 분류
```
🏷️ 예측 클래스: 🔴 사과
   신뢰도: 66.7%
```

## 🎯 실습 활동 예시

### 활동 1: K값 실험
1. K=1로 분류 실행 → 정확도 확인
2. K=5로 분류 실행 → 정확도 비교
3. K=15로 분류 실행 → 결정 경계 관찰
4. **질문**: 어떤 K값이 가장 좋았나요?

### 활동 2: 데이터 분산도
1. 분산도 "밀집"으로 데이터 생성 → 분류 실행
2. 분산도 "분산"으로 데이터 생성 → 분류 실행
3. **질문**: 정확도는 어떻게 변했나요?

### 활동 3: 추론 체험
1. [🔮 추론 모드]로 전환
2. 슬라이더를 경계 근처로 이동
3. **관찰**: 경계 근처에서 신뢰도는 어떤가요?

## ❓ 자주 묻는 질문

**Q: K값을 짝수로 하면 안 되나요?**
→ 동점이 발생할 수 있어요. 홀수를 권장합니다!

**Q: 결정 경계가 안 보여요**
→ "결정 경계 표시" 체크박스를 확인하세요.

**Q: 정확도가 너무 낮아요**
→ K값을 조절하거나 분산도를 "밀집"으로 설정해보세요.

**Q: 클래스가 겹치는 영역은 어떻게 되나요?**
→ 그 영역의 데이터는 잘못 분류될 확률이 높아요. 이것이 KNN의 한계입니다.

| **K값** | 참조할 이웃의 수 |
