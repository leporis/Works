# 🎯 KNN 분류 시뮬레이터 개발일지

**문서 버전: v2.1.0.0**

## 1. 프로젝트 개요

### 1.1 목적

K-최근접 이웃(K-Nearest Neighbors) 분류 알고리즘의 원리를 **시각적으로 체험**할 수 있는 교육용 웹 시뮬레이터입니다. 데이터를 훈련/테스트로 분할하고, K값 변화에 따른 결정 경계와 분류 정확도 변화를 관찰하여 **과적합과 K값 선택**의 개념을 직관적으로 이해할 수 있습니다.

### 1.2 교육 목표

| 수준 | 학습 목표 |
|------|----------|
| **이해** | KNN 알고리즘의 "다수결 투표" 원리 파악 |
| **체험** | K값·거리 측정 방식·분할 비율 조작 → 결과 변화 관찰 |
| **확인** | Train/Test 분할로 과적합 탐지, 최적 K값 발견 |

### 1.3 기술 스택

| 항목 | 기술 |
|------|------|
| 프레임워크 | Vanilla HTML/CSS/JavaScript (단일 파일) |
| 차트 라이브러리 | Chart.js 4.4.1 (CDN) |
| 폰트 | Noto Sans KR, JetBrains Mono (Google Fonts) |
| Canvas API | 산점도, 결정 경계, 추론 차트 직접 렌더링 |

## 2. 시스템 설계

### 2.1 입출력 설계

```
[입력]                          [처리]                      [출력]
──────                          ──────                      ──────
데이터셋 선택 (9종)             Fisher-Yates 셔플           산점도 (Train/Test 구분)
 ├─ 합성 4종                    Train/Test 분할             결정 경계 시각화
 └─ 실 데이터 5종               KNN 분류 (유클리드/맨해튼)  K값별 테스트 정확도 테이블
K값 (1~15, 홀수)                K=1~15 전수 평가            K값별 정확도 차트
거리 측정 (유클리드/맨해튼)                                  혼동행렬
훈련:테스트 비율 (60~80%)                                   추론 모드 (클릭 분류)
CSV 파일 업로드
```

### 2.2 알고리즘

**KNN 분류 과정:**

```
1. 테스트 점 p에 대해
2. 훈련 데이터 전체와의 거리 계산
   - 유클리드: √((x₁-x₁')² + (x₂-x₂')²)
   - 맨해튼: |x₁-x₁'| + |x₂-x₂'|
3. 거리 오름차순 정렬 → 상위 K개 이웃 선택
4. 다수결 투표 → 예측 클래스 결정
5. 신뢰도 = (최다 득표 / K) × 100%
```

**Train/Test 분할:**

```
1. Fisher-Yates 셔플로 데이터 무작위 섞기
2. 비율(60/70/80%)에 따라 훈련·테스트 분리
3. 훈련 데이터만으로 결정 경계 생성
4. 테스트 데이터로 정확도 평가 (과적합 방지)
```
