## 2. 교육 목표와 역량

### 2.1 핵심 학습 목표

| # | 학습 목표 | 시뮬레이터 기능 |
|:-:|----------|----------------|
| 1 | KNN 알고리즘의 "다수결 투표" 원리를 설명할 수 있다 | 추론 모드 (클릭 → 이웃 연결선 관찰) |
| 2 | K값이 분류 결과에 미치는 영향을 비교할 수 있다 | K값 재생 (경계 + 정확도 변화) |
| 3 | 과적합과 일반화의 개념을 구분할 수 있다 | Train/Test 분할 + K=1 vs K=15 비교 |
| 4 | 적절한 K값을 실험적으로 탐색할 수 있다 | K값별 정확도 테이블 + 🏆 최적 K |
| 5 | 혼동행렬을 읽고 분류 성능을 해석할 수 있다 | 혼동행렬 패널 |

### 2.2 연계 가능한 교육 역량

| 역량 | KNN 시뮬레이터에서의 발현 |
|------|-------------------------|
| **데이터 리터러시** | 데이터 수집(CSV 업로드), 분할 전략, 결과 해석 |
| **계산적 사고** | 알고리즘 이해, 파라미터 조작, 패턴 인식 |
| **비판적 사고** | "왜 K=1이 항상 나쁘지 않은가?", "정확도가 높으면 항상 좋은가?" |
| **실험 설계** | 독립 변수(K값) 조작 → 종속 변수(정확도) 관찰 → 결론 도출 |

## 4. 데이터셋별 활용 전략

### 4.1 수업 단계별 추천 데이터셋

| 수업 단계 | 추천 데이터셋 | 이유 |
|----------|-------------|------|
| **첫 시연** | 🍎 과일 분류 | 직관적 특성명(당도, 무게), 깔끔한 클러스터 |
| **K값 실험** | 🌸 꽃 분류 | K=3~5에서 안정적, 분산도 변화 효과 명확 |
| **과적합 체험** | 🐧 동물 분류 | 분산도 높이면 클래스 겹침 → 과적합 쉽게 관찰 |
| **무작위 실험** | 🎲 랜덤 데이터 | 매번 다른 패턴 → 반복 실험의 변동성 체험 |
| **실 데이터 입문** | 🌺 붓꽃 (Iris) | 가장 유명한 ML 데이터셋, 교과서와 직결 |
| **변수 선택 실험** | 🐧 팔머 펭귄 | 4변수 중 어떤 조합이 가장 좋은지 탐색 |
| **2클래스 분류** | 🏥 유방암 진단 | 2클래스(양성/악성)로 단순한 분류 체험. 의료 AI 토론 소재 |

### 4.2 데이터셋 조합 시나리오

**"쉬운 것 → 어려운 것" 순서가 핵심입니다.**

```
[1단계] 합성 데이터로 개념 잡기
        🍎 과일 분류 (분산도 2~3, 클래스 2~3)
        → 깔끔한 경계, 높은 정확도 → 성공 경험
                ↓
[2단계] 난이도 높이기
        🐧 동물 분류 (분산도 4~5, 클래스 4)
        → 경계 겹침, 정확도 하락 → "왜?" 질문 유발
                ↓
[3단계] 실 데이터 도전
        🌺 붓꽃 → 변수 선택에 따라 정확도 차이 큼
        → 실전 데이터의 복잡성 체험
                ↓
[4단계] 자유 탐구
        CSV 업로드 또는 다른 실 데이터셋
        → 자기 주도 실험
```

### 4.3 분산도를 활용한 난이도 조절

합성 데이터의 분산도(1~5)는 수업 난이도를 조절하는 핵심 도구입니다.

| 분산도 | 난이도 | 수업 활용 |
|:------:|:------:|----------|
| 1~2 | 쉬움 | 개념 도입, 첫 성공 경험. 정확도 90%+ 기대 |
| 3 | 보통 | 표준 실습. K값 차이가 명확하게 드러남 |
| 4~5 | 어려움 | 과적합 체험, 한계 토론. 정확도 60~70% 정상 |

> 💡 **교수 팁**: 처음에는 분산도 2로 "성공 경험"을 준 뒤, 분산도 4로 올려서 "왜 정확도가 떨어졌을까?" 토론을 이끌어내세요.

## 5. 실습 활동지

### 5.1 활동지 1: K값의 영향 탐색

**학습 목표**: K값이 결정 경계와 분류 정확도에 미치는 영향을 관찰한다.

**준비**: 🍎 과일 분류, 데이터 60개, 클래스 3, 분산도 3

**단계:**

1. [ ] K=1로 설정 → **[⚡ 분류 실행]** → 결정 경계와 정확도 기록
2. [ ] K=5로 변경 → **[⚡ 분류 실행]** → 결정 경계와 정확도 기록
3. [ ] K=15로 변경 → **[⚡ 분류 실행]** → 결정 경계와 정확도 기록
4. [ ] 세 결과를 비교하여 관찰 기록 작성

**기록표:**

| K값 | 테스트 정확도 | 결정 경계 모양 | 관찰 (경계가 어떻게 달라졌나?) |
|:---:|:----------:|:-------------:|--------------------------|
| 1 | ___% | □ 울퉁불퉁 □ 부드러움 | |
| 5 | ___% | □ 울퉁불퉁 □ 부드러움 | |
| 15 | ___% | □ 울퉁불퉁 □ 부드러움 | |

**질문:**
- K=1일 때 경계가 복잡한 이유는 무엇일까요?
- K=15일 때 정확도가 낮아졌다면, 왜 그런 걸까요?
- 세 결과 중 가장 좋은 K는 몇이고, 그 이유는?

---

### 5.2 활동지 2: 최적 K 탐색

**학습 목표**: K값 재생 기능을 사용하여 최적 K를 실험적으로 찾는다.

**준비**: 활동지 1과 동일한 데이터 유지

**단계:**

1. [ ] **[▶️ K값 재생]** 클릭 → K=1부터 15까지 자동 재생 관찰
2. [ ] 재생 중 K별 정확도 스트립에서 **녹색(🏆)** 칩 확인
3. [ ] 재생 완료 후 결과 테이블에서 전체 K의 정확도 기록
4. [ ] 혼동행렬 분석 — 어떤 클래스가 가장 혼동되는지 확인

**기록표:**

| K | 1 | 3 | 5 | 7 | 9 | 11 | 13 | 15 |
|:-:|:--:|:--:|:--:|:--:|:--:|:--:|:--:|:--:|
| 정확도(%) | | | | | | | | |

- 🏆 최적 K = ___, 정확도 = ___% 
- 가장 혼동되는 두 클래스: ___ ↔ ___

**질문:**
- 최적 K가 1이나 15의 극단값이 아닌 이유는?
- 이 결과를 다시 실행하면 최적 K가 달라질까요? 왜 달라질 수 있을까요?

---

### 5.3 활동지 3: Train/Test 비율 실험

**학습 목표**: 데이터 분할 비율이 모델 평가에 미치는 영향을 이해한다.

**준비**: 🌸 꽃 분류, 데이터 80개, 클래스 3, 분산도 3, K=5 고정

**단계:**

1. [ ] 비율 **60:40**으로 설정 → 분류 실행 → 정확도와 Train/Test 개수 기록
2. [ ] 비율 **70:30**으로 변경 → 분류 실행 → 기록
3. [ ] 비율 **80:20**으로 변경 → 분류 실행 → 기록

**기록표:**

| 비율 | 훈련 개수 | 테스트 개수 | K=5 정확도 | 혼동행렬의 오분류 합계 |
|:----:|:---------:|:----------:|:----------:|:------------------:|
| 60:40 | ___ | ___ | ___% | ___ |
| 70:30 | ___ | ___ | ___% | ___ |
| 80:20 | ___ | ___ | ___% | ___ |

**질문:**
- 훈련 데이터가 많을수록 항상 좋은 결과가 나올까요?
- 테스트 데이터가 적을 때 정확도의 신뢰도는 어떨까요?
- 실무에서는 보통 어떤 비율을 사용할까요? (힌트: 70:30 또는 80:20)

---

### 5.4 활동지 4: 거리 측정 비교

**학습 목표**: 유클리드 거리와 맨해튼 거리의 차이가 분류에 미치는 영향을 비교한다.

**준비**: 🌾 밀 씨앗, K=5, 비율 70:30

**단계:**

1. [ ] 거리 방식 **유클리드** → 분류 실행 → 정확도 및 혼동행렬 기록
2. [ ] 거리 방식 **맨해튼**으로 변경 → 분류 실행 → 기록
3. [ ] 두 결과 비교

**기록표:**

| 거리 방식 | 정확도 | 가장 잘 분류된 클래스 | 가장 많이 혼동된 클래스 쌍 |
|:---------:|:------:|:-------------------:|:----------------------:|
| 유클리드 | ___% | ___ | ___ ↔ ___ |
| 맨해튼 | ___% | ___ | ___ ↔ ___ |

**질문:**
- 두 거리 방식의 정확도 차이는 얼마인가요?
- 어떤 데이터에서 맨해튼 거리가 더 유리할까요? (힌트: 격자형 구조)

---

### 5.5 활동지 5: 실 데이터 자유 분석 (심화)

**학습 목표**: 실 데이터셋에서 최적의 분류 설정을 자기 주도적으로 탐색한다.

**준비**: 🌺 붓꽃 (Iris), 🐧 팔머 펭귄, 🍷 와인 품종 중 1개 선택

**보고서 양식:**

```
■ 선택한 데이터셋: _______________
■ 선택한 변수 조합: X₁ = ___________, X₂ = ___________
■ 선택 이유: _______________________________________________

■ 실험 결과 요약:
  - 최적 K: ___
  - 최적 K의 정확도: ___%
  - 사용한 거리 방식: ___________
  - 분할 비율: ___:___

■ 혼동행렬 분석:
  - 가장 정확한 클래스: ___________
  - 가장 혼동되는 클래스 쌍: ___________ ↔ ___________
  - 혼동의 원인 추측: _______________________________________________

■ 다른 변수 조합으로 바꾸면 결과가 달라지나요?
  - 시도한 다른 조합: X₁ = ___________, X₂ = ___________
  - 정확도 변화: ___% → ___%
  - 결론: _______________________________________________
```
