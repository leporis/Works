# KNN 분류 시뮬레이터 개발일지

**문서 버전: v1.1.0.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
KNN 분류 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 **K-최근접 이웃(K-Nearest Neighbors)** 분류 알고리즘의 원리를 직관적으로 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표
| 목표 | 설명 |
|------|------|
| KNN 이해 | 가장 가까운 K개 이웃으로 분류하는 원리 학습 |
| 거리 개념 | 유클리드 거리 계산 및 시각화 |
| K값 영향 | K값 변화에 따른 분류 결과 차이 관찰 |
| 결정 경계 | 다수결로 결정되는 분류 영역 이해 |

### 1.4 기반 버전
사과 가격 예측 선형 회귀 시뮬레이터 v1.2.4.1 디자인 기반

## 2. v1.1.0.0 핵심 기능: CSV 파일 업로드 및 내장 데이터셋

### 2.1 데이터 입력 탭 시스템

| 탭 | 설명 |
|----|------|
| **프리셋** | 과일/꽃/동물/랜덤 데이터 자동 생성 |
| **CSV** | 사용자 CSV 파일 업로드 및 변수 선택 |
| **내장** | 아이리스/펭귄/와인/씨앗 데이터셋 |

### 2.2 CSV 파일 업로드

#### 지원 기능
- 드래그 앤 드롭 업로드
- 클릭하여 파일 선택
- 자동 컬럼 분석 (수치형/범주형 분리)
- 변수 선택 UI (X₁, X₂, 클래스)

#### 변수 선택 드롭다운
```
📊 변수 선택
┌─────────────────────────┐
│ X₁: [sepal_length ▼]    │
│ X₂: [sepal_width  ▼]    │
│ 클래스: [species   ▼]    │
│ [📥 데이터 적용]         │
└─────────────────────────┘
```

### 2.3 내장 데이터셋

| 데이터셋 | 샘플 수 | 클래스 수 | 특성 |
|----------|---------|-----------|------|
| 🌺 아이리스 | 150 | 3 (Setosa, Versicolor, Virginica) | 꽃받침/꽃잎 길이·너비 |
| 🐧 펭귄 | 100 | 3 (Adelie, Chinstrap, Gentoo) | 부리 길이·깊이, 지느러미, 체중 |
| 🍷 와인 | 60 | 3 (품종 1, 2, 3) | 알코올, 말산, 회분, 플라보노이드, 색상 |
| 🌾 씨앗 | 63 | 3 (Kama, Rosa, Canadian) | 면적, 둘레, 압축률, 길이, 너비 |

### 2.4 데이터 정규화

모든 입력 데이터는 0~100 범위로 정규화하여 시각화:

```javascript
정규화값 = ((원본값 - 최소값) / (최대값 - 최소값)) × 100
```

## 3. 시스템 설계

### 3.1 입출력 변수

| 구분 | 변수명 | 범위 | 설명 |
|:----:|--------|------|------|
| 입력 | X₁ | 0 ~ 100 | 첫 번째 특성 (정규화) |
| 입력 | X₂ | 0 ~ 100 | 두 번째 특성 (정규화) |
| 출력 | 클래스 | 0 ~ N | 분류 결과 |

### 3.2 거리 계산

```javascript
거리 = √((x1₁ - x1₂)² + (x2₁ - x2₂)²)
```

### 3.3 분류 알고리즘

```javascript
1. 새 점에서 모든 학습 데이터까지 거리 계산
2. 거리순 정렬
3. 가장 가까운 K개 이웃 선택
4. K개 이웃의 클래스 중 다수결로 분류
```

## 4. 주요 기능
