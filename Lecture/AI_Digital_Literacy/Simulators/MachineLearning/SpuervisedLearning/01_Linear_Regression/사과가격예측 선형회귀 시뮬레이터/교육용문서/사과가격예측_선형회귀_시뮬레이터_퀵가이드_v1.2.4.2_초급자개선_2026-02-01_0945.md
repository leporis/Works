## 📐 품질 점수 계산

당도와 무게를 **하나의 품질 점수**로 통합합니다:

```
품질 = 당도점수(50%) + 무게점수(50%)

🍯 당도점수 = (당도 - 10) / 8 × 50
   예: 14 Brix → (14-10)/8 × 50 = 25점

⚖️ 무게점수 = (무게 - 150) / 200 × 50
   예: 250g → (250-150)/200 × 50 = 25점
```

| 당도 범위 | 무게 범위 | 품질 범위 |
|----------|----------|----------|
| 10~18 Brix | 150~350g | 0~100점 |

## 📐 학습된 모델 해석

```
가격 = 30.5 × 품질 + 985
```

이 수식의 의미:
- 품질이 1점 오르면 → 가격 약 30원 상승
- 품질이 0점일 때 기본 가격은 약 985원
- 품질 100점이면 → 약 4,035원 예측

## 🔮 추론 과정 시각화 (v1.1.0 변경)

### 1단계: 품질 점수 계산
```
🍯 당도 점수 (14.0 Brix)      25.0점
⚖️ 무게 점수 (250g)           25.0점
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⭐ 총 품질 점수                50.0점
```

### 2단계: 가격 예측
```
🏷️ 기본 가격 (b)              985원
⭐ 품질 기여 (50.0 × 30.5)   +1,525원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💰 최종 가격                  2,510원
```

**차트에서 시각화:**
- 회귀선: 품질과 가격의 관계를 보여주는 직선
- 예측점: 노란 점으로 표시
- **수직선**: 품질 점수에서 회귀선까지
- **수평선**: 회귀선에서 가격 축까지

## ❓ 자주 묻는 질문

### 📊 데이터 관련

**Q: 데이터를 몇 개 생성해야 하나요?**
→ 처음에는 20~30개로 시작하세요. 너무 적으면(5개 미만) 학습이 불안정하고, 너무 많으면(100개) 화면이 복잡해집니다.

**Q: 노이즈는 왜 있나요?**
→ 실제 데이터는 완벽하지 않습니다. 같은 품질의 사과도 가격이 조금씩 다를 수 있죠. 노이즈는 이런 현실의 불확실성을 표현합니다.

**Q: 데이터를 다시 생성하면 어떻게 되나요?**
→ 새로운 랜덤 데이터가 생성되고, 이전 학습 결과는 초기화됩니다. 다시 학습을 실행해야 합니다.

### 🧠 학습 관련

**Q: 손실 그래프가 올라갔다 내려갔다 해요**
→ 학습률이 너무 큽니다. 0.01 이하로 낮춰보세요.

**Q: R² 점수가 음수예요**
→ 모델이 평균보다 못한 예측을 합니다. 데이터를 더 생성하거나 에포크를 늘려보세요.

**Q: 학습이 너무 오래 걸려요**
→ 에포크를 줄여보세요. 보통 500회면 충분합니다.

**Q: 손실 그래프가 처음부터 평평해요**
→ 학습률이 너무 작거나, 이미 좋은 초기값으로 시작했을 수 있습니다. 초기화 후 다시 시도하거나 학습률을 약간 높여보세요.

**Q: 학습할 때마다 결과가 달라요**
→ 정상입니다! 가중치의 초기값이 랜덤이기 때문입니다. 하지만 충분히 학습하면 비슷한 결과에 도달합니다.

### 🔮 추론 관련

**Q: 예측 가격이 이상해요**
→ 학습이 제대로 되지 않았을 수 있습니다. 초기화 버튼을 누르고 다시 학습해보세요.

**Q: 추론 모드에서 슬라이더를 움직여도 반응이 없어요**
→ 학습 모드에서 먼저 학습을 실행해야 합니다. 학습되지 않은 모델로는 예측할 수 없습니다.

## 🔑 핵심 용어

| 용어 | 의미 | 쉬운 설명 |
|------|------|----------|
| **품질 점수** | 당도+무게를 0~100점으로 통합한 점수 | 사과의 종합 점수 |
| **가중치 (Weight)** | 품질이 가격에 미치는 영향력 (기울기) | 품질 1점이 가격에 얼마나 영향주는지 |
| **편향 (Bias)** | 기본값, 품질이 0일 때의 예측 가격 | 품질과 상관없는 기본 가격 |
| **손실 (Loss)** | 예측과 실제의 차이 (작을수록 좋음) | AI가 얼마나 틀렸는지 |
| **오차선 (잔차)** | 실제값과 예측값의 차이를 시각화한 선 | 틀린 정도를 보여주는 선 |
| **학습률** | 가중치 업데이트 크기 | 한 걸음에 얼마나 배울지 |
| **에포크** | 전체 데이터로 학습하는 횟수 | 전체 데이터를 몇 번 볼지 |
| **R² 점수** | 모델 적합도 (1에 가까울수록 좋음) | 모델이 얼마나 잘 맞는지 (100점 만점) |
| **MSE** | Mean Squared Error, 평균 제곱 오차 | 틀린 정도의 평균 (작을수록 좋음) |
| **회귀선** | 품질과 가격의 관계를 나타내는 직선 | 예측을 위한 직선 |
