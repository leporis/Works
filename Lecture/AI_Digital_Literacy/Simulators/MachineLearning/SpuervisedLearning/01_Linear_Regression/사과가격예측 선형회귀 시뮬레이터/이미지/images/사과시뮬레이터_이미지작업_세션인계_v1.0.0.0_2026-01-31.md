## 4. 핵심 기술 정보

### 4.1 시뮬레이터 수치 범위 (이미지와 일치해야 함)

| 항목 | 범위 | 기본값 |
|------|------|--------|
| 당도 | 10~18 Brix | 14 |
| 무게 | 150~350g | 250 |
| 품질 | 0~100점 | 계산값 |
| 학습률 | 0.001~0.1 | 0.01 |
| 에포크 | 100~2000 | 500 |
| 노이즈 | 5%/10%/20% | 10% |

### 4.2 핵심 공식

```
품질 = 당도점수(50%) + 무게점수(50%)
당도점수 = (당도 - 10) / 8 × 50
무게점수 = (무게 - 150) / 200 × 50

회귀식: 가격 = w × 품질 + b
```

### 4.3 UI 레이아웃

```
학습/추론 모드 공통: 3열 레이아웃
├── 왼쪽: 240px (데이터/예측 입력)
├── 중앙: 가변 (차트)
└── 오른쪽: 260px (학습 설정/예측 결과)
```

### 4.4 SVG 스타일 규칙

| 항목 | 값 |
|------|-----|
| 배경 그라디언트 | #FFFBF5 → #FFF3E0 |
| 주요 색상 | #E53935 (빨강), #43A047 (녹색), #2196F3 (파랑) |
| 폰트 | Noto Sans KR, JetBrains Mono |
| 하단 패딩 | 10px 이상 확보 |
| 그림자 | filter로 적용 |

## 5. PNG 캡처 방법 (Puppeteer)

### 5.1 환경

```bash
# Chrome 설치됨
/usr/bin/google-chrome-stable (v144)

# Node.js 패키지
puppeteer-core 설치됨
```

### 5.2 주의사항

- **CDN 접근 불가**: Chart.js를 HTML에 인라인으로 삽입 필요
- 캡처용 HTML 파일: `/home/claude/simulator_local.html`
- 해상도: 1600 × 1200px

### 5.3 캡처 스크립트

