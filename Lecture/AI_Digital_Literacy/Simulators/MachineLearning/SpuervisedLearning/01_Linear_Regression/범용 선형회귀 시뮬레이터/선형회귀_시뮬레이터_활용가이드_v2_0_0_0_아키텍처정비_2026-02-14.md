## 2. 수업 시나리오

### 2.1 시나리오 A: 기본 수업 (45분)

> **목표**: "데이터로부터 패턴을 찾아 예측하는 원리"를 체험한다.

| 단계 | 시간 | 교수자 활동 | 학생 활동 | 시뮬레이터 기능 |
|:----:|:----:|------------|----------|----------------|
| 도입 | 5분 | "집이 클수록 가격이 비싼 건 어떻게 아나요?" | 경험 공유 | - |
| 개념 | 10분 | Y = wX + b 의미, 산점도 위 직선 설명 | 시뮬레이터 화면 관찰 | 랜덤 프리셋(주택가격) 표시 |
| 탐색 | 10분 | 학습률 변경 시연 → "손실 그래프를 관찰하세요" | 학습률 0.001 vs 0.1 비교 | 슬라이더 + 손실 그래프 |
| 실습 | 15분 | 활동지 1 배포 | Iris/펭귄 데이터로 필터링 실험 | 범주형 필터 + R² 비교 |
| 정리 | 5분 | 훈련MSE vs 테스트MSE 비교 → 과적합 개념 | 관찰 결과 발표 | 결과 테이블 |

**도입 대화 예시**:

> "여러분, 부동산 앱에서 '이 동네 평균 시세'가 어떻게 계산되는지 생각해 본 적 있나요? 오늘 우리는 컴퓨터가 데이터에서 패턴을 스스로 찾아내는 과정을 직접 눈으로 보겠습니다."

### 2.2 시나리오 B: 심화 수업 (90분, 2차시)

> **목표**: 하이퍼파라미터 튜닝, 데이터 편향, 모델 평가의 실제 과정을 경험한다.

**1차시 (45분) — 기본 학습**

| 단계 | 시간 | 활동 | 시뮬레이터 기능 |
|:----:|:----:|------|----------------|
| 도입 | 5분 | "AI가 '학습'한다는 게 무슨 뜻일까요?" | - |
| 개념 | 10분 | 경사 하강법 원리: "눈 감고 산에서 내려가기" | 학습 과정 재생 |
| 탐색 | 15분 | 활동지 2: 학습률-에포크 조합 실험 | 슬라이더 + 재생 |
| 실습 | 10분 | 추론 모드에서 예측 실험 | 추론 차트 + 예측값 |
| 정리 | 5분 | "좋은 모델이란?" 토론 | R² + 오차선 |

**2차시 (45분) — 데이터 탐색과 편향**

| 단계 | 시간 | 활동 | 시뮬레이터 기능 |
|:----:|:----:|------|----------------|
| 복습 | 5분 | 1차시 핵심 요약 | - |
| 탐색 | 15분 | 활동지 3: 앤스컴 4종 비교 → "R²만 믿으면 안 되는 이유" | 앤스컴 데이터 1~4 전환 |
| 실습 | 15분 | 활동지 4: 필터로 부분 데이터 회귀 → 편향 관찰 | Iris/팁 데이터 + 범주형 필터 |
| 토론 | 5분 | "필터 전후 R²가 다른 이유" | 필터 배너 + 워터마크 |
| 정리 | 5분 | 데이터 대표성과 모델 신뢰도 정리 | - |

### 2.3 시나리오 C: 단발 체험 (15분)

> **목표**: "AI 학습이 무엇인지" 한 번에 체험한다. (오리엔테이션, 특강 등)

| 단계 | 시간 | 활동 |
|:----:|:----:|------|

| 체험 | 5분 | 학생 각자 보스턴 주택가격 데이터로 학습 실행 |

## 3. 학습 질문 설계

### 3.1 블룸 분류 체계 기반

#### 🟢 초급 — 이해 (Remember & Understand)

| # | 질문 | 기대 답변 키워드 |
|:-:|------|----------------|
| 1 | 산점도의 파란 점과 빨간 점은 각각 무엇인가요? | 훈련 데이터, 테스트 데이터 |
| 2 | 회귀선의 기울기(w)가 양수이면 X와 Y의 관계는? | X가 커지면 Y도 커진다 (양의 상관) |
| 3 | R² = 0.85는 무슨 뜻인가요? | 모델이 데이터 변동의 85%를 설명 |
| 4 | 학습률을 매우 크게 하면 손실 그래프에 어떤 현상이 나타나나요? | 요동(발산), 안정적으로 내려가지 않음 |

#### 🟡 중급 — 적용 (Apply & Analyze)

| # | 질문 | 기대 관찰 |
|:-:|------|----------|
| 5 | 학습률 0.001과 0.05로 같은 데이터를 학습시켜보세요. 손실 곡선의 차이는? | 0.001: 느리지만 안정, 0.05: 빠르지만 요동 가능 |
| 6 | 에포크를 100에서 1,000으로 늘리면 R²가 개선되나요? 항상 그런가요? | 수렴 후에는 더 늘려도 변화 없음 |
| 7 | Iris에서 setosa만 필터링하면 R²가 어떻게 변하나요? 왜? | 부분 데이터의 선형 경향이 전체와 다름 |
| 8 | 훈련 MSE는 매우 낮은데 테스트 MSE가 높다면 어떤 문제인가요? | 과적합 (훈련 데이터에만 특화) |

#### 🔴 고급 — 분석 & 평가 (Evaluate & Create)

| # | 질문 | 핵심 통찰 |
|:-:|------|----------|
| 9 | 앤스컴 데이터 4종의 R²가 모두 약 0.67인데, 산점도가 전혀 다릅니다. 이것이 의미하는 바는? | 수치 지표만으로 모델을 평가하면 안 됨, 시각화 필수 |
| 10 | 팁 데이터에서 흡연자만 필터링한 회귀 결과로 "흡연자가 팁을 더 많이 준다"고 결론내릴 수 있나요? | 교란 변수(식사 금액 등) 미통제, 인과≠상관 |
| 11 | 다이아몬드 데이터에서 Ideal 등급만 필터링하면 캐럿-가격 관계가 달라지나요? 왜? | 심슨의 역설과 유사 — 부분과 전체의 경향 차이 |
| 12 | 이 시뮬레이터로는 할 수 없는 분석은 무엇인가요? | 비선형 관계, 다중 회귀(직접적), 시계열 패턴 |

### 3.2 질문 활용 팁

| 원칙 | 설명 |
|------|------|
| **먼저 조작, 나중에 질문** | "학습률을 바꿔보세요" → (관찰 후) → "왜 그런 차이가 생겼나요?" |
| **비교 질문** | 단일 실행보다 "A vs B" 비교가 학습 효과 높음 |
| **예측 → 확인** | "학습률을 높이면 어떻게 될 것 같나요?" → 실행 → 비교 |
| **오답 활용** | 예상과 다른 결과가 나오면 → "왜 예상과 달랐나요?" 토론 |

## 4. 학생 활동지

### 4.1 활동지 1: 첫 만남 — 선형 회귀 체험

**목표**: 선형 회귀의 기본 워크플로우를 체험한다.

**준비**: 시뮬레이터를 브라우저에서 연다.

| 단계 | 지시 | 확인 |
|:----:|------|:----:|
| 1 | 데이터셋 `📚 공부시간과 성적` 선택 | 산점도에 점이 보이나요? |
| 2 | 학습률 `0.01`, 에포크 `500` 확인 | - |
| 3 | **⚡ 학습 실행** 클릭 | 손실 그래프가 L자로 내려가나요? |
| 4 | 결과 패널에서 R² 확인 | R² = ______ |

**기록표**:

| 시도 | 데이터셋 | 학습률 | 에포크 | R² | 관찰 |
|:----:|---------|:------:|:------:|:---:|------|
| 1 | 공부시간과 성적 | 0.01 | 500 | | |
| 2 | 경력과 연봉 | 0.01 | 500 | | |
| 3 | 보스턴 주택가격 | 0.01 | 500 | | |

**질문**: 세 데이터셋의 R²가 서로 다릅니다. R²가 가장 높은 데이터는 무엇이고, 그 이유는 무엇일까요?

|----------|------|------------|
| "R²가 0.3이라 실패했어요" | 성공/실패 이분법 | "0.3이면 30%를 설명한 겁니다. 선형 관계가 약한 데이터일 수 있어요." |
| 학습률을 0.1로 올렸더니 발산 | 큰 값 = 좋다는 직관 | "학습률은 '보폭'입니다. 보폭이 너무 크면 목표를 지나쳐요." |
| 필터로 데이터 3개만 남김 | 필터 과다 적용 | "3명만 조사해서 전교생에 대한 결론을 내릴 수 있나요?" |
| 앤스컴 데이터 보고 혼란 | R²가 같은데 왜 다르지? | "이것이 바로 '항상 그래프를 봐야 하는 이유'입니다." |
| 추론 모드에서 "--" | 학습을 안 함 | "먼저 학습 모드에서 모델을 훈련시켜야 예측할 수 있어요." |

### 6.2 난이도 조절

| 수준 | 데이터 | 기능 | 질문 |
|:----:|--------|------|------|
| ⭐ | 랜덤 프리셋, 공부-성적 | 학습 실행 + R² 확인만 | "R²가 높으면 좋은 건가요?" |
| ⭐⭐ | Iris, 보스턴 주택 | + 학습률/에포크 조절 | "학습률을 바꾸면?" |
| ⭐⭐⭐ | 펭귄, 팁, 앤스컴 | + 범주형 필터 + 오차선 | "필터 전후 차이는 왜?" |
| ⭐⭐⭐⭐ | 다이아몬드, 택시, CSV | + 다중 변수 가중치 + 재생 | "편향, 교란 변수, 인과" |

### 6.3 시간이 부족할 때

| 사용 가능 시간 | 활동 | 핵심 메시지 |
|:-------------:|------|------------|
| 5분 | 주택가격 랜덤 → 학습 → R² 확인 | "AI는 데이터에서 패턴을 찾는다" |
| 15분 | + 학습률 비교 (0.001 vs 0.1) | "학습 속도를 조절해야 한다" |
| 30분 | + Iris 필터링 실험 | "데이터 구성이 결과를 바꾼다" |
| 45분 | 시나리오 A 전체 | 기본 개념 + 실습 + 과적합 |
| 90분 | 시나리오 B 전체 | 기본 + 앤스컴 + 편향 |

### 7.1 수업 후 퀴즈 예시

**객관식 (5지선다)**

Q1. 학습률(Learning Rate)을 너무 크게 설정하면 나타나는 현상은?
1. 손실이 매우 천천히 감소한다
2. R²가 항상 1에 가까워진다
3. 손실이 요동치거나 발산한다 ✅
4. 테스트 MSE가 0이 된다
5. 데이터 수가 줄어든다

Q2. R² = 0.85일 때, 이 모델은 데이터 변동의 몇 %를 설명하나요?
1. 15%
2. 50%
3. 75%
4. 85% ✅
5. 100%

Q3. 훈련 MSE가 매우 낮은데 테스트 MSE가 높다면 어떤 상태인가요?
1. 과소적합 (Underfitting)
2. 과적합 (Overfitting) ✅
3. 정규화 (Regularization)
4. 최적 상태
5. 데이터 부족

**서술형**

Q4. 앤스컴 데이터 4종은 R²가 모두 비슷하지만 산점도가 전혀 다릅니다. 이 사실이 데이터 분석에 시사하는 바를 2문장으로 설명하세요.

> (모범 답안 방향: 수치 지표만으로 모델을 평가하면 안 되며, 항상 시각화를 함께 확인해야 한다. 같은 통계량이 전혀 다른 데이터 패턴을 숨길 수 있다.)

Q5. Iris 데이터에서 전체 150건의 R²와 setosa 50건만 필터링한 R²가 다릅니다. 왜 이런 차이가 발생하는지, 그리고 이것이 실제 데이터 분석에서 어떤 주의점을 알려주는지 설명하세요.

> (모범 답안 방향: 전체 데이터의 경향과 부분 데이터의 경향이 다를 수 있다. 특정 그룹만 선택하면 대표성이 달라진다. 표본 편향/심슨의 역설과 연결.)

### 7.2 실습 평가 루브릭

| 평가 항목 | 우수 (A) | 보통 (B) | 미흡 (C) |
|----------|---------|---------|---------|
| 시뮬레이터 조작 | 데이터 선택, 필터, 학습, 추론 모두 수행 | 기본 학습은 수행하나 필터/추론 미활용 | 학습 실행만 수행 |

### 8.1 "왜 이 데이터셋인가?" — 선택 근거

| 데이터셋 | 교육적 의도 |
|---------|------------|
| 공부시간-성적 | 가장 직관적인 양의 상관 → 첫 경험용 |

| Iris | 피셔의 1936년 데이터 → 범주형 필터의 자연스러운 도입 |
| 다이아몬드 | 등급별 필터 시 경향 역전 가능 → 심슨의 역설 체험 |
| 팁 | 4개 범주형 컬럼 → 다중 필터의 AND 조합 탐색 |

### 8.2 선형 회귀의 한계를 가르칠 때

시뮬레이터는 단순 선형 회귀(Y = wX + b)만 지원합니다. 다음 주제는 **이 시뮬레이터로는 불가능**하므로 별도 설명이 필요합니다:

| 주제 | 시뮬레이터 대응 | 보충 방법 |
|------|:--------------:|----------|
| 비선형 관계 | ❌ | 앤스컴 2(곡선)로 한계 시연 후 설명 |
| 다중 선형 회귀 | △ (가중치 합산) | 가중치 슬라이더로 개념만 전달 |
| 범주형 변수 인코딩 | ❌ | 필터로 "그룹별 별도 회귀"만 가능 |
| 정규화(L1/L2) | ❌ | 과적합 개념 설명 후 이론적 소개 |
| 시계열 패턴 | ❌ | 항공 승객 데이터로 한계 시연 |

### 8.3 다른 과목과의 연계

| 연계 과목 | 연결 주제 | 시뮬레이터 활용 |
|----------|----------|----------------|
| 수학 (일차함수) | Y = aX + b의 기울기와 절편 | 슬라이더로 a, b 값이 직선에 미치는 영향 관찰 |
| 통계학 | 상관계수, 분산, 잔차 | R², MSE, 오차선 시각화 |
| 사회과학 방법론 | 표본 편향, 대표성 | 범주형 필터로 표본 구성 변경 실험 |
| 데이터 과학 | 훈련/테스트 분할, 모델 평가 | 80:20 분할 + 재분할 기능 |
| 철학/윤리 | AI의 편향과 공정성 | 필터로 특정 그룹 제외 시 결과 변화 |
