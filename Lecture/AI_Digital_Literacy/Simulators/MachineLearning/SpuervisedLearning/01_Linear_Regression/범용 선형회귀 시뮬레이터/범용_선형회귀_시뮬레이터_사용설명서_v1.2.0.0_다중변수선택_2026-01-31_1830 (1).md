## 1. 개요

### 1.1 시뮬레이터 소개

**선형 회귀 학습 시뮬레이터**는 머신러닝의 기초 알고리즘인 선형 회귀(Linear Regression)의 원리를 직관적으로 학습할 수 있는 교육용 도구입니다.

다양한 데이터셋으로 **Y = wX + b** 관계를 학습하고, 학습된 모델로 새로운 값을 예측해볼 수 있습니다.

### 1.2 학습 목표

| 목표 | 설명 |
|------|------|
| 선형 관계 이해 | 두 변수 사이의 선형 관계 파악 |
| 경사 하강법 체험 | 손실 함수를 최소화하는 최적화 과정 시각화 |
| 예측 원리 학습 | 학습된 모델로 새로운 데이터 예측 |
| 다중 변수 결합 | 여러 변수를 하나의 점수로 통합하는 방법 |

### 1.3 실행 환경

| 항목 | 권장 사양 |
|------|----------|
| 브라우저 | Chrome, Edge, Firefox (최신 버전) |
| 화면 크기 | 1200px 이상 권장 |
| 인터넷 | 필요 (Chart.js, 폰트 로드) |

## 3. 화면 구성

### 3.1 전체 레이아웃

시뮬레이터는 **3열 구성**으로 되어 있습니다:

```
┌──────────┬─────────────────────────┬──────────┐
│  왼쪽    │         중앙            │  오른쪽  │
│ (240px)  │        (가변)           │ (260px)  │
│          │                         │          │
│ 데이터   │  📉 손실    📈 산점도   │  학습    │
│ 입력     │  그래프    (회귀선)     │  설정    │
└──────────┴─────────────────────────┴──────────┘
```

### 3.2 모드별 화면

#### 🧠 학습 모드

| 영역 | 구성 요소 |
|------|----------|
| 왼쪽 | 데이터 패널 (프리셋/CSV/직접 입력) |
| 중앙 | 손실 그래프 + 산점도 |
| 오른쪽 | 학습 설정 (학습률, 에포크) |
| 하단 | 학습 결과 패널 (학습 후 표시) |

```
┌──────────┬─────────────────────────┬──────────┐
│  데이터   │  📉 손실    📈 산점도   │  학습    │
│ [프리셋] │  그래프     (회귀선)    │  설정    │
│ [CSV]   │                         │          │
│ [직접]  │                         │          │
└──────────┴─────────────────────────┴──────────┘
           ↓  학습 실행 후 표시  ↓
┌─────────────────────────────────────────────────┐
│  📊 학습결과  │  📏 오차분석  │  📽️ 재생     │
└─────────────────────────────────────────────────┘
```

#### 🔮 추론 모드

| 영역 | 구성 요소 |
|------|----------|
| 왼쪽 | 예측 입력 (X값 슬라이더) |
| 중앙 | 예측 시각화 (산점도 + 회귀선 + 예측점) |
| 오른쪽 | 예측 결과 (Y값, 추론 과정) |

```
┌──────────┬─────────────────────────┬──────────┐
│  X값     │   📈 예측 시각화        │  예측 Y  │
│  입력    │  (회귀선 + 예측점)      │  결과    │
│          │  (수직선 + 수평선)      │  추론    │
└──────────┴─────────────────────────┴──────────┘
```

### 3.3 모드 전환

화면 상단의 탭 버튼으로 모드를 전환합니다:

- **🧠 학습 모드**: 데이터 생성 및 모델 학습
- **🔮 추론 모드**: 학습된 모델로 예측

## 4. 데이터 입력

### 4.1 데이터 입력 방법 3가지

왼쪽 패널 상단의 탭으로 입력 방법을 선택합니다:

| 탭 | 용도 | 특징 |
|----|------|------|
| **프리셋** | 미리 정의된 시나리오 | 초보자 추천 |
| **CSV** | 파일 업로드 또는 내장 데이터 | 다중 변수 선택 가능 |
| **직접** | 수동 입력 또는 차트 클릭 | 소량 데이터 |

### 4.2 프리셋 탭

#### 시나리오 선택

| 버튼 | X 변수 | Y 변수 | 설명 |
|------|--------|--------|------|
| 🏠 주택 | 면적(㎡) | 가격(만원) | 면적↑ → 가격↑ |
| 📝 시험 | 공부시간(h) | 점수 | 공부↑ → 점수↑ |
| 💰 급여 | 경력(년) | 연봉(만원) | 경력↑ → 연봉↑ |
| 🎲 랜덤 | X | Y | 임의 데이터 |

#### 파라미터 설정

| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| 데이터 개수 | 5~100 | 30 | 생성할 데이터 수 |
| 노이즈 수준 | 5%/10%/20% | 10% | 데이터 산포도 |

#### 사용 방법

1. 프리셋 버튼 클릭 (예: 🏠 주택)
2. 데이터 개수/노이즈 설정
3. [✨ 데이터 생성] 클릭

### 4.3 CSV 탭

#### CSV 파일 업로드

```csv
면적,가격
50,800
70,1200
90,1600
```

- **첫 행**: 변수명 (헤더)
- **이후**: 데이터 (콤마 구분)
- **인코딩**: UTF-8 권장

#### 내장 데이터셋 (v1.1.0+)

| 버튼 | 데이터셋 | 변수 | 샘플 수 |
|------|----------|------|---------|
| 🐧 펭귄 | Palmer Penguins | 부리길이, 부리깊이, 날개길이, 체중 | 30 |
| 🌸 아이리스 | Iris | 꽃받침길이/너비, 꽃잎길이/너비 | 30 |
| 🚗 자동차 | Cars | 속도, 제동거리 | 50 |
| 🏘️ 보스턴 | Boston Housing | 방개수, 저소득층비율, 학생교사비율, 주택가격 | 30 |

#### 다중 독립변수 선택 (v1.2.0 신규)

CSV 또는 내장 데이터셋 로드 후:

1. **독립변수(X) 선택**: 체크박스로 1개 이상 선택
2. **종속변수(Y) 선택**: 드롭다운에서 1개 선택

```
📊 독립변수(X) 선택
┌─────────────────────────┐
│ ☑ 부리길이(mm)          │
│ ☑ 부리깊이(mm)          │
│ ☐ 날개길이(mm)          │
│ ☐ 체중(g)  ← 종속변수   │
└─────────────────────────┘
```

#### 가중치 슬라이더 (2개 이상 선택 시)

독립변수를 2개 이상 선택하면 가중치 슬라이더가 표시됩니다:

```
⚖️ 변수별 가중치
┌─────────────────────────┐
│ 부리길이    [====] 50%  │
│ 부리깊이    [====] 50%  │
├─────────────────────────┤
│ X = 50%×부리길이 + 50%× │
│     부리깊이             │
│ [🔄 가중치 적용]        │
└─────────────────────────┘
```

**가중치 조정 방법**:
- 슬라이더를 드래그하여 비율 조정
- 한 변수의 가중치를 변경하면 나머지가 자동 조정됨
- 가중치 합계는 항상 100%
- [🔄 가중치 적용] 버튼으로 데이터 재계산

**조합점수 계산 공식**:
```
X = w₁×정규화(X₁) + w₂×정규화(X₂) + ...

정규화(Xᵢ) = (Xᵢ - min) / (max - min) × 100
```

### 4.4 직접 입력 탭

#### 수동 입력

1. X, Y 값 입력
2. [+ 추가] 버튼 클릭

#### 차트 클릭

1. [직접] 탭 선택
2. 산점도 영역 클릭
3. 클릭 위치에 데이터 추가

💡 **팁**: 차트 클릭은 [직접] 탭에서만 작동합니다.

## 5. 모델 학습

### 5.1 학습 설정

오른쪽 패널에서 학습 파라미터를 설정합니다:

| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| 학습률 | 0.001~0.1 | 0.01 | 가중치 업데이트 크기 |
| 에포크 | 100~2000 | 500 | 전체 학습 반복 횟수 |

#### 학습률 선택 가이드

| 학습률 | 특징 | 권장 상황 |
|--------|------|----------|
| 0.001 | 매우 느림, 안정 | 발산 방지 |
| 0.01 | 적당한 속도 | **일반 추천** |
| 0.1 | 빠름, 불안정 | 빠른 테스트 |

### 5.2 학습 실행

[⚡ 학습 실행] 버튼을 클릭하면:

1. 모델이 빠르게 학습됨
2. 손실 그래프에 MSE 변화 표시
3. 산점도에 회귀선 표시
4. 학습 결과 패널이 하단에 표시됨

### 5.3 학습 결과 확인

학습 완료 후 하단에 3개 섹션이 표시됩니다:

#### 📊 학습 결과

| 항목 | 설명 |
|------|------|
| 에포크 | 총 학습 횟수 |
| 손실(MSE) | 최종 평균 제곱 오차 |
| R² | 결정 계수 (1에 가까울수록 좋음) |
| 상태 | ✅ 양호 / ⚠️ 보통 / ❌ 부족 |

**R² 해석**:
| R² 값 | 상태 | 의미 |
|-------|------|------|
| > 0.8 | ✅ 양호 | 모델이 데이터를 잘 설명함 |
| > 0.5 | ⚠️ 보통 | 어느 정도 설명함 |
| ≤ 0.5 | ❌ 부족 | 개선 필요 |

**회귀식 표시**:
```
Y = 15.23 × X + 842.5
```

#### 📏 오차 분석

**오차선 표시**:
1. [오차선 표시] 체크박스 클릭
2. 산점도에 점선으로 오차선 표시됨
3. 색상: 🟢 오차 작음 → 🔴 오차 큼

**오차 통계**:
| 지표 | 설명 |
|------|------|
| 평균 오차 | 오차들의 평균 |
| 최대 오차 | 가장 큰 오차 |
| MSE | Mean Squared Error |

💡 **교육 포인트**: "AI는 항상 정확하지 않다"를 시각적으로 확인!

#### 📽️ 학습 과정 재생

학습 과정을 애니메이션으로 재현합니다:

| 버튼 | 기능 |
|------|------|
| ▶️ 재생 | 회귀선이 데이터에 맞춰가는 과정 재생 |
| ⏹️ 정지 | 재생 중지, 최종 상태로 복원 |

**재생 속도**: 슬라이더로 느림~빠름 10단계 조절

### 6.3 예측 결과 확인

#### 중앙: 시각화

```
┌─────────────────────────────────────────┐
│                                         │
│    데이터 점들                           │
│         ●  ●                            │
│      ●     ●   회귀선                   │
│   ●     ●   ──────────                  │
│ ●    ●         │                        │
│        ┄┄┄┄┄┄┄┄┼─────  예측점 ●        │
│               │       (노란색)          │
│               │                         │
│               X값                       │
└─────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 점선 (수직) | X값 → 회귀선 |
| 점선 (수평) | 회귀선 → Y축 |
| 노란 원 | 예측점 위치 |

#### 오른쪽: 추론 과정

```
🏷️ 기본값 (b)              842.5

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💰 최종 예측값              2,362.5
```

**회귀식**:
```
Y = 15.23 × X + 842.5
```

## 7. 자주 묻는 질문

### Q1: 손실 그래프가 내려가다가 다시 올라가요

**원인**: 학습률이 너무 큼  
**해결**: 학습률을 0.01 이하로 낮추세요

### Q2: R² 점수가 음수예요

**원인**: 모델이 평균보다 못한 예측을 함  
**해결**: 
- 데이터 개수를 늘리세요
- 에포크를 늘리세요
- 노이즈를 낮추세요

### Q3: 차트 클릭이 안 돼요

**원인**: [직접] 탭이 아닌 상태  
**해결**: [직접] 탭을 선택한 후 클릭하세요

### Q4: CSV 파일이 안 열려요

**원인**: 파일 형식 또는 인코딩 문제  
**해결**:
- UTF-8 인코딩으로 저장
- 콤마(,)로 구분
- 첫 행에 헤더 포함

### Q5: 가중치 슬라이더가 안 보여요

**원인**: 독립변수를 1개만 선택함  
**해결**: CSV 탭에서 독립변수를 2개 이상 체크하세요

### Q6: 학습 후 추론 모드에서 아무것도 안 보여요

**원인**: 학습이 완료되지 않음  
**해결**: 학습 모드에서 [⚡ 학습 실행]을 먼저 실행하세요

## 8. 문제 해결

### 8.1 증상별 해결 방법

| 증상 | 원인 | 해결 방법 |
|------|------|----------|
| 화면이 깨짐 | 브라우저 호환성 | Chrome/Edge 최신 버전 사용 |
| 차트가 안 보임 | 인터넷 연결 | 인터넷 연결 확인 (Chart.js 로드) |
| 손실이 발산 | 학습률 과다 | 학습률 0.01 이하로 설정 |
| 학습이 안 됨 | 데이터 없음 | 데이터 생성 먼저 실행 |
| 예측이 이상함 | 초기화 필요 | [🗑️ 초기화] 후 재학습 |
| 가중치 합이 100% 안 됨 | 버그 | [🔄 가중치 적용] 재클릭 |
