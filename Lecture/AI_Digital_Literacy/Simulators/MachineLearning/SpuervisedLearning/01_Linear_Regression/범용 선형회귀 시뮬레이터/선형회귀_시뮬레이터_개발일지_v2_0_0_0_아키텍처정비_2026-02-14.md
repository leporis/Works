# 📊 선형 회귀 학습 시뮬레이터 개발일지

**문서 버전: v2.0.0.0**

---

## 1. 프로젝트 개요

### 1.1 목적

경사 하강법(Gradient Descent) 기반 단순 선형 회귀(Simple Linear Regression)의 학습 원리를 **직접 조작하며 체험**할 수 있는 웹 기반 교육용 시뮬레이터입니다. Y = wX + b 모델의 파라미터(w, b)가 데이터에 맞춰 최적화되는 과정을 실시간 시각화하여, 비전공자도 머신러닝의 핵심 개념을 직관적으로 이해할 수 있도록 설계되었습니다.

### 1.2 교육 목표

| 수준 | 목표 | 시뮬레이터 기능 |
|:----:|------|----------------|
| 초급 | 선형 회귀의 의미 이해 | 산점도 + 회귀선 시각화 |
| 중급 | 학습률·에포크가 학습에 미치는 영향 체험 | 하이퍼파라미터 슬라이더 + 손실 그래프 |
| 고급 | 과적합·일반화 개념 이해 | 훈련/테스트 분할, 테스트 MSE 비교 |
| 심화 | 데이터 편향과 표본 구성의 영향 탐구 | 범주형 필터 + 다중 변수 가중치 |

### 1.3 개발 배경

v1.4.0.0(공개 데이터셋 23종)에서 기능적 완성도를 확보한 후, v2.0.0.0에서는 **코드 아키텍처 정비**를 목표로 메이저 업그레이드를 수행했습니다. 유지보수성·확장성을 높이기 위해 소프트코딩 체계(CONFIG 객체, CSS 변수 2단계 계층)를 전면 도입하고, 범주형 필터 시스템을 신설하여 데이터 탐색 기능을 강화했습니다.

## 2. 시스템 설계

### 2.1 입출력

| 구분 | 항목 | 설명 |
|------|------|------|
| **입력** | 데이터 | 내장 데이터셋 23종 + 랜덤 프리셋 4종 + CSV 업로드 + 직접 입력 |
| | 하이퍼파라미터 | 학습률 (0.001~0.1), 에포크 수 (100~2,000) |
| | 변수 선택 | 독립변수(X) 다중 선택 + 종속변수(Y) 단일 선택 |
| | 범주형 필터 | 데이터셋 내 범주형 컬럼별 값 선택/제외 |
| **출력** | 회귀 모델 | Y = wX + b (기울기, 절편) |
| | 성능 지표 | 훈련 MSE, 테스트 MSE, R² |
| | 시각화 | 산점도, 손실 그래프, 오차선, 추론 차트 |

### 2.2 알고리즘

경사 하강법을 사용한 MSE(평균 제곱 오차) 최소화:

```
for each epoch:
    gradW = (2/n) × Σ(w·xᵢ + b - yᵢ) × xᵢ
    gradB = (2/n) × Σ(w·xᵢ + b - yᵢ)
    w = w - lr × gradW
    b = b - lr × gradB
```

### 2.3 데이터 흐름

```
[데이터 소스] → rawCsvData (원본 보존)
                    │
                    ▼
            getFilteredRawData() ←── categoricalFilters (범주형 필터)
                    │
                    ▼
            applyVariableSelection() ←── selectedXVars, selectY, varWeights
                    │
                    ▼
                data [{x, y}] (회귀용 수치 데이터)
                    │
                    ▼
              splitData() ←── Fisher-Yates 셔플, 80:20 분할
                    │
            ┌───────┴───────┐
            ▼               ▼
        trainData         testData
            │               │
            ▼               ▼
        trainOneEpoch()   calculateRealMSE()
```

| 기능 | 설명 | 상세 |
|------|------|------|
| 내장 데이터셋 | 공개 데이터셋 23종 + 랜덤 4종 | Iris, 펭귄, MPG, 보스턴 주택, 당뇨병, 캘리포니아 주택, 와인, 팁, 다이아몬드, 광고-매출, 공부-성적, 경력-연봉, 간헐천, 항공 승객, 교통사고, 의료비, 앤스컴 4종, 택시, 체력 측정, 사과 품질 |
