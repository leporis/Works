- 필요한 기능만 **추가**하는 방식으로 개발

### 4.2 다중 변수 → 단일 X 변환
```
X = Σ(wᵢ × 정규화(Xᵢ))

정규화: (값 - min) / (max - min) × 100
가중치 합: w₁ + w₂ + ... + wₙ = 1
```

### 4.3 가중치 슬라이더 연동
- 한 변수 가중치 변경 시 나머지는 **비례 배분**으로 자동 조정
- 예: A를 70%로 변경 → B(30%), C(20%)가 B(18%), C(12%)로 조정
