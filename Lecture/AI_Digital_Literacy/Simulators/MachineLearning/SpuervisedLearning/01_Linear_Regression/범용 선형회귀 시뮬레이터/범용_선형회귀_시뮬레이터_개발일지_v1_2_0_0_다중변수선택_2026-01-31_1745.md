## 2. v1.2.0.0 핵심 기능: 다중 독립변수 선택

### 2.1 개념

원본 사과 시뮬레이터의 "품질 점수" 개념을 일반화:
- **원본**: 당도 + 무게 → 품질점수 → 가격 예측
- **범용**: X₁ + X₂ + ... + Xₙ → 조합점수(X) → Y 예측

```
조합점수(X) = w₁×정규화(X₁) + w₂×정규화(X₂) + ... + wₙ×정규화(Xₙ)

여기서:
- 정규화(Xᵢ) = (Xᵢ - min) / (max - min) × 100  (0~100 스케일)
- w₁ + w₂ + ... + wₙ = 1 (가중치 합계)
```

### 2.2 사용자 인터페이스

#### 독립변수 선택 (체크박스)
```
📊 독립변수(X) 선택
┌─────────────────────────┐
│ ☑ 부리길이(mm)          │
│ ☑ 부리깊이(mm)          │
│ ☐ 날개길이(mm)          │
│ ☐ 체중(g)  ← 종속변수    │
└─────────────────────────┘
```

#### 가중치 슬라이더 (2개 이상 선택 시)
```
⚖️ 변수별 가중치
┌─────────────────────────┐
│ 부리길이    [====] 50%  │
│ 부리깊이    [====] 50%  │
├─────────────────────────┤
│ X = 50%×부리 + 50%×부리 │
│ [🔄 가중치 적용]        │
└─────────────────────────┘
```

### 2.3 가중치 연동 로직

한 변수의 가중치를 변경하면 나머지 변수들이 **비례 배분**으로 자동 조정:

```javascript
// 변수 A의 가중치를 70%로 변경 시
// 변수 B(기존 30%), C(기존 20%)의 비율 유지하며 나머지 30% 배분
B_new = (B_old / (B_old + C_old)) × 30% = 18%
C_new = (C_old / (B_old + C_old)) × 30% = 12%
```

### 2.4 정규화 방식

각 변수를 0~100 스케일로 정규화하여 단위가 다른 변수들을 동일 기준으로 비교:

| 변수 | 원본 범위 | 정규화 범위 |
|------|----------|------------|
| 부리길이(mm) | 34~46 | 0~100 |
| 부리깊이(mm) | 17~22 | 0~100 |
| 체중(g) | 3200~4675 | 0~100 |

### 3.1 새로 추가된 전역 변수

```javascript
let numericColumns = [];    // 수치형 컬럼 목록
```

### 3.3 가중 평균 X 계산 로직

```javascript
function calculateWeightedX(row) {
    // 1. 각 변수의 min, max 계산
    const varStats = {};
    selectedXVars.forEach(col => {
        const values = rawCsvData.map(r => parseFloat(r[col]));
        varStats[col] = { min: Math.min(...values), max: Math.max(...values) };
    });

    // 2. 정규화(0~100) 후 가중 평균
    let weightedSum = 0;
    selectedXVars.forEach(col => {
        const val = parseFloat(row[col]);
        const { min, max } = varStats[col];
        const normalized = ((val - min) / (max - min)) * 100;
        weightedSum += normalized * varWeights[col];
    });

    return weightedSum;
}
```

## 5. 사용 시나리오

### 시나리오 1: 단일 변수 선택
1. 🐧 펭귄 데이터셋 로드
2. 독립변수: 부리길이 ☑ (1개만)
3. 종속변수: 체중
4. → 기존 단순 선형회귀와 동일하게 동작

### 시나리오 2: 다중 변수 동일 비율
1. 🐧 펭귄 데이터셋 로드
2. 독립변수: 부리길이 ☑, 부리깊이 ☑ (2개)
3. 가중치: 50% : 50% (기본값)
4. → 조합점수 = 50%×정규화(부리길이) + 50%×정규화(부리깊이)

### 시나리오 3: 가중치 조정
1. 시나리오 2 상태에서
2. 부리길이 가중치를 70%로 조정
3. 부리깊이 가중치 자동으로 30%
4. → 조합점수 = 70%×정규화(부리길이) + 30%×정규화(부리깊이)

## 6. 교육적 의의

### 6.1 다변량 → 단변량 축소
- 여러 입력 변수를 하나의 "점수"로 통합하는 개념 학습
- 원본 사과 시뮬레이터의 "품질 점수" 개념과 연결

### 6.2 가중치의 의미
- 각 변수가 결과에 미치는 상대적 중요도
- 슬라이더로 직접 조절하며 결과 변화 관찰

### 6.3 정규화의 필요성
- 단위가 다른 변수들을 동일 기준으로 비교
- 예: mm 단위 부리길이 vs g 단위 체중
