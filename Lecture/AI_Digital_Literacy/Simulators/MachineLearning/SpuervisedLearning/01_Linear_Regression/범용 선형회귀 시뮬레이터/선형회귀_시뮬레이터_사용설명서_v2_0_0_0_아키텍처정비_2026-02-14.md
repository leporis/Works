## 1. 개요

### 1.1 시뮬레이터 소개

선형 회귀 학습 시뮬레이터는 **경사 하강법(Gradient Descent)**을 사용하여 Y = wX + b 모델을 학습하는 과정을 실시간으로 시각화하는 웹 기반 교육 도구입니다. 별도의 설치 없이 브라우저에서 바로 실행됩니다.

### 1.2 주요 특징

| 특징 | 설명 |
|------|------|
| 단일 HTML 파일 | 설치 불필요, 오프라인 사용 가능 (CDN 제외) |
| 27종 내장 데이터셋 | 실제 공개 데이터 기반 학습 |
| 2가지 모드 | 학습 모드(모델 훈련) + 추론 모드(예측 실험) |
| 범주형 필터 | 데이터 부분집합으로 회귀 실험 |
| 다중 변수 가중치 | 여러 독립변수를 조합한 회귀 |
| 학습 과정 재생 | 에포크별 모델 변화 애니메이션 |

### 1.3 동작 환경

| 항목 | 요구 사항 |
|------|----------|
| 브라우저 | Chrome, Firefox, Edge, Safari (최신 버전) |
| 화면 | 1024px 이상 권장 (반응형 지원) |
| 네트워크 | 최초 로드 시 CDN 접근 필요 (Chart.js, Google Fonts) |

### 1.4 관련 문서

| 문서 | 용도 | 사용 시점 |
|------|------|----------|
| **퀵가이드** | 빠른 참조, 즉시 실행 | 시뮬레이터 사용 **중** |
| **사용설명서 (본 문서)** | 전체 기능 상세 설명 | 사용 **전/후** 참조 |
| **활용가이드** | 수업 적용 방법 | 수업 준비 시 |
| **개발일지** | 기술 구현 상세 | 개발·확장 시 |

## 2. 시작하기

### 2.1 실행 방법

HTML 파일을 브라우저로 열면 즉시 실행됩니다. 초기 화면은 **학습 모드**이며, `📚 공부시간과 성적` 데이터셋이 기본 선택되어 있습니다.

### 2.2 기본 워크플로우

```
[1] 데이터 선택/로드
    ↓
[2] X 변수, Y 변수 선택
    ↓
[3] (선택) 범주형 필터 적용
    ↓
[4] 학습률, 에포크 설정
    ↓
[5] ⚡ 학습 실행
    ↓
[6] 결과 확인 (MSE, R²)
    ↓
[7] (선택) 🔮 추론 모드에서 예측 실험
```

## 3. 화면 구성

### 3.1 전체 레이아웃

시뮬레이터는 3열 그리드 구조로 구성됩니다.

```
┌──────────────────────────────────────────────────────────────────┐
│  📊 선형 회귀 학습 시뮬레이터  v2.0.0.0                         │
│  데이터의 패턴을 찾아 Y = wX + b 관계를 학습해보세요!           │
├──────────────────────────────────────────────────────────────────┤
│  [🧠 학습 모드]     [🔮 추론 모드]                              │
├──────────┬───────────────────────────┬───────────────────────────┤
│ 좌측      │       중앙 차트 영역       │       우측               │
│ 사이드바  │                           │       사이드바            │
│ (240px)  │                           │       (260px)            │
│          │   📉 손실  │  📈 산점도    │                          │
│ · 데이터  │   그래프   │              │  · 하이퍼파라미터         │
│ · 변수    │           │              │  · 학습/재분할 버튼       │
│ · 필터    │   학습 결과 패널          │  · 오차선/재생            │
│ · 테이블  │                           │                          │
└──────────┴───────────────────────────┴───────────────────────────┘
```

### 3.2 모드 탭

화면 상단의 모드 탭으로 두 가지 모드를 전환합니다.

| 모드 | 탭 | 좌측 사이드바 | 우측 사이드바 | 차트 영역 |
|------|:--:|:-----------:|:-----------:|:---------:|
| **🧠 학습** | 빨간 활성 | 데이터 패널 | 학습 설정 패널 | 손실 + 산점도 |
| **🔮 추론** | 빨간 활성 | 예측 입력 패널 | 예측 결과 패널 | 추론 차트 |

### 3.3 반응형 동작

| 화면 너비 | 레이아웃 |
|:---------:|---------|
| 1200px 초과 | 3열 (좌측 240px + 차트 + 우측 260px) |
| 1024~1200px | 차트 + 좌/우 사이드바 축소 |
| 1024px 이하 | 1열 세로 배치 |

## 4. 데이터 입력

좌측 사이드바 상단에 3개의 탭이 있습니다.

### 4.1 데이터셋 탭

**내장 데이터셋 23종 + 랜덤 프리셋 4종**을 드롭다운에서 선택합니다.

데이터셋을 선택하면 다음이 자동으로 수행됩니다:

1. 데이터 로드 및 데이터 테이블 표시
2. X/Y 변수 드롭다운 자동 구성
3. 범주형 컬럼이 있으면 필터 UI 생성
4. 산점도에 데이터 포인트 표시
5. `📊 [데이터셋명] 데이터 로드 (N건)` 토스트 알림

**랜덤 프리셋** (드롭다운 상단 4종):

| 프리셋 | X축 | Y축 | 데이터 수 |
|--------|-----|-----|:---------:|
| 주택가격 | 면적(㎡) | 가격(만원) | 5~100 (조절) |
| 시험점수 | 공부시간(h) | 점수 | 5~100 |
| 급여 | 경력(년) | 연봉(만원) | 5~100 |
| 랜덤 | X | Y | 5~100 |

랜덤 프리셋 선택 시 추가 옵션:

| 옵션 | 설명 | 기본값 |
|------|------|:------:|
| 데이터 수 | 생성할 데이터 포인트 수 (5~100) | 30 |
| 노이즈 수준 | 데이터의 산포 정도 | 10% (보통) |

노이즈 수준 3단계:

| 수준 | 값 | 효과 |
|------|:--:|------|
| 낮음 | 5% | 점들이 회귀선에 밀착 → R² 높음 |
| 보통 | 10% | 적당한 산포 |
| 높음 | 20% | 점들이 넓게 퍼짐 → R² 낮음 |

### 4.2 CSV 탭

사용자가 직접 CSV 파일을 업로드합니다.

**지원 형식**: 쉼표(,) 구분 CSV, 첫 행은 헤더

**자동 감지 기능**:
- **수치형 컬럼**: 숫자로 변환 가능한 컬럼 → X/Y 변수 후보
- **범주형 컬럼**: 문자열 컬럼 → 필터 후보
- **스마트 범주형 감지**: 수치형이지만 유니크값이 20개 이하인 컬럼도 범주형으로 인식 (예: 평점 1~5)

업로드 성공 시 상태 표시: `✅ N개 행, M개 수치 변수, K개 범주형`

### 4.3 직접 탭

수동으로 데이터를 입력합니다.

**입력 방법 2가지**:

1. **수치 입력**: X값과 Y값을 입력 필드에 타이핑 후 추가
2. **차트 클릭**: 산점도 영역을 클릭하면 해당 좌표에 데이터 포인트 추가

## 5. 변수 선택 및 가중치

### 5.1 변수 선택

내장 데이터셋이나 CSV를 로드하면 좌측 사이드바에 변수 선택 UI가 나타납니다.

| 항목 | 설명 |
|------|------|
| **Y변수** (종속) | 드롭다운에서 1개 선택 |
| **X변수** (독립) | 체크박스로 1개 이상 선택 |

**제약 조건**:
- Y변수와 같은 변수를 X변수로 선택할 수 없습니다.
- 최소 1개의 X변수가 선택되어야 합니다.

### 5.2 다중 변수 가중치

X변수를 2개 이상 선택하면 **가중치 슬라이더**가 나타납니다.

각 변수는 0~100으로 정규화(Min-Max Scaling)된 후, 가중치에 따라 합산되어 하나의 X값을 만듭니다.

```
X = w₁ × norm(변수1) + w₂ × norm(변수2) + ...
```

가중치 조절 시 합성 변수 수식이 실시간으로 표시됩니다:

```
X = 0.60 × 꽃받침길이 + 0.40 × 꽃잎길이
```

## 6. 범주형 필터

### 6.1 개요

일부 데이터셋에는 범주형(categorical) 컬럼이 포함되어 있습니다. 필터를 사용하면 특정 범주만 선택하여 부분 데이터로 회귀를 실행할 수 있습니다.

### 6.2 필터 대상 데이터셋

| 데이터셋 | 범주형 컬럼 수 | 컬럼 목록 |
|---------|:------------:|----------|
| 🌸 Iris | 1 | 품종 |
| 🐧 팔머 펭귄 | 3 | 종, 섬, 성별 |
| 🚗 자동차 연비 | 1 | 제조국 |
| 🍷 와인 | 1 | 품종 |
| 🍽️ 식당 팁 | 4 | 성별, 흡연, 요일, 시간대 |
| 💎 다이아몬드 | 3 | 컷등급, 색상등급, 투명도등급 |
| 🚨 교통사고 | 1 | 주 |
| 🏥 의료비 | 1 | 국가 |
| 🚕 뉴욕 택시 | 2 | 차량색상, 결제방식 |

### 6.3 필터 UI

범주형 컬럼이 있는 데이터셋을 선택하면 좌측 사이드바에 **🏷️ 범주형 필터** 섹션이 나타납니다.

| UI 요소 | 동작 |
|---------|------|
| **파란 칩** | 해당 범주 포함 (활성) |
| **회색 칩** | 해당 범주 제외 (비활성) |
| **전체선택/해제 버튼** | 해당 컬럼의 모든 값을 일괄 토글 |
| **↺ 필터 초기화** | 모든 컬럼의 필터를 전체선택으로 복원 |

**다중 필터**: 여러 컬럼의 필터를 동시에 적용하면 **AND 조건**으로 결합됩니다.

> 예: 펭귄 데이터에서 `종=Adelie` AND `성별=MALE` → Adelie 수컷만 표시

### 6.4 안전장치

필터로 데이터가 줄어들 때 다음 안전장치가 작동합니다:

| 조건 | 표시 | 동작 |
|------|------|------|
| 필터 활성 | 주황색 배너: `필터 적용 중 · [조건] → N/M건` | 차트 위 워터마크 표시 |
| 10건 이하 | 노란 경고: `⚠️ 소표본으로 회귀 결과가 불안정할 수 있습니다` | 경고만 |
| 3건 이하 | 빨간 경고: `🚫 학습 불가능합니다. 필터를 넓혀주세요` | **학습 버튼 비활성화** |

### 6.5 교육적 주의사항

> ⚠️ 필터는 표본 구성을 변경하는 행위입니다. 필터 적용 시 대표성이 달라지고, 회귀 결과가 전체 데이터와 다를 수 있습니다. 결과 해석 시 필터 조건을 반드시 함께 보고하세요.

## 7. 학습 모드

### 7.1 하이퍼파라미터 설정

우측 사이드바에서 2가지 하이퍼파라미터를 조절합니다.

| 파라미터 | 범위 | 기본값 | 단위 | 설명 |
|---------|:----:|:------:|:----:|------|
| **학습률** | 0.001 ~ 0.1 | 0.01 | 0.001 | 경사 하강법의 스텝 크기 |
| **에포크** | 100 ~ 2,000 | 500 | 100 | 전체 데이터 반복 학습 횟수 |

**학습률 가이드**:

| 값 | 현상 | 적합 상황 |
|:--:|------|----------|
| 0.001 | 매우 느리게 수렴, 안정적 | 데이터가 복잡하거나 발산할 때 |
| 0.01 | 적당한 속도로 수렴 | 대부분의 경우 (기본값) |
| 0.05~0.1 | 빠르지만 불안정 가능 | 빠른 탐색, 단순 데이터 |

### 7.2 학습 실행

**⚡ 학습 실행** 버튼을 클릭하면 다음이 수행됩니다:

1. 데이터를 훈련(80%) / 테스트(20%)로 분할 (Fisher-Yates 셔플)
2. 정규화(Min-Max) 적용
3. 경사 하강법으로 w, b 최적화
4. 에포크별 훈련 MSE + 테스트 MSE 기록
5. 학습 완료 후 결과 패널 표시
6. 토스트 알림: `학습 완료! R² = 0.XXXX ✅/⚠️/❌`

### 7.3 훈련/테스트 분할

| 항목 | 값 |
|------|-----|
| 분할 비율 | 훈련 80% : 테스트 20% |
| 셔플 방식 | Fisher-Yates 알고리즘 (균일 랜덤) |
| 최소 데이터 | 5개 미만 시 전체를 훈련에 사용 (분할 없음) |

**🔀 데이터 재분할** 버튼으로 셔플을 다시 수행하고 자동 재학습합니다 (데이터 5개 이상 필요).

### 7.4 학습 결과

학습 완료 후 차트 아래에 결과 패널이 나타납니다.

**회귀식 표시**:

```
Y = [w] × X + [b]
```

**결과 테이블**:

| 항목 | 설명 |
|------|------|
| 에포크 | 실행된 총 에포크 수 |
| 훈련 MSE | 훈련 데이터에 대한 평균 제곱 오차 (실제 스케일) |
| 테스트 MSE | 테스트 데이터에 대한 평균 제곱 오차 (실제 스케일) |
| R² | 결정 계수 (0~1, 1에 가까울수록 좋음) |
| 상태 | ✅ 양호 (R²>0.8), ⚠️ 보통 (0.5~0.8), ❌ 부족 (<0.5) |

**테스트 MSE 하이라이트**: 분할이 수행된 경우, 테스트 MSE가 별도 강조 표시됩니다.

### 8.2 예측 입력

좌측 사이드바에 **🎯 예측 입력** 패널이 나타납니다.

| UI 요소 | 설명 |
|---------|------|
| X값 슬라이더 | 데이터의 X 범위 내에서 조절 |
| 현재값 표시 | 슬라이더 아래에 최소/현재/최대 표시 |

### 8.3 예측 과정 표시

우측 사이드바에 **🎯 예측 결과** 패널이 나타납니다.

```
절편(b)          →  [b값]
+ 기울기 기여     →  ±[w×X 값]
─────────────────────────
= 예측 Y         →  [결과값]
```

수식도 함께 표시됩니다: `Y = [w] × X + [b]`

### 8.4 추론 차트

중앙 영역에 추론 전용 차트가 표시됩니다:
- 전체 데이터 포인트 (회색)
- 회귀선 (빨간)
- **예측점** (노란 하이라이트 원)
- **가이드선** (수직선: X→회귀선, 수평선: 회귀선→Y축)

슬라이더를 움직이면 예측점과 가이드선이 실시간으로 이동합니다.

## 13. 트러블슈팅

| # | 증상 | 원인 | 해결책 |
|:-:|------|------|--------|
| 1 | 손실 그래프가 요동친다 | 학습률이 너무 큼 | 학습률을 0.001~0.005로 낮추기 |
| 2 | 손실이 충분히 내려가지 않는다 | 에포크 부족 또는 학습률이 너무 작음 | 에포크 1,000 이상, 학습률 0.05로 시도 |
| 3 | R²가 0 근처이다 | 데이터에 선형 관계가 없음 | 다른 X/Y 조합 시도, 앤스컴 2(곡선) 확인 |
| 4 | 학습 버튼이 비활성화됨 | 필터로 데이터 3건 이하 | ↺ 필터 초기화 클릭 |

| 6 | 추론 모드에서 "--"만 보임 | 학습을 실행하지 않음 | 학습 모드에서 ⚡ 학습 실행 후 전환 |
