# 범용 선형 회귀 학습 시뮬레이터 개발일지

**문서 버전: v1.1.0.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
범용 선형 회귀 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 **선형 회귀(Linear Regression)**의 원리를 다양한 데이터셋으로 학습할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 기반 버전
사과 가격 예측 선형 회귀 시뮬레이터 v1.2.4.1

### 1.4 개발 원칙
**원본 구조 최대 유지**: 원본 v1.2.4.1의 CSS, 레이아웃, JavaScript 로직을 100% 유지하고, 필요한 부분만 수정

---

## 2. v1.1.0.0 변경 사항

### 2.1 프리셋 축 레이블 표시

**변경 전**: 프리셋 선택 시 X, Y로만 표시
**변경 후**: 프리셋별 실제 변수명 표시

| 프리셋 | X축 레이블 | Y축 레이블 |
|--------|-----------|-----------|
| 🏠 주택 | 면적(㎡) | 가격(만원) |
| 📝 시험 | 공부시간(h) | 점수 |
| 💰 급여 | 경력(년) | 연봉(만원) |
| 🎲 랜덤 | X | Y |

### 2.2 CSV 변수 선택 기능

**변경 전**: CSV 첫 두 열을 자동으로 X, Y로 사용
**변경 후**: 
- CSV 로드 시 모든 수치형 컬럼 자동 감지
- 드롭다운으로 독립변수(X)와 종속변수(Y) 선택 가능
- 선택 변경 시 즉시 차트 업데이트

### 2.3 내장 데이터셋 추가

| 데이터셋 | 설명 | 변수 |
|----------|------|------|
| 🐧 펭귄 | Palmer Penguins | 부리길이, 부리깊이, 날개길이, 체중 |
| 🌸 아이리스 | Iris 꽃 | 꽃받침길이/너비, 꽃잎길이/너비 |
| 🚗 자동차 | 제동거리 | 속도, 제동거리 |
| 🏘️ 보스턴 | Boston Housing | 방개수, 저소득층비율, 학생교사비율, 주택가격 |

### 2.4 직접 입력 UI 개선

**변경 전**: X, Y 입력 박스 가로 배치
**변경 후**: 
- 입력 박스 수직(위/아래) 배치
- 레이블(X:, Y:) 명시적 표시
