# 📊 선형 회귀 시뮬레이터 퀵가이드

**문서 버전: v1.0.0.0**

---

## 한눈에 보기

| 항목 | 내용 |
|------|------|
| **목적** | 선형 회귀로 Y = wX + b 관계 학습 |
| **입력** | X (독립 변수) |
| **출력** | Y (종속 변수) |
| **핵심 개념** | 기울기(w), 절편(b), 경사 하강법, R², 오차선 |

## 🚀 빠른 시작 (3단계)

### 1️⃣ 데이터 준비
```
[프리셋 탭] → 🏠 주택 클릭 → [✨ 데이터 생성]
```

### 2️⃣ 학습 실행
```
[⚡ 학습 실행] 클릭 → 결과 패널 확인
```

### 3️⃣ 예측하기
```
[🔮 추론 모드] → X값 슬라이더 조절 → 자동 예측
```

---

## 📁 데이터 입력 방법

### 프리셋 (권장)
| 버튼 | X 변수 | Y 변수 |
|------|--------|--------|
| 🏠 주택 | 면적(㎡) | 가격(만원) |
| 📝 시험 | 공부시간(h) | 점수 |
| 💰 급여 | 경력(년) | 연봉(만원) |
| 🎲 랜덤 | X | Y |

### CSV 업로드
```csv
면적,가격
50,800
70,1200
```
- 첫 행: 변수명 (헤더)
- 이후: 데이터 (콤마 구분)

### 직접 입력
- X, Y 값 입력 후 [+] 버튼
- 또는 **차트 클릭**으로 데이터 추가

## 📐 회귀식 해석

```
Y = 15.0 × X + 50.0
```

- **기울기 15.0**: X가 1 증가 → Y가 15 증가
- **절편 50.0**: X가 0일 때 Y의 값
