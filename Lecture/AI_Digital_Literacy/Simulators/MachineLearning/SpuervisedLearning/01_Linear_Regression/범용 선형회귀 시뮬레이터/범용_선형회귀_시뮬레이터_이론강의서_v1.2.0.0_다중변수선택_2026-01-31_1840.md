## 1. 도입

이 강의를 마치면 다음을 할 수 있습니다:

| 목표 | 내용 |
|------|------|
| **이해** | 선형 회귀의 개념과 원리를 설명할 수 있다 |
| **계산** | 회귀식 Y = wX + b로 예측값을 계산할 수 있다 |
| **해석** | 기울기와 절편의 의미를 해석할 수 있다 |
| **평가** | R² 점수로 모델의 성능을 평가할 수 있다 |
| **적용** | 다중 변수를 조합하여 예측 모델을 구성할 수 있다 |

### 1.2 선수 지식

| 필수 | 권장 |
|------|------|
| 좌표평면 (x, y) | 일차함수 y = ax + b |

### 1.3 왜 선형 회귀를 배우나요?

**선형 회귀(Linear Regression)**는 머신러닝의 가장 기본적인 알고리즘입니다.

#### 실생활 예시

| 상황 | X (원인) | Y (결과) |
|------|----------|----------|
| 🏠 부동산 | 집 면적 | 집 가격 |
| 📝 공부 | 공부 시간 | 시험 점수 |
| 💰 급여 | 경력 연수 | 연봉 |
| 🌡️ 날씨 | 온도 | 아이스크림 판매량 |

이처럼 **원인(X)**과 **결과(Y)** 사이의 관계를 수식으로 나타내고, 새로운 X값에 대한 Y값을 **예측**하는 것이 선형 회귀입니다.

## 2. 핵심 개념

### 2.1 선형 관계란?

두 변수 X와 Y 사이에 **직선 형태**의 관계가 있을 때 "선형 관계"라고 합니다.

```
Y
│          ●
│        ●
│      ●        ← 점들이 직선 형태로 분포
│    ●
│  ●
└────────────── X
```

### 2.2 회귀식: Y = wX + b

**회귀식(Regression Equation)**은 X와 Y의 관계를 수식으로 표현한 것입니다.

```
Y = w × X + b

Y : 예측하려는 값 (종속 변수)
X : 알고 있는 값 (독립 변수)
w : 기울기 (가중치, weight)
b : 절편 (편향, bias)
```

#### 예시: 주택 가격 예측

```
가격(만원) = 15 × 면적(㎡) + 500

면적 50㎡ → 가격 = 15 × 50 + 500 = 1,250만원
면적 80㎡ → 가격 = 15 × 80 + 500 = 1,700만원
```

### 2.3 기울기(w)의 의미

**기울기(w)**는 X가 1 증가할 때 Y가 얼마나 변하는지를 나타냅니다.

| 기울기 | 의미 | 그래프 |
|--------|------|--------|
| w > 0 | X↑ → Y↑ (양의 상관) | 오른쪽 위로 상승 |
| w < 0 | X↑ → Y↓ (음의 상관) | 오른쪽 아래로 하강 |
| w = 0 | X와 Y는 무관 | 수평선 |

```
w = 15의 의미:
면적이 1㎡ 늘어날 때마다 가격이 15만원 상승
```

### 2.4 절편(b)의 의미

**절편(b)**은 X가 0일 때 Y의 값입니다.

```
b = 500의 의미:
면적이 0㎡일 때의 기본 가격은 500만원
(실제로는 해석이 어려울 수 있음)
```

```
Y
│
│  ●
│  │ ← 절편(b): Y축과 만나는 점
│  │
└──┼────────── X
   0
```

## 3. 수학적 원리

### 3.1 손실 함수 (Loss Function)

**손실(Loss)**은 예측값과 실제값의 차이를 나타냅니다.

#### 오차(Error)

```
오차 = 실제값 - 예측값
     = yᵢ - (w × xᵢ + b)
```

#### 평균 제곱 오차 (MSE: Mean Squared Error)

```
MSE = (1/n) × Σ(오차)²
    = (1/n) × Σ(yᵢ - (w × xᵢ + b))²
```

**왜 제곱하나요?**

| 이유 | 설명 |
|------|------|
| 부호 제거 | 양수/음수 오차가 상쇄되는 것 방지 |
| 큰 오차 강조 | 오차가 클수록 더 크게 반영 |
| 미분 용이 | 최적화 계산이 편리함 |

### 3.2 경사 하강법 (Gradient Descent)

**경사 하강법**은 손실을 최소화하는 w와 b를 찾는 방법입니다.

#### 직관적 이해

산꼭대기에서 골짜기(최저점)로 내려가는 것과 같습니다:

```
손실
│\
│ \  ← 현재 위치
│  \ 
│   \    
│    \___  ← 최저점 (목표)
└─────────── w
```

1. 현재 위치에서 **기울기(경사)** 계산
2. 기울기의 **반대 방향**으로 이동
3. 최저점에 도달할 때까지 반복

#### 수식

```
w := w - α × (∂MSE/∂w)
b := b - α × (∂MSE/∂b)

α(알파): 학습률 (Learning Rate)
```

### 3.3 학습률 (Learning Rate)

**학습률(α)**은 한 번에 얼마나 이동할지 결정합니다.

| 학습률 | 특징 | 위험 |
|--------|------|------|
| 너무 작음 | 느리지만 안정적 | 학습이 오래 걸림 |
| 적절함 | 빠르고 안정적 | - |
| 너무 큼 | 빠름 | **발산** (최저점을 지나침) |

```
학습률이 너무 크면:

손실
│  /\    /\
│ /  \  /  \  ← 발산 (왔다갔다)
│/    \/    
└──────────── 에포크
```

### 3.4 에포크 (Epoch)

**에포크(Epoch)**는 전체 데이터를 한 번 학습하는 것입니다.

```
에포크 1: 모든 데이터로 w, b 업데이트
에포크 2: 모든 데이터로 w, b 업데이트
...
에포크 500: 모든 데이터로 w, b 업데이트
```

일반적으로 에포크가 늘어날수록 손실이 감소합니다.

## 4. 알고리즘

### 4.1 전체 흐름

```
┌─────────────────┐
│  1. 데이터 준비  │
└────────┬────────┘
         ↓
┌─────────────────┐
│  2. 모델 초기화  │  w, b를 임의의 값으로 설정
└────────┬────────┘
         ↓
┌─────────────────┐
│  3. 예측 수행    │  Ŷ = wX + b
└────────┬────────┘
         ↓
┌─────────────────┐
│  4. 손실 계산    │  MSE = Σ(Y - Ŷ)² / n
└────────┬────────┘
         ↓
┌─────────────────┐
│  5. 기울기 계산  │  ∂MSE/∂w, ∂MSE/∂b
└────────┬────────┘
         ↓
┌─────────────────────────────┐
│  6. 가중치 업데이트          │
│  w := w - α × ∂MSE/∂w       │
│  b := b - α × ∂MSE/∂b       │
└────────┬────────────────────┘
         ↓
    ┌────┴────┐
    │에포크 끝?│───No──→ 3번으로 돌아감
    └────┬────┘
         │Yes
         ↓
┌─────────────────┐
│  7. 학습 완료    │
└─────────────────┘
```

### 4.2 의사 코드 (Pseudo Code)

```
함수 선형회귀학습(데이터, 학습률, 에포크수):
    
    # 1. 초기화
    w = 랜덤값
    b = 0
    
    # 2. 학습 반복
    반복 (에포크 = 1 부터 에포크수 까지):
        
        총손실 = 0
        w기울기 = 0
        b기울기 = 0
        
        # 3. 각 데이터에 대해
        각 (x, y) in 데이터:
            예측값 = w × x + b
            오차 = 예측값 - y
            
            w기울기 += 오차 × x
            b기울기 += 오차
            총손실 += 오차²
        
        # 4. 평균 기울기 계산
        w기울기 = w기울기 / 데이터개수
        b기울기 = b기울기 / 데이터개수
        
        # 5. 가중치 업데이트
        w = w - 학습률 × w기울기
        b = b - 학습률 × b기울기
        
        # 6. 손실 기록
        평균손실 = 총손실 / 데이터개수
    
    반환 w, b
```

### 4.3 예측 (추론)

학습이 완료되면, 새로운 X값에 대해 Y를 예측합니다:

```
함수 예측(X, w, b):
    반환 w × X + b

예시:
- 학습된 모델: Y = 15 × X + 500
- 새로운 X: 60 (면적 60㎡)
- 예측 Y: 15 × 60 + 500 = 1,400 (만원)
```

## 5. 시각화 해석

### 5.1 산점도와 회귀선

```
가격(Y)
│         ● 
│    ●     ●     
│  ●    ●    ──────  ← 회귀선
│    ●  ●   ●
│  ●  ●
│ ●
└──────────────────── 면적(X)
```

| 요소 | 의미 |
|------|------|
| 점 (●) | 실제 데이터 |
| 직선 (──) | 학습된 회귀선 |
| 점과 선의 거리 | 오차 (잔차) |

### 5.2 손실 그래프

```
손실(MSE)
│\
│ \
│  \
│   \___________  ← 수렴 (학습 완료)
│
└──────────────── 에포크
```

| 패턴 | 의미 | 조치 |
|------|------|------|
| 계속 감소 | 정상 학습 중 | 계속 진행 |
| 평평해짐 | 수렴 완료 | 학습 중단 가능 |
| 증가함 | 발산 (문제!) | 학습률 낮추기 |
| 진동함 | 학습률 과다 | 학습률 낮추기 |

### 5.3 오차선 (잔차)

```
가격(Y)
│         ● 
│    ●    ┆●     
│  ●    ●┆───────  회귀선
│    ● ┆●   ●
│  ●  ●
│ ●
└──────────────────── 면적(X)

┆ = 오차선 (데이터점과 회귀선 사이)
```

**오차선 색상**:
- 🟢 녹색: 오차가 작음 (예측 정확)
- 🔴 빨강: 오차가 큼 (예측 부정확)

### 5.4 R² 점수 (결정 계수)

**R²(R-squared)**는 모델이 데이터를 얼마나 잘 설명하는지 나타냅니다.

```
R² = 1 - (SS_res / SS_tot)

SS_res = Σ(실제값 - 예측값)²  (잔차 제곱합)
SS_tot = Σ(실제값 - 평균값)²  (총 변동)
```

| R² 값 | 해석 |
|-------|------|
| 1.0 | 완벽한 예측 (모든 점이 선 위에) |
| 0.8~1.0 | 매우 좋음 |
| 0.5~0.8 | 보통 |
| 0~0.5 | 개선 필요 |
| < 0 | 평균보다 못한 예측 |

## 6. 다중 변수 처리

### 6.1 다중 변수란?

하나의 Y를 예측하기 위해 **여러 개의 X 변수**를 사용하는 것입니다.

```
단일 변수: 집 가격 = w × 면적 + b

다중 변수: 집 가격 = w₁×면적 + w₂×방개수 + w₃×층수 + b
```

### 6.2 시뮬레이터의 다중 변수 처리

이 시뮬레이터는 여러 X 변수를 **하나의 조합점수(X)**로 통합합니다:

```
조합점수(X) = w₁×정규화(X₁) + w₂×정규화(X₂) + ... + wₙ×정규화(Xₙ)
```

#### 정규화 (Normalization)

단위가 다른 변수들을 0~100 범위로 맞춥니다:

```
정규화(Xᵢ) = (Xᵢ - min) / (max - min) × 100
```

**예시: 펭귄 데이터**

| 변수 | 원본 범위 | 정규화 후 |
|------|----------|----------|
| 부리길이(mm) | 34~46 | 0~100 |
| 체중(g) | 3200~4675 | 0~100 |

**왜 정규화하나요?**

정규화하지 않으면 단위가 큰 변수가 지배적이 됩니다:
- 부리길이: 40mm
- 체중: 4000g ← 숫자가 커서 영향력 과대

### 6.3 가중치의 의미

**가중치(w₁, w₂, ...)**는 각 변수의 **상대적 중요도**입니다.

```
조합점수 = 70%×부리길이 + 30%×부리깊이

→ 부리길이가 부리깊이보다 2배 이상 중요하게 취급됨
```

**가중치 조절의 효과**:
- 가중치를 높이면: 해당 변수의 영향력 증가
- 가중치를 낮추면: 해당 변수의 영향력 감소
- 모든 가중치의 합은 항상 1 (100%)

### 6.4 원본 사과 시뮬레이터와의 연결

원본 **사과 가격 예측 시뮬레이터**의 "품질 점수"가 바로 이 개념입니다:

```
품질점수 = 50%×당도점수 + 50%×무게점수

당도점수 = (당도 - 10) / 8 × 50
무게점수 = (무게 - 150) / 200 × 50
```

범용 시뮬레이터에서는 이 개념을 **일반화**하여 어떤 변수 조합에도 적용할 수 있게 했습니다.

## 7. 핵심 용어 정리

| 용어 | 영어 | 정의 |
|------|------|------|
| **선형 회귀** | Linear Regression | X와 Y 사이의 직선 관계를 찾는 알고리즘 |
| **회귀식** | Regression Equation | Y = wX + b 형태의 예측 공식 |
| **기울기** | Slope, Weight (w) | X가 1 증가할 때 Y의 변화량 |
| **절편** | Intercept, Bias (b) | X=0일 때 Y의 값 |
| **손실** | Loss | 예측값과 실제값의 차이 |
| **MSE** | Mean Squared Error | 평균 제곱 오차 |
| **경사 하강법** | Gradient Descent | 손실을 최소화하는 최적화 방법 |
| **학습률** | Learning Rate (α) | 한 번에 이동하는 크기 |
| **에포크** | Epoch | 전체 데이터를 한 번 학습하는 것 |
| **R²** | R-squared | 모델의 설명력 (결정 계수) |
| **오차선** | Residual Line | 데이터점과 회귀선 사이의 거리 |
| **정규화** | Normalization | 변수를 동일한 범위로 변환 |
| **가중치** | Weight | 변수의 상대적 중요도 |
| **독립변수** | Independent Variable (X) | 원인, 입력 변수 |
| **종속변수** | Dependent Variable (Y) | 결과, 출력 변수 |
