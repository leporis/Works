## 한눈에 보기

| 항목 | 내용 |
|------|------|
| **목적** | 선형 회귀로 Y = wX + b 관계 학습 |

### 3️⃣ 예측하기

## 📁 데이터 입력 방법 (3가지)

### [프리셋] 탭 - 권장
| 버튼 | X 변수 | Y 변수 |
|------|--------|--------|
| 🏠 주택 | 면적(㎡) | 가격(만원) |
| 📝 시험 | 공부시간(h) | 점수 |
| 💰 급여 | 경력(년) | 연봉(만원) |
| 🎲 랜덤 | X | Y |

### [CSV] 탭 - 다중 변수 지원
```csv
면적,방개수,가격
50,2,800
70,3,1200
```

#### 내장 데이터셋 (v1.1.0+)
| 버튼 | 데이터셋 | 변수 |
|------|----------|------|
| 🐧 펭귄 | 부리길이, 부리깊이, 날개길이, 체중 |
| 🌸 아이리스 | 꽃받침길이/너비, 꽃잎길이/너비 |
| 🚗 자동차 | 속도, 제동거리 |
| 🏘️ 보스턴 | 방개수, 저소득층비율, 학생교사비율, 주택가격 |

### [직접] 탭
- X, Y 값 입력 후 [+] 버튼
- 또는 **차트 클릭**으로 데이터 추가

## 🆕 다중 변수 선택 (v1.2.0 신규)

### 사용 방법
1. [CSV 탭] → 내장 데이터셋 로드 (예: 🐧 펭귄)
2. **독립변수(X)**: 체크박스로 **2개 이상** 선택
3. **종속변수(Y)**: 드롭다운에서 1개 선택

```
📊 독립변수(X) 선택
┌─────────────────────────┐
│ ☑ 부리길이(mm)          │
│ ☑ 부리깊이(mm)          │
│ ☐ 날개길이(mm)          │
│ ☐ 체중(g)  ← 종속변수   │
└─────────────────────────┘
```

### 가중치 슬라이더 (2개+ 선택 시)
```
⚖️ 변수별 가중치
┌─────────────────────────┐
│ 부리길이    [====] 50%  │
│ 부리깊이    [====] 50%  │
├─────────────────────────┤
│ X = 50%×부리 + 50%×부리 │
│ [🔄 가중치 적용]        │
└─────────────────────────┘
```

**가중치 조정**:
- 슬라이더 드래그 → 나머지 자동 조정
- 가중치 합계 = 항상 100%
- [🔄 가중치 적용] 버튼으로 반영

## 📐 회귀식 해석

```
Y = 15.0 × X + 50.0
```

| 요소 | 의미 |
|------|------|
| **기울기 15.0** | X가 1 증가 → Y가 15 증가 |
| **절편 50.0** | X가 0일 때 Y의 기본값 |

## ❓ FAQ

| 질문 | 해결 |
|------|------|
| 손실이 발산해요 | 학습률 0.01 이하로 낮추기 |
| 차트 클릭 안 돼요 | [직접] 탭에서만 작동 |
| CSV 오류 | UTF-8 인코딩, 콤마 구분 확인 |
| 가중치 슬라이더 안 보여요 | 독립변수 2개 이상 선택 |
| 추론이 안 돼요 | 학습 먼저 실행 |

## 🔑 핵심 용어

| 용어 | 의미 |
|------|------|
| **선형 회귀** | X와 Y의 직선 관계를 찾는 알고리즘 |
| **기울기 (w)** | X가 1 증가할 때 Y의 변화량 |
| **절편 (b)** | X=0일 때 Y의 기본값 |
| **MSE** | 평균 제곱 오차 (손실 함수) |
| **R²** | 모델의 설명력 (1에 가까울수록 좋음) |
| **학습률** | 가중치 업데이트 크기 |
| **에포크** | 전체 데이터 학습 1회 |
| **가중치** | 다중 변수의 상대적 중요도 |
| **정규화** | 변수를 0~100 범위로 통일 |
