## 📁 데이터 입력 방법

### 프리셋 (초보자 권장)

| 버튼 | X 변수 | Y 변수 |
|------|--------|--------|
| 🏠 주택 | 면적(㎡) | 가격(만원) |
| 📝 시험 | 공부시간(h) | 점수 |
| 💰 급여 | 경력(년) | 연봉(만원) |
| 🎲 랜덤 | X | Y |

### CSV 탭 - 내장 데이터셋 (v1.1.0+)

| 버튼 | 데이터셋 | 변수 |
|------|----------|------|
| 🐧 펭귄 | Palmer Penguins | 부리길이, 부리깊이, 날개길이, 체중 |
| 🌸 아이리스 | Iris | 꽃받침길이/너비, 꽃잎길이/너비 |
| 🚗 자동차 | Cars | 속도, 제동거리 |
| 🏘️ 보스턴 | Boston Housing | 방개수, 저소득층비율, 주택가격 |

### CSV 업로드
```csv
면적,가격
50,800
70,1200
```
- 첫 행: 변수명 (헤더)
- 이후: 데이터 (콤마 구분)

### 직접 입력
- X, Y 값 입력 후 [+] 버튼
- 또는 **차트 클릭**으로 데이터 추가 (직접 탭에서만)

## 🔢 다중 변수 선택 (v1.2.0 신규)

### 독립변수 선택
CSV 또는 내장 데이터셋 로드 후 **체크박스**로 선택:

```
📊 독립변수(X) 선택
┌─────────────────────────┐
│ ☑ 부리길이(mm)          │
│ ☑ 부리깊이(mm)          │
│ ☐ 날개길이(mm)          │
│ ☐ 체중(g)  ← 종속변수   │
└─────────────────────────┘
```

### 가중치 슬라이더 (2개 이상 선택 시)

```
⚖️ 변수별 가중치
┌─────────────────────────┐
│ 부리길이    [====] 50%  │
│ 부리깊이    [====] 50%  │
├─────────────────────────┤
│ X = 50%×부리 + 50%×부리 │
│ [🔄 가중치 적용]        │
└─────────────────────────┘
```

| 포인트 | 설명 |
|--------|------|
| 가중치 합 | 항상 100% |
| 조정 방법 | 한 변수 변경 → 나머지 자동 조정 |
| 적용 | [🔄 가중치 적용] 버튼 클릭 |

### 조합점수 계산 공식
```
X = w₁×정규화(X₁) + w₂×정규화(X₂) + ...

정규화 = (값 - 최소) / (최대 - 최소) × 100
```

## 📐 회귀식 해석

```
Y = 15.0 × X + 50.0
```

- **기울기 15.0**: X가 1 증가 → Y가 15 증가
- **절편 50.0**: X가 0일 때 Y의 값

## 🔑 핵심 용어

| 용어 | 의미 |
|------|------|
| **기울기 (w)** | X가 1 증가할 때 Y 변화량 |
| **절편 (b)** | X=0일 때 Y값 |
| **손실 (MSE)** | 예측과 실제의 차이 제곱 평균 |
| **R²** | 모델 적합도 (1에 가까울수록 좋음) |
| **정규화** | 변수를 0~100 범위로 변환 |
| **가중치** | 변수의 상대적 중요도 |

## ❓ FAQ

**Q: 손실이 발산해요**  
→ 학습률을 0.01 이하로 낮추세요

**Q: 차트 클릭이 안 돼요**  
→ [직접] 탭에서만 작동합니다

**Q: CSV 업로드 오류**  
→ UTF-8 인코딩, 콤마 구분 확인

**Q: 가중치 슬라이더가 안 보여요**  
→ CSV 탭에서 독립변수 2개 이상 선택하세요

**Q: R²가 음수예요**  
→ 데이터 늘리거나 에포크 증가
