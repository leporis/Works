## 1. 프로젝트 개요

### 1.1 프로젝트명
경사하강법(Gradient Descent) 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 **경사하강법**의 7단계 학습 과정을 직접 체험하며 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표

| 목표 | 설명 |
|------|------|
| 학습 과정 이해 | 데이터 준비 → 가중치 초기화 → 예측 → 손실 → 기울기 → 업데이트 → 완료 |
| 경사하강법 체험 | 손실 함수의 기울기를 따라 최적값을 찾아가는 과정 시각화 |
| 하이퍼파라미터 영향 | 학습률이 학습 속도와 수렴에 미치는 영향 체험 |
| 손실 곡면 이해 | w, b에 따른 MSE 변화를 등고선으로 시각화 |

### 1.4 개발 배경
- 사과 가격 예측 시뮬레이터(v1.2.4.1)의 디자인 패턴 기반
- 첨부된 "선형 회귀 학습 과정" 흐름도 이미지 참고
- **v1.1.0.0**: ML시뮬레이터_디자인설계가이드_v1.0.0.0 완전 준수

## 3. 시스템 설계

### 3.1 7단계 학습 과정

| 단계 | 이름 | 색상 | 설명 |
|:----:|------|------|------|
| 1 | 데이터 준비 | 🟢 녹색 | (X, Y) 데이터셋 생성 |
| 2 | 가중치 초기화 | 🔵 파랑 | w=0, b=0 설정 |
| 3 | 예측 계산 | 🟣 보라 | ŷ = w×x + b |
| 4 | 손실 계산 | 🔴 빨강 | MSE = Σ(y-ŷ)²/n |
| 5 | 기울기 계산 | 🔴 빨강 | ∇w = ∂MSE/∂w, ∇b = ∂MSE/∂b |
| 6 | 가중치 업데이트 | 🟠 주황 | w = w - η×∇w, b = b - η×∇b |
| 7 | 학습 완료 | 🟢 녹색 | 최적의 w, b 획득 |

**반복 루프**: 3~6단계를 에포크 횟수만큼 반복

### 3.2 시뮬레이션 설정

| 항목 | 범위 | 기본값 |
|------|------|--------|
| 데이터 개수 | 5 ~ 100 | 30개 |
| 노이즈 수준 | 5% / 10% / 20% | 10% |
| 학습률 (η) | 0.001 ~ 0.1 | 0.01 |
| 에포크 | 100 ~ 2000 | 500회 |

### 5.2 핵심 알고리즘

#### 손실 함수 (MSE)
```javascript
function calcMSE(w, b) {
    let total = 0;
    data.forEach(d => {
        const pred = w * d.x + b;
        total += (d.y - pred) ** 2;
    });
    return total / data.length;
}
```

#### 기울기 계산
```javascript
function calculateGradients() {
    let gradW = 0, gradB = 0;
    const n = data.length;
    
    data.forEach(d => {
        const pred = model.w * d.x + model.b;
        const error = pred - d.y;
        gradW += error * d.x;
        gradB += error;
    });
    
    model.gradW = (2 / n) * gradW;
    model.gradB = (2 / n) * gradB;
}
```

#### 가중치 업데이트
```javascript
function updateWeights() {
    const lr = learningRate;
    model.w -= lr * model.gradW;
    model.b -= lr * model.gradB;
}
```

### 5.3 정규화

```javascript
function normalize(arr) {
    const mean = arr.reduce((a,b) => a+b, 0) / arr.length;
    const std = Math.sqrt(arr.reduce((a,b) => a + (b-mean)**2, 0) / arr.length) || 1;
    return { mean, std };
}
```

### 5.4 R² 점수 계산

```javascript
function calculateR2() {
    const predictions = data.map(d => model.wReal * d.x + model.bReal);
    const actuals = data.map(d => d.y);
    const meanActual = mean(actuals);

    const ssRes = actuals.reduce((sum, a, i) => sum + (a - predictions[i]) ** 2, 0);
    const ssTot = actuals.reduce((sum, a) => sum + (a - meanActual) ** 2, 0);

    return 1 - ssRes / ssTot;
}
```
