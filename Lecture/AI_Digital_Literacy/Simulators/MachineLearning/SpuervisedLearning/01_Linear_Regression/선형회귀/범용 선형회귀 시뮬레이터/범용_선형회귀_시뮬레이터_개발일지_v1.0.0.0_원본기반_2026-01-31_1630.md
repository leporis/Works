## 2. 원본(v1.2.4.1) 대비 변경사항

### 2.1 ✅ 유지된 요소 (변경 없음)

| 구분 | 내용 |
|------|------|
| **레이아웃** | 240px - 1fr - 260px 3열 구조 |
| **CSS 클래스** | .panel, .btn, .input-group, .chart-container 등 전체 |
| **학습 알고리즘** | Z-score 정규화 + 경사 하강법 + 역정규화 |
| **손실 그래프** | Chart.js 기반 MSE 시각화 |
| **산점도** | Canvas API, 점 크기 1.5px/1.25px (원본 그대로) |
| **오차선 분석** | 토글, 색상 변화(녹→적), 평균/최대/MSE 통계 |
| **학습 과정 재생** | 100프레임 히스토리, 10단계 속도 조절 |
| **추론 시각화** | 수직선, 수평선, 예측점(노란원 12px) |
| **학습 결과 패널** | 차트 아래 3열 그리드 (결과/오차/재생) |
| **반응형** | 1200px, 1024px 브레이크포인트 |

### 2.2 🔄 변경된 요소 (최소 수정)

## 3. 프리셋 데이터셋

| 프리셋 | X 변수 | Y 변수 | X 범위 | 기본 공식 | 노이즈 기준 |
|--------|--------|--------|--------|-----------|------------|
| 🏠 주택 | 면적(㎡) | 가격(만원) | 20~150 | Y = 15X + 50 | 100 |
| 📝 시험 | 공부시간(h) | 점수 | 0~10 | Y = 7X + 30 | 5 |
| 💰 급여 | 경력(년) | 연봉(만원) | 0~20 | Y = 200X + 2500 | 300 |
| 🎲 랜덤 | X | Y | 0~100 | Y = 2X + 10 | 10 |
