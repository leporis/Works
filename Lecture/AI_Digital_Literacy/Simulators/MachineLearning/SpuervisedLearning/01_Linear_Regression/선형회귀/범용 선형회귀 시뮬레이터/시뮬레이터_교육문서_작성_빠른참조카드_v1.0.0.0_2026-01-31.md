# 시뮬레이터 교육문서 작성 - 빠른 참조 카드

**v1.0.0.0**

---

## 🎯 작업 절차 (4단계)

```
1️⃣ 시뮬레이터 분석 → 2️⃣ 문서 작성 → 3️⃣ 그림 제작 → 4️⃣ 검토/완성
```

---

## 📄 문서 4종 세트

| 문서 | 핵심 질문 | 필수 섹션 |
|------|----------|----------|
| **사용설명서** | How? | 시작하기, 화면 구성, 기능별 사용법, FAQ |
| **활용가이드** | Why? | 학습 목표, 수업 설계, 탐구 활동 3개+, 평가 루브릭 |
| **이론강의서** | What? | 개념 정의, 수학 공식, 알고리즘 단계, 용어 정리 |
| **수업지도안** | When? | 학습 목표(지식/기능/태도), 교수학습 과정표, 평가 계획 |

## 🎨 그림 제작 규칙

### 파일명
```
[문서명]_[번호]_[내용].svg

예: 이론강의서_01_품질점수계산.svg
예: 사용설명서_02_UI구성_학습모드.svg
```

### SVG 설정
```
viewBox: 800×400~500
폰트: Noto Sans KR, JetBrains Mono
색상: 시뮬레이터 테마 그대로 사용
효과: 그라데이션 배경 + 드롭쉐도우
```

## 💬 Claude 프롬프트

### 분석 요청
```
첨부한 [시뮬레이터] HTML을 분석해주세요.
(입력/출력 변수, 알고리즘, UI, 색상 테마)
```

### 문서 작성
```
[시뮬레이터]의 [문서유형]을 작성해주세요.
버전: vX.Y.Z.W, 대상: [대상자]
문서버전관리가이드 적용
```

### 그림 제작
```
[시뮬레이터]의 [문서유형]용 그림을 제작해주세요.
파일명: [문서명]_[번호]_[내용].svg
시뮬레이터 색상 테마 적용
```
