## 3. 시뮬레이터 핵심 기능 (v1.2.0.0)

문서 검토 시 아래 기능들이 정확하게 설명되어 있는지 확인 필요:

### 3.1 데이터 입력 방식

| 방식 | 설명 | UI 요소 |
|------|------|---------|
| **프리셋** | 4가지 시나리오 선택 | 🏠 주택, 📝 성적, 💰 매출, 🎲 랜덤 |
| **CSV** | 파일 업로드 또는 내장 데이터셋 | 🐧 펭귄, 🌸 아이리스, 🚗 자동차, 🏘️ 보스턴 |
| **직접** | 수동 입력 또는 차트 클릭 | 입력 필드, 클릭 추가 |

### 3.2 다중 변수 기능 (v1.2.0.0 신규)

| 기능 | 설명 |
|------|------|
| **변수 선택** | 체크박스로 독립변수 다중 선택 |
| **가중치 조절** | 슬라이더로 변수별 비율 조정 |
| **정규화** | 0~100 스케일로 변환 |
| **조합점수** | 가중 평균으로 X값 계산 |

### 3.3 학습 모드

| 요소 | 설명 |
|------|------|
| 하이퍼파라미터 | 학습률 (0.001~0.1), 에포크 (100~2000) |
| 시각화 | 손실 그래프, 산점도+회귀선 |
| 결과 | R², 회귀식, 오차선 표시 |
| 재생 | 학습 과정 애니메이션 |

### 3.4 추론 모드

| 요소 | 설명 |
|------|------|
| 입력 | X값 슬라이더 또는 다중 변수 조합 |
| 시각화 | 수직선→회귀선→수평선→예측점 |
| 결과 | 예측값, 계산 과정 표시 |

## 4. 검토 체크리스트

### 4.1 정합성 검토

| # | 검토 항목 | 확인 사항 |
|:-:|----------|----------|
| 1 | **버전 일치** | 모든 문서가 v1.2.0.0 기준인지 |
| 2 | **기능 설명** | 시뮬레이터 실제 기능과 일치하는지 |
| 3 | **UI 요소** | 버튼명, 레이블, 아이콘이 정확한지 |
| 4 | **용어 통일** | 문서 간 용어가 일관되는지 |
| 5 | **수식 정확성** | Y = wX + b, MSE, R² 등 수식이 정확한지 |

### 4.2 용어 통일 확인

| 용어 | 올바른 표현 | 잘못된 표현 |
|------|------------|------------|
| 입력 변수 | 독립변수(X) | 입력값, 특성 |
| 출력 변수 | 종속변수(Y) | 출력값, 타겟 |
| 가중치 | w (기울기) | weight, 계수 |
| 편향 | b (절편) | bias, 상수항 |
| 조합점수 | X (가중 평균) | 품질점수 (사과 시뮬레이터 용어) |

### 4.3 문서별 검토 포인트

| 문서 | 중점 검토 사항 |
|------|---------------|
| **퀵가이드** | 3단계 빠른 시작이 실제 조작과 일치하는지 |
| **사용설명서** | UI 구성 설명이 실제 화면과 일치하는지 |
| **활용가이드** | 탐구 활동 절차가 시뮬레이터로 수행 가능한지 |
| **이론강의서** | 개념 설명이 시뮬레이터 동작과 연결되는지 |
| **수업지도안** | 교수학습 활동이 현실적인 시간 배분인지 |

### 4.4 교차 검토

| 검토 쌍 | 확인 사항 |
|---------|----------|
| 퀵가이드 ↔ 사용설명서 | 핵심 내용 일치, 상세도 차이 |
| 활용가이드 ↔ 수업지도안 | 탐구 활동과 교수학습 과정 연계 |
| 이론강의서 ↔ 시뮬레이터 | 개념 설명과 UI 동작 연결 |
