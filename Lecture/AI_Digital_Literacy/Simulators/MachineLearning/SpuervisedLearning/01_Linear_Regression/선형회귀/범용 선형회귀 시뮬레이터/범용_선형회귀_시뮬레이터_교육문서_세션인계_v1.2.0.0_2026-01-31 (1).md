## 6. 이미지 제작 가이드

### 6.1 SVG 기본 설정

```xml
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 800 [높이]">
  <defs>
    <!-- 배경 그라데이션 -->
    <linearGradient id="bgGrad" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%" style="stop-color:#FFFBF5"/>
      <stop offset="100%" style="stop-color:#FFF3E0"/>
    </linearGradient>
    
    <!-- 그림자 효과 -->
    <filter id="shadow" x="-20%" y="-20%" width="140%" height="140%">
      <feDropShadow dx="2" dy="3" stdDeviation="3" flood-opacity="0.12"/>
    </filter>
  </defs>
  
  <!-- 배경 -->
  <rect width="800" height="[높이]" fill="url(#bgGrad)"/>
  
  <!-- 내용 -->
  ...
</svg>
```

### 6.2 파일명 규칙

```
[문서명]_[번호]_[내용].svg

예: 이론강의서_01_선형관계.svg
예: 사용설명서_05_다중변수선택.svg
```

### 6.3 권장 크기

| 용도 | viewBox | 비고 |
|------|---------|------|
| 개념도/다이어그램 | 800×400 | 가로형 |
| 흐름도 | 800×500 | 세로 여유 |
| UI 스크린샷 | 800×450 | 화면 비율 |
| 비교표 | 800×400 | 3열 비교 |

### 6.4 폰트

```css
font-family: 'Noto Sans KR', sans-serif;  /* 본문 */
font-family: 'JetBrains Mono', monospace; /* 코드/숫자 */
```
