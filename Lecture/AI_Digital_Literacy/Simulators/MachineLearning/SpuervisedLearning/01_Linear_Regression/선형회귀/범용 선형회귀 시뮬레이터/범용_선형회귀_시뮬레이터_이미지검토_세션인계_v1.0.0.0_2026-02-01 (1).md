## 4. 검토 체크리스트

### 4.1 정합성 검토

| 검토 항목 | 확인 사항 |
|----------|----------|
| 색상 일치 | 시뮬레이터 CSS와 동일한 색상 사용 여부 |
| 레이아웃 | 문서에 명시된 구조와 일치 여부 |
| UI 요소 | 프리셋 아이콘(🏠📝💰🎲), 데이터셋 아이콘(🐧🌸🚗🏘️) 등 |
| 용어 | 문서와 동일한 용어 사용 (품질→조합점수 등) |
| 수식 | Y = wX + b 형식 일관성 |

### 4.2 품질 검토

| 검토 항목 | 확인 사항 |
|----------|----------|
| 텍스트 겹침 | 텍스트가 다른 요소와 겹치지 않는지 |
| 텍스트 잘림 | 긴 텍스트가 영역 밖으로 나가지 않는지 |
| 색상 대비 | 텍스트 가독성 확보 |
| 요소 배치 | 균형 잡힌 레이아웃 |

```
첨부한 인계 문서를 참고하여 이미지 검토를 진행합니다.

1. 색상이 시뮬레이터와 일치하는지

4. 용어와 수식이 정확한지

문제가 발견되면 수정해주세요.
```
