## 2. 수업 설계

### 2.1 차시 구성 (총 90분, 2차시)

| 차시 | 시간 | 주제 | 활동 |
|:----:|:----:|------|------|
| 1 | 45분 | 선형 회귀 기초 | 개념 이해 + 모델 학습 |
| 2 | 45분 | 다중 변수와 예측 | 가중치 조절 + 예측 실습 |

### 2.2 1차시 상세 (45분) - 선형 회귀 기초

#### 학습 목표
- 선형 회귀의 개념(기울기, 절편)을 설명할 수 있다
- 시뮬레이터를 사용하여 모델을 학습시킬 수 있다
- 손실 그래프를 해석할 수 있다

#### 교수학습 과정

| 단계 | 시간 | 교수 활동 | 학습 활동 |
|------|:----:|----------|----------|
| **도입** | 5분 | "키와 발크기의 관계는?" 질문 | 경험 공유, 예측 |
| **전개1** | 10분 | Y = wX + b 개념 설명 | 개념 노트 작성 |
| **전개2** | 20분 | 시뮬레이터 시연 | [탐구 활동 1] 실습 |
| **전개3** | 5분 | 오차선 기능 소개 | 오차의 의미 토론 |
| **정리** | 5분 | 핵심 개념 정리 | 학습 결과 공유 |

#### 준비물
- 컴퓨터/태블릿 (학생 1인 1기기)
- 시뮬레이터 HTML 파일
- 학습지

### 2.3 2차시 상세 (45분) - 다중 변수와 예측

#### 학습 목표
- 여러 변수를 조합하여 예측 모델을 만들 수 있다
- 가중치의 의미를 이해하고 조절할 수 있다
- 학습된 모델로 새로운 값을 예측할 수 있다

#### 교수학습 과정

| 단계 | 시간 | 교수 활동 | 학습 활동 |
|------|:----:|----------|----------|
| **도입** | 5분 | 1차시 복습, 다중 변수 소개 | 복습 퀴즈 |
| **전개1** | 15분 | 다중 변수 선택 방법 시연 | [탐구 활동 2] 실습 |
| **전개2** | 15분 | 가중치 조절 시연 | [탐구 활동 3] 실습 |
| **정리** | 10분 | 추론 모드 활용 | 예측 결과 발표 |

#### 준비물
- 1차시와 동일
- 내장 데이터셋 (펭귄, 아이리스 등)

## 3. 탐구 활동

### 3.1 탐구 활동 1: 학습률 실험

#### 목표
학습률이 모델 학습에 미치는 영향을 탐구한다

#### 절차

| 단계 | 활동 |
|:----:|------|
| 1 | 🏠 주택 프리셋으로 데이터 30개 생성 |
| 2 | 학습률 **0.001**로 설정 후 학습 실행 |
| 3 | 손실 그래프, R² 기록 |
| 4 | 초기화 후 학습률 **0.01**로 재실험 |
| 5 | 초기화 후 학습률 **0.1**로 재실험 |
| 6 | 결과 비교 및 분석 |

#### 기록표

| 학습률 | 손실 그래프 모양 | 최종 손실 | R² | 관찰 |
|--------|-----------------|----------|-----|------|
| 0.001 | | | | |
| 0.01 | | | | |
| 0.1 | | | | |

#### 토론 질문
1. 학습률이 너무 작으면 어떤 문제가 생기나요?
2. 학습률이 너무 크면 어떤 현상이 나타나나요?
3. "적절한" 학습률이란 무엇일까요?

#### 예상 결과
- **0.001**: 천천히 수렴, 안정적 (에포크 부족 가능)
- **0.01**: 적당한 속도로 수렴 (권장)
- **0.1**: 빠르게 수렴하거나 발산 (불안정)

### 3.2 탐구 활동 2: 다중 변수 선택

#### 목표
여러 변수를 조합하여 예측 정확도를 향상시킬 수 있는지 탐구한다

#### 절차

| 단계 | 활동 |
|:----:|------|
| 1 | [CSV 탭] → [🐧 펭귄] 내장 데이터셋 로드 |
| 2 | 종속변수(Y): **체중(g)** 선택 |
| 3 | 독립변수: **부리길이** 1개만 선택 → 학습 → R² 기록 |
| 4 | 독립변수: **부리길이 + 부리깊이** 2개 선택 → 학습 → R² 기록 |
| 5 | 독립변수: **부리길이 + 날개길이** 2개 선택 → 학습 → R² 기록 |
| 6 | 어떤 조합이 가장 좋은지 분석 |

#### 기록표

| 선택한 변수 | R² | 관찰 |
|------------|-----|------|
| 부리길이 | | |
| 부리길이 + 부리깊이 | | |
| 부리길이 + 날개길이 | | |

#### 토론 질문
1. 변수를 많이 선택할수록 항상 좋은가요?
2. 어떤 변수 조합이 가장 효과적이었나요? 왜 그럴까요?
3. 실제 펭귄의 체중과 관련이 있을 것 같은 변수는 무엇인가요?

#### 예상 결과
- 날개길이가 체중과 상관관계가 높아 R²가 높게 나올 가능성
- 무관한 변수 추가 시 오히려 성능 저하 가능

### 3.3 탐구 활동 3: 가중치 조절 실험

#### 목표
가중치의 의미를 이해하고, 조절에 따른 예측 변화를 관찰한다

#### 절차

| 단계 | 활동 |
|:----:|------|
| 1 | [🐧 펭귄] 데이터셋 로드 |
| 2 | 독립변수: **부리길이 + 부리깊이** 선택 |
| 3 | 가중치 **50% : 50%** (기본) → 학습 → R² 기록 |
| 4 | 가중치 **70% : 30%** (부리길이 높임) → 학습 → R² 기록 |
| 5 | 가중치 **30% : 70%** (부리깊이 높임) → 학습 → R² 기록 |
| 6 | 결과 비교 및 분석 |

#### 기록표

| 부리길이 가중치 | 부리깊이 가중치 | R² | 관찰 |
|----------------|----------------|-----|------|
| 50% | 50% | | |
| 70% | 30% | | |
| 30% | 70% | | |

#### 토론 질문
1. 가중치를 변경하면 무엇이 바뀌나요?
2. 최적의 가중치 조합은 어떻게 찾을 수 있을까요?
3. 가중치는 각 변수의 "중요도"를 나타낸다고 할 수 있을까요?

#### 예상 결과
- 각 변수와 체중의 상관관계에 따라 최적 가중치가 다름
- 한쪽에 치우친 가중치가 더 좋을 수 있음

### 3.4 탐구 활동 4: 노이즈의 영향

#### 목표
데이터의 불확실성(노이즈)이 모델 성능에 미치는 영향을 탐구한다

#### 절차

| 단계 | 활동 |
|:----:|------|
| 1 | 🏠 주택 프리셋 선택, 데이터 30개 |
| 2 | 노이즈 **5%** → 데이터 생성 → 학습 → R² 기록 |
| 3 | 초기화 후 노이즈 **10%** → 학습 → R² 기록 |
| 4 | 초기화 후 노이즈 **20%** → 학습 → R² 기록 |
| 5 | 오차선 표시하여 비교 |

#### 기록표

| 노이즈 수준 | R² | 오차선 관찰 |
|------------|-----|------------|
| 5% | | |
| 10% | | |
| 20% | | |

#### 토론 질문
1. 노이즈가 클수록 R²는 어떻게 변하나요?
2. 실제 데이터에도 노이즈가 있을까요? 왜 그럴까요?
3. AI 예측이 "100% 정확"하지 않은 이유는 무엇일까요?

### 3.5 탐구 활동 5: 나만의 데이터 만들기

#### 목표
직접 데이터를 설계하고 예측 모델을 만들어본다

#### 절차

| 단계 | 활동 |
|:----:|------|
| 1 | 관심 있는 X-Y 관계 정하기 (예: 운동시간-칼로리 소모) |
| 2 | [직접 입력] 탭에서 데이터 10개 이상 입력 |
| 3 | 모델 학습 |
| 4 | 새로운 X값으로 예측해보기 |
| 5 | 발표 및 토론 |

#### 아이디어 예시

| X 변수 | Y 변수 | 예상 관계 |
|--------|--------|----------|
| 운동시간(분) | 소모 칼로리 | 양의 상관 |
| 수면시간(h) | 집중력 점수 | 양의 상관? |
| 게임시간(h) | 성적 | 음의 상관? |
| 온도(°C) | 아이스크림 판매량 | 양의 상관 |

## 4. 평가

### 4.1 평가 루브릭

#### 지식 영역 (선형 회귀 이해)

| 평가 기준 | 상 (3점) | 중 (2점) | 하 (1점) |
|----------|---------|---------|---------|
| 기울기(w) 이해 | X가 1 증가할 때 Y의 변화량을 정확히 설명 | 기울기의 의미를 대략적으로 설명 | 기울기의 의미를 설명하지 못함 |
| 절편(b) 이해 | X=0일 때 Y값임을 정확히 설명 | 절편의 의미를 대략적으로 설명 | 절편의 의미를 설명하지 못함 |
| 손실 함수 이해 | 예측값과 실제값의 차이 제곱 평균임을 설명 | 손실이 줄어야 좋다는 것을 이해 | 손실의 의미를 모름 |

## 5. 확장 활동

### 5.1 심화 학습

#### AI와 일상생활

| 주제 | 탐구 질문 |
|------|----------|
| 추천 알고리즘 | 유튜브/넷플릭스는 어떻게 나에게 맞는 영상을 추천할까? |
| 날씨 예측 | 기상청의 날씨 예측은 얼마나 정확할까? |
| 가격 예측 | 주식/부동산 가격을 AI로 예측할 수 있을까? |

#### 다른 머신러닝 알고리즘

| 알고리즘 | 용도 | 시뮬레이터 |
|----------|------|-----------|
| KNN | 분류 (고양이 vs 개) | 동물분류_KNN_시뮬레이터 |
| K-means | 군집화 (고객 분류) | K평균_클러스터링_시뮬레이터 |
| 의사결정트리 | 분류/회귀 | 의사결정트리_시뮬레이터 |

### 5.2 실생활 연계

#### 프로젝트 아이디어

| 프로젝트 | 데이터 수집 | 예측 목표 |
|----------|------------|----------|
| 우리 반 키-발크기 관계 | 학생 설문 | 키로 발크기 예측 |
| 용돈-저축액 관계 | 학생 설문 | 용돈으로 저축액 예측 |
| 독서량-어휘력 관계 | 테스트 | 독서량으로 어휘력 예측 |
| 등교시간-지각횟수 | 기록 | 출발시간으로 지각 예측 |

#### 데이터 수집 가이드

1. **주제 선정**: 관계가 있을 것 같은 두 변수 선정
2. **데이터 수집**: 최소 20개 이상 (학급 전체 등)
3. **CSV 변환**: 엑셀에서 CSV로 저장
4. **시뮬레이터 적용**: 업로드 → 학습 → 예측
5. **결과 발표**: 발견한 관계와 예측 결과 공유

### 5.3 토론 주제

| 주제 | 찬반 논점 |
|------|----------|
