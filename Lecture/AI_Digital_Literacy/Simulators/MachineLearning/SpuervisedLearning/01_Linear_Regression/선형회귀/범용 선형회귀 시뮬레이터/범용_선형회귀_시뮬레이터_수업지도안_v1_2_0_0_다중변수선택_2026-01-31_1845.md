## 6. 지도상 유의점

### 6.1 기술적 유의점

| 상황 | 유의점 |
|------|--------|
| 시뮬레이터 실행 | Chrome/Edge 최신 버전 권장, 인터넷 연결 필요 |

| CSV 업로드 | UTF-8 인코딩, 콤마 구분 확인 |

### 6.2 교수학습 유의점

| 상황 | 유의점 |
|------|--------|
| 수학적 공식 | 공식 유도보다 **직관적 이해**에 초점 |
| 용어 | 영어 용어와 한글 용어 병기 (학습률 = Learning Rate) |
| 오류 경험 | 발산, 과적합 등 **실패 경험도 학습**임을 강조 |
| AI 한계 | "AI는 항상 맞지 않는다"를 오차선으로 시각적으로 확인 |
| 다중 변수 | 원본 사과 시뮬레이터의 "품질 점수"와 연결하여 설명 |

### 6.3 개별화 지도

| 수준 | 지도 방안 |
|------|----------|
| **상** | 자유 탐구, 나만의 데이터셋 만들기, 심화 토론 참여 |
| **중** | 기본 탐구 활동 완수, 결과 발표 |
| **하** | 1:1 시연, 단계별 안내, 동료 도움 매칭 |

## 7. 판서 계획

### 7.1 1차시 판서

```
┌─────────────────────────────────────────────────────────────────────┐
│                    📊 선형 회귀 (Linear Regression)                 │
├───────────────────────────────┬─────────────────────────────────────┤
│                               │                                     │
│  【회귀식】                    │  【학습 과정】                       │
│                               │                                     │
│  Y = w × X + b               │   ① 데이터 준비                     │
│                               │   ② 모델 초기화 (w, b)              │
│  • Y: 예측값 (종속변수)        │   ③ 예측 → 오차 계산               │
│  • X: 입력값 (독립변수)        │   ④ 가중치 업데이트                 │
│  • w: 기울기 (가중치)          │   ⑤ 반복 (에포크)                   │
│  • b: 절편 (편향)              │                                     │
│                               │                                     │
│  【예시: 주택 가격】           │  【손실 함수 (MSE)】                 │
│                               │                                     │
│  가격 = 15 × 면적 + 500       │   오차 = 실제 - 예측                │
│                               │   MSE = 평균(오차²)                 │
│  면적 50㎡                     │                                     │
│  → 15×50+500 = 1,250만원      │   손실↓ = 학습 진행 중              │
│                               │                                     │
```
