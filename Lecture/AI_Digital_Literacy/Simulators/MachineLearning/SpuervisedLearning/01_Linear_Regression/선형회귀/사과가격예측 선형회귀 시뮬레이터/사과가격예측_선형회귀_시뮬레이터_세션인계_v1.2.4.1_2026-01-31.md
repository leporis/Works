## 3. 현재 시뮬레이터 핵심 구조

### 3.1 모델 (단순 선형 회귀)

```
가격 = w × 품질 + b

품질 = 당도점수(50%) + 무게점수(50%)
- 당도: 10~18 Brix → 0~50점
- 무게: 150~350g → 0~50점
- 총 품질: 0~100점
```

### 3.2 정규화 방식

```javascript
// Z-score 정규화
normParams = {
    quality: { mean, std },
    price: { mean, std }
}

// 학습용 가중치와 표시용 가중치 분리
model = {
    w: 0,      // 정규화된 가중치 (학습용)
    b: 0,      // 정규화된 편향 (학습용)
    wReal: 0,  // 실제 스케일 (표시/예측용)
    bReal: 0   // 실제 스케일 (표시/예측용)
}
```

### 3.3 레이아웃 구조

**학습 모드**:
```
┌──────────┬─────────────────────────┬──────────┐
│  데이터   │  📉 손실    📈 산점도   │  학습    │
│  생성    │  그래프     (회귀선)    │  설정    │
└──────────┴─────────────────────────┴──────────┘
           ↓  학습 후 표시  ↓
┌─────────────────────────────────────────────────┐
│  📊 학습결과  │  📏 오차분석  │  📽️ 재생     │
└─────────────────────────────────────────────────┘
```

**추론 모드**:
```
┌──────────┬─────────────────────────┬──────────┐
│  예측    │   📈 예측 시각화        │  예측    │
│  입력    │  (산점도+회귀선+예측점)  │  결과    │
└──────────┴─────────────────────────┴──────────┘
```

## 5. 범용 선형회귀 시뮬레이터 확장 방향

### 5.1 목표

"사과 가격 예측" 테마를 제거하고, **다양한 데이터셋에 적용 가능한 범용 시뮬레이터**로 확장

### 5.2 확장 아이디어

| 기능 | 설명 |
|------|------|
| **데이터 입력** | CSV 파일 업로드 또는 직접 입력 |
| **변수 선택** | X축, Y축 변수 드롭다운 선택 |
| **다중 회귀** | 2개 이상 입력 변수 지원 |
| **테마 전환** | 사과/주택가격/시험점수 등 프리셋 |
| **데이터 클릭 추가** | 차트 클릭으로 데이터 직접 추가 |

### 5.3 현재 코드에서 재사용 가능한 부분

| 모듈 | 재사용 가능 |
|------|------------|
| 경사 하강법 학습 | ✅ |
| 정규화 (Z-score) | ✅ |
| 손실 그래프 (Chart.js) | ✅ |
| 산점도 + 회귀선 (Canvas) | ✅ (변수명만 일반화) |
| 오차선 시각화 | ✅ |
| 학습 과정 재생 | ✅ |
| 추론 시각화 | ✅ (변수명만 일반화) |
| UI 레이아웃 | ✅ (패널 구조 유지) |

### 5.4 수정 필요한 부분

| 모듈 | 수정 내용 |
|------|----------|
| 데이터 생성 | 고정 공식 → 사용자 데이터 입력 |
| 변수명 | "당도", "품질" → 일반 변수명 |
| 품질 계산 | 제거 또는 선택적 |
| 테마 색상 | 사과 테마 → 중립적 색상 |
