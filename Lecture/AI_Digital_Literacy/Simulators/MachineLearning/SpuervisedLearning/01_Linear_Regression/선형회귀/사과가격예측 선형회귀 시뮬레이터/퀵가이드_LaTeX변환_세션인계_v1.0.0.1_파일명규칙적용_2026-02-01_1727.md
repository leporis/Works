## 4. 핵심 기술 결정사항

### 4.1 LaTeX 설정

| 항목 | 결정 | 이유 |
|------|------|------|
| 컴파일러 | XeLaTeX | 한글 폰트 지원 |
| 한글 폰트 | Noto Sans CJK KR | 시스템에 설치됨 |
| 한글 패키지 | fontspec (xeCJK 미사용) | xeCJK.sty 미설치 |
| 이미지 형식 | SVG → PNG 변환 | LaTeX에서 PNG 직접 삽입 |

### 4.2 색상 팔레트 (파란색 테마)

```latex
\definecolor{mainblue}{HTML}{1565C0}      % 메인 색상 (기존 applered 대체)
\definecolor{applegreen}{HTML}{43A047}    % 학습 버튼
\definecolor{honeyyellow}{HTML}{FFC107}   % 예측점
\definecolor{warmcream}{HTML}{FFFBF5}     % 배경
\definecolor{blueprimary}{HTML}{1976D2}   % 하이퍼링크
```

### 4.3 tcolorbox 스타일

| 박스명 | 용도 | 테두리 색상 |
|--------|------|-------------|
| infobox | 정보/안내 | mainblue |
| successbox | 성공/실습 | applegreen |
| warningbox | 경고/트러블슈팅 | honeyyellow |
| tipbox | 팁 | blueprimary |

### 4.4 이미지 파일명 규칙 (문서버전관리지침 적용)

```
[문서종류]_[번호]_[내용]_v[버전]_[수정키워드]_[날짜]_[시간].확장자
```

| 구성요소 | 값 | 설명 |
|----------|---|------|
| 문서종류 | 퀵가이드 | 이미지가 삽입될 문서 |
| 번호 | 06 | 화면 구성 섹션 번호 |
| 내용 | 학습모드화면구성 / 추론모드화면구성 | 이미지 내용 |
| 버전 | v1.0.0.0 | 이미지 버전 |
| 수정키워드 | 초안작성 | 해당 버전의 변경 사항 |
| 날짜 | 2026-02-01 | YYYY-MM-DD |
| 시간 | 1721 | HHMM (24시간) |
