# 사과 가격 예측 선형 회귀 시뮬레이터 개발일지

**문서 버전: v1.2.4.1**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
사과 가격 예측 선형 회귀 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 머신러닝의 핵심 알고리즘인 **선형 회귀(Linear Regression)**의 원리를 직관적으로 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표
| 목표 | 설명 |
|------|------|
| 선형 회귀 이해 | 입력 변수와 출력 변수 간의 선형 관계 학습 |
| 경사 하강법 체험 | 손실 함수를 최소화하는 최적화 과정 시각화 |
| 특성 중요도 파악 | 각 입력 변수가 결과에 미치는 영향력 비교 |
| 예측 원리 학습 | 학습된 모델로 새로운 데이터 예측 체험 |

### 1.4 개발 배경
- 이전 세션(v1.2.2.2)의 인계 문서를 기반으로 새로 제작
- 2변수(당도, 무게)로 단순화하여 선형 회귀의 핵심 원리에 집중
- 3D 시각화 없이 2D 평면에서 등가선으로 회귀 평면 표현

## 2. 시스템 설계

### 2.1 입출력 변수

| 구분 | 변수명 | 단위 | 범위 | 설명 |
|:----:|--------|------|------|------|
| 입력 | 당도 | Brix | 10 ~ 18 | 사과의 당도 (높을수록 달콤) |
| 입력 | 무게 | g | 150 ~ 350 | 사과의 무게 |
| **중간** | **품질** | **점** | **0 ~ 100** | **당도+무게 조합 점수** |
| 출력 | 가격 | 원 | 계산값 | 예측 판매 가격 |

### 2.2 품질 점수 계산 (v1.1.0 신규)

```
품질 = 당도점수(50%) + 무게점수(50%)

당도점수 = (당도 - 10) / 8 × 50   (10~18 Brix → 0~50점)
무게점수 = (무게 - 150) / 200 × 50  (150~350g → 0~50점)
```

### 2.3 선형 회귀 모델

```
가격 = w × 품질 + b

w: 품질 가중치 (학습 파라미터)
b: 편향 (학습 파라미터)
```

**변경 사항 (v1.1.0):**
- 기존: 2변수 입력 (당도, 무게) → **1변수 입력 (품질)**
- X축: 품질 점수 (0~100)
- Y축: 가격 (원)
- 단순 선형 회귀로 변경하여 직관적 이해 향상

### 2.4 실제 가격 산정 공식 (데이터 생성용)

```javascript
실제가격 = 1000 + (30 × 품질) ± 노이즈
```

- 기본 가격: 1,000원
- 품질 기여: 1점당 30원

### 2.4 시뮬레이션 기본 설정

| 항목 | 기본값 | 조절 범위 |
|------|--------|-----------|
| 데이터 개수 | 25개 | 5 ~ 100개 |
| 노이즈 수준 | 10% | 5% / 10% / 20% |
| 학습률 | 0.01 | 0.001 ~ 0.1 |
| 에포크 | 500회 | 100 ~ 2,000회 |
| 시각화 간격 | 50 에포크 | 10 ~ 100 |

### 5.2 핵심 알고리즘

#### 정규화 (Z-score Normalization)
```javascript
normalized_value = (value - mean) / std
```

#### 경사 하강법 (Gradient Descent)
```javascript
for each epoch:
    for each data point:
        prediction = w1 * x1 + w2 * x2 + b
        error = prediction - actual
        grad_w1 += error * x1
        grad_w2 += error * x2
        grad_b += error
    
    w1 -= learning_rate * grad_w1 / n
    w2 -= learning_rate * grad_w2 / n
    b  -= learning_rate * grad_b / n
```

#### R² 점수 계산
```javascript
R² = 1 - (SS_res / SS_tot)

SS_res = Σ(actual - predicted)²
SS_tot = Σ(actual - mean)²
```

### 5.3 핵심 기술 결정사항

| 결정 | 이유 |
|------|------|
| **품질 점수 통합** | **당도+무게를 하나의 변수로 통합하여 단순 선형 회귀로 시각화** |
| **X축 품질, Y축 가격** | **2D 평면에서 회귀선을 직선으로 명확하게 표현 가능** |
