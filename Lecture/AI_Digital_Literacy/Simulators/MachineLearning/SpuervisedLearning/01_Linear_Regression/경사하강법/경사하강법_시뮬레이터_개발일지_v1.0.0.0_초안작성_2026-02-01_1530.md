# 경사하강법 시뮬레이터 개발일지

**문서 버전: v1.0.0.0**

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
경사하강법(Gradient Descent) 학습 시뮬레이터

### 1.2 목적
초등 예비교사를 대상으로 **경사하강법**의 7단계 학습 과정을 직접 체험하며 이해할 수 있도록 교육용 시뮬레이터를 개발한다.

### 1.3 교육 목표

| 목표 | 설명 |
|------|------|
| 학습 과정 이해 | 데이터 준비 → 가중치 초기화 → 예측 → 손실 → 기울기 → 업데이트 → 완료 |
| 경사하강법 체험 | 손실 함수의 기울기를 따라 최적값을 찾아가는 과정 시각화 |
| 하이퍼파라미터 영향 | 학습률이 학습 속도와 수렴에 미치는 영향 체험 |
| 손실 곡면 이해 | w, b에 따른 MSE 변화를 등고선으로 시각화 |

### 1.4 개발 배경
- 사과 가격 예측 시뮬레이터(v1.2.4.1)의 디자인 패턴 기반
- 첨부된 "선형 회귀 학습 과정" 흐름도 이미지 참고
- 7단계 학습 과정을 단계별로 직접 실행하는 방식

## 2. 시스템 설계

### 2.1 7단계 학습 과정

| 단계 | 이름 | 색상 | 설명 |
|:----:|------|------|------|
| 1 | 데이터 준비 | 🟢 녹색 | (X, Y) 데이터셋 생성 |
| 2 | 가중치 초기화 | 🔵 파랑 | w=0, b=0 설정 |
| 3 | 예측 계산 | 🟣 보라 | ŷ = w×x + b |
| 4 | 손실 계산 | 🔴 빨강 | MSE = Σ(y-ŷ)²/n |
| 5 | 기울기 계산 | 🔴 빨강 | ∇w = ∂MSE/∂w, ∇b = ∂MSE/∂b |
| 6 | 가중치 업데이트 | 🟠 주황 | w = w - η×∇w, b = b - η×∇b |
| 7 | 학습 완료 | 🟢 녹색 | 최적의 w, b 획득 |

**반복 루프**: 3~6단계를 에포크 횟수만큼 반복

### 2.2 데이터 생성

```javascript
// 실제 관계: Y = 2X + 10 (+ 노이즈)
const trueW = 2;
const trueB = 10;

for (let i = 0; i < count; i++) {
    const x = Math.random() * 10;
    const noise = (Math.random() - 0.5) * 2 * (noisePercent / 100) * 20;
    const y = trueW * x + trueB + noise;
    data.push({ x, y });
}
```

### 2.3 시뮬레이션 설정

| 항목 | 범위 | 기본값 |
|------|------|--------|
| 데이터 개수 | 5 ~ 50 | 20개 |
| 노이즈 수준 | 0 ~ 50% | 15% |
| 학습률 (η) | 0.001 ~ 0.5 | 0.1 |
| 목표 에포크 | 10 ~ 500 | 100회 |

| 차트 | 설명 |
|------|------|
| 산점도 & 회귀선 | 데이터 점 + 현재 회귀선 + 오차선 |

| 가중치 궤적 | 손실 곡면 위 (w, b) 이동 경로 |
| 기울기 벡터 | 현재 위치에서 업데이트 방향 화살표 |

### 3.3 자동 학습

| 기능 | 설명 |
|------|------|
| ▶ 재생 | 3~6단계 자동 반복 실행 |
| ⏸ 일시정지 | 자동 실행 중지 |
| ↺ 리셋 | 학습 초기화 (1단계로) |
| 재생 속도 | 느림~빠름 10단계 조절 |

### 3.4 파라미터 표시

| 항목 | 설명 |
|------|------|
| 가중치 (w) | 현재 기울기 값 |
| 편향 (b) | 현재 절편 값 |
| 손실 (MSE) | 현재 평균 제곱 오차 |
| 기울기 (∇w, ∇b) | 손실 함수의 편미분 값 |
| 회귀식 | ŷ = w × x + b |

### 5.2 핵심 알고리즘

#### 손실 함수 (MSE)
```javascript
function calcMSE(w, b) {
    let total = 0;
    data.forEach(d => {
        const pred = w * d.x + b;
        total += (d.y - pred) ** 2;
    });
    return total / data.length;
}
```

#### 기울기 계산
```javascript
function calculateGradients() {
    let gradW = 0, gradB = 0;
    const n = data.length;
    
    data.forEach(d => {
        const pred = model.w * d.x + model.b;
        const error = pred - d.y;
        gradW += error * d.x;
        gradB += error;
    });
    
    model.gradW = (2 / n) * gradW;
    model.gradB = (2 / n) * gradB;
}
```

#### 가중치 업데이트
```javascript
function updateWeights() {
    const lr = learningRate;
    model.w -= lr * model.gradW;
    model.b -= lr * model.gradB;
}
```

### 5.3 손실 곡면 시각화

- **히트맵 방식**: w, b 격자점마다 MSE 계산
- **해상도**: 50×50 그리드
- **색상**: 파랑(낮은 손실) → 빨강(높은 손실)
- **궤적**: 노란색 선으로 (w, b) 이동 경로 표시
- **기울기 벡터**: 빨간 화살표로 업데이트 방향 표시
