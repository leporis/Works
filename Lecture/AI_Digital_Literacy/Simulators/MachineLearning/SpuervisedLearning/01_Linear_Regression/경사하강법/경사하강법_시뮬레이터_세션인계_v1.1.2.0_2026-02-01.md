## 5. 시뮬레이터 주요 기능

### 5.1 학습 모드
| 기능 | 설명 |
|------|------|
| 데이터 입력 | 프리셋 4종, CSV 업로드, 내장 데이터셋 8종, 직접 입력 |
| 자동 학습 | ⚡ 학습 실행 버튼으로 전체 에포크 빠른 학습 |
| 단계별 실행 | ◀ 이전 / 다음 ▶ 버튼으로 한 단계씩 수동 진행 |
| 손실 추적 | 에포크별 손실 변화 실시간 모니터링 |
| 결과 분석 | 학습 결과, 오차 분석, 학습 재생 3열 |

### 5.2 추론 모드
| 기능 | 설명 |
|------|------|
| X 슬라이더 | 데이터 범위에 맞게 자동 조정 |
| 자동 추론 | 슬라이더 변경 시 즉시 예측 |
| 예측 분해 | X → w×X → +b → ŷ 과정 표시 |
| 시각화 | 예측점, 수직/수평 가이드라인 |

### 5.3 7단계 학습 과정
```
1. 데이터 준비      → (X, Y) 쌍 생성
2. 가중치 초기화    → w=0, b=0
3. 예측 계산        → ŷ = w×X + b
4. 손실 계산        → MSE = Σ(y-ŷ)²/n  📊 그래프 업데이트
5. 기울기 계산      → ∇w, ∇b
6. 가중치 업데이트  → w = w - η×∇w  ✅ 학습 완료 버튼
   ↓ "다음▶"                        ↓ "✅ 학습 완료"
   3단계로 반복                     7. 학습 완료!
```
