# 📘 전개 1: 기계학습의 정의

---

## 슬라이드 8: 기계학습이란?

**Machine Learning (기계학습)**

> 💡 데이터로부터 패턴을 학습하여
> 새로운 상황에서 예측하는 AI 기술

**핵심 키워드**
- 📊 데이터
- 🔍 패턴 발견
- 🎯 예측

---

## 슬라이드 9: 전통적 프로그래밍 vs 기계학습

**전통적 프로그래밍**
```
규칙 + 데이터 → 결과

사람이 규칙 작성 ✍️
컴퓨터가 실행 🖥️
```

**기계학습**
```
데이터 + 결과 → 규칙 발견

사람이 데이터 제공 📊
컴퓨터가 스스로 규칙 발견 🤖
```

---

## 슬라이드 10: 왜 기계학습이 필요한가?

**문제 상황**

❌ **규칙 작성이 불가능한 경우**
- 고양이 vs 개 구분 규칙을 어떻게 작성?
- 귀 모양? 털 색깔? 크기?
- 예외가 너무 많음!

✅ **기계학습의 해결책**
- 수천 장의 사진 제공
- 컴퓨터가 스스로 패턴 발견
- 새로운 사진도 잘 구분!

---

🎬 **영화 추천**

## 슬라이드 15: 1️⃣ 지도학습

**지도학습 (Supervised Learning)**

> 💡 정답(라벨)이 있는 데이터로 학습

**비유**
- 선생님이 정답을 알려주며 가르치는 것

**데이터 구조**
```
고양이 사진 → "고양이" (정답)
개 사진 → "개" (정답)
```

---

## 슬라이드 16: 지도학습의 2가지 유형

**분류 (Classification)**
- 출력: 범주 (고양이/개)
- 질문: "어느 것?"

**회귀 (Regression)**
- 출력: 숫자 (3억원, 85점)
- 질문: "얼마나?"

---

## 슬라이드 17: 분류 vs 회귀 비교

**회귀 vs 분류 비교표**

| 구분 | 회귀 | 분류 |
|:---:|:---|:---|
| **질문** | "얼마나?" | "어느 것?" |
| **답 형태** | 숫자 (연속적) | 범주 (구분) |
| **예시 1** | 시험 점수: 85점 | 합격/불합격: 합격 |
| **예시 2** | 집값: 3억 5천 | 아파트/빌라: 아파트 |
| **예시 3** | 기온: 18도 | 더위/추위: 더위 |

## 슬라이드 19: 비지도학습 - 라벨이 없는 이유

# 📙 전개 3: 머신러닝 4단계 프로세스

## 슬라이드 25: 머신러닝 4단계

**학습 과정 4단계**

1️⃣ **데이터 수집** (Data Collection)
- 양질의 데이터 준비

2️⃣ **학습** (Training)
- 패턴 발견

3️⃣ **평가** (Testing)
- 성능 확인

4️⃣ **예측** (Prediction)
- 실제 적용

---

## 슬라이드 26: 1단계 - 데이터 수집

**데이터 수집의 3가지 원칙**

**[그림 13] 편향된 데이터 vs 좋은 데이터**

![편향된 데이터 vs 좋은 데이터](AI디지털리터러시_1주차2차시_13_편향된데이터vs좋은데이터.png)

1️⃣ **양** (Quantity)
- 충분히 많은 데이터 (수천~수만 개)

2️⃣ **질** (Quality)
- 정확하고 깨끗한 데이터

3️⃣ **다양성** (Diversity)
- 다양한 상황의 데이터

---

## 슬라이드 27: 2단계 - 학습 (Training)

**학습 과정**

**[그림 12] 학습 곡선**

![학습 곡선](AI디지털리터러시_1주차2차시_12_학습곡선.png)

**에포크 (Epoch)**
- 전체 데이터를 1회 학습하는 단위
- 에포크 ↑ → 정확도 ↑

**가중치 (Weights)**
- AI가 학습하며 조정하는 수치

---

## 슬라이드 28: 3단계 - 평가 (Testing)

**데이터 분할**

**[그림 11] 데이터 분할**

![데이터 분할](AI디지털리터러시_1주차2차시_11_데이터분할.png)

- **Train**: 70% (학습용)
- **Validation**: 15% (검증용)
- **Test**: 15% (최종 평가용)

**왜 나눌까?**
→ 새로운 데이터에도 잘 작동하는지 확인!

---

## 슬라이드 29: 과적합 (Overfitting)

**과적합이란?**

❌ **과적합 상태**
- 훈련 데이터는 100점
- 새 데이터는 50점
- 암기만 하고 이해 못함

✅ **좋은 학습**
- 훈련 데이터 85점
- 새 데이터 82점
- 일반화 능력 우수

---

## 슬라이드 30: 4단계 - 예측 (Prediction)

**실전 배치**

**모델 배포**
1️⃣ 학습 완료된 모델 저장
2️⃣ 앱/웹사이트에 적용
3️⃣ 실시간 예측 시작

**지속적 개선**
- 새 데이터 수집
- 재학습 (Re-training)
- 성능 모니터링

---

## 슬라이드 31: 머신러닝 4단계 정리

**전체 과정 요약**

```
1️⃣ 데이터 수집
   ↓
2️⃣ 학습 (패턴 발견)
   ↓
3️⃣ 평가 (성능 확인)
   ↓
4️⃣ 예측 (실전 적용)
```
