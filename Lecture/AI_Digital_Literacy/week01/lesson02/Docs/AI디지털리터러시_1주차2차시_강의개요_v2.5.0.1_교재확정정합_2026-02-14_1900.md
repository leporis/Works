### 5.4 전개 3: 머신러닝 학습 과정 (5분)

| 순서 | 단계 | 설명 | 엔트리 연결 |
|:---:|:---|:---|:---|
| 1 | **데이터 수집** | 학습할 데이터 모으기 (양·질·다양성) | 웹캠 사진 촬영 |
| 2 | **학습 (Training)** | 패턴 찾기 (에포크, 가중치) | "모델 학습하기" 버튼 |
| 3 | **평가 (Testing)** | 새 데이터로 정확도 확인 | 정확도 % 확인 |
| 4 | **예측 (Prediction)** | 실전 적용 | 새 사진 분류 |

**사용 그림**: 그림 10 머신러닝프로세스, 그림 11 편향된데이터vs좋은데이터

**핵심 강조**:
- 좋은 데이터 = 좋은 AI (편향 주의)
- 사과농장도 이 4단계를 따름: 사과 사진 수집 → CNN 학습 → 정확도 확인 → 수확 판단

### 5.5 정리 (5분)

**핵심 개념 4가지 요약**:

1. **AI ⊃ ML ⊃ DL**: 포함 관계
2. **기계학습**: 데이터 → 패턴 학습
3. **3가지 학습 유형**: 지도(정답) / 비지도(라벨 없음) / 강화(보상)
4. **4단계 과정**: 데이터 수집 → 학습 → 평가 → 예측

**엔트리 AI 연결**: 3~4주차 엔트리 이미지 분류 = 지도학습

**다음 주 퀴즈 안내**: 5지선다 10문항(0.5점) + 단답형 5문항(1점) = 10점

## 9. 주요 용어 해설

| 용어 (한글) | 영문 | 간단 정의 | 비유/예시 |
|:---|:---|:---|:---|
| **에포크** | Epoch | 전체 데이터를 1회 학습하는 단위 | 교과서를 한 번 읽기 = 1에포크 |
| **가중치** | Weight | 각 특징의 중요도 수치 | 귀 모양(중요) = 높은 가중치 |
| **정확도** | Accuracy | 전체 중 맞춘 비율 | 90/100 = 90% |
| **라벨** | Label | 데이터의 정답 표시 | "고양이" 스티커 |
| **특징** | Feature | 데이터의 속성 | 귀 모양, 수염, 눈 모양 |
| **군집화** | Clustering | 유사한 데이터끼리 그룹화 | 비슷한 친구끼리 모으기 |
| **보상** | Reward | 행동의 좋고 나쁨을 알려주는 피드백 | 게임 점수 |
