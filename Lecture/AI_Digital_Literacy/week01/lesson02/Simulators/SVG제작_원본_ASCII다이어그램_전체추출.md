```
    │

[2] 정규화 파라미터 계산 (훈련 데이터만)
    x_mean, x_std, y_mean, y_std
    │
    ▼
[3] 경사 하강법 반복 (에포크 × 훈련 데이터)
    각 에포크:
    ├─ 훈련 데이터로 기울기 계산 (gradW, gradB)
    ├─ w -= lr × gradW / n
    ├─ b -= lr × gradB / n
    ├─ 훈련 MSE 기록
    ├─ 테스트 MSE 기록 (테스트 데이터 있을 때)
    └─ 100프레임 간격으로 히스토리 저장
    │
    ▼
[4] 역정규화 (원래 스케일로 변환)
    wReal = w × y_std / x_std
    bReal = b × y_std + y_mean - wReal × x_mean
    │
    ▼
[5] 결과 표시
    ├─ 손실 그래프 업데이트 (훈련 + 테스트)
    ├─ 산점도 + 회귀선 그리기
    ├─ 방정식 표시: Y = wReal × X + bReal
    ├─ R² 계산 (훈련/테스트)
    └─ 결과 테이블 표시
```

### 다이어그램 5

**섹션:** ### 7.4 R² (결정 계수) 해석

```
R² = 1 - (SS_res / SS_tot)
   = 1 - Σ(실제 - 예측)² / Σ(실제 - 평균)²
```

### 다이어그램 6

**섹션:** ### 11.1 경사 하강법 (Gradient Descent)

```
입력: 정규화된 x̂ = (x - μx) / σx
      정규화된 ŷ = (y - μy) / σy

모델: ŷ_pred = w × x̂ + b

손실: MSE = (1/n) × Σ(ŷ_pred - ŷ)²

기울기:
  ∂MSE/∂w = (1/n) × Σ(error × x̂)
  ∂MSE/∂b = (1/n) × Σ(error)

업데이트:
  w ← w - lr × ∂MSE/∂w
  b ← b - lr × ∂MSE/∂b
```

### 다이어그램 7

**섹션:** ### 11.1 경사 하강법 (Gradient Descent)

```
wReal = w × σy / σx
bReal = b × σy + μy - wReal × μx
```

### 다이어그램 8

**섹션:** ### 11.3 R² 계산

```
SS_res = Σ(y_actual - y_predicted)²  (잔차 제곱합)
SS_tot = Σ(y_actual - y_mean)²       (총 제곱합)
R² = 1 - SS_res / SS_tot
```
