## 1. 개요

### 1.1 소프트웨어 정보

| 항목 | 내용 |
|------|------|
| 이름 | 선형 회귀 학습 시뮬레이터 |
| 버전 | v1.4.0.0 |
| 유형 | 교육용 머신러닝 시뮬레이터 (단일 HTML 파일) |
| 대상 | 대학생 (예비 초등 교사), 머신러닝 입문자 |
| 목적 | 선형 회귀(Y = wX + b)의 학습 과정을 시각적으로 체험 |
| 핵심 알고리즘 | 경사 하강법 (Gradient Descent)을 통한 선형 회귀 학습 |

### 1.2 주요 기능 요약

| 기능 | 설명 |
|------|------|
| 2가지 모드 | 학습 모드 (모델 훈련) + 추론 모드 (예측 수행) |
| 3가지 데이터 입력 | 내장 데이터셋(23종) / CSV 업로드 / 직접 입력 |
| 다중 변수 지원 | 독립변수 다중 선택 + 변수별 가중치 조절 |
| 데이터 분할 | 훈련(80%) / 테스트(20%) 자동 분할 |
| 시각화 3종 | 손실 그래프 + 산점도(회귀선) + 추론 차트 |
| 오차 분석 | 오차선 시각화 + 평균/최대/MSE 통계 |
| 학습 과정 재생 | 에포크별 회귀선 변화 애니메이션 |
| 추론 과정 분해 | 절편 + 기울기 기여 → 예측값 단계별 표시 |

### 1.3 설계 원칙

- **단일 HTML**: 모든 코드(HTML/CSS/JS)와 데이터(23개 데이터셋)를 하나의 파일에 포함
- **정규화 학습**: 내부적으로 Z-score 정규화 후 학습, 결과는 원래 스케일로 역변환
- **훈련/테스트 분리**: Fisher-Yates 셔플 기반 80:20 분할로 과적합 검증 지원
- **반응형 레이아웃**: 데스크톱(3열) → 태블릿(2열) → 모바일(1열)

## 4. 데이터 입력

### 4.1 데이터셋 탭

23개 내장 데이터셋을 7개 카테고리로 제공합니다.

**데이터 로드 절차:**
1. 드롭다운에서 데이터셋 선택
2. 데이터셋 정보 박스에 출처·설명·권장 변수 조합 표시
3. [📊 데이터 로드] 클릭
4. 다변수 데이터셋: 변수 선택 UI 자동 표시

**랜덤 데이터 생성:**
- 드롭다운 최하단 "🎲 랜덤 데이터 생성" 선택 시 추가 설정 표시
- 데이터 개수: 5~100개 (슬라이더)
- 노이즈 수준: 5%(낮음) / 10%(보통) / 20%(높음)
- 4개 프리셋 중 랜덤 선택: 주택가격, 시험점수, 급여, 랜덤

### 4.2 CSV 탭

외부 CSV 파일을 업로드하여 사용합니다.

| 항목 | 요구사항 |
|------|---------|
| 파일 형식 | `.csv` |
| 첫 행 | 헤더 (변수명) |
| 데이터 타입 | 수치형만 자동 인식 (문자열 컬럼은 제외) |
| 인코딩 | UTF-8 |

업로드 후 수치형 컬럼이 자동 감지되어 변수 선택 UI가 표시됩니다.

### 4.3 직접 입력 탭

두 가지 방법으로 데이터를 추가합니다:

| 방법 | 사용법 |
|------|--------|
| 수치 입력 | X, Y 값 입력 후 [+ 추가] 클릭 |
| 차트 클릭 | 산점도 캔버스 클릭 → 해당 좌표에 데이터 추가 |

- 차트 클릭은 학습 모드에서만 동작합니다
- 데이터가 5개 미만이면 전체를 훈련에 사용합니다 (분할 안 함)

## 5. 변수 선택 및 가중치

### 5.1 변수 선택

다변수 데이터셋(컬럼 3개 이상) 로드 시 변수 선택 UI가 표시됩니다.

| 요소 | 기능 |
|------|------|
| 독립변수(X) 체크박스 | 1개 이상 선택 (다중 선택 가능) |
| 종속변수(Y) 드롭다운 | 1개만 선택 |

### 5.2 다중 변수 가중치

독립변수를 2개 이상 선택하면 가중치 섹션이 표시됩니다.

**가중치 계산 방식:**
1. 각 독립변수를 0~100 범위로 정규화
2. 변수별 가중치(0~1)를 곱함
3. 가중 합산하여 단일 X값 생성

```
X_combined = Σ(normalize(x_i) × weight_i) / Σ(weight_i)
```

| 요소 | 기능 |
|------|------|
| 가중치 슬라이더 | 0~1 범위, 기본값 1.0 |
| 합산 공식 표시 | `X = w₁ × 변수1 + w₂ × 변수2 + ...` |
| [🔄 가중치 적용] | 가중치 변경 후 데이터 재계산 |

## 6. 학습 설정 및 실행

### 6.1 학습 파라미터

| 파라미터 | ID | 범위 | 기본값 | 단위 |
|---------|-----|------|:------:|------|
| 학습률 | `learningRate` | 0.001 ~ 0.1 | 0.01 | step 0.001 |
| 에포크 | `epochs` | 100 ~ 2000 | 500 | step 100 |

### 6.2 데이터 분할

| 항목 | 값 |
|------|-----|
| 분할 비율 | 훈련 80% : 테스트 20% |
| 셔플 알고리즘 | Fisher-Yates |
| 최소 데이터 | 5개 미만 시 분할 없이 전체 훈련 |
| 재분할 | [🔀 재분할 & 재학습] 버튼으로 재셔플 + 자동 재학습 |

### 6.3 학습 실행 흐름

```
[⚡ 학습 실행] 클릭
    │
    ▼
[1] 모델 초기화
    w = random(-0.05, 0.05)
    b = 0
    │
    ▼
[2] 정규화 파라미터 계산 (훈련 데이터만)
    x_mean, x_std, y_mean, y_std
    │
    ▼
[3] 경사 하강법 반복 (에포크 × 훈련 데이터)
    각 에포크:
    ├─ 훈련 데이터로 기울기 계산 (gradW, gradB)
    ├─ w -= lr × gradW / n
    ├─ b -= lr × gradB / n
    ├─ 훈련 MSE 기록
    ├─ 테스트 MSE 기록 (테스트 데이터 있을 때)
    └─ 100프레임 간격으로 히스토리 저장
    │
    ▼
[4] 역정규화 (원래 스케일로 변환)
    wReal = w × y_std / x_std
    bReal = b × y_std + y_mean - wReal × x_mean
    │
    ▼
[5] 결과 표시
    ├─ 손실 그래프 업데이트 (훈련 + 테스트)
    ├─ 산점도 + 회귀선 그리기
    ├─ 방정식 표시: Y = wReal × X + bReal
    ├─ R² 계산 (훈련/테스트)
    └─ 결과 테이블 표시
```

## 7. 학습 결과 분석

### 7.1 결과 테이블

| 항목 | 설명 |
|------|------|
| 에포크 | 실행된 총 에포크 수 |
| 훈련 MSE | 훈련 데이터에 대한 실제 스케일 평균 제곱 오차 |
| 테스트 MSE | 테스트 데이터에 대한 실제 스케일 평균 제곱 오차 |
| R² | 결정 계수 (훈련 데이터 기준) |
| 상태 | ✅ 양호(>0.8) / ⚠️ 보통(>0.5) / ❌ 부족(≤0.5) |

### 7.2 회귀 방정식

학습 완료 후 `Y = w × X + b` 형태로 표시됩니다.

- **w** (기울기): X가 1 증가할 때 Y의 변화량
- **b** (절편): X=0일 때의 Y 예측값

### 7.3 오차 분석

[오차선 표시] 체크박스 활성화 시:

| 통계 | 계산 방법 |
|------|----------|
| 평균 오차 | 실제값과 예측값 차이의 평균 |
| 최대 오차 | 가장 큰 오차 |
| MSE | 오차 제곱의 평균 |

**오차선 시각화:**
- 각 데이터 점에서 회귀선까지 수직선 표시
- 색상 그라데이션: 오차 작음(초록) → 오차 큼(빨강)

### 7.4 R² (결정 계수) 해석

```
R² = 1 - (SS_res / SS_tot)
   = 1 - Σ(실제 - 예측)² / Σ(실제 - 평균)²
```

| R² 범위 | 상태 | 색상 | 해석 |
|:-------:|:----:|:----:|------|
| > 0.8 | ✅ 양호 | 🟢 | 모델이 데이터 변동의 80% 이상 설명 |
| 0.5 ~ 0.8 | ⚠️ 보통 | 🟡 | 설명력이 있으나 다른 요인도 존재 |
| ≤ 0.5 | ❌ 부족 | 🔴 | 선형 관계가 약하거나 적합하지 않음 |

### 8.3 추론 과정 분해

| 📐 기울기 기여 (w × X) | X에 의한 추가분 | wReal × X |
| 🎯 예측 Y | 최종 결과 | bReal + wReal × X |

### 8.4 추론 차트 시각화

큰 캔버스에 다음 요소가 표시됩니다:

| 요소 | 시각 표현 |

| 테스트 데이터 점 | 사각형 (□) |
| 회귀선 | 빨간 직선 |
| 예측점 | **노란색 큰 원** (반경 8px) |
| 수직 가이드선 | 회색 점선 (X → 회귀선) |
| 수평 가이드선 | 회색 점선 (회귀선 → Y축) |
| 예측값 레이블 | 예측점 위에 수치 표시 |

## 9. 학습 과정 재생

### 9.1 기능

## 11. 알고리즘 상세

### 11.1 경사 하강법 (Gradient Descent)

**내부 학습 공간 (정규화 후):**

```
입력: 정규화된 x̂ = (x - μx) / σx
      정규화된 ŷ = (y - μy) / σy

모델: ŷ_pred = w × x̂ + b

손실: MSE = (1/n) × Σ(ŷ_pred - ŷ)²

기울기:
  ∂MSE/∂w = (1/n) × Σ(error × x̂)
  ∂MSE/∂b = (1/n) × Σ(error)

업데이트:
  w ← w - lr × ∂MSE/∂w
  b ← b - lr × ∂MSE/∂b
```

**역정규화 (결과 표시용):**

```
wReal = w × σy / σx
bReal = b × σy + μy - wReal × μx
```

### 11.2 정규화 방법

| 방법 | 공식 | 적용 |
|------|------|------|
| Z-score 정규화 | (x - μ) / σ | 학습 전 X, Y에 적용 |
| 파라미터 기준 | **훈련 데이터만** | 테스트 데이터에도 훈련 데이터의 μ, σ 사용 |

### 11.3 R² 계산

```
SS_res = Σ(y_actual - y_predicted)²  (잔차 제곱합)
SS_tot = Σ(y_actual - y_mean)²       (총 제곱합)
R² = 1 - SS_res / SS_tot
```

## 14. 문제 해결

### 14.1 학습 관련

| 증상 | 원인 | 해결 |
|------|------|------|
| 손실이 NaN으로 표시 | 학습률 과도 → 발산 | 학습률 0.001~0.01로 낮춤 |
| 손실이 줄지 않음 | 학습률 너무 낮거나 에포크 부족 | 학습률 0.01 + 에포크 1000+ |
| R²가 음수 | 모델이 평균보다 못한 예측 | 변수 조합 변경 또는 학습률 조정 |
| 훈련 MSE ≪ 테스트 MSE | 과적합 가능성 | 재분할 시도, 데이터 추가 |
| R²가 0에 가까움 | 선형 관계 없음 | 다른 변수 조합 시도 |

### 14.2 데이터 관련

| 증상 | 원인 | 해결 |
|------|------|------|
| 차트에 점 안 나옴 | 데이터 미로드 | [📊 데이터 로드] 확인 |
| CSV 업로드 후 변수 안 나옴 | 수치형 컬럼 없음 | CSV에 숫자 데이터 확인 |
| 데이터 분할 바 안 보임 | 데이터 5개 미만 | 정상 (전체 훈련 사용) |
| "최소 2개 이상" 알림 | 데이터 부족 | 데이터 추가 필요 |

### 14.3 표시 관련

| 증상 | 원인 | 해결 |
|------|------|------|
| 추론 모드에서 빈 차트 | 학습 미완료 | 학습 모드에서 먼저 학습 실행 |
| 차트 글씨 깨짐 | 폰트 미로드 | 인터넷 연결 확인 |
| 오차선이 안 보임 | 체크박스 미체크 | [오차선 표시] 체크 확인 |
| 재생 버튼 비활성 | 학습 미완료 | 학습 먼저 실행 |
