## 🚀 빠른 시작

### 4가지 모드

| 모드 | 용도 | 설명 |
|:---:|------|------|
| 📊 대시보드 | AI 훈련 및 모니터링 | 학습 진행, 차트, 파라미터 조절의 메인 화면 |
| ⚔️ AI 대전 | 두 AI의 대국 관전 | 학습된 X·O 에이전트가 서로 대국하는 모습 관찰 |
| 👤 사람 대전 | AI와 직접 대국 | 학습된 AI와 직접 틱택토 대결 |
| 🔄/🔵/🔴 학습 방식 | 학습 전략 선택 | 자기대국·X단독·O단독 중 선택 |

### 학습→대전 흐름

```
① 학습 방식 선택 → ② 학습 시작 → ③ 성장 관찰 → ④ 사람 대전으로 확인
```

| 단계 | 어디서 | 무엇을 |
|:---:|--------|--------|
| ① | 좌측 📋 학습 방식 | 🔄 자기대국 / 🔵 X 단독 / 🔴 O 단독 중 선택 |
| ② | 좌측 **▶️ 학습 시작** | 클릭 → AI가 틱택토를 반복 학습 시작 |
| ③ | 우측 대시보드 | 단계(🌱→🔍→📚→🎯→🏆), 승률·ε·강도 추이 관찰 |
| ④ | 좌측 **👤 사람 대전** | 학습된 AI와 직접 대국 |

### 사람 대전 모드

| 조작 | 방법 |
|------|------|
| 돌 놓기 | 빈 칸 **클릭** |
| X/O 선택 | 상단 🔵X / 🔴O 버튼 클릭 (선·후공 결정) |
| 새 판 | **🔄 새 게임** 클릭 |

## 🔍 이런 장면을 찾아보세요!

| 학습 진행 | 관찰 포인트 | 교육적 의미 |
|-----------|------------|------------|
| 🌱 초기 (0~500판) | ε=100%, 랜덤 착수, 승률 불규칙 | 아직 아무것도 모르는 상태 |
| 🔍 탐색 (500~5,000판) | ε 감소 시작, 승률 변동 | 시행착오를 통한 경험 축적 |
| 📚 학습 (5,000~20,000판) | 무승부율 상승, 강도 증가 | 좋은 전략을 발견하기 시작 |
| 🎯 응용 (20,000~50,000판) | ε→1%, 무승부 비율 높음 | 최적에 가까운 전략 습득 |
| 🏆 숙련 (50,000판+) | 등급 S, 무승부 지배적 | 완벽한 플레이 = 항상 무승부 |

> 💡 **핵심 관찰:** 틱택토는 양쪽 모두 최적 플레이하면 **항상 무승부**입니다. 무승부율이 올라가는 것이 올바른 학습의 신호입니다!

## ⚙️ 파라미터 가이드

### 학습 파라미터

| 파라미터 | 기본값 | 범위 | 역할 |
|----------|:------:|:----:|------|
| 학습률 α | 0.30 | 0.01~1.0 | Q값 업데이트 크기 (높을수록 최근 경험 반영 ↑) |
| 할인율 γ | 0.90 | 0.1~1.0 | 미래 보상 반영 정도 (높을수록 장기 전략 ↑) |
| ε 감소율 | 0.9995 | 0.99~1.0 | 탐험률 감소 속도 (낮을수록 빨리 활용 전환) |
| ε 초기값 | 1.0 | (고정) | 시작 시 100% 랜덤 탐험 |
| ε 최소값 | 0.01 | (고정) | 최소 1% 탐험 유지 |

### 보상 구조

| 상황 | 보상값 | 의미 |
|------|:------:|------|
| 승리 | +1.0 | 이기면 큰 보상 |
| 패배 | -1.0 | 지면 큰 패널티 |
| 무승부 | +0.3 | 무승부도 긍정적 보상 |
| 매 수 | -0.01 | 빠른 결판 유도 |

### 학습 방식 비교

| 방식 | 🔄 자기대국 | 🔵 X 단독 | 🔴 O 단독 |
|------|:-----------:|:---------:|:---------:|
| X 학습 | ✅ | ✅ | ❌ (랜덤) |
| O 학습 | ✅ | ❌ (랜덤) | ✅ |
| 특징 | 양쪽 상호 발전 | X만 집중 훈련 | O만 집중 훈련 |
| 권장 용도 | 기본 학습 | 선공 전략 실험 | 후공 전략 실험 |

### 파라미터 동기화 (🔗동기 / 🔓별도)

| 모드 | 동작 | 용도 |
|------|------|------|
| 🔗 동기 | X·O 파라미터를 동일하게 유지 | 일반 학습 (기본) |
| 🔓 별도 | X·O 파라미터를 개별 조절 | 비대칭 실험 |

## 🆘 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 학습이 시작되지 않는다 | 모드가 idle 상태 | **▶️ 학습 시작** 클릭 |
| 승률이 계속 불규칙하다 | 학습 초기 단계 | 5,000판 이상 학습 (속도 1ms 권장) |
| 사람 대전에서 AI가 너무 약하다 | 충분히 학습되지 않음 | 20,000판 이상 학습 후 재도전 |
| 무승부만 나온다 | 최적 학습 완료! | 이것이 정상 — 최적 틱택토는 항상 무승부 |
| AI가 명백히 나쁜 수를 둔다 | ε이 아직 높음 | ε이 5% 이하로 내려갈 때까지 학습 지속 |
| ε이 잘 안 줄어든다 | 감소율이 너무 높음 | ε 감소율을 0.999로 낮춰보기 |
| X만 계속 이긴다 | 단독 학습 모드 확인 | 자기대국으로 변경 (양쪽 모두 학습) |
| 화면이 안 보인다 | 테마 문제 | ☀️/🌙 테마 전환 시도 |

## 🔑 핵심 용어 5

| 용어 | 한 줄 설명 |
|------|-----------|
| **Q-테이블** | 각 상황에서 각 수의 가치(Q값)를 기록한 표. AI의 기억 장치 |
| **ε (탐험률)** | 새로운 수를 시도할 확률. 100%→1%로 감소하며 탐험→활용 전환 |
| **대칭성 정규화** | 회전·반사로 같은 보드를 하나의 상태로 통합 (8배 효율 향상) |
| **에피소드** | 한 판의 게임. 시작부터 승리·패배·무승부까지 |
| **강도 (Strength)** | 랜덤 상대에 대한 승률 점수 (0~100). 높을수록 AI가 강함 |
