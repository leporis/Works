## 📖 이 문서 사용법

> **이 퀵가이드는 AI 사과 농장 이야기 애니메이션과 함께 보는 내비게이션입니다.**

| 상황 | 참조 섹션 |
|------|----------|
| 처음 보는데 빨리 시작하고 싶다 | ⚡ 초속성 실습 |
| 전체 이야기 흐름을 알고 싶다 | 🚀 빠른 시작 |
| 버튼 위치를 모르겠다 | 🖥️ 화면 구성 |
| 각 장에서 뭘 관찰해야 하는지 모르겠다 | 🔍 관찰 포인트 |
| 소리가 안 나오거나 문제가 생겼다 | 🆘 트러블슈팅 |
| AI 용어가 헷갈린다 | 🔑 핵심 용어 |

## ⚡ 30초 초속성 실습

| 단계 | 행동 | 확인 |
|:---:|------|:----:|
| 1 | 화면 하단의 빨간 ▶ 버튼을 클릭하세요 | 🎵 차임벨 소리가 들리나요? |
| 2 | 내레이션 음성이 나오면 화면 하단 자막을 읽으세요 | 👨‍🌾 농부 김사과가 말하나요? |
| 3 | 자동으로 다음 장면으로 넘어가면 🎉 성공! | 하단 점(●)이 이동했나요? |

> 💡 **30초 안에 이야기가 시작됩니다!** 나머지는 자동으로 진행되니 편하게 감상하세요.

## 🚀 빠른 시작 (3단계)

### 1️⃣ 재생하기

화면 하단 컨트롤 바에서 **빨간 ▶ 버튼**을 클릭하면 애니메이션이 시작됩니다. 차임벨 전환음과 함께 프롤로그가 시작되고, TTS(음성 합성)로 내레이션이 자동 재생됩니다. 재생 중에 같은 버튼(⏸)을 누르면 일시정지됩니다.

### 2️⃣ 이야기 감상하기

애니메이션은 총 **7개 장면**(프롤로그 → 1~5장 → 에필로그)으로 구성되어 있습니다. 각 장면은 자동으로 전환되며, 농부 김사과(👨‍🌾)와 AI 로봇 알파(🤖)의 대화로 5가지 AI 기술을 하나씩 소개합니다. 화면 중앙의 일러스트와 하단의 자막이 동시에 표시됩니다.

| 장 | 제목 | AI 기술 | 핵심 장면 |
|:--:|------|---------|----------|
| 프롤로그 | 🍎 AI 사과 농장 이야기 | - | 김사과와 알파의 만남 |
| 1장 | 🌳 수확의 시작 | 딥러닝 (CNN) | 익은 사과 92% 확률 판별 |
| 2장 | 🔍 품종이 섞였다! | KNN 분류 | 가장 가까운 3개로 품종 결정 |
| 3장 | 📦 크기별 등급 분류 | K-평균 클러스터링 | 정답 없이 자동 3그룹 분류 |
| 4장 | 💰 가격을 매기자 | 선형 회귀 | 가격 = 30 × 품질 + 1000 |
| 5장 | 🚚 어디로 보낼까? | 결정 트리 | 예/아니오 질문으로 출하처 결정 |
| 에필로그 | 🎉 스마트 과수원 완성! | 종합 정리 | 5가지 AI 기술 협력 요약 |

### 3️⃣ 원하는 장면으로 이동하기

3가지 방법으로 원하는 장면으로 바로 이동할 수 있습니다:

- **장면 점(●) 클릭**: 하단 컨트롤 바 오른쪽의 7개 점 중 원하는 위치를 클릭
- **프로그레스 바 클릭**: 하단 진행 바의 원하는 위치를 클릭
- **자동 진행**: 그냥 두면 장면이 자동으로 넘어감

│                                                         │
│   ┌─────────────────────────────────────────────────┐   │

### 컨트롤 바 요소

| 요소 | 위치 | 기능 |
|------|------|------|
| **▶ / ⏸** (재생/일시정지) | 맨 왼쪽 빨간 원형 버튼 | 애니메이션 재생 또는 일시정지 |
| **🔊 / 🔇** (음소거) | 재생 버튼 오른쪽 | TTS 음성과 전환 효과음 켜기/끄기 |
| **프로그레스 바** | 중앙 | 현재 장면 내 진행률 표시, 클릭하여 이동 가능 |
| **장면 이름** | 프로그레스 바 아래 왼쪽 | 현재 재생 중인 장면 이름 (예: "1장: 수확") |
| **시간 표시** | 프로그레스 바 아래 오른쪽 | 현재 시간 / 장면 전체 시간 |
| **속도 조절** | 오른쪽 | 느리게(0.85x) / 보통(1x) / 빠르게(1.15x) |
| **TTS 상태** | 속도 오른쪽 | 준비중 / 준비됨 / 재생중 상태 표시 (점 색상 변화) |
| **장면 점(●)** | 맨 오른쪽 7개 점 | 현재 장면 위치 표시, 클릭하여 해당 장면으로 이동 |

### 내레이션 표시 색상

| 화자 | 아이콘 | 자막 배경색 | 음성 특징 |
|------|:------:|-----------|----------|
| 내레이터 | 📖 | 진한 갈색 반투명 | 보통 속도, 보통 톤 |
| 농부 김사과 | 👨‍🌾 | 갈색 반투명 | 느린 속도, 낮은 톤 |
| AI 로봇 알파 | 🤖 | 파란색 반투명 | 빠른 속도, 높은 톤 |

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| 지도학습 | Supervised Learning | 정답(라벨)이 있는 데이터로 학습 | "정답 스티커 붙여서 공부하기" |
| 비지도학습 | Unsupervised Learning | 정답 없이 데이터의 패턴을 찾는 학습 | "정답 없이 비슷한 것끼리 모으기" |
| 딥러닝 | Deep Learning | 인공 신경망을 사용한 기계학습 | "사람 뇌를 흉내 낸 AI의 학습법" |
| CNN | Convolutional Neural Network | 이미지 인식에 특화된 딥러닝 모델 | "사진을 보고 무엇인지 알아내는 AI 눈" |
| KNN | K-Nearest Neighbors | 가장 가까운 K개 이웃으로 분류 | "비슷한 친구 K명에게 물어보고 다수결" |
| K-평균 | K-Means Clustering | 데이터를 K개 그룹으로 자동 분류 | "비슷한 것끼리 K개 모둠 만들기" |
| 선형 회귀 | Linear Regression | 직선 관계식으로 값을 예측 | "y = ax + b 직선 긋기" |
| 결정 트리 | Decision Tree | 예/아니오 질문을 반복하여 분류 | "스무고개 게임으로 답 찾기" |
| TTS | Text-to-Speech | 텍스트를 음성으로 변환하는 기술 | "글자를 읽어주는 기능" |

## 💭 생각해보기

> **Q1.** 1장(수확)과 3장(등급)에서 AI가 사과를 분류하는 방식이 다릅니다. 어떤 차이가 있나요?
>
> - 힌트: 1장은 **정답을 알려주고** 학습했고, 3장은 **정답 없이** 학습했습니다.

> **Q2.** 4장에서 "가격 = 30 × 품질 + 1000"이라는 공식이 나옵니다. 만약 품질이 80점이라면 가격은 얼마일까요?
>
> - 힌트: 공식에 80을 대입해보세요.

> **Q3.** 5장의 결정 트리에서 "상품인가?" → "예" → 백화점으로 결정됩니다. 만약 "상품이 아니고, 중품도 아닌" 사과라면 어디로 갈까요?
>
> - 힌트: 하품 중에서도 크기에 따라 **온라인** 또는 **주스공장**으로 갈립니다.

> **Q4.** 이 애니메이션에 등장하는 5가지 AI 기술 중 **지도학습**에 해당하는 것은 몇 개이고, **비지도학습**은 몇 개인가요?
>
> - 힌트: 3장의 K-평균만 비지도학습이고, 나머지 4개(CNN, KNN, 선형회귀, 결정트리)는 지도학습입니다.
