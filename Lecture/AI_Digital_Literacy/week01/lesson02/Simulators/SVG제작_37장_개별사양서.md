## 제작 규칙 요약

1. **defs 블록:** `SVG제작_템플릿_패턴.svg`에서 복사 (bgGrad, headerGrad, shadow, shadowLight, 폰트)
2. **색상:** 아래 시뮬레이터별 팔레트 적용 (계획서가 아닌 실제 HTML CSS 기준)
3. **교육 레이블:** 모든 UI 레이아웃에 ①②③④ 영역 뱃지 포함
4. **파일명:** `[약칭]_[문서종류]_[번호]_[내용요약].svg`
5. **저장:** `/mnt/user-data/outputs/`에 직접 저장
