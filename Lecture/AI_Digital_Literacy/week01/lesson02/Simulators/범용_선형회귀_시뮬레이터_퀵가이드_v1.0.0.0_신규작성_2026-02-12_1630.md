## 🚀 빠른 시작 (3단계)

### 1️⃣ 데이터 준비

3가지 방법 중 하나를 선택하세요:

| 방법 | 탭 | 사용법 | 권장 상황 |
|------|:---:|--------|---------|
| 데이터셋 | **데이터셋** | 23개 내장 데이터 중 선택 → [📊 데이터 로드] | 처음 사용 시 (권장) |
| CSV 업로드 | **CSV** | .csv 파일 업로드 (첫 행: 헤더) | 내 데이터 사용 시 |
| 직접 입력 | **직접** | X, Y 수치 입력 또는 **차트 클릭** | 간단한 실험 시 |

데이터 로드 후 **독립변수(X)**와 **종속변수(Y)**를 선택합니다. 다중 변수 선택도 가능합니다.

### 2️⃣ 학습 실행

| 설정 | 기본값 | 권장 범위 | 역할 |
|------|:------:|----------|------|
| 학습률 | 0.01 | 0.001 ~ 0.05 | 한 번에 얼마나 배울지 (너무 크면 발산) |
| 에포크 | 500 | 200 ~ 1000 | 몇 번 반복 학습할지 |

[⚡ 학습 실행] 클릭 → 결과 확인:
- **회귀 방정식**: Y = w × X + b (w: 기울기, b: 절편)
- **R² 점수**: 1에 가까울수록 좋음 (🟢 >0.8 우수, 🟡 >0.5 보통, 🔴 ≤0.5 부족)
- **손실 그래프**: L자로 수렴하면 정상

### 3️⃣ 추론 (예측)

상단 [🔮 추론 모드] 탭 → X 슬라이더로 값 입력 → 예측 결과 확인

| 표시 항목 | 의미 |
|----------|------|
| 🏷️ 절편 (b) | 기본값 (X=0일 때 Y) |
| 📐 기울기 기여 (w × X) | X에 의한 추가분 |
| 🎯 예측 Y | 최종 예측 결과 |

## 🆘 트러블슈팅

| 증상 | 원인 | 해결책 |
|------|------|--------|
| 손실 그래프가 폭등 📈 | 학습률이 너무 높음 | 학습률을 **0.01 이하**로 낮추세요 |
| 손실이 안 줄어듦 | 학습률이 너무 낮거나 에포크 부족 | 학습률 0.01 + 에포크 1000으로 재시도 |
| R²가 0에 가까움 🔴 | 데이터에 선형 관계가 없음 | 다른 X-Y 변수 조합을 선택하세요 |
| 직선이 점들을 안 관통함 | 이상치(outlier) 영향 | 앤스컴 III/IV 데이터로 이상치 효과 체험 |
| 차트에 점이 안 나옴 | 데이터 미로드 | [📊 데이터 로드] 버튼을 클릭했는지 확인 |
| 추론 모드가 안 됨 | 학습 미실행 | 학습 모드에서 [⚡ 학습 실행] 먼저 |
| 변수 선택 드롭다운 안 보임 | 2변수 데이터(직접/랜덤) | 다변수 데이터셋 선택 시에만 표시 |
| 훈련 MSE ≪ 테스트 MSE | 과적합 가능성 | [🔀 재분할 & 재학습] 시도 |

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| 선형 회귀 | Linear Regression | 데이터의 직선 관계(Y=wX+b) 학습 | "점들 사이 가장 잘 맞는 직선 찾기" |
| 기울기 | Weight (w) | X가 1 증가할 때 Y의 변화량 | "경력 1년 → 연봉 200만원 증가" |
| 절편 | Bias (b) | X=0일 때의 Y값 | "신입 연봉 2500만원" |
| 학습률 | Learning Rate | 한 번에 수정하는 양 | "큰 걸음 vs 작은 걸음" |
| 에포크 | Epoch | 전체 데이터를 한 번 학습하는 단위 | "문제집 한 바퀴 푸는 것" |
| MSE | Mean Squared Error | 평균 제곱 오차 (작을수록 좋음) | "평균적으로 얼마나 빗나갔는지" |
| R² | R-squared | 결정 계수 (1에 가까울수록 좋음) | "모델이 데이터를 몇 % 설명하는지" |
| 오차선 | Error Line / Residual | 실제값과 예측값의 차이 | "빗나간 거리" |
| 과적합 | Overfitting | 훈련 데이터에만 잘 맞는 상태 | "문제집 답 외웠지만 실전에서 틀림" |
