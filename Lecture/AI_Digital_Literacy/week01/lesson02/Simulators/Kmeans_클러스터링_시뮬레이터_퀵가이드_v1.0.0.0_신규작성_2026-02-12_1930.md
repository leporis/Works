## 🚀 빠른 시작

### 학습 모드 흐름

```
① 데이터 설정 → ② 데이터 생성 → ③ 학습 설정 → ④ 클러스터링 실행 → ⑤ 과정 재생
```

| 단계 | 어디서 | 무엇을 |
|:---:|--------|--------|
| ① | 좌측 패널 "📊 데이터 생성" | 데이터 개수(20~200), K값(2~8), 분산도(밀집/보통/넓음) 조절 |
| ② | 좌측 **✨ 데이터 생성** 버튼 | 클릭 → 산점도에 회색 점 표시 |
| ③ | 우측 패널 "⚙️ 학습 설정" | 최대 반복 횟수(10~100), 초기화 방법(랜덤/K-means++) 선택 |
| ④ | 우측 **⚡ 클러스터링 실행** 버튼 | 클릭 → 점들이 K개 색상으로 분류 + 중심점(◉) 표시 |
| ⑤ | 하단 결과 패널 | 반복 횟수·Inertia·수렴 상태 확인 + **▶️ 재생**으로 과정 관찰 |

### 추론 모드 흐름

```
① 학습 완료 확인 → ② 추론 모드 전환 → ③ 좌표 입력 → ④ 예측 결과 확인
```

| 단계 | 어디서 | 무엇을 |
|:---:|--------|--------|
| ① | 학습 모드에서 클러스터링 실행 완료 | 결과 테이블에 ✅ 수렴 표시 확인 |
| ② | 상단 **🔮 추론 모드** 탭 클릭 | 화면이 추론 모드로 전환 |
| ③ | 좌측 X좌표·Y좌표 슬라이더 조절 | 예측할 위치를 지정 |
| ④ | 우측 "🎯 예측 결과" 패널 | 할당된 클러스터 번호, 데이터 개수, 중심점 좌표, 거리 확인 |

## ⚙️ 학습 설정 가이드

### 데이터 생성 설정

| 설정 | 범위 | 기본값 | 설명 |
|------|------|:------:|------|
| 데이터 개수 | 20~200 | 100 | 많을수록 패턴이 뚜렷, 적으면 빠르게 관찰 |
| K (클러스터 수) | 2~8 | 3 | 데이터를 몇 개 그룹으로 나눌지 결정 |
| 분산도 | 밀집/보통/넓음 | 보통 | 밀집: 그룹이 뚜렷, 넓음: 그룹 경계가 모호 |

### 학습 설정

| 설정 | 범위 | 기본값 | 설명 |
|------|------|:------:|------|
| 최대 반복 횟수 | 10~100 | 50 | 알고리즘이 최대 몇 번 반복할지 (수렴하면 일찍 종료) |
| 초기화 방법 | 랜덤 / K-means++ | K-means++ | 중심점의 첫 위치를 정하는 방식 |

### 초기화 방법 비교

| 방법 | 원리 | 장점 | 단점 |
|------|------|------|------|
| **랜덤** | 데이터 중 K개를 무작위 선택 | 단순, 빠름 | 결과 불안정, 나쁜 초기값 가능 |
| **K-means++** | 중심점끼리 최대한 멀리 배치 | 안정적, 좋은 결과 | 약간 더 복잡 |

> 💡 **팁**: K-means++로 실행한 뒤, 같은 데이터에서 랜덤으로 여러 번 실행해보면 초기화 방법의 차이를 체감할 수 있습니다.

### 재생 속도

| 설정 | 범위 | 기본값 | 설명 |
|------|------|:------:|------|
| 재생 속도 | 1~10 | 5(보통) | 1: 매우 느림(500ms), 10: 최고속(50ms) |

## 📐 Inertia 이해하기

**Inertia**(관성)는 클러스터링의 품질을 측정하는 값입니다.

```
Inertia = Σ (각 데이터점 ~ 소속 중심점까지 거리)²
```

| Inertia 값 | 의미 |
|:---:|------|
| **작을수록** | 데이터가 중심점 가까이 모여 있음 → 좋은 클러스터링 |
| **클수록** | 데이터가 중심점에서 멀리 퍼져 있음 → 느슨한 클러스터링 |

> 💡 **주의**: Inertia는 K를 늘리면 자연스럽게 줄어듭니다. K=데이터 수이면 Inertia=0이 되지만, 이것은 의미 없는 분류입니다!

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| 클러스터 | Cluster | 비슷한 데이터끼리 모인 그룹 | "같은 동네 친구들" |
| 클러스터링 | Clustering | 데이터를 그룹으로 나누는 작업 | "반 편성하기" |
| K값 | K (number of clusters) | 몇 개 그룹으로 나눌지 결정하는 수 | "반을 몇 개로 나눌까?" |
| 중심점 | Centroid | 각 클러스터의 중심 좌표 | "각 반의 대표 위치" |
| 관성 | Inertia | 클러스터 내 데이터 흩어진 정도 | "같은 반 친구들이 얼마나 모여 있나?" |
| 수렴 | Convergence | 더 이상 변화가 없어 알고리즘이 멈춤 | "이제 자리를 다 잡았다!" |
| 초기화 | Initialization | 중심점의 첫 위치를 정하는 방법 | "시작 위치를 어디로 할까?" |
| K-means++ | K-means++ | 중심점을 멀리 떨어뜨려 시작하는 방법 | "골고루 퍼져서 시작하자!" |
| 비지도 학습 | Unsupervised Learning | 정답 없이 패턴을 찾는 학습 방식 | "선생님 없이 스스로 그룹 짓기" |
| 할당 | Assignment | 각 데이터를 가장 가까운 중심점 그룹에 배정 | "가장 가까운 반으로 배정" |
