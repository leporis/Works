## ⚙️ 파라미터 가이드

### 학습 파라미터 (RL)

| 파라미터 | 범위 | 기본값 | 쉬운 설명 |
|---------|------|:------:|----------|
| 학습률 (α) | 0.01~0.5 | 0.10 | 새로운 경험을 얼마나 빨리 받아들이나 |
| 할인율 (γ) | 0.5~0.99 | 0.95 | 미래 보상을 얼마나 중요하게 보나 |
| 탐험률 (ε) | 0.01~1.0 | 1.00 | 무작위 탐험 vs 학습된 행동 비율 |
| ε 감소율 | 0.9~0.999 | 0.995 | 매 에피소드마다 ε를 얼마나 줄이나 |
| ε 최솟값 | 0.001~0.1 | 0.01 | ε가 줄어도 이 값 이하로는 안 내려감 |

### 보상 설정

| 보상 | 범위 | 기본값 | 의미 |
|------|------|:------:|------|
| 먹이 보상 | +1 ~ +50 | +10 | 🍎 먹이 획득 시 |
| 충돌 벌점 | -50 ~ -1 | -10 | 💀 벽/몸통 충돌 시 |
| 접근 보상 | 0 ~ +5 | +1 | 먹이에 한 칸 가까워졌을 때 |
| 멀어짐 벌점 | -5 ~ 0 | -1 | 먹이에서 한 칸 멀어졌을 때 |

> 💡 **실험 팁**: 먹이 보상을 +50, 충돌 벌점을 -1로 설정하면 어떻게 될까요? 반대로 하면?

|:---:|:---:|------|:---:|
| 0 | 🥚 | 미학습 | 0 이상 |
| 1 | 🐣 | 방황 중 | 0.1 이상 |
| 2 | 🐥 | 눈치 있음 | 0.5 이상 |

## 🆘 도와주세요!

| 증상 | 원인 | 해결책 |
|------|------|--------|
| AI가 벽에 계속 부딪힘 | ε=1.0 상태 (순수 탐험) | 100~200 에피소드 기다리기 (ε가 자연 감소) |
| 점수가 전혀 오르지 않음 | 학습률 너무 낮거나 보상 불균형 | α=0.10, 보상 기본값으로 리셋 |
| 학습이 너무 느림 | FPS가 낮음 | 속도를 60+(터보)로 올리기 |
| 직접 플레이에서 조작 안 됨 | 모드가 AI 학습임 | **🎮 직접 플레이** 탭으로 전환 |
| AI 시연에서 바보같이 움직임 | 학습 부족 (에피소드 < 100) | AI 학습 모드로 돌아가서 더 학습 |
| 그리드가 너무 작음/큼 | 맵 크기 설정 | 🗺️ 맵 설정에서 크기 변경 (10~30) |
| 레벨이 마스터까지 안 감 | 그리드가 커서 난이도 높음 | 그리드를 10×10으로 줄이고 학습 |
| 차트가 안 보임 | 에피소드 0회 | 학습을 시작해야 차트에 데이터 쌓임 |

## 🔑 핵심 용어

| 용어 | 학술 용어 | 의미 | 쉬운 설명 |
|------|----------|------|----------|
| 강화학습 | Reinforcement Learning | 보상/벌점으로 행동을 학습하는 방식 | "칭찬받은 행동은 더 하고, 혼난 행동은 안 하기" |
| Q-Learning | Q-Learning | 상태-행동 쌍의 가치를 학습하는 알고리즘 | "이 상황에서 어느 방향이 가장 좋은지 점수 매기기" |
| Q-테이블 | Q-Table | 모든 (상태, 행동) 조합의 가치를 저장하는 표 | "경험 노트: 이 상황에서 위로 가면 +5점" |
| 에피소드 | Episode | 게임 시작~사망까지의 한 판 | "한 번의 게임 플레이" |
| 탐험률 (ε) | Epsilon | 무작위 행동을 선택할 확률 | "모험심: 높으면 새로운 길 시도, 낮으면 아는 길 고수" |
| 할인율 (γ) | Gamma (Discount Factor) | 미래 보상의 현재 가치 비율 | "참을성: 높으면 미래 보상 중시, 낮으면 당장의 보상 중시" |
| 학습률 (α) | Learning Rate (Alpha) | 새 경험을 반영하는 비율 | "적응력: 높으면 새 경험에 빨리 적응, 낮으면 천천히" |
| 보상 | Reward | 행동 결과에 대한 피드백 신호 | "성적표: +점은 잘한 것, -점은 못한 것" |
| ε-greedy | Epsilon-Greedy | ε 확률로 탐험, 1-ε 확률로 최적 행동 | "가끔은 새로운 시도, 대부분은 최선의 행동" |
| 수렴 | Convergence | 학습이 안정되어 성능 변화가 적어짐 | "실력이 완성되어 더 이상 크게 변하지 않는 상태" |
