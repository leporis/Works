| 상황 | 참조 섹션 |

## ⚡ 30초 초속성 실습

| 단계 | 행동 | 확인 |
|:---:|------|:----:|
| 1 | 상단 데이터셋 드롭다운에서 **🐧 펭귄** 확인 (기본 선택) | 우측에 데이터 정보가 표시되나요? |
| 2 | 좌측 **🌳 트리 생성** 버튼 클릭 | 트리가 노드별로 자라나는 애니메이션이 보이나요? |
| 3 | 상단 **🔍 추론 모드** 탭 → 테스트 데이터 선택 → 추론 경로 확인 | 트리에서 노란색 경로가 표시되나요? |

## 🚀 빠른 시작

### 학습 & 분석 모드

```
① 데이터셋 선택 → ② 트리 설정 → ③ 트리 생성 → ④ 결과 분석
```

| 단계 | 어디서 | 무엇을 |
|:---:|--------|--------|
| ① | 상단 데이터셋 드롭다운 | 🐧 펭귄 또는 🌺 붓꽃 선택 (CSV 업로드도 가능) |
| ② | 좌측 패널 "⚙️ 트리 설정" | 최대 깊이(1~8, 기본 4), 최소 샘플 수(1~10, 기본 2), 사용할 특성 선택 |
| ③ | 좌측 **🌳 트리 생성** 버튼 | 클릭 → 트리 성장 애니메이션 재생 |
| ④ | 중앙 + 우측 패널 | 트리 시각화, 2D 결정 경계, 정확도, 혼동행렬, 특성 중요도, 학습된 규칙 |

### 추론 모드

```
① 추론 모드 전환 → ② 입력 방법 선택 → ③ 결과 확인
```

| 단계 | 어디서 | 무엇을 |
|:---:|--------|--------|
| ① | 상단 **🔍 추론 모드** 탭 | 클릭 → 추론 화면 전환 |
| ② | 좌측 패널 탭 선택 | **📋 테스트 데이터** (드롭다운 선택) 또는 **✏️ 직접 입력** (슬라이더) |
| ③ | 중앙 트리 + 우측 결과 | 추론 경로(노란색), 단계별 조건, 최종 분류 결과 |

## 🖥️ 화면 구성

### 학습 & 분석 모드 (3열 + 하단)

```
┌──────────┬──────────────────────────────┬─────────┐
│  좌측     │        중앙 영역              │  우측    │
│ ⚙️ 트리  │ ┌────────────────────────┐  │ 📊 데이터│
│   설정    │ │  🌳 의사결정트리 시각화   │  │   정보  │
│          │ │  (성장 애니메이션)        │  │         │
│ 최대 깊이 │ ├────────────────────────┤  │ 📏 모델 │
│ 최소 샘플 │ │  🎯 2D 결정 경계         │  │   성능  │
│ 특성 선택 │ │  (산점도 + 경계 영역)     │  │         │
│          │ └────────────────────────┘  │ 📊 특성 │
│ 🌳 트리  │                             │  중요도  │
│   생성   │                             │         │
│ 🎬 애니  │                             │         │
│  메이션  │                             │         │
├──────────┴──────────────────────────────┴─────────┤
│ 📜 학습된 규칙          │ 📊 테스트 결과 상세        │
└──────────────────────────────────────────────────┘
```

### 추론 모드

```
┌──────────┬──────────────────────────────┬─────────┐
│ 🔍 추론  │   🔍 추론 경로 시각화          │ 💡 추론 │
│   입력   │  (트리에 노란색 경로 표시)      │   결과  │
│          │                             │         │
│ 📋 테스트│                             │ 예측    │
│  데이터  │                             │ 클래스  │
│ ✏️ 직접  │                             │         │
│  입력   │  📋 추론 과정 (단계별)         │         │
│          │  단계1 → 단계2 → 최종 분류    │         │
└──────────┴──────────────────────────────┴─────────┘
```

## 📂 데이터 입력

### 내장 데이터셋

| 데이터셋 | 설명 | 데이터 수 | 클래스 | 특성 수 |
|---------|------|:--------:|--------|:------:|
| 🐧 펭귄 | Palmer Penguins 종 분류 | 약 130 | Adelie, Chinstrap, Gentoo | 4 |
| 🌺 붓꽃 | Iris 품종 분류 | 150 | Setosa, Versicolor, Virginica | 4 |

### CSV 업로드

상단 **📂 CSV 업로드** 버튼으로 외부 데이터를 불러올 수 있습니다.
1. CSV 파일 선택 → 미리보기 모달 표시
2. 클래스(분류 대상) 열 지정 → **✅ 적용**
3. 나머지 숫자형 열은 자동으로 특성으로 등록

## ⚙️ 트리 설정 가이드

| 설정 | 범위 | 기본값 | 효과 |
|------|------|--------|------|
| 최대 깊이 | 1~8 | 4 | 클수록 복잡한 트리 (과적합 위험), 작을수록 단순 (과소적합) |
| 최소 샘플 수 | 1~10 | 2 | 클수록 분할 조건이 엄격해져 단순한 트리 |
| 사용할 특성 | 체크박스 | 전체 | 선택한 특성만 분할 기준으로 사용 |

> 💡 **팁**: 깊이 1~2로 시작하여 트리가 어떻게 생기는지 관찰한 뒤, 점차 깊이를 늘려 보세요.

### 결과 패널 읽기

| 결과 | 위치 | 의미 |
|------|------|------|
| 학습/테스트 정확도 | 우측 📏 모델 성능 | 80:20 분할 후 각 데이터셋의 정확도 |
| 혼동행렬 | 우측 📏 모델 성능 | 클래스별 정분류/오분류 교차표 |
| 특성 중요도 | 우측 📊 | 각 특성이 분류에 기여하는 비율 막대 차트 |
| 학습된 규칙 | 하단 📜 | IF-THEN 형식의 분류 규칙 목록 |
| 테스트 결과 | 하단 📊 | 테스트 데이터 각각의 예측/실제/정오 상세 |

## 🆘 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 트리가 안 보임 | 트리 미생성 | **🌳 트리 생성** 클릭 |
| "특성을 1개 이상 선택해주세요" | 특성 미체크 | 좌측 체크박스에서 1개 이상 선택 |
| 추론 모드 빈 화면 | 트리 미생성 | 학습 모드에서 트리 먼저 생성 |
| 테스트 정확도가 학습보다 많이 낮음 | 과적합 | 최대 깊이 줄이기 또는 최소 샘플 수 높이기 |
| 결정 경계가 안 보임 | 특성 1개만 선택 | 특성 2개 이상 선택해야 2D 경계 표시 |
| 애니메이션이 너무 빠름/느림 | 속도 기본값 | 🐢-🐇 슬라이더로 속도 조절 |
| CSV 업로드 실패 | 형식 오류 | UTF-8 인코딩 + 쉼표 구분 CSV 확인 |

## 🔑 핵심 용어

| 용어 | 의미 |
|------|------|
| **의사결정트리** | 조건(질문)에 따라 데이터를 반복적으로 나누어 분류하는 트리 구조 알고리즘 |
| **노드** | 트리의 각 분기점. 내부 노드(조건 분기)와 리프 노드(최종 분류)로 구분 |
| **지니 불순도** | 데이터가 얼마나 섞여 있는지를 나타내는 지표 (0=순수, 0.5=최대 혼합) |
| **최대 깊이** | 트리가 성장할 수 있는 최대 레벨 수 |
| **최소 샘플 수** | 노드를 더 분할하기 위한 최소 데이터 수 |
| **특성 중요도** | 각 특성이 분류 정확도 향상에 기여하는 비율 |
| **결정 경계** | 분류 결과가 바뀌는 경계선 (2D 산점도에서 색상 영역) |
| **혼동행렬** | 실제 클래스와 예측 클래스의 교차 빈도표 |
| **과적합** | 트리가 너무 깊어 훈련 데이터에만 맞고, 새 데이터에 취약한 상태 |
| **리프 노드** | 트리의 끝 노드. 최종 분류 결과를 담고 있음 |
