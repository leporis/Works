## 1. 교육 활용 개요

### 1.1 교육적 가치

K-means 클러스터링 시뮬레이터는 다음 세 가지 측면에서 교육적 가치를 제공합니다.

**① 비지도 학습의 핵심 개념 체험:**
"정답 없이 데이터의 구조를 발견한다"는 비지도 학습의 본질을 직관적으로 경험할 수 있습니다. 레이블 없는 회색 점들이 K개의 색상 그룹으로 자동 분류되는 과정이 이를 시각적으로 보여줍니다.

**② 반복적 최적화 과정 시각화:**
중심점이 반복적으로 이동하며 최적 위치를 찾아가는 과정을 재생 기능으로 관찰할 수 있어, "알고리즘이 스스로 학습한다"는 개념을 구체적으로 이해할 수 있습니다.

**③ 하이퍼파라미터(K)의 역할 탐구:**
K값 선택이 결과에 결정적 영향을 미치는 점, 그리고 "최적의 K"를 찾기 위해 여러 값을 시도해야 한다는 점을 실험적으로 깨달을 수 있습니다. 이는 머신러닝 전반의 모델 선택 문제와 연결됩니다.

### 1.2 대상 학습자

| 대상 | 적합도 | 활용 방법 |
|------|:------:|----------|
| 초등 5~6학년 | ⭐⭐ | "비슷한 친구끼리 모이기" 비유로 군집화 개념 체험 |
| 예비교사 | ⭐⭐⭐ | 비지도 학습 원리 이해 + 지도학습과의 차이 비교 |
| 현직교사 연수 | ⭐⭐⭐ | K-means 원리 + 실생활 군집화 사례 연결 |
| 통계·데이터 교육 | ⭐⭐⭐ | 클러스터링 알고리즘, Inertia, 엘보 방법 심화 |

## 3. 수업 시나리오

### 시나리오 A: 비지도 학습 입문 — "정답 없이 분류하기" (15분)

**목표:** 비지도 학습의 개념을 이해하고, K-means로 데이터를 그룹으로 나눌 수 있다.

| 시간 | 활동 | 교사 발문 |
|:----:|------|----------|
| 0~2분 | 동기유발: "선생님 없이 스스로 그룹 짓기" | "정답을 모르는 상태에서 비슷한 것끼리 모을 수 있을까요?" |
| 2~5분 | 데이터 100개, K=3, 분산도 "낮음" → 생성 | "회색 점들에는 이름표(레이블)가 없어요. 어떻게 나눌 수 있을까요?" |
| 5~8분 | **⚡ 클러스터링 실행** → 결과 관찰 | "회색 점이 세 가지 색으로 나뉘었어요! 컴퓨터는 어떻게 나눈 걸까요?" |
| 8~12분 | **▶️ 재생** → 중심점 이동 과정 관찰 | "큰 원(중심점)이 어떻게 움직이나요? 왜 그렇게 움직일까요?" |
| 12~15분 | 정리: 지도학습(KNN)과 비교 토론 | "KNN과 K-means의 차이는 무엇일까요? 데이터에 이름표가 있었나요?" |

**학생 도달 기대:**
- "비지도 학습은 정답 없이 비슷한 데이터끼리 묶는 것"이라고 설명할 수 있음
- 중심점이 반복적으로 이동하며 그룹을 찾아가는 과정을 이해함

### 시나리오 B: K값 탐구 실험 — "몇 개로 나누는 게 좋을까?" (25분)

**목표:** K값 변화가 클러스터링 결과에 미치는 영향을 실험으로 확인하고, 적절한 K를 선택하는 기준을 탐구한다.

| 시간 | 활동 | 교사 지시 |
|:----:|------|----------|
| 0~3분 | 실험 안내 및 기록표 배부 | "같은 데이터를 K=2, 3, 4, 5로 나눠보고 Inertia를 기록해봅시다" |
| 3~8분 | 데이터 100개, 분산도 "보통" 생성 → K=2로 실행 | "Inertia 값과 클러스터 모양을 기록하세요" |
| 8~13분 | 동일 데이터에서 K=3, 4, 5 순차 실행 | "K를 늘릴 때마다 Inertia가 어떻게 변하나요?" |
| 13~18분 | K=8로 실행 → 과도한 분할 관찰 | "K=8이면 Inertia가 더 낮은데, 이게 좋은 분류일까요?" |
| 18~22분 | Inertia 변화 그래프 그리기 (엘보 방법) | "Inertia가 급격히 줄다가 완만해지는 지점은 어디인가요?" |
| 22~25분 | 정리 토론 | "왜 Inertia가 가장 낮은 K가 항상 좋은 건 아닐까요?" |

**📝 학생 기록표:**

### 시나리오 C: 초기화 방법 비교 — "출발점이 중요하다!" (20분)

**목표:** K-means의 확률적 특성을 이해하고, 초기화 방법이 결과 품질에 미치는 영향을 탐구한다.

| 시간 | 활동 | 교사 발문 |
|:----:|------|----------|
| 0~3분 | 데이터 100개, K=3, 분산도 "보통" → 생성 | "이제 같은 데이터에서 두 가지 방법으로 시작해봅시다" |
| 3~8분 | 초기화 "K-means++" → 5회 실행, Inertia 기록 | "5번 실행한 Inertia 값이 비슷한가요, 많이 다른가요?" |
| 8~13분 | 초기화 "랜덤" → 5회 실행, Inertia 기록 | "랜덤 방법의 Inertia 값은 어떤가요? K-means++와 비교하면?" |
| 13~16분 | **▶️ 재생**으로 랜덤의 좋은 결과 vs 나쁜 결과 비교 | "중심점이 처음에 어디서 시작했는지 보세요. 어떤 차이가 있나요?" |
| 16~20분 | 정리: 왜 초기값이 중요한지 토론 | "출발점에 따라 결과가 달라지는 것은 일상에서 어떤 상황과 비슷할까요?" |

### 시나리오 D: 추론 모드 탐구 — "새 데이터는 어느 그룹?" (10분)

**목표:** 학습된 클러스터링 결과를 활용하여 새 데이터의 그룹을 예측하는 과정을 체험한다.

## 4. 실습 활동지

### 활동 1: K-means 기본 체험

**목표:** K-means 알고리즘의 기본 작동을 이해한다.

**단계:**
1. [ ] 데이터 100개, K=3, 분산도 "낮음" → **✨ 데이터 생성** 클릭
2. [ ] 초기화 방법 "K-means++" 확인 → **⚡ 클러스터링 실행** 클릭
3. [ ] 결과 확인: 반복 횟수 ____회, Inertia ____, 상태 ________
4. [ ] **▶️ 재생** 클릭 → 중심점이 움직이는 방향 관찰
5. [ ] 추론 모드 전환 → (50, 50) 좌표의 할당 클러스터 확인: 클러스터 ____

**질문:**
- 중심점은 어떤 방향으로 이동했나요?
- 왜 그 방향으로 움직였다고 생각하나요?

### 활동 2: 분산도 실험

**목표:** 데이터 분산도가 클러스터링 품질에 미치는 영향을 탐구한다.

|:----:|:------:|:--------:|:-------:|:---------:|:---:|

**질문:**
- 분산도가 높을 때 그룹 구분이 어려워지는 이유는 무엇일까요?
- 실생활에서 "구분이 모호한 그룹"의 예시를 들어보세요.

### 활동 3: 실생활 연결 사고

**상황:** 학교 도서관에서 1,000권의 책을 주제별로 분류하려고 합니다.

- 이 상황에서 K-means의 "데이터"는 무엇인가요?
- K값은 어떻게 결정할 수 있을까요?
- 분류 결과가 매번 다르다면 어떻게 해야 할까요?

## 5. 교수·학습 팁

### 5.1 비유 활용

| K-means 개념 | 쉬운 비유 |
|-------------|----------|
| 클러스터링 | 체육 시간에 "비슷한 키끼리 모여보세요!" |
| 중심점(Centroid) | 각 모둠의 "대표 자리" |
| 중심점 이동 | 대표가 모둠원들 가운데로 자리를 옮기기 |
| 수렴 | "더 이상 자리를 옮길 필요가 없어요!" |
| K값 선택 | "몇 모둠으로 나눌까?" 결정하기 |
| 비지도 학습 | 선생님 없이 학생들끼리 스스로 모둠 만들기 |

### 5.2 흔한 오개념과 교정

| 오개념 | 교정 |
|--------|------|
| "K-means가 정답을 찾는다" | K-means는 "정답"이 없는 상황에서 패턴을 발견하는 것. 동일 데이터에 여러 "좋은 분류"가 가능 |
| "K가 클수록 좋다" | K를 늘리면 Inertia는 줄지만, 의미 없는 과도한 분할이 됨. 적절한 K를 찾는 것이 핵심 |
| "K-means는 항상 같은 결과를 낸다" | 초기화가 랜덤이므로 매번 다른 결과 가능. 여러 번 실행 후 최적 결과 선택 |
| "Inertia가 0이면 완벽한 분류" | K=데이터 수이면 Inertia=0이지만 의미 없음. 적절한 그룹화가 목표 |

### 5.3 지도학습과의 비교 교수법

K-means를 KNN 분류 시뮬레이터와 함께 사용하면 지도/비지도 학습의 차이를 효과적으로 교수할 수 있습니다.

| 비교 항목 | KNN (지도학습) | K-means (비지도 학습) |
|----------|:---:|:---:|
| 데이터에 레이블 | ✅ 있음 (색상별 분류) | ❌ 없음 (모두 회색) |
| 학습 목표 | 새 데이터의 클래스 예측 | 데이터의 그룹 구조 발견 |
| K의 의미 | 이웃 수 (참조 범위) | 그룹 수 (분류 개수) |
| 결과 평가 | 정확도 (정답과 비교) | Inertia (그룹 내 응집도) |
| 결정 방식 | 이웃의 다수결 투표 | 가장 가까운 중심점 |
