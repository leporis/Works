### 4.1 전개 1: 데이터의 역할 (10분)

**핵심 질문**: "AI가 학습하려면 무엇이 필요할까요?"

| 순서 | 내용 | 핵심 메시지 |
|:---:|:---|:---|
| 1 | 1주차 복습: 머신러닝 4단계 | 데이터 수집 → 학습 → 평가 → 예측 |
| 2 | 데이터 = AI 학습의 연료 | 좋은 연료 → 좋은 결과 |
| 3 | 데이터 유형: 이미지, 텍스트, 소리 | 다양한 형태의 데이터가 AI에 활용 |
| 4 | 양과 질의 중요성 | 양 ↑ 질 ↑ → 좋은 AI, 양 ↓ 질 ↓ → 나쁜 AI |

**사용 그림**: 그림 01 머신러닝_4단계_프로세스 (재사용), 그림 02 데이터유형_이미지텍스트소리 (신규)

**교수 전략**:

**1) 머신러닝 4단계 복습 (3분)**
- 1주차 2차시 학습 내용 상기
- "기억나시죠? 데이터 수집 → 학습 → 평가 → 예측"
- **초점**: "오늘은 이 중에서 첫 번째, **데이터**에 집중합니다"

**2) 데이터 유형 소개 (4분)**
- **이미지**: 고양이/개 사진, 의료 X-ray
- **텍스트**: 이메일, 뉴스 기사, SNS 게시물
- **소리**: 음성 인식, 음악 분석
- **비유**: "데이터 = AI의 교과서. 어떤 교과서로 공부하느냐가 실력을 결정합니다"

**3) 양과 질 강조 (3분)**
- **양**: 데이터가 많을수록 다양한 패턴 학습 가능
- **질**: 정확하고 다양한 데이터 = 좋은 AI
- **반례**: 양만 많고 질이 나쁘면? → 편향 문제 (전개 2로 연결)

### 4.2 전개 2: 데이터 편향 (10분)

**핵심 질문**: "데이터가 한쪽에 치우치면 어떤 일이 벌어질까요?"

| 순서 | 내용 | 핵심 메시지 |
|:---:|:---|:---|
| 1 | 편향 개념 상기 | 1주차 2차시 편향 내용 복습 |
| 2 | 실제 사례 3가지 | 얼굴인식, 채용 AI, 번역 AI |
| 3 | 원인과 해결 | 한쪽에 치우친 데이터 → 다양한 데이터 수집 |
| 4 | 교육적 의미 | AI 교육에서 데이터 다양성 강조 필요 |

**사용 그림**: 그림 03 편향된데이터vs좋은데이터 (재사용)

**교수 전략**:

**1) 편향 개념 상기 (2분)**
- "1주차에서 편향이란 단어, 기억나시죠?"
- **편향 = 데이터가 특정 방향에 치우친 상태**
- 비유: "한쪽 눈만 가리고 세상을 보는 것"

**2) 실제 사례 3가지 (5분)**

| # | 사례 | 편향 원인 | 결과 |
|:---:|:---|:---|:---|
| 1 | 얼굴인식 AI | 밝은 피부 데이터 위주 학습 | 어두운 피부 인식 정확도 저하 |
| 2 | 채용 AI | 과거 남성 위주 채용 데이터 | 여성 지원자 불이익 |
| 3 | 번역 AI | 성별 고정관념 포함 텍스트 | "간호사=she, 의사=he" 고정 번역 |

**3) 원인-해결 정리 (3분)**

| 원인 | 해결 방향 |
|:---|:---|
| 특정 집단 데이터만 수집 | 다양한 집단의 데이터 균형 수집 |
| 과거 편견 포함 데이터 | 편향 감지 도구로 사전 점검 |
| 데이터 수집자의 무의식적 편향 | 수집 과정에서 다양성 체크리스트 활용 |

### 4.3 전개 3: 딥러닝과 신경망 (15분)

**핵심 질문**: "AI의 뇌는 어떻게 생겼을까요?"

| 순서 | 내용 | 핵심 메시지 | 비고 |
|:---:|:---|:---|:---:|
| 1 | 신경망 기본 개념 | 입력층 → 은닉층 → 출력층 구조 | 핵심 CK |
| 2 | 신경망과일분류시뮬레이터 시연 | 학습 과정과 결과를 직접 관찰 | 핵심 CK |
| 3 | 딥러닝 = 깊은 신경망 → CNN 개념 | 은닉층이 많을수록 복잡한 패턴 학습 | ⭐ 심화 |
| 4 | CNN 이미지필터 시뮬레이터 시연 | 필터가 이미지에서 특징을 추출하는 과정 체험 | ⭐ 심화 |

> ⭐ **심화 학습 안내**: 순서 3~4(CNN 개념 및 시뮬레이터)와 그림 04(뉴런·퍼셉트론)는 딥러닝 이해를 돕기 위한 **심화 내용**입니다. 2주차 퀴즈 출제 범위에 포함되지 않습니다.

**사용 그림**: 그림 04 뉴런과퍼셉트론 (신규, ⭐심화), 그림 05 신경망기본구조_입력은닉출력 (신규), 그림 06 딥러닝특징추출_단계별 (신규)

**교수 전략**:

**1) 신경망 기본 개념 (5분)**
- **비유**: "AI의 뇌 구조"
  - 입력층 = 눈 (정보를 받아들임)
  - 은닉층 = 뇌 (정보를 처리/분석)
  - 출력층 = 입/손 (결과를 내놓음)
- 1주차 AI/ML/DL 포함관계 상기: "DL이 ML 안에 있다고 했죠?"
- **딥러닝 = 은닉층이 여러 개인 신경망**
  - 층이 많을수록 복잡한 특징 학습 가능
- ⭐ **심화**: 뉴런과 퍼셉트론 비교 (수상돌기↔입력×가중치, 세포체↔합산, 축삭↔활성화함수)
  - "생물학적 뇌에서 영감을 얻었지만, 실제 뇌와는 매우 다릅니다"

**2) 신경망과일분류시뮬레이터 시연 (5분)**

> **시뮬레이터**: 신경망과일분류시뮬레이터 v1.0.0.0
> - **구조**: 입력 2개(크기, 빨간 정도) → 은닉 6개 → 출력 4개(🍎사과/🍊오렌지/🍋레몬/🍇포도)
> - **학습 데이터**: 12개 (과일 4종 × 3개)
> - **학습 단계**: 6단계 (초기화 → 순전파 → 오차계산 → 역전파 → 가중치갱신 → 반복)

**시연 3단계**:

| 단계 | 시간 | 활동 | 관찰 포인트 |
|:---:|:---:|:---|:---|
| ① 구조 관찰 | 1분 | 입력/은닉/출력 노드와 연결선 확인 | "입력 2개 → 은닉 6개 → 출력 4개, 모두 연결!" |
| ② 학습 과정 | 2분 | [학습 시작] 클릭 → 가중치 변화 관찰 | "연결선 굵기가 변한다 = 가중치 조정 중!" |
| ③ 학습 후 테스트 | 2분 | 테스트 카드 → 랜덤 → 경계 영역 | "새로운 과일도 분류 가능 = 학습 완료!" |

**3) ⭐ 심화: 딥러닝과 CNN 개념 연결 + 시뮬레이터 시연 (5분)**
- "지금 본 시뮬레이터는 은닉층 1개. 이걸 여러 개로 늘리면?"
- **딥러닝 = Deep(깊은) Learning**: 은닉층이 여러 개
- **CNN (합성곱 신경망)**: 이미지 처리에 특화된 딥러닝
  - 각 은닉층이 **필터(커널)**로 이미지의 특징을 추출
  - 1층: 선/윤곽선 → 2층: 모양/질감 → 3층: 물체 전체
- "필터가 어떻게 동작하는지 직접 체험해 봅시다!"

**CNN 이미지필터 시뮬레이터 시연**:

> **시뮬레이터**: CNN 이미지필터 체험 시뮬레이터 v1.0.0.0
> - **입력**: 10×10 픽셀 그리드 (클릭/드래그로 그리기, 흑백)
> - **프리셋**: 8종 (가로선, 세로선, 십자, 대각선, 사각형, 원형, T글자, 화살표)
> - **필터(커널)**: 7종 3×3 커널 — 원본(항등), 가로윤곽선, 세로윤곽선, 전체윤곽선(라플라시안), 선명하게, 흐리게(블러), 엠보싱
> - **출력**: 8×8 결과 그리드 (양수=파랑, 음수=빨강, 0=회색)
> - **시각화**: 합성곱 연산 단계별 애니메이션, 계산 과정 실시간 표시

**시연 3단계**:

| 단계 | 시간 | 활동 | 관찰 포인트 |
|:---:|:---:|:---|:---|
| ① 그리기 + 프리셋 | 1분 | 십자(+) 프리셋 선택 또는 직접 그리기 | "10×10 칸에 흑백으로 그림을 입력!" |
| ② 필터 적용 | 2분 | 가로윤곽선 → 세로윤곽선 → 전체윤곽선 순서 | "같은 그림인데 필터마다 결과가 다르다!" |
| ③ 애니메이션 관찰 | 1분 | [단계별 재생] → 3×3 영역 하이라이트 | "필터가 이미지 위를 훑으며 특징을 추출!" |

**핵심 메시지**: "CNN은 이런 필터를 스스로 학습합니다. 어떤 필터가 좋은지 AI가 데이터로부터 배워요!"

### 4.4 정리 (5분)

**핵심 개념 3가지 요약**:

1. **데이터의 역할**: AI 학습의 연료 — 양과 질이 모두 중요
2. **데이터 편향**: 치우친 데이터 → 불공정한 AI → 다양한 데이터 수집이 해결책
3. **딥러닝과 신경망**: 여러 층의 신경망으로 복잡한 패턴 학습 (입력층 → 은닉층 → 출력층)

**딥러닝 성공 사례**:

| 연도 | 사건 | 의미 |
|:---:|:---|:---|
| 2012 | ImageNet 대회: 딥러닝 이미지 인식 압도적 성능 | 딥러닝 가능성 증명 |
| 2016 | AlphaGo: 바둑 세계 챔피언 승리 | 딥러닝 + 강화학습 |
| 2022 | ChatGPT 출시: LLM 대중화 | 딥러닝 기반 생성형 AI |

- 1주차 "제3의 봄"과 연결: "이 모든 성공의 비결이 바로 딥러닝이에요!"

**교육과정 연결**:
- [6실05-04]: 데이터 유형(이미지/텍스트/소리)과 AI 학습의 관계
- [6실05-05]: AI(딥러닝) 원리 이해

**엔트리 AI 연결**:
- 3~4주차에 배울 엔트리 AI 모델에서 데이터 수집·학습 과정을 직접 체험
- "오늘 배운 데이터의 중요성, 편향 문제가 엔트리 실습에서도 그대로 나타납니다!"

**다음 차시 예고**: 2주차 2차시 - 생성형 AI 심화 (LLM 원리, 증강/자동화, 프롬프트 엔지니어링)

## 6. 교수 전략 및 유의사항

### 6.1 학생 수준 고려

**예비교사 대상 특성**:
- 수학적·기술적 세부사항(역전파, 활성화 함수, 합성곱 수식 등)은 다루지 않음
- 개념적 이해와 교육적 의미에 집중
- "초등학생에게 어떻게 설명할까?" 관점 유지

### 6.2 개념 설명 전략

**추상적 개념을 구체화**:
- 데이터 = 요리 재료 (양과 질)
- 편향 = 한쪽에 치우친 안경 (세상을 삐뚤게 봄)
- 신경망 = AI의 뇌 구조 (눈→뇌→손)
- 딥러닝 = "깊은 학습" (층이 많아서 복잡한 것도 이해)
- CNN 필터 = 돋보기 (이미지에서 특정 특징만 골라 봄)

**1주차 내용과의 연결 강조**:
- 전개 1: 1주차 2차시 머신러닝 4단계 중 "데이터 수집" 심화
- 전개 2: 1주차 2차시 편향 개념 상기 및 사례 심화
- 전개 3: 1주차 1차시 AI/ML/DL 포함관계 중 "DL" 심화

### 6.3 퀴즈 운영 유의사항

| 항목 | 유의 사항 |
|:---|:---|
| 시간 관리 | 퀴즈 10분 엄수 → 강의 시간 확보 |
| 분위기 전환 | 퀴즈 후 "수고했어요!" → 부드러운 강의 시작 |
| 정답 공유 | 상세 해설보다 정답만 빠르게 공유 (시간 절약) |

### 6.4 시뮬레이터 운영 유의사항

| 항목 | 신경망과일분류시뮬레이터 | CNN 이미지필터 시뮬레이터 |
|:---|:---|:---|
| 사전 준비 | 브라우저에 HTML 파일 미리 로드 | 동일 |
| 화면 공유 | 교수 화면 프로젝터 투사 | 동일 |
| 학생 참여 | "다음에 무슨 과일이 나올까요?" 예측 유도 | "이 필터를 적용하면 어떤 결과가 나올까요?" 예측 유도 |
| 시간 초과 방지 | 학습 에포크 수 미리 설정 | 프리셋 이미지 사용 (직접 그리기 시간 절약) |
| 오류 대응 | 새로고침으로 초기화 가능 | 동일 |

### 6.5 학생 오개념 대응

| 오개념 | 올바른 이해 |
|:---|:---|
| "데이터가 많으면 무조건 좋다" | 편향된 데이터는 양이 많아도 문제 → 다양성이 핵심 |
| "딥러닝 = AI 전체" | 딥러닝은 기계학습의 한 방법 (AI ⊃ ML ⊃ DL) |
| "신경망 = 실제 뇌와 같다" | 영감만 얻었을 뿐, 실제 뇌 작동 방식과는 매우 다름 |
| "은닉층이 많으면 무조건 좋다" | 적절한 깊이가 필요, 과도한 층은 오히려 성능 저하 가능 |
| "CNN 필터는 사람이 일일이 만든다" | 전통적 방법과 달리 CNN은 필터를 데이터로부터 자동 학습 |

## 7. 주요 용어 해설

2주차 1차시에서 새로 등장하는 핵심 전문용어를 정리합니다.

| 용어 (한글) | 영문 | 간단 정의 | 비유/예시 |
|:---|:---|:---|:---|
| **데이터** | Data | AI가 학습하는 재료 (숫자, 이미지, 텍스트, 소리 등) | 요리 재료 — 좋은 재료 = 좋은 요리 |
| **편향** | Bias | 데이터가 특정 유형에 치우친 상태 | 한쪽만 보는 안경 — 세상을 삐뚤게 봄 |
| **딥러닝** | Deep Learning | 여러 층의 신경망으로 학습하는 기계학습 방법 | "깊은 학습" — 층이 깊어서 복잡한 것도 파악 |
| **신경망** | Neural Network | 입력-은닉-출력 층으로 구성된 AI 모델 구조 | AI의 뇌 — 입력(눈), 은닉(뇌), 출력(입) |
| **입력층** | Input Layer | 데이터가 들어오는 첫 번째 층 | 눈: 정보를 받아들이는 곳 |
| **은닉층** | Hidden Layer | 데이터에서 특징을 추출하는 중간 층 | 뇌: 정보를 분석하는 곳 |
| **출력층** | Output Layer | 최종 결과가 나오는 마지막 층 | 입/손: 결과를 내놓는 곳 |
| **합성곱** | Convolution | 필터를 이미지 위에서 이동하며 특징을 추출하는 연산 | 돋보기로 이미지를 훑으며 살펴보기 |
| **필터/커널** | Filter/Kernel | 이미지에서 특정 특징(윤곽선 등)을 감지하는 작은 격자 | 특수 안경 — 각 안경마다 다른 것이 보임 |

**교수 전략:**
- 용어 첫 등장 시 한글 + 영문 병기 후 간단히 설명
- 예시: "딥러닝(Deep Learning)은 '깊은 학습'이라는 뜻이에요. 왜 깊다고 할까요? 신경망의 층이 많기 때문이에요!"
- 1주차에서 배운 용어(기계학습, 지도학습 등)와의 관계 강조
