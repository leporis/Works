## 3. 발견 사항 상세

### 3.1 ❌ [그림 정확성] 그림 03 내부 번호 불일치

| 항목 | 내용 |
|:---|:---|
| **심각도** | 높음 |
| **위치** | AI디지털리터러시_2주차1차시_03_편향된데이터vs좋은데이터.png |
| **문제** | 이미지 파일 내부에 **"그림 13:"**이라고 표시되어 있음 |
| **원인** | 1주차 2차시에서 그림 13번으로 사용된 이미지를 2주차 1차시에서 그림 03으로 재사용하면서 이미지 내부 번호를 갱신하지 않음 |
| **영향** | 슬라이드·교재에서 "그림 03"으로 참조하나, 이미지 자체에 "그림 13"이 표시되어 학생 혼동 유발 |
| **조치** | 이미지 내부 텍스트를 **"그림 03: 편향된 데이터 vs 좋은 데이터"**로 수정하거나, 내부 번호를 제거 |

### 3.2 ⚠️ [일관성] 시뮬레이터 입력 용어 불일치

| 항목 | 내용 |
|:---|:---|
| **심각도** | 낮음 |
| **위치** | 강의노트 — Part 2 시뮬레이터 사양 테이블 |
| **문제** | 강의노트에서 시뮬레이터 입력을 **"크기, 색상"**으로 기재 |
| **기준** | 강의개요·강의교재에서는 **"크기, 빨간 정도(Redness)"**로 표기 |
| **영향** | "색상"은 RGB 전체를 의미할 수 있어 시뮬레이터 실제 입력("빨간 정도")과 차이 |
| **조치** | 강의노트의 "색상" → **"빨간 정도"**로 통일 |
