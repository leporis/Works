### 슬라이드 3: 퀴즈 안내

- 범위: 1주차 전체 (1차시 + 2차시)
- 형식: 5지선다 10문항(0.5점×10) + 단답형 5문항(1점×5) = 10점
- 시간: 10분

### 슬라이드 4: 머신러닝 4단계 복습

- **[그림 01]** 머신러닝 4단계 프로세스
- 핵심 메시지: "가장 먼저 나오는 것이 **데이터 수집**입니다!"

### 슬라이드 5: 데이터 = AI의 연료

| 자동차 🚗 | AI 🤖 |
|:---:|:---:|
| 좋은 연료 → 잘 달린다 | 좋은 데이터 → 정확한 AI |
| 나쁜 연료 → 고장 | 나쁜 데이터 → 부정확한 AI |
| 연료 없음 → 멈춤 | 데이터 없음 → 학습 불가 |

> "Garbage In, Garbage Out"

### 슬라이드 6: 데이터 유형

- **[그림 02]** 이미지·텍스트·소리

### 슬라이드 7: 데이터의 양과 질

| 양↑ + 질↑ | ✅ 이상적 |
| 양↑ + 질↓ | ⚠️ 편향 확대 |
| 양↓ + 질↑ | 🔶 제한적이지만 정확 |
| 양↓ + 질↓ | ❌ 학습 어려움 |

### 슬라이드 8: 데이터 편향

- **[그림 03]** 편향된 데이터 vs 좋은 데이터
- 비유: "한쪽에 치우친 안경"

### 슬라이드 9: 실제 편향 사례 3가지

| 사례 | 문제 | 원인 |
|:---:|:---|:---|
| 👤 얼굴인식 | 피부색에 따른 정확도 차이 | 학습 데이터 77%가 밝은 피부 |
| 💼 채용 AI | 여성 지원자에게 불이익 | 과거 10년 남성 위주 채용 기록 |
| 🌐 번역 AI | 직업별 성별 고정 | 인터넷 텍스트의 고정관념 |

### 슬라이드 10: 편향 해결 방향

| 원인 | 해결 |
|:---|:---|
| 수집 범위 편향 | 다양한 집단 균형 수집 |
| 역사적 편향 | 편향 감지 도구로 사전 점검 |
| 고정관념 포함 | 데이터 전처리에서 편향 제거 |

### 슬라이드 11: 신경망이란?

- AI ⊃ ML ⊃ **DL** (복습)
- 신경망 = 인간의 뇌에서 **영감**을 얻은 AI 모델
- ⚠️ "영감"이지 "같다"가 아닙니다!

### 슬라이드 12: ⭐심화 — 뉴런 vs 퍼셉트론

- **[그림 04]** 생물학적 뇌 vs 인공지능
- 심화 표기 안내

### 슬라이드 13: 신경망 기본 구조

- **[그림 05]** 입력층 → 은닉층 → 출력층
- 비유: 눈 → 뇌 → 손

### 슬라이드 14: 학습 = 가중치 조정

- 순전파 → 오차 계산 → 역전파 → 반복
- 비유: "시험 → 채점 → 복습 → 반복"

### 슬라이드 15: 시뮬레이터 시연 — 과일분류

| 항목 | 내용 |
|:---|:---|
| 입력 | 크기, 빨간 정도 (2개) |
| 은닉층 | 뉴런 6개 |
| 출력 | 사과·오렌지·레몬·포도 (4개) |
| 관찰 | 연결선 굵기 변화 = 학습! |

### 슬라이드 16: 딥러닝 = 깊은 신경망

### 슬라이드 17: ⭐심화 — CNN 이미지 전문가

- CNN = 이미지 처리 특화 딥러닝
- 필터(커널): 특정 특징을 감지하는 작은 격자
- "돋보기로 이미지를 훑어보는 것"

### 슬라이드 18: ⭐심화 — CNN 시뮬레이터 시연

| 항목 | 내용 |
|:---|:---|
| 입력 | 10×10 픽셀 그리드 |
| 필터 | 7종 3×3 커널 |
| 출력 | 8×8 결과 그리드 |
| 핵심 | 같은 그림 + 다른 필터 → 다른 결과! |

### 슬라이드 19: 핵심 3가지

1. 데이터의 역할: AI 학습의 연료 — 양과 질 모두 중요
2. 데이터 편향: 치우친 데이터 → 불공정한 AI → 다양한 데이터가 해결책
3. 딥러닝과 신경망: 여러 층의 신경망으로 복잡한 패턴 학습
