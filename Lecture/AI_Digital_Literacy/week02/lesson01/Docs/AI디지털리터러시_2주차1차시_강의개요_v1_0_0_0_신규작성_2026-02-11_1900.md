### 4.2 전개 1: 데이터의 역할 (10분)

**핵심 질문**: "기계학습은 '무엇'으로 배울까요?"

| 순서 | 내용 | 핵심 포인트 |
|:---:|:---|:---|
| 1 | 1주차 복습 연결: 머신러닝 4단계 중 첫 번째 | 데이터 수집 → 학습 → 평가 → 예측 |
| 2 | 데이터 = AI 학습의 연료 | 자동차에 연료가 필요하듯, AI에는 데이터가 필요 |
| 3 | 데이터의 3가지 유형 | 이미지(사진), 텍스트(글), 소리(음성) |
| 4 | 데이터의 양: 많을수록 유리 | 공부할 문제가 많으면 실력이 오르듯 |
| 5 | 데이터의 질: 정확하고 다양해야 | 틀린 답으로 공부하면 시험에서도 틀림 |

**사용 그림**: 그림 01 데이터유형_이미지텍스트소리, 그림 02 데이터양과질_비교

**비유 활용**:
- **데이터 = 요리 재료**: 좋은 재료(신선, 다양) → 맛있는 요리. 나쁜 재료(상한, 한 종류) → 맛없는 요리
- **데이터 = 공부 교재**: 문제집이 많고 다양하면 실력 향상. 같은 문제만 반복하면 응용력 부족

**핵심 개념**:
- **데이터 유형**: AI는 이미지, 텍스트, 소리 등 다양한 형태의 데이터를 학습
- **양(Quantity)**: 데이터가 많을수록 패턴을 정확하게 파악 가능
- **질(Quality)**: 정확하고 다양한 데이터가 좋은 AI를 만듦

**전환**: "그런데, 데이터가 많으면 항상 좋을까요? 아닌 경우를 볼까요."

### 4.3 전개 2: 데이터 편향 (10분)

**핵심 질문**: "데이터가 많은데도 AI가 틀리는 이유는?"

| 순서 | 내용 | 핵심 포인트 |
|:---:|:---|:---|
| 1 | 편향(Bias)이란? | 한쪽으로 치우친 것 — 데이터가 특정 유형에 집중된 상태 |
| 2 | 사례 1: 얼굴 인식 AI의 편향 | 특정 인종 데이터 편중 → 다른 인종 인식률 저하 |
| 3 | 사례 2: 채용 AI의 편향 | 남성 이력서 중심 학습 → 여성 지원자 차별 |
| 4 | 편향의 원인 | 수집 단계의 편향, 라벨링 편향, 선택 편향 |
| 5 | 편향 해결 방안 | 다양한 데이터 수집, 편향 검사, 지속적 모니터링 |

**사용 그림**: 그림 03 편향된데이터사례_얼굴인식, 그림 04 좋은데이터수집_다양성

**교수 전략**:

**1) 편향 개념 도입 (3분)**
- **비유**: "서울 학생들의 사진만으로 '한국 초등학생 얼굴' AI를 만들면?"
  - → 지방 학생을 잘 인식 못할 수 있음
  - → 데이터가 서울에 편향되어 있기 때문
- 초등 교실 비유: "교실에 남학생만 있는 사진으로 학습하면, 여학생을 학생으로 인식 못하는 AI가 됩니다"

**2) 실제 사례 (4분)**
- **얼굴 인식**: 밝은 피부 데이터 중심 학습 → 어두운 피부 인식률 30% 이상 차이
- **채용 AI**: 과거 채용 데이터(남성 위주) 학습 → 여성 이력서에 낮은 점수
- **번역 AI**: "간호사는 여성, 의사는 남성"이라는 편향 반영

**3) 왜 중요한가? — 교육적 의미 (3분)**
- AI가 사회적 편견을 학습하고 강화할 수 있음
- **교사로서 알아야 할 이유**: 학생들이 AI를 올바르게 이해하고 비판적으로 활용하도록 지도
- 데이터 윤리: "AI가 공정하려면, 먼저 데이터가 공정해야 합니다"

**전환**: "데이터의 중요성을 알았으니, 이제 AI가 데이터를 어떻게 '깊이' 학습하는지 알아볼까요?"

### 4.4 전개 3: 딥러닝과 신경망 (15분)

**핵심 질문**: "AI가 고양이와 개를 구분하는 '눈'은 어떤 구조일까요?"

| 순서 | 내용 | 핵심 포인트 |
|:---:|:---|:---|
| 1 | 딥러닝(Deep Learning) 정의 | 여러 층의 신경망을 활용한 학습 — ML의 한 방법 |
| 2 | 왜 "딥(Deep)"인가? | 층(Layer)이 여러 개 = 깊다(Deep) |
| 3 | 신경망의 영감: 인간의 뇌 | 뉴런의 연결 구조에서 영감을 얻음 |
| 4 | 신경망 기본 구조 | 입력층 → 은닉층 → 출력층 |
| 5 | 각 층의 역할 | 입력(데이터) → 은닉(특징 추출) → 출력(결과) |
| 6 | 은닉층의 수 = 깊이(Depth) | 층이 많을수록 복잡한 패턴 학습 가능 |
| 7 | AI/ML/DL 관계 재확인 | AI ⊃ ML ⊃ DL — 딥러닝은 기계학습의 한 종류 |

**사용 그림**: 그림 05 신경망기본구조_3층, 그림 06 은닉층깊이비교_얕은vs깊은, 그림 07 고양이특징추출과정

**교수 전략**:

**1) 신경망 개념 (5분)**
- **비유**: "AI의 뇌 구조"
  - 입력층 = 눈, 귀 (정보를 받아들임)
  - 은닉층 = 뇌 (정보를 처리하고 분석)
  - 출력층 = 입, 손 (결과를 내놓음)
- 1주차의 AI/ML/DL 포함관계 그림 떠올리기
  - "DL이 ML 안에 있다고 했죠? 딥러닝은 신경망이라는 특별한 구조를 사용하는 기계학습이에요"

**2) 신경망 구조 상세 (5분)**
- **입력층 (Input Layer)**:
  - 데이터가 들어오는 곳
  - 예: 고양이 사진의 각 픽셀(점) 값
- **은닉층 (Hidden Layer)**:
  - 데이터에서 특징을 추출하는 곳
  - 1번째 은닉층: 선, 모서리 같은 단순한 특징
  - 2번째 은닉층: 눈, 귀 같은 복잡한 특징
  - 3번째 은닉층: "고양이 얼굴" 같은 전체적 특징
  - **"깊다(Deep)" = 은닉층이 많다**
- **출력층 (Output Layer)**:
  - 최종 결과가 나오는 곳
  - 예: "고양이 95%, 개 5%"

**3) 딥러닝 성공 사례 (5분)**

| 연도 | 사건 | 의미 |
|:---:|:---|:---|
| 2012 | **ImageNet 대회**: 딥러닝이 이미지 인식에서 압도적 성능 | 딥러닝의 가능성 증명 |
| 2016 | **AlphaGo**: 바둑에서 인간 세계 챔피언 승리 | 딥러닝 + 강화학습의 힘 |
| 2022 | **ChatGPT 출시**: 대규모 언어 모델(LLM) 대중화 | 딥러닝 기반 생성형 AI |

- 1주차 "제3의 봄"과 연결: "이 모든 성공의 비결이 바로 딥러닝이에요!"
- 다음 차시 예고: "ChatGPT 같은 생성형 AI가 어떻게 작동하는지 2차시에 알아볼 거예요"

## 6. 교수 전략 및 유의사항

### 6.1 학생 수준 고려

**예비교사 대상 특성**:
- 수학적·기술적 세부사항(역전파, 활성화 함수 등)은 다루지 않음
- 개념적 이해와 교육적 의미에 집중
- "초등학생에게 어떻게 설명할까?" 관점 유지

### 6.2 개념 설명 전략

**추상적 개념을 구체화**:
- 데이터 = 요리 재료 (양과 질)
- 편향 = 한쪽에 치우친 안경 (세상을 삐뚤게 봄)
- 신경망 = AI의 뇌 구조 (눈→뇌→손)
- 딥러닝 = "깊은 학습" (층이 많아서 복잡한 것도 이해)

**1주차 내용과의 연결 강조**:
- 전개 1: 1주차 2차시 머신러닝 4단계 중 "데이터 수집" 심화
- 전개 3: 1주차 1차시 AI/ML/DL 포함관계 중 "DL" 심화

### 6.3 퀴즈 운영 유의사항

| 항목 | 유의 사항 |
|:---|:---|
| 시간 관리 | 퀴즈 10분 엄수 → 강의 시간 확보 |
| 분위기 전환 | 퀴즈 후 "수고했어요!" → 부드러운 강의 시작 |
| 정답 공유 | 상세 해설보다 정답만 빠르게 공유 (시간 절약) |

### 6.4 학생 오개념 대응

| 오개념 | 올바른 이해 |
|:---|:---|
| "데이터가 많으면 무조건 좋다" | 편향된 데이터는 양이 많아도 문제 → 다양성이 핵심 |
| "딥러닝 = AI 전체" | 딥러닝은 기계학습의 한 방법 (AI ⊃ ML ⊃ DL) |
| "신경망 = 실제 뇌와 같다" | 영감만 얻었을 뿐, 실제 뇌의 작동 방식과는 매우 다름 |
| "은닉층이 많으면 무조건 좋다" | 적절한 깊이가 필요, 과도한 층은 오히려 성능 저하 가능 |

| 용어 (한글) | 영문 | 간단 정의 | 비유/예시 |
|:---|:---|:---|:---|
| **데이터** | Data | AI가 학습하는 재료 (숫자, 이미지, 텍스트, 소리 등) | 요리 재료 — 좋은 재료 = 좋은 요리 |
| **편향** | Bias | 데이터가 특정 유형에 치우친 상태 | 한쪽만 보는 안경 — 세상을 삐뚤게 봄 |
| **딥러닝** | Deep Learning | 여러 층의 신경망으로 학습하는 기계학습 방법 | "깊은 학습" — 층이 깊어서 복잡한 것도 파악 |
| **신경망** | Neural Network | 입력-은닉-출력 층으로 구성된 AI 모델 구조 | AI의 뇌 — 입력(눈), 은닉(뇌), 출력(입) |
| **입력층** | Input Layer | 데이터가 들어오는 첫 번째 층 | 눈: 정보를 받아들이는 곳 |
| **은닉층** | Hidden Layer | 데이터에서 특징을 추출하는 중간 층 | 뇌: 정보를 분석하는 곳 |
| **출력층** | Output Layer | 최종 결과가 나오는 마지막 층 | 입/손: 결과를 내놓는 곳 |
| **뉴런** | Neuron | 신경망의 기본 단위 (입력 → 처리 → 출력) | 뇌세포 하나 — 정보를 받고 전달 |

**교수 전략:**
- 용어 첫 등장 시 한글 + 영문 병기 후 간단히 설명
- 예시: "딥러닝(Deep Learning)은 '깊은 학습'이라는 뜻이에요. 왜 깊다고 할까요? 신경망의 층이 많기 때문이에요!"
- 1주차에서 배운 용어(기계학습, 지도학습 등)와의 관계 강조
