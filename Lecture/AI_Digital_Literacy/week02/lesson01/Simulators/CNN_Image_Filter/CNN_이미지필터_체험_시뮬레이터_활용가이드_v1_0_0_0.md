# 🔍 CNN 이미지 필터 체험 시뮬레이터 활용가이드

## 1. 개요

### 1.1 목적

이 활용가이드는 CNN 이미지 필터 체험 시뮬레이터를 **수업에 효과적으로 적용**하기 위한 안내서입니다. 수업 시나리오, 학습 질문, 활동지를 제공하여 교수자가 바로 활용할 수 있도록 합니다.

### 1.2 대상

| 대상 | 설명 |
|------|------|
| 1차 사용자 | 교육대학교 학생 (초등 예비교사) |
| 교과목 | 컴퓨팅 사고와 AI 교육 |
| 선수 지식 | 디지털 이미지의 기본 개념 (픽셀, 밝기값) |

### 1.3 관련 문서

| 문서 | 활용 시점 |
|------|----------|
| **퀵가이드** | 수업 중 시뮬레이터 조작 참조 |
| **사용설명서** | 수업 전 기능 전체 파악 |
| **이론해설서** | 수업 전 CNN·합성곱 이론 학습 |

## 2. 수업 활용 시나리오

### 2.1 수업 흐름 (45분)

| 단계 | 시간 | 활동 | 시뮬레이터 기능 |
|:----:|:----:|------|----------------|
| 도입 | 5분 | "사진 앱 필터는 어떻게 작동할까?" 동기 유발 | — |
| 개념 | 10분 | 합성곱 연산 원리 설명 (이론해설서 활용) | 📖 설명 패널, 💡 필터 정보 |
| 탐색 | 10분 | 프리셋 이미지 + 윤곽선 필터로 결과 관찰 | 프리셋 선택, ⚡ 즉시 적용 |
| 실습 | 15분 | 활동지 기반 필터 비교 실험 | 전체 기능 활용 |

### 2.2 단계별 교수 전략

#### 🎬 도입 (5분)

**발문:** "여러분, 스마트폰 카메라 앱에서 '흑백', '선명하게', '부드럽게' 같은 필터를 써 본 적 있나요? 이런 필터들은 내부에서 **숫자 계산**을 하고 있어요. 오늘은 그 계산이 어떻게 이루어지는지 직접 체험해 볼 거예요."

> 💡 학생의 일상 경험(스마트폰 필터)에서 출발하여 기술적 원리로 자연스럽게 연결합니다.

#### 📚 개념 (10분)

시뮬레이터의 **📖 CNN 필터(커널)란?** 설명 패널을 함께 읽으며 4단계 원리를 설명합니다.

| 단계 | 핵심 질문 | 시뮬레이터 활동 |
|:---:|----------|----------------|
| 1 | "필터는 무엇인가요?" | 커널 그리드(3×3) 가리키며 "작은 숫자 격자"임을 확인 |
| 2 | "합성곱은 어떻게 계산하나요?" | **[⏭️ 한 칸 이동]** → 📐 계산 과정에서 곱셈·합산 관찰 |
| 3 | "필터마다 왜 결과가 다른가요?" | 가로 윤곽선 → 세로 윤곽선으로 전환하며 차이 비교 |
| 4 | "CNN은 뭐가 다른가요?" | 💡 필터 정보: "CNN은 필터를 자동 학습" 언급 |

> 💡 **[▶️ 단계별 재생]**은 속도를 `600ms~800ms`로 느리게 설정하면 슬라이딩 과정을 시각적으로 따라가기 좋습니다.

#### 🔍 탐색 (10분)

교수자가 시연하며 학생들이 함께 관찰합니다.

**시연 시나리오:**

| 순서 | 교수자 행동 | 학생 관찰 포인트 |
|:---:|-----------|----------------|
| 1 | **[━ 가로선]** 프리셋 선택 | 입력 그리드에 가로 줄무늬 확인 |
| 2 | **가로 윤곽선** 필터 선택 → **[⚡ 즉시 적용]** | 출력에 파란색이 나타남 = 가로선 검출 성공! |
| 3 | **세로 윤곽선** 필터로 변경 | 출력이 거의 회색 → "왜 안 보일까?" |
| 4 | **[┃ 세로선]** 프리셋으로 변경 | 이번엔 세로 윤곽선에 강하게 반응! |
| 5 | **전체 윤곽선** 필터 선택 | 가로선이든 세로선이든 다 찾아냄 |

> **핵심 관찰:** 같은 이미지라도 **어떤 필터를 쓰느냐**에 따라 결과가 완전히 다르다!

#### ✋ 실습 (15분)

학생 개인 또는 짝꿍 단위로 활동지(§3 참조)를 수행합니다. 교수자는 순회하며 관찰 포인트를 안내합니다.

#### 📝 정리 (5분)

**마무리 발문:**

- "필터마다 잘 찾는 특징이 다르다는 것을 확인했죠? 그렇다면 고양이 사진에서 '귀 모양'을 찾으려면 어떤 필터가 필요할까요?"
- "우리는 직접 필터를 골랐지만, CNN은 이 과정을 **자동으로** 합니다. 수천 장의 사진을 보면서 최적의 필터를 스스로 찾아내는 거예요!"

## 3. 학생 활동지

### 📝 활동 1: 필터 탐정 — "이 필터는 무엇을 찾을까?"

**목표:** 서로 다른 필터가 서로 다른 특징을 검출한다는 것을 실험적으로 확인한다.

**준비물:** CNN 이미지 필터 체험 시뮬레이터 (브라우저)

**단계:**

1. [ ] **[━ 가로선]** 프리셋을 선택합니다.
2. [ ] 아래 표의 필터를 하나씩 선택한 뒤, **[⚡ 즉시 적용]**을 눌러 결과를 관찰합니다.
3. [ ] 출력 그리드에서 **파란색이 진하게 나타나면 ◎**, **약간 보이면 △**, **거의 안 보이면 ✕**로 기록합니다.

**기록표 A: 가로선 입력**

| 필터 | 반응 정도 | 관찰 메모 |
|------|:--------:|----------|
| ━ 가로 윤곽선 | | |
| ┃ 세로 윤곽선 | | |
| ✦ 전체 윤곽선 | | |
| 🌫️ 흐리게 | | |
| 🏔️ 엠보싱 | | |

4. [ ] 이번에는 **[┃ 세로선]** 프리셋으로 바꾸고, 같은 실험을 반복합니다.

**기록표 B: 세로선 입력**

| 필터 | 반응 정도 | 관찰 메모 |
|------|:--------:|----------|
| ━ 가로 윤곽선 | | |
| ┃ 세로 윤곽선 | | |
| ✦ 전체 윤곽선 | | |
| 🌫️ 흐리게 | | |
| 🏔️ 엠보싱 | | |

**질문:**

- 기록표 A와 B를 비교해 보세요. 가로 윤곽선 필터와 세로 윤곽선 필터의 반응이 **반대**인 이유는 무엇일까요?
- 전체 윤곽선(✦) 필터는 두 경우 모두 반응했나요? 왜 그럴까요?

---

### 📝 활동 2: 나만의 그림 실험

**목표:** 직접 그린 이미지에서 필터의 반응을 예측하고 검증한다.

**단계:**

1. [ ] **[🗑️ 지우기]**를 눌러 입력 그리드를 비웁니다.
2. [ ] 입력 그리드에 아래 도형 중 하나를 **직접 그립니다** (좌클릭으로 그리기).

| 도형 제안 | 힌트 |
|----------|------|
| ㄱ 자 | 가로 + 세로가 모두 포함된 도형 |
| + 모양 | 얇은 십자 (프리셋 십자와 다르게 1줄 두께) |
| 계단 모양 | 대각선을 계단식으로 표현 |
| 나만의 자유 그림 | 원하는 대로! |

3. [ ] 그림을 완성한 뒤, **가로 윤곽선** 필터를 적용하기 **전에** 결과를 예측합니다.

| 예측 | 내 예측 | 실제 결과 | 맞았나요? |
|------|---------|----------|:---------:|
| 가로 윤곽선 필터로 어떤 부분이 강하게 나올까? | | | |
| 세로 윤곽선 필터로는? | | | |

4. [ ] **[⚡ 즉시 적용]**으로 실제 결과를 확인하고, 예측과 비교합니다.

**질문:** 예측이 맞았나요? 틀렸다면 왜 틀렸는지 생각해 보세요.

---

### 📝 활동 3: 합성곱 따라하기

**목표:** 합성곱 계산 과정을 한 단계씩 직접 따라가며 이해한다.

**단계:**

1. [ ] **[━ 가로선]** 프리셋을 선택합니다.
2. [ ] **가로 윤곽선** 필터를 선택합니다.
3. [ ] ⚙️ 옵션에서 **[셀 값 표시]**와 **[계산 과정 표시]**를 모두 켭니다.
4. [ ] **[⏭️ 한 칸 이동]**을 눌러 첫 번째 위치의 계산을 확인합니다.

**기록:**

| 항목 | 기록 |
|------|------|
| 첫 위치는 어디인가요? (행, 열) | |
| 입력 영역의 9개 값은? | |
| 필터 커널의 9개 값은? | |
| 곱셈 결과 9개는? | |
| 모두 합산하면? | |
| 📐 계산 과정 패널의 결과와 같나요? | |

5. [ ] **[⏭️ 한 칸 이동]**을 3~4번 더 눌러 다른 위치도 관찰합니다.

**질문:** 필터가 가로선의 **위쪽 경계**와 **아래쪽 경계**에서 결과 부호가 다른 이유를 설명해 보세요.

## 5. 수업 운영 팁

### 5.1 자주 하는 실수와 대처

| 학생 실수 | 교수자 대처 |
|----------|-----------|
| 출력 그리드에 그리려고 함 | "출력은 자동 계산 결과예요. 왼쪽 입력 그리드에 그려보세요!" |
| 필터를 바꿨는데 출력이 안 바뀜 | "아직 적용 전이에요. [⚡ 즉시 적용]을 눌러보세요." (단, 기존 출력이 있으면 자동 갱신됨) |
| "왜 출력이 입력보다 작아요?" | "3×3 필터가 가장자리에서 벗어나지 않으려면, 양쪽 1칸씩 줄어야 해요." |
| 계산 과정의 숫자가 너무 많아 혼란 | "처음엔 결과 색상만 보세요. 파란색=찾았다, 회색=못 찾았다!" |

### 5.2 난이도별 활용

| 난이도 | 활용 범위 | 권장 활동 |
|:------:|----------|----------|
| 기초 | 프리셋 + 윤곽선 필터 2종만 | 활동 1 (기록표만) |
| 표준 | 모든 프리셋 + 필터 5종 | 활동 1 + 활동 2 |
| 심화 | 직접 그리기 + 계산 과정 추적 | 활동 1 + 2 + 3 |

### 5.3 짝꿍/모둠 활동 제안

| 활동명 | 방법 | 시간 |
|--------|------|:----:|
| **필터 퀴즈** | A가 입력+필터를 선택해 결과를 보여주면, B가 어떤 필터인지 맞추기 | 5분 |
| **패턴 예측 대결** | 같은 입력에 서로 다른 필터를 적용한 결과를 **먼저 예측**하고, 시뮬레이터로 검증 | 10분 |
| **설명하기 챌린지** | 합성곱 계산 과정을 짝꿍에게 설명하기 (📐 패널 참조) | 5분 |

## 6. 평가 연계

### 6.1 형성 평가 예시 (5지선다)

**Q1.** 3×3 필터가 이미지 위를 한 칸씩 이동하며 계산하는 과정을 무엇이라 하나요?

① 분류 ② 군집화 ③ 합성곱 ④ 역전파 ⑤ 풀링

**정답:** ③

**Q2.** 10×10 이미지에 3×3 필터를 적용하면 출력 크기는?

① 7×7 ② 8×8 ③ 9×9 ④ 10×10 ⑤ 12×12

**정답:** ②

**Q3.** 가로 윤곽선 필터를 세로선 이미지에 적용하면 출력은 주로 어떤 색인가요?

① 진한 파란색 ② 진한 빨간색 ③ 회색 ④ 흰색 ⑤ 검정색

**정답:** ③

### 6.2 서술형 평가 예시

**Q.** 같은 "십자(╋)" 이미지에 "가로 윤곽선 필터"와 "세로 윤곽선 필터"를 각각 적용했을 때 결과가 어떻게 다른지 설명하고, CNN이 이러한 여러 필터를 동시에 사용하는 이유를 서술하세요.

**채점 기준:**

| 요소 | 배점 |
|------|:----:|
| 가로 필터: 가로선 부분에만 강하게 반응한다고 설명 | 2점 |
| 세로 필터: 세로선 부분에만 강하게 반응한다고 설명 | 2점 |
| CNN이 다양한 필터를 동시 사용하여 모든 방향의 특징을 포괄적으로 검출한다고 설명 | 3점 |
