### 슬라이드 3: 📝 오늘의 실습 과제

**과제:**
> **"초등학교 5학년 수학 학습지 1장을 생성형 AI로 제작하세요."**

| 항목 | 내용 |
|:---:|:---|
| 제출물 | 최종 결과물 (학습지) + 프롬프트 전문 |
| 제출 기한 | 수업 종료 후 10분 이내 |
| 배점 | 총 15점 |

---

### 슬라이드 4: 📋 과제 요구사항

**🏫 기본 요구사항:**
- 대상: 초등학교 5학년
- 교과: 수학
- 분량: A4 1장
- 문제 수: 5~10개

**📊 난이도 구성:**
- 쉬움: 30%
- 중간: 40%
- 어려움: 30%

**💻 사용 가능 도구:**
- ChatGPT
- Claude
- Gemini

---

### 슬라이드 5: ⭐ 평가 기준

**총 15점**

| 항목 | 배점 | 평가 내용 |
|:---:|:---:|:---|
| **프롬프트의 질** | 5점 | 4대 원칙 적용 정도 |
| **결과물의 질** | 10점 | 완성도, 정확성, 적절성 |

**⚠️ 중요:**
- 프롬프트 전문을 **반드시 함께 제출**해야 합니다!
- 결과물만 제출하면 프롬프트 평가(5점)를 받을 수 없습니다.

---

### 슬라이드 6: 📝 프롬프트 평가 기준 (5점)

| 점수 | 기준 |
|:---:|:---|
| **5점** | 4대 원칙 모두 적용, 체계적으로 구조화 |
| 4점 | 3가지 원칙 적용, 구조화 시도 |
| 3점 | 2가지 원칙 적용 |
| 2점 | 1가지 원칙만 적용 |
| **1점** | 원칙 적용 없이 단순 요청 |

**💡 4대 원칙 복습:**
1. 구체적으로 작성하기
2. 맥락과 배경 정보 제공하기
3. 원하는 출력 형식 지정하기
4. 단계별로 나누어 요청하기

---

### 슬라이드 7: 📄 결과물 평가 기준 (10점)

| 항목 | 배점 | 세부 기준 |
|:---:|:---:|:---|
| 완성도 | 4점 | 요청한 형식/분량 충족, 체계적 구성 |
| 정확성 | 3점 | 내용 오류 없음, **수학 문제 정답 정확** |
| 적절성 | 3점 | 학년 수준에 맞음, 교육적 가치 있음 |

**⭐ 핵심:**
- 수학 학습지이므로 **정답이 맞는지 반드시 확인**해야 합니다!

---

### 슬라이드 8: ❌ 나쁜 프롬프트 vs ✅ 좋은 프롬프트

**❌ 나쁜 프롬프트:**
```
수학 학습지 만들어줘
```
- 학년 미지정
- 단원 미지정
- 문제 수 미지정
- 난이도 미지정

**✅ 좋은 프롬프트:**
```
초등 5학년 약분과 통분 단원,
A4 1장, 문제 8개,
쉬움 3 / 중간 3 / 어려움 2,
정답 포함하여 만들어줘
```
- 구체적 정보 포함
- 형식 지정

### 슬라이드 10: ⚠️ 검토 시 주의사항

**👻 환각(Hallucination) 현상:**
- AI가 **사실이 아닌 내용을 사실처럼 생성**하는 현상
- 수학에서는 **정답이 틀릴 수 있습니다!**

**🔍 반드시 확인할 항목:**
- ✅ 수학 문제의 **정답이 맞는가?**
- ✅ **학년 수준**에 맞는가?
- ✅ **문제가 성립**하는가? (조건 충족)
- ✅ **풀이 과정**이 논리적인가?

### 슬라이드 11: 📥 제출 방법

| 항목 | 내용 |
|:---:|:---|
| 제출 기한 | 수업 종료 후 **10분 이내** (절대 기한) |
| 제출 방법 | LMS 과제 제출 |
| 제출물 1 | 최종 결과물 (학습지) |
| 제출물 2 | 사용한 프롬프트 전문 |

**✅ 제출 전 체크리스트:**
- ☐ 학습지가 완성되었는가?
- ☐ 수학 문제의 정답을 직접 확인했는가?
- ☐ 프롬프트 전문을 저장했는가?
- ☐ 2개 파일이 모두 준비되었는가?

### 슬라이드 12: 🤝 증강(Augmentation) 관점

**💡 증강이란?**
> 인간이 **주도**하고, AI가 **보조**하는 방식

| 단계 | 주체 | 역할 |
|:---|:---:|:---|
| 과제 파악 | 인간 | 어떤 학습지가 필요한지 판단 |
| 프롬프트 작성 | 인간 | AI에게 요구사항 전달 |
| 초안 생성 | AI | 학습지 초안 생성 |
| 검토 및 수정 | 인간 | 오류 확인, 수정 |
| 최종 확정 | **인간** | **최종 책임** |
