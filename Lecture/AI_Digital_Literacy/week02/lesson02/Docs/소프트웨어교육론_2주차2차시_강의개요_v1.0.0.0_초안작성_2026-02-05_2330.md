# 소프트웨어교육론 2주차 2차시 강의개요

**주제: 자료 제작 실습**

**문서 버전: v1.0.0.0**

---

## 1. 기본 정보

| 항목 | 내용 |
|:---:|:---|
| **과목명** | 소프트웨어교육론 |
| **주차/차시** | 2주차 2차시 |
| **주제** | 자료 제작 실습 |
| **수업 시간** | 50분 |
| **수업 형태** | 실습 중심 |
| **수업 대상** | 교육대학교 4학년 |

---

## 2. 수업 목표

- 생성형 AI를 활용하여 다양한 형태의 교육 자료를 제작할 수 있다.
- 1차시에서 배운 프롬프트 작성 원칙을 실제 자료 제작에 적용할 수 있다.
- 결과물의 품질을 평가하고 프롬프트를 개선할 수 있다.

---

## 3. 시간 배분

| 단계 | 시간 | 내용 |
|:---:|:---:|:---|
| **도입** | 5분 | 실습 과제 안내 및 평가 기준 설명 |
| **전개** | 35분 | 다양한 자료 제작 실습 (개별 실습 + 순회 지도) |
| **정리** | 10분 | 실습 결과 정리 및 과제 제출 |

---

## 4. 단계별 세부 내용

### 4.1 도입 (5분)

**활동**: 실습 과제 안내

**진행 방법**:
1. 오늘의 실습 과제 제시
2. 평가 기준 안내 (프롬프트 질 5점 + 결과물 품질 10점 = 15점)
3. 제출 방법 안내 (수업 종료 후 10분 이내)

**과제 내용**:
- 초등학교 수업에 활용할 수 있는 자료를 생성형 AI로 제작
- 자료 유형: 학습지, 안내문, 평가 문항, 표/데이터 정리 중 선택
- 프롬프트 전문과 결과물을 함께 제출

---

### 4.2 전개 (35분)

**활동**: 다양한 자료 제작 실습

#### 실습 활동 구성

| 활동 | 시간 | 내용 |
|:---:|:---:|:---|
| 과제 선택 및 프롬프트 설계 | 5분 | 제작할 자료 유형 선택, 프롬프트 초안 작성 |
| 1차 시도 및 결과 확인 | 10분 | AI에게 요청, 결과물 확인 및 문제점 파악 |
| 프롬프트 개선 및 재시도 | 15분 | 프롬프트 수정, 반복 시도, 최적 결과물 도출 |
| 결과물 다듬기 | 5분 | 최종 결과물 정리, 필요 시 직접 수정 |

#### 자료 유형별 실습 가이드

| 자료 유형 | 프롬프트에 포함할 요소 | 예시 |
|:---|:---|:---|
| **학습지** | 학년, 과목, 단원, 문항 수, 난이도, 형식 | 3학년 수학 분수 학습지 |
| **안내문** | 대상, 목적, 포함 정보, 톤, 분량 | 현장학습 안내문 |
| **평가 문항** | 과목, 성취기준, 문항 유형, 난이도, 수량 | 과학 단원평가 10문항 |
| **표/데이터** | 데이터 종류, 열/행 구성, 정렬 기준 | 학생 성적 분석표 |

#### 순회 지도 포인트

- 프롬프트 작성에 어려움을 겪는 학생 지원
- 1차시 프롬프트 원칙(명확성, 맥락, 형식, 단계, 예시) 적용 여부 확인
- 결과물에 대한 비판적 검토 유도 ("이 결과물을 실제 수업에 바로 쓸 수 있을까?")

---

### 4.3 정리 (10분)

**활동**: 실습 결과 정리 및 과제 제출

| 활동 | 시간 | 내용 |
|:---:|:---:|:---|
| 우수 사례 공유 | 3분 | 1~2명의 프롬프트와 결과물 전체 공유 |
| 과제 정리 | 5분 | 프롬프트 전문 + 결과물 정리 |
| 제출 안내 | 2분 | 수업 종료 후 10분 이내 LMS 제출 |

---

## 5. 평가

| 평가 항목 | 배점 | 세부 기준 |
|:---:|:---:|:---|
| **프롬프트 질** | 5점 | 명확성, 구체성, 형식 지정, 맥락 제공 |
| **결과물 품질** | 10점 | 목표 달성도(5), 완성도(3), 교육 현장 활용 가능성(2) |
| **합계** | **15점** | |

**제출물**: 프롬프트 전문 + 최종 결과물 (수업 종료 후 10분 이내)

---

## 6. 준비물

### 교수자 준비물
- 실습 과제 안내 슬라이드
- 평가 루브릭 (학생 배포용)
- 순회 지도 체크리스트

### 학습자 준비물
- 노트북 또는 태블릿
- 생성형 AI 도구 계정 (ChatGPT, Claude 등)
- 1차시 프롬프트 작성 원칙 메모

---

## 7. 수업 운영 Tip

### 실습 환경
- 수업 시작 전 AI 도구 접속 상태 확인
- 네트워크 문제 대비 핫스팟 또는 대체 도구 준비

### 개별 실습 운영
- 자료 유형 선택에 고민이 많은 학생에게는 "학습지"를 권장 (가장 구조화하기 쉬움)
- 같은 유형을 선택한 학생끼리 결과물 비교 유도

### 과제 제출
- 프롬프트 전문을 반드시 포함하도록 안내 (프롬프트 없이 결과물만 제출 시 감점)
- 여러 번 시도한 경우, 최종 프롬프트와 중간 과정도 포함하면 가산점

## 8. 1차시와의 연결

| 1차시 (이론) | 2차시 (실습) |
|:---|:---|
| 프롬프트 작성 5원칙 학습 | 5원칙을 실제 자료 제작에 적용 |
| 모호한/명확한 프롬프트 비교 시연 | 직접 프롬프트를 작성하고 개선 |
| 생성형 AI 소개 | 생성형 AI를 도구로 활용 |
