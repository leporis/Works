### 슬라이드 11: ⚠️ LLM의 한계 — 환각(Hallucination)

**주의할 점:**
- LLM이 실제로 내용을 "이해"하는 것은 아닙니다!

**환각이란?**
- AI가 **그럴듯하지만 틀린 정보**를 생성하는 현상
- 존재하지 않는 논문 인용
- 틀린 역사적 사실 서술
- 가짜 통계 데이터 제시

**대응 방법:**
- AI의 결과물은 항상 **검증**이 필요!
- 사실 여부 확인
- 출처 확인
- 전문 자료와 대조

### 슬라이드 12: ✍️ 프롬프트란?

**정의:**
- **프롬프트(Prompt)** = AI에게 보내는 요청, 질문, 명령

**쉬운 비유: "음식 주문 기술"**
- ❌ "아무거나 주세요" → 원하는 음식이 나올 확률 낮음
- ✅ "매운맛 제육볶음, 밥 많이요" → 딱 원하는 대로!

**프롬프트 엔지니어링** = 효과적인 AI 활용을 위한 질문/명령 기술

---

### 슬라이드 13: ❌ vs ✅ 나쁜 프롬프트 vs 좋은 프롬프트

**❌ 나쁜 프롬프트:**
```
3학년 분수 학습지 만들어줘
```
- ? 어떤 내용?
- ? 문제 몇 개?
- ? 난이도는?
- ? 어떤 형식?
- AI가 다 추측해야 함!

**✅ 좋은 프롬프트:**
```
초등 3학년 수학 학습지를 만들어줘.

[기본 정보]
- 단원: 분수의 이해
- 분량: A4 1장

[문제 구성]
1. 그림 보고 분수 쓰기 (3문제)
2. 분수 크기 비교 (3문제)
3. 분수 색칠하기 (4문제)

[형식]
- 문제 번호 ①②③
- 하단에 정답란
```

### 슬라이드 15: 1️⃣ 원칙 1: 명확하고 구체적으로

**AI가 추측해야 할 것을 줄여주세요!**

| ❌ 모호한 표현 | ✅ 구체적 표현 |
|:---|:---|
| "학습지 만들어줘" | "초등 3학년 수학 분수 학습지" |
| "문제 좀 내줘" | "5지선다 객관식 10문제" |
| "쉬운 수준으로" | "받아올림 없는 두 자리 수 덧셈" |
| "적당한 길이로" | "A4 1장 분량으로" |
| "여러 개" | "5개" |

### 슬라이드 16: 2️⃣ 원칙 2: 맥락과 배경 제공

**❌ 맥락 없음:**
- "소풍 안내문 써줘"
- 어느 학교? 몇 학년? 어디로? 언제?

**✅ 맥락 있음:**
```
초등 1학년 학부모에게 보내는 봄 소풍 안내문을 써줘.
- 일시: 4월 15일 화요일
- 장소: 서울대공원
- 도시락 지참, 버스비 5,000원
- 톤: 공식적이면서 따뜻하게
```

**역할 부여도 효과적!**
- "당신은 10년 경력의 초등학교 3학년 담임교사입니다."

---

### 슬라이드 17: 3️⃣ 원칙 3: 출력 형식 지정

**AI에게 결과물의 형태를 미리 알려주세요!**

**❌ 형식 미지정:**
- "광합성 설명해줘"
- 대학 수준일 수도, 너무 간략할 수도 있음

**✅ 형식 지정:**
- "광합성을 **초등 5학년 수준**으로 설명해줘. **3문장**으로 요약하고, 이해를 돕는 **비유를 하나** 포함해줘."

| 형식 지정 | 사용 예시 |
|:---|:---|
| 표 형식 | "표로 정리해줘" |
| 특정 분량 | "300자 이내로 요약해줘" |
| 문서 형태 | "가정통신문 형식으로" |
| 언어 수준 | "초등 2학년이 이해할 수 있게" |

### 슬라이드 18: 4️⃣ 원칙 4: 단계별로 나누기

**❌ 한꺼번에 요청:**
- "수업지도안 작성해줘. 도입-전개-정리, 발문, 평가, 학습지 다 포함해서."

**✅ 단계별 요청:**

|:---:|:---|

| 2 | "이 목표로 도입-전개-정리 흐름을 잡아줘" |
| 3 | "전개의 핵심 발문 5개를 만들어줘" |
| 4 | "형성평가 문제 3개를 만들어줘" |

**각 단계를 확인하고 수정한 뒤 다음 단계로!**

### 슬라이드 19: 🔧 실전 팁

**1️⃣ 예시 제공 (Few-shot):**
- 원하는 형태를 보여주기
- "다음 형식으로 5개 더 만들어줘. [예시] 문제: 1/3 + 1/3 = ? 정답: 2/3"

**2️⃣ 반복 개선 (Iterative):**
- 수정 요청으로 다듬기
- "난이도 낮춰줘", "3번을 서술형으로", "분량 절반으로"

**3️⃣ 제약 조건:**
- "~하지 마" 형태로 범위 좁히기
- "전문 용어 쓰지 마", "영어 사용하지 마", "3줄 이내로"
