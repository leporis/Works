## 2. 수업 목표

- 생성형 AI를 활용하여 다양한 형태의 교육 자료를 제작할 수 있다.
- 1차시에서 배운 프롬프트 작성 원칙을 실제 자료 제작에 적용할 수 있다.
- 결과물의 품질을 평가하고 프롬프트를 개선할 수 있다.

---

## 4. 세부 내용

### 4.1 도입 (5분)

**오늘의 실습 과제 안내**

- 실습 과제: "초등학교 5학년 수학 학습지 1장 제작"
- 평가 기준: 프롬프트의 질(5점) + 결과물의 질(10점) = 총 15점
- 제출 방법: 수업 종료 후 10분 이내 제출
- 제출물: 최종 결과물 + 프롬프트 전문

### 4.2 전개 (35분)

**다양한 자료 제작 실습**

| 자료 유형 | 예시 | 활용 방법 |
|:---:|:---|:---|
| **문서 자료** | 학습지, 안내문, 평가 문항 | 프롬프트에 학년, 주제, 난이도 명시 |
| **표와 데이터** | 학습 계획표, 평가 기준표 | 표 형식 지정, 구체적 조건 제시 |
| **이미지 생성** | 수업 자료용 그림 (선택) | 이미지 생성 AI 활용 |
| **코드 생성** | 간단한 자동화 스크립트 (선택) | 반복 작업 자동화 요청 |

**실습 진행 방식**

1. 개인별 노트북/태블릿으로 ChatGPT 또는 Claude 접속
2. 과제에 맞는 프롬프트 작성 및 결과 확인
3. 결과가 불만족스러우면 프롬프트 수정 후 재시도
4. 최종 결과물과 사용한 프롬프트 정리

**순회 지도 포인트**

- 프롬프트가 너무 막연하지 않은지 확인
- 1차시에서 배운 원칙(구체성, 맥락, 형식, 단계별)을 적용했는지 확인
- 결과물의 정확성(환각 현상) 검토 안내

### 4.3 정리 (10분)

**실습 결과 및 프롬프트 전문 정리**

- 최종 결과물 파일 정리
- 사용한 프롬프트 전문 복사
- 제출 파일 준비 (결과물 + 프롬프트)

**과제 제출 안내**

- 제출 기한: 수업 종료 후 10분 이내
- 제출 방법: LMS 과제 제출
- 제출물: ① 최종 결과물 ② 사용한 프롬프트 전문
