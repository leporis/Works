## 4. 주요 내용

### 4.1 퀴즈 (10분)

- 1주차 내용 확인 (AI 개요, 역사, 교육적 의의)
- 객관식 5지선다 15문항
- 수업 시작과 동시에 시작, 10분 후 자동 제출
- 문항당 1점 (총 15점)

### 4.2 도입 (5분)

**"AI에게 그림을 그려달라고 하면?"**

- 같은 과제를 모호한 프롬프트 vs 명확한 프롬프트로 비교
- 결과물 차이를 통해 프롬프트의 중요성 인식
- "도구를 잘 쓰려면 명확한 지시가 필요하다"

### 4.3 전개 1: 생성형 AI의 등장 (15분)

**A. 전통적 AI vs 생성형 AI**
- 전통적 AI: 분류, 인식, 예측 (정해진 답 중 선택)
- 생성형 AI: 새로운 콘텐츠를 "생성" (텍스트, 이미지, 코드 등)

**B. 주요 텍스트 생성 AI 서비스**
- ChatGPT (OpenAI), Claude (Anthropic), Gemini (Google)

**C. 주요 이미지 생성 AI 서비스**
- DALL-E, Midjourney, Stable Diffusion 등

**D. 생성형 AI의 작동 원리 (간략)**
- 대규모 언어 모델(LLM): "다음 단어 예측"
- 패턴 학습을 통한 콘텐츠 생성
- 한계: 할루시네이션 (그럴듯하지만 틀린 정보)

### 4.4 전개 2: 효과적인 프롬프트 작성 원칙 (15분)

**프롬프트 작성 4원칙**

| 원칙 | 설명 |
|------|------|
| ① 명확하고 구체적으로 | 모호한 표현 대신 정확한 요구사항 기술 |
| ② 맥락과 배경 정보 제공 | 상황, 대상, 목적 등 설명 |
| ③ 원하는 출력 형식 지정 | 표, 목록, 문단, 분량 등 형식 명시 |
| ④ 단계별로 나누어 요청 | 복잡한 과제는 분할하여 요청 |

- 각 원칙별 Before/After 예시 비교
- 교육 현장 프롬프트 예시 (학습지, 안내문 등)

### 4.5 정리 (5분)

- 프롬프트 작성 4원칙 요약
- 2차시 자료 제작 실습 예고
