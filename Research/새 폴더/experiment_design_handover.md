## 1. 연구 개요

### 연구 주제
**생성형 AI 질문 역량(AI Questioning Competency)** 연구
- AI에게 효과적으로 질문하는 능력의 예측변인 탐색
- Vibe Coding 교육의 효과 검증

### 핵심 개념
- **Vibe Coding**: AI와 대화하며 프로그램을 만드는 교육 방식
- **협업적 추상화(Collaborative Abstraction)**: 의도 표현 → AI 응답 평가 → 반복 정련

## 5. AI 질문 역량 루브릭

### 예비교사용 (5영역, 각 1-4점)

| 영역 | 평가 내용 |
|------|----------|
| 초기 의도 표현 | 목표, 맥락, 제약조건 명시 |
| 대화적 구체화 | 구체적 수정/설명 요청 |
| 반복적 정련 | 체계적 개선 시도 |
| 메타인지적 모니터링 | 불일치 인식, 전략 수정 |
| 최종 통합 | 의도와 결과 일치도 |

### 초등학생용 (4영역, 각 1-4점)

| 영역 | 1점 | 4점 |
|------|-----|-----|
| 처음 말하기 | 거의 말 안 함 | 자세히 설명 |
| 고쳐달라고 하기 | 수정 요청 안 함 | 여러 번 구체적 수정 |
| 확인하기 | 확인 안 함 | 꼼꼼히 확인+수정 |
| 완성하기 | 미완성 | 창의적 완성 |

## 9. 예상 결과 (가상 데이터 기반)

### 논문 1: 예측 연구
- AI 경험 β = .43***
- 메타인지 β = .31***
- R² = .333

### 논문 2: 효과 연구
- 사전 M = 10.12 → 사후 M = 12.64
- Cohen's d = 1.48 (큰 효과)

### 논문 3: 통합 연구
- 메타인지 상 집단: 향상 3.21점
- 메타인지 하 집단: 향상 1.79점
- F(2,87) = 6.17, p = .003
