## 1단계: 사전 준비 (연구 시작 4주 전)

### 1.1 IRB(연구윤리위원회) 승인

**필요 서류:**
- [ ] 연구계획서
- [ ] 연구 참여 동의서
- [ ] 설문지 및 측정 도구
- [ ] 개인정보 처리 방침
- [ ] 데이터 보관 및 폐기 계획

**IRB 신청 시 포함할 내용:**
```
1. 연구 목적: 생성형 AI 활용 역량의 예측변인 탐색
2. 연구 대상: A 교육대학교 '컴퓨팅 사고와 교육' 수강생 90명
3. 연구 방법: 설문조사 + AI 협업 과제
4. 예상 위험: 최소 위험 연구 (심리적 불편감 없음)
5. 개인정보 보호: 익명화 처리, 3년 보관 후 폐기
6. 참여 보상: 문화상품권 10,000원 (선택사항)
```

**예상 소요 시간:** 2-4주 (기관마다 다름)

- 네이버 오피스 폼 (무료)

### 5.2 채점자 간 신뢰도 계산 (Python)

```python
from scipy.stats import pearsonr
import numpy as np

# 두 채점자의 점수
rater1 = [15, 12, 18, 10, ...]  # 채점자 1의 점수
rater2 = [14, 13, 17, 11, ...]  # 채점자 2의 점수

# 피어슨 상관계수
r, p = pearsonr(rater1, rater2)
print(f"채점자 간 상관: r = {r:.3f}, p = {p:.4f}")

# ICC 계산 (간단 버전)
# ICC = (MSB - MSW) / (MSB + MSW)
```
