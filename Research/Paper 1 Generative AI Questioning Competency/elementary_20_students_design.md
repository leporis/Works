# 20명 초등학생 대상 연구 설계

## Vibe Coding 교육이 초등학생의 AI 질문 역량에 미치는 효과

---

## 연구 개요

| 항목 | 내용 |
|------|------|
| **연구 유형** | 사전-사후 효과 연구 |
| **대상** | 초등학교 5-6학년 20명 |
| **기간** | 5주 |
| **분석** | 대응표본 t검정, 효과크기 |

## 2-4주차: Vibe Coding 교육 (각 40분)

### 2주차: AI 소개하기

| 시간 | 활동 | 내용 |
|------|------|------|
| 10분 | 도입 | AI란 무엇인가? (영상 시청) |
| 10분 | 시범 | 선생님이 AI와 대화하는 것 보여주기 |
| 15분 | 실습 | AI에게 자기소개하고 질문하기 |
| 5분 | 정리 | 오늘 배운 것 이야기하기 |

### 3주차: 고쳐달라고 하기

| 시간 | 활동 | 내용 |
|------|------|------|
| 5분 | 복습 | 지난 시간 배운 것 |
| 10분 | 설명 | "고쳐달라고 하는 방법" |
| 20분 | 실습 | AI가 만든 것 수정 요청하기 |
| 5분 | 정리 | 어떻게 고쳐달라고 했는지 발표 |

**핵심 문장:**
- "이 부분을 더 재미있게 해줘"
- "주인공 이름을 OO로 바꿔줘"
- "더 길게/짧게 해줘"

### 4주차: 내 생각 말하기

| 시간 | 활동 | 내용 |
|------|------|------|
| 5분 | 복습 | 지난 시간 배운 것 |
| 10분 | 설명 | "내가 원하는 것 자세히 말하기" |
| 20분 | 실습 | 자세한 요청으로 이야기 만들기 |
| 5분 | 정리 | 가장 좋았던 대화 공유 |

**자세히 말하기 예시:**
```
❌ "이야기 만들어줘"
⭕ "마법 학교에 다니는 여자아이가 용을 친구로 만드는 이야기를 만들어줘. 
    재미있고 감동적이면 좋겠어."
```

## 평가 루브릭 (초등학생용, 간소화)

### AI 질문 역량 평가 (총 4-16점)

| 영역 | 1점 | 2점 | 3점 | 4점 |
|------|-----|-----|-----|-----|

## 예상 결과 (가상 데이터 기반)

### 기술통계

| 시점 | M | SD |
|------|------|------|
| 사전 | 8.5 | 2.3 |
| 사후 | 11.2 | 2.8 |
| 향상도 | 2.7 | 1.8 |

### 대응표본 t검정

```
t(19) = 6.71, p < .001
Cohen's d = 1.50 (큰 효과)
```

### 해석

"20명의 초등학생을 대상으로 5주간 Vibe Coding 교육을 실시한 결과,
AI 질문 역량이 사전(M=8.5)에서 사후(M=11.2)로 유의하게 향상되었다
(t(19)=6.71, p<.001). 효과크기는 Cohen's d=1.50으로 큰 효과에 해당하였다."
