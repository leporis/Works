## 1. 논문 구성 표준

### 1.1 논문 구성 순서

모든 투고 논문은 아래 순서를 반드시 준수한다.

| 순서 | 항목 | 비고 |
|:----:|------|------|
| ① | 국문 제목 | — |
| ② | 국문 저자명 · 소속 | 심사용: 삭제 필수 |
| ③ | 국문 요약 | 공백 제외 400자 내외 |
| ④ | 국문 주제어 (키워드) | — |
| ⑤ | 영문 제목 | — |
| ⑥ | 영문 저자명 · 소속 | 심사용: 삭제 필수 |
| ⑦ | 영문 Abstract | — |
| ⑧ | 영문 Keywords | — |
| ⑨ | 본문 | 섹션 번호 1. ~ 5. |
| ⑩ | 참고문헌 | References |
| ⑪ | 저자소개 | 최종 제출 시 전원 기재 |

> ⚠ **심사용 제출**: 저자 관련 정보 모두 삭제 (자기 인용 표현도 익명성 위반)
> ⚠ **최종 제출**: 저자소개에 성명·소속·직위 전원 기재

### 1.2 국문 요약 작성 기준

| 항목 | 기준 |
|------|------|
| 분량 | 공백 제외 **400자 내외** |
| 포함 내용 | 연구 목적 · 방법 · 결과 · 결론 (4요소) |
| 금지 사항 | 인용 문헌 표기 금지, 표·그림 언급 금지 |
| 주제어 수 | 3~6개 권장 |

### 1.3 제목 번호 체계

```
1.        대제목
1.1       중제목
1.1.1     소제목
```

> ❌ **금지**: Ⅰ, Ⅱ, Ⅲ 로마자 번호 사용
> ❌ **금지**: 1), (1), 가. 나. 형식 사용

### 1.4 논문 구조·분량 기준 (일반)

| 섹션 번호 | 한국어 명칭 | 영문 명칭 | 권장 분량 |
|:--------:|-----------|---------|:-------:|
| 1. | 서론 | Introduction | 2~3쪽 |
| 2. | 이론적 배경 / 관련 연구 | Theoretical Background / Related Work | 3~5쪽 |
| 3. | 연구 방법 | Research Methods | 3~4쪽 |
| 4. | 연구 결과 | Results | 4~5쪽 |
| 5. | 결론 및 논의 | Conclusion and Discussion | 3~4쪽 |
| — | 참고문헌 | References | 1~2쪽 |
| **합계** | | | **16~23쪽** |

> ⚠ 논문 유형(실험·조사·개발)에 따라 분량이 달라질 수 있음

## 2. 표 작성 표준

### 2.1 표 형식 규칙

| 항목 | 규칙 | 예시 |
|------|------|------|
| 제목 위치 | **표 위** (표 바로 위에 표기) | — |
| 제목 형식 | `<Table N> 영문 제목` | `<Table 1> Participants of the Study` |
| 번호 | 논문 내 아라비아 숫자 연속 번호 | `<Table 1>`, `<Table 2>`, `<Table 3>`… |
| 제목 언어 | **영문** | — |
| 유의수준 | 표 **하단** 줄에 표기 | `*p<.05  **p<.01  ***p<.001` |
| 소수점 자릿수 | 논문 내 동일 통계값 동일 자릿수 유지 | M·SD: 2자리, β·r: 2자리 |

> ❌ **금지**: 표 제목을 표 아래에 표기
> ❌ **금지**: 표 제목을 한국어로 표기

### 2.2 표 제목 작성 패턴

| 표 유형 | 제목 패턴 예시 |
|--------|-------------|
| 연구 참여자 현황 | `<Table N> Participants of the Study` |
| 측정 도구 및 신뢰도 | `<Table N> Measurement Instruments and Reliability (Cronbach's α)` |
| 기술통계 | `<Table N> Descriptive Statistics of [변인명]` |
| 기술통계 + 상관행렬 | `<Table N> Descriptive Statistics and Correlation Matrix of [변인명]` |
| 대응표본 t검정 | `<Table N> Pre- and Post-test Comparison of [변인명]` |
| 독립표본 t검정 | `<Table N> Comparison of [변인명] by [집단변인]` |
| 일원분산분석(ANOVA) | `<Table N> Comparison of [변인명] by [분류기준] (One-Way ANOVA)` |
| 위계적 회귀분석 | `<Table N> Results of Hierarchical Regression Analysis on [종속변인]` |
| 단순/다중 회귀분석 | `<Table N> Results of Regression Analysis on [종속변인]` |
| 분반/집단별 비교 | `<Table N> [변인명] Comparison by [분류기준]` |
| 교육 프로그램 구성 | `<Table N> Composition of [프로그램명] Program` |
| 평가 루브릭 | `<Table N> Assessment Rubric for [평가 대상] (N Domains)` |
| 이질성 검정 | `<Table N> Heterogeneity of Effect Sizes` |
| 효과크기 요약 | `<Table N> Summary of Average Effect Size` |

### 2.3 열 헤더 영문 약어 표준

| 한국어 | 영문 약어 | 한국어 | 영문 약어 |
|--------|---------|--------|---------|
| 평균 | *M* | 표준편차 | *SD* |
| 사전점수 평균 | *M*(pre) | 사후점수 평균 | *M*(post) |
| 표준오차 | *SE* | 분산 | Var |
| t값 | *t* | F값 | *F* |
| 유의확률 | *p* | 자유도 | *df* |
| 효과크기 (Cohen) | Cohen's *d* | 에타 제곱 | η² |
| 표준화 회귀계수 | β | 비표준화 회귀계수 | *B* |
| 결정계수 | *R*² | R² 변화량 | Δ*R*² |
| 95% 신뢰구간 | *CI*(95%) | 상관계수 | *r* |
| 향상도 | Gain | 인원 | *N* (전체) / *n* (부분) |
| 분반 | Section | Cronbach α | α |
| 급내상관계수 | ICC | 안전계수 | *N*f,s |
| 이질성 지수 | *I*² | AI 사용 경험 | AI Exp. |

### 2.4 유의수준 표기 표준

표 하단 고정 문구:

```
*p<.05  **p<.01  ***p<.001
```

> ⚠ 별표(*)는 셀 안이 아닌 **표 하단**에 설명
> ⚠ `p<.0001`이 필요한 경우: `***p<.0001` 로 변경

### 3.1 그림 형식 규칙

| 항목 | 규칙 | 예시 |

## 4. 통계 보고 표준

### 4.1 공통 표기 원칙

| 항목 | 규칙 |
|------|------|
| p값 형식 | 앞 0 생략 → `.001` (O) / `0.001` (X) |
| 유의 표기 | `p < .001` 또는 `p = .023` (등호·부등호 모두 사용) |
| 비유의 표기 | `p = .109 (ns)` 또는 `p = .109` 로 명시 |
| 통계 기호 | 이탤릭: *t*, *F*, *M*, *SD*, *p*, *r*, *N*, *n* / 비이탤릭: β, α |
| 소수점 자릿수 | *M*·*SD*·*B*: 2자리 / β·*r*: 2자리 / *t*·*F*: 2자리 / *p*: 3자리 |

### 4.2 기술통계

```
M = [값], SD = [값], N = [값]
예) M = 12.64, SD = 2.58, N = 90
```

### 4.3 대응표본 t검정

```
t(df) = [값], p [부등호] [값], Cohen's d = [값]
예) t(89) = 14.28, p < .001, Cohen's d = 1.48
```

### 4.4 독립표본 t검정

```
t(df) = [값], p [부등호] [값]
예) t(88) = 2.36, p = .020
```

### 4.5 일원분산분석 (One-Way ANOVA)

```
F(df집단, df오차) = [값], p [부등호] [값]
예) F(2, 87) = 6.17, p = .003

사후검정(Tukey HSD): [집단A] > [집단B] (p = [값])
예) Tukey HSD: High > Low (p = .002)
```

### 4.6 위계적 회귀분석

```
각 단계: R² = [값], F(df1, df2) = [값], p [부등호] [값]
2단계~: ΔR² = [값], ΔF(df1, df2) = [값], p [부등호] [값]
개별 변인: B = [값], SE = [값], β = [값], t = [값], p [부등호] [값]
```

### 4.7 상관분석

```
r = [값], p [부등호] [값]
예) r = .41, p < .001
비유의: r = .17, p = .109 (ns)
```

### 4.8 신뢰도

```
Cronbach's α = [값]
예) α = .83

채점자 간 신뢰도: ICC = [값] (95% CI [하한], [상한]), p < .001
```

> ⚠ ICC는 **.80 이상** 확인 후 기재

### 4.9 효과크기 해석 기준 (Cohen, 1992)

| 효과크기 (d) | 해석 | 비고 |
|:-----------:|------|------|
| d < 0.20 | 무시할 수 있는 효과 | — |
| 0.20 ≤ d < 0.50 | 작은 효과 (small) | — |
| 0.50 ≤ d < 0.80 | 중간 효과 (medium) | — |
| d ≥ 0.80 | 큰 효과 (large) | — |
| **d > 1.0** | **이례적으로 큰 효과** | **설계 한계 논의 필수** |

> ⚠ d > 1.0: 통제집단 부재·소표본·출판편향 가능성을 논의에서 명시

## 5. 참고문헌 작성 표준

### 5.1 본문 내 인용 형식

| 형식 | 예시 |
|------|------|
| 단독 인용 | 메타인지는 학습 효과를 조절하는 핵심 변인이다`[3]`. |
| 복수 인용 | 여러 선행 연구에서 확인되었다`[1, 3, 7]`. |
| 범위 인용 | 연속된 경우 `[2-5]` 형식도 허용 |

> ❌ **금지**: APA 방식 `(저자명, 연도)` 사용 — 본 학회는 번호 방식
> ❌ **금지**: 각주 방식 인용

### 5.2 참고문헌 나열 순서 및 형식

**나열 순서**: ① 국내 문헌 (가나다 순) → ② 국외 문헌 (알파벳 순) → ③ 웹문서

---

**① 국내 학술지**
```
[N] 저자(연도), 논문제목, 학술지명, 권(호), 쪽수
예) [1] 양창모(2022), AI 학습 프로그램의 효과, 한국정보교육학회논문지, 26(1), 35-44
```

**② 국내 단행본**
```
[N] 저자(연도), 도서명, 발행소, 발행지
예) [2] 홍길동(2020), 메타인지와 학습, 학지사, 서울
```

**③ 국내 학위논문**
```
[N] 저자(연도), 논문제목, 학위종류, 대학교 대학원
예) [3] 홍길순(2023), 예비교사의 AI 역량 연구, 석사학위논문, XX교육대학교 대학원
```

**④ 국외 학술지**
```
[N] Author, A. B.(Year), Title of article, Journal Name, Vol(No), pp.xx-xx
예) [4] Flavell, J. H.(1979), Metacognition and cognitive monitoring, American Psychologist, 34(10), 906-911
```

**⑤ 국외 단행본**
```
[N] Author, A. B.(Year), Book title, Publisher, City
예) [5] Cohen, J.(1992), Statistical power analysis, Erlbaum, Hillsdale
```

**⑥ 웹문서**
```
[N] 저자 또는 기관(연도), 문서 제목, URL
예) [6] 교육부(2022), 2022 개정 교육과정 총론, https://www.moe.go.kr
```

## 6. 자주 하는 실수 및 금지 사항

| # | 항목 | ❌ 잘못된 예 | ✔ 올바른 예 |
|:-:|------|------------|------------|
| 1 | 표 제목 위치 | 표 **아래**에 `<Table 1>` 표기 | 표 **위**에 `<Table 1>` 표기 |
| 2 | 그림 제목 위치 | 그림 **위**에 `(Fig. 1)` 표기 | 그림 **아래**에 `(Fig. 1)` 표기 |
| 3 | 제목 번호 체계 | `Ⅰ. 서론` | `1. 서론` |
| 4 | 참고문헌 인용 | `메타인지(Flavell, 1979)는…` | `메타인지는…[4]` |
| 5 | p값 앞 0 | `p = 0.023` | `p = .023` |
| 6 | 표 제목 언어 | `<표 1> 연구 참여자` | `<Table 1> Participants of the Study` |
| 7 | 그림 제목 언어 | `(그림 1) 연구 모형` | `(Fig. 1) Research Model` |
| 8 | 유의수준 위치 | 셀 안에 `**` 별표 혼용 | 표 하단: `**p<.01` |
| 9 | 국문 요약 분량 | 600자 이상 또는 200자 이하 | 공백 제외 400자 내외 |
| 10 | 저자 정보 (심사용) | 심사용에 저자명 기재 | 저자 정보 완전 삭제 |

## 7. 제출 전 최종 체크리스트

### 7.1 형식 체크

- [ ] 논문 구성 순서 (①~⑪) 준수
- [ ] 제목 번호 체계: `1.` / `1.1` / `1.1.1` 형식
- [ ] 표 제목: **표 위**, `<Table N>` 형식, **영문**
- [ ] 그림 제목: **그림 아래**, `(Fig. N)` 형식, **영문**
- [ ] 유의수준: `*p<.05  **p<.01  ***p<.001` 표 하단 기재
- [ ] 참고문헌 인용: `[번호]` 형식 (APA 방식 금지)
- [ ] 국문 요약: 공백 제외 400자 내외
- [ ] 심사용 제출: 저자 관련 정보 전면 삭제

### 7.2 통계 체크

- [ ] 기술통계: *M*, *SD*, *N* 모두 보고
- [ ] t검정: `t(df)`, `p`, `Cohen's d` 보고
- [ ] ANOVA: `F(df1, df2)`, `p`, 사후검정 결과 보고
- [ ] 위계적 회귀분석: 단계별 `R²`, `ΔR²`, 개별 변인 `β`·`t`·`p` 보고
- [ ] 상관분석: `r`, `p` 보고 / 비유의 시 `(ns)` 병기
- [ ] 신뢰도: Cronbach's α, ICC ≥ .80 확인 후 기재
- [ ] p값 형식: `.001` (앞 0 생략)
- [ ] 소수점 자릿수: 논문 내 동일 통계값 동일 자릿수

### 7.3 내용 체크

- [ ] 연구 문제와 결과 섹션 1:1 대응
- [ ] 한계점 3가지 이상 솔직하게 기술
- [ ] 후속 연구 제언 2가지 이상
- [ ] 교육적 시사점 구체적 기술 (추상적 제안 지양)
- [ ] 결측치 처리 방법 및 사례 수 명시
- [ ] 횡단 설계: "예측" 표현 사용 (인과 표현 금지)
- [ ] 단일집단 설계: 통제집단 부재 한계 논의 포함
- [ ] d > 1.0: 설계 한계·출판편향 가능성 논의 포함
